# **Коришћење Phi-3 у Microsoft Foundry**

Са развојем генеративне вештачке интелигенције, надамо се да ћемо користити јединствену платформу за управљање различитим LLM и SLM, интеграцију корпоративних података, фино подешавање/RAG операције, и евалуацију различитих корпоративних бизниса након интеграције LLM и SLM, итд., како би се генеративна вештачка интелигенција боље имплементирала у паметне апликације. [Microsoft Foundry](https://ai.azure.com) је корпоративна платформа за генеративне AI апликације.

![aistudo](../../../../translated_images/sr/aifoundry_home.f28a8127c96c7d93.webp)

Помоћу Microsoft Foundry, можете евалуирати одговоре модела великог језика (LLM) и оркестрирати компоненте апликације са prompt flow ради бољег перформанса. Платформа омогућава скалабилност за трансформацију концепата у пуну производњу са лакоћом. Континуирано праћење и префињавање подржавају дугорочни успех.

Можемо брзо депановати Phi-3 модел на Microsoft Foundry кроз једноставне кораке, а затим користити Microsoft Foundry да завршимо Playground/Chat, фино подешавање, евалуацију и друге релевантне радове везане за Phi-3.

## **1. Припрема**

Ако већ имате инсталиран [Azure Developer CLI](https://learn.microsoft.com/azure/developer/azure-developer-cli/overview?WT.mc_id=aiml-138114-kinfeylo) на вашем уређају, коришћење овог шаблона је једноставно као покретање ове команде у новом директоријуму.

## Ручно креирање

Креирање пројекта и хаба у Microsoft Foundry је одличан начин да организујете и управљате својим AI радом. Ево детаљног водича који ће вам помоћи да започнете:

### Креирање пројекта у Microsoft Foundry

1. **Идите на Microsoft Foundry**: Пријавите се у портал Microsoft Foundry.
2. **Креирајте пројекат**:
   - Ако сте унутар неког пројекта, изаберите "Microsoft Foundry" у горњем левом углу странице да бисте отишли на почетну страницу.
   - Изаберите "+ Create project".
   - Унесите име пројекта.
   - Ако већ имате хаб, он ће бити подразумевано изабран. Ако имате приступ више од једног хаба, можете изабрати други са падајуће листе. Ако желите да креирате нови хаб, изаберите "Create new hub" и унесите име.
   - Изаберите "Create".

### Креирање хаба у Microsoft Foundry

1. **Идите на Microsoft Foundry**: Пријавите се својим Azure налогом.
2. **Креирајте хаб**:
   - Изаберите Управљачки центар (Management center) из левог менија.
   - Изаберите "All resources", затим стрелицу доле поред "+ New project" и изаберите "+ New hub".
   - У дијалогу "Create a new hub" унесите име за ваш хаб (нпр. contoso-hub) и по жељи измените остала поља.
   - Изаберите "Next", прегледајте информације, а затим изаберите "Create".

За детаљнија упутства можете погледати званичну [Microsoft документацију](https://learn.microsoft.com/azure/ai-studio/how-to/create-projects).

Након успешног креирања, можете приступити студију који сте креирали преко [ai.azure.com](https://ai.azure.com/)

На једном AI Foundry-у може бити више пројеката. Креирајте пројекат у AI Foundry-у као припрему.

Креирајте Microsoft Foundry [QuickStarts](https://learn.microsoft.com/azure/ai-studio/quickstarts/get-started-code)

## **2. Деплој Phi модела у Microsoft Foundry**

Кликните на опцију Explore пројекта да уђете у Model Catalog и изаберите Phi-3.

Изаберите Phi-3-mini-4k-instruct.

Кликните 'Deploy' да бисте депловали Phi-3-mini-4k-instruct модел.

> [!NOTE]
>
> Можете одабрати рачунарску снагу приликом деплоирања

## **3. Playground Chat Phi у Microsoft Foundry**

Идите на страницу деплоирања, изаберите Playground и ћаскајте са Phi-3 у Microsoft Foundry.

## **4. Деплој модела из Microsoft Foundry**

Да бисте депловали модел из Azure Model Catalog-а, пратите ове кораке:

- Пријавите се у Microsoft Foundry.
- Изаберите модел који желите да деплујете из Microsoft Foundry model catalog-а.
- На страници са детаљима модела изаберите Deploy, а затим Serverless API са Azure AI Content Safety.
- Изаберите пројекат у ком желите да деплујете модел. Да бисте користили Serverless API понуду, ваш workspace мора бити у региону East US 2 или Sweden Central. Можете прилагодити име деплоирања.
- У чаробњаку за деплоирање изаберите Pricing and terms да бисте сазнали цене и услове коришћења.
- Изаберите Deploy. Сачекајте док деплоирање не буде спремно и док вас страница не преусмери на страницу Deployments.
- Изаберите Open in playground да почнете интеракцију са моделом.
- Можете се вратити на страницу Deployments, изабрати деплоирање и приметити Target URL и Secret Key који се користе за позив деплоирања и генерисање резултата.
- Увек можете пронаћи детаље ендпоинта, URL и приступне кључеве тако што ћете отићи на Build таб и изабрати Deployments из секције Components.

> [!NOTE]
> Имајте у виду да ваш налог мора имати Azure AI Developer улогу са одговарајућим дозволама на Resource Group да бисте извршили ове кораке.

## **5. Коришћење Phi API у Microsoft Foundry**

Можете приступити https://{Your project name}.region.inference.ml.azure.com/swagger.json преко Postman GET захтева и ускладити га са кључем да бисте се упознали са датим интерфејсима.

Веома лако можете добити параметре захтева као и параметре одговора.
