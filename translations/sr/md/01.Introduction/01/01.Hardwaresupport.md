# Подршка за Phi Хардвер

Microsoft Phi је оптимизован за ONNX Runtime и подржава Windows DirectML. Добро функционише на разним типовима хардвера, укључујући GPU-ове, CPU-ове и чак мобилне уређаје.

## Хардвер уређаја
Конкретно, подржани хардвер укључује:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPU-а, 128 GiB меморије)

## Мобилни SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 или новији са A16/A17 процесором

## Спецификације Phi хардвера

- Минимална потребна конфигурација.
- Windows: GPU компатибилан са DirectX 12 и минимум 4GB укупне RAM меморије

CUDA: NVIDIA GPU са Compute Capability >= 7.02

## Покретање onnxruntime на више GPU-ова

Тренутно доступни Phi ONNX модели су само за 1 GPU. Могуће је подржати мулти-GPU за Phi модел, али ORT са 2 GPU-а не гарантује да ће пружити већу пропусност у поређењу са 2 инстанце ORT-а. За најновија ажурирања погледајте [ONNX Runtime](https://onnxruntime.ai/).

На [Build 2024 the GenAI ONNX тим](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) је најавио да су омогућили мулти-инстанце уместо мулти-GPU за Phi моделе.

Тренутно ово омогућава покретање једне onnxruntime или onnxruntime-genai инстанце са CUDA_VISIBLE_DEVICES окружењском променљивом на овај начин.

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

Слободно истражујте даље Phi у [Microsoft Foundry](https://ai.azure.com)
