## Phi породица у Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) је поуздана платформа која омогућава програмерима да покрећу иновације и обликују будућност уз помоћ вештачке интелигенције на безбедан, сигуран и одговоран начин.

[Microsoft Foundry](https://ai.azure.com) је дизајниран за програмере да:

- Креирају генеративне AI апликације на платформи корпоративног нивоа.
- Истражују, креирају, тестирају и пуштају у рад користећи најсавременије AI алате и ML моделе, засноване на одговорним AI праксама.
- Сарађују са тимом кроз цео животни циклус развоја апликације.

Помоћу Microsoft Foundry, можете истраживати широк спектар модела, сервиса и могућности, и почети правити AI апликације које најбоље одговарају вашим циљевима. Платформа Microsoft Foundry омогућава скалабилност за трансформацију прототипа у потпуне продукцијске апликације с лакоћом. Континуирано надгледање и усавршавање подржава дугорочни успех.

### **Покретање Python кода за позив Microsoft Foundry Phi-4**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Укључи резултат говора ако је говор омогућен
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
