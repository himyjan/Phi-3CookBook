# Семейство Phi от Microsoft

Модели Phi — самые мощные и экономичные малые языковые модели (SLM), превосходящие модели того же размера и следующего уровня по разнообразным языковым, логическим, кодировочным, аудио, визуальным и математическим задачам. Этот выпуск расширяет выбор качественных моделей для клиентов, предлагая более практичные варианты для создания и построения генеративных ИИ-приложений.

Семейство Phi началось с Phi-1 для генерации кода на Python, продолжилось Phi-1.5 /2 на основе текстового и чатового дополнения, Phi-3-mini/small/medium-instruct и Phi-3.5/4-mini-instruct, и развилось до Phi-3/3.5-vision для визуальных задач, Phi-4 на базе мощного логического мышления, Phi-3.5-MoE для MoE, и сейчас полноформатная модель Phi-4-multimodal. Благодаря высококачественным наборам данных, модели могут обучаться, сопоставимым с моделями с большим количеством параметров.
