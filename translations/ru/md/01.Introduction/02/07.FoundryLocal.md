## Начало работы с моделями семейства Phi в Foundry Local

### Введение в Foundry Local

Foundry Local — это мощное решение для локального выполнения ИИ-инференса, предоставляющее возможности корпоративного уровня ИИ прямо на вашем локальном оборудовании. Это руководство поможет вам настроить и использовать модели семейства Phi с Foundry Local, предоставляя полный контроль над вашими ИИ-задачами при сохранении конфиденциальности и снижении затрат.

Foundry Local обеспечивает производительность, конфиденциальность, настройку и экономию за счёт запуска ИИ-моделей локально на вашем устройстве. Он бесшовно интегрируется в ваши текущие рабочие процессы и приложения через удобный CLI, SDK и REST API.


![arch](../../../../../translated_images/ru/foundry-local-arch.8823e321dd8258d7.webp)

### Почему выбирают Foundry Local?

Понимание преимуществ Foundry Local поможет вам принимать обоснованные решения о стратегии развертывания ИИ:

- **Инференс на устройстве:** Запускайте модели локально на вашем оборудовании, снижая затраты и сохраняя все данные на вашем устройстве.

- **Настройка моделей:** Выбирайте из предустановленных моделей или используйте собственные для удовлетворения конкретных требований и сценариев использования.

- **Экономия средств:** Исключите повторяющиеся расходы на облачные сервисы, используя существующее оборудование, делая ИИ более доступным.

- **Бесшовная интеграция:** Подключайтесь к вашим приложениям через SDK, API-эндпоинты или CLI с возможностью лёгкого масштабирования до Microsoft Foundry по мере роста потребностей.

> **Примечание по началу работы:** В этом руководстве рассматривается использование Foundry Local через интерфейсы CLI и SDK. Вы изучите оба подхода, чтобы выбрать лучший метод для вашего случая.

## Часть 1: Настройка Foundry Local CLI

### Шаг 1: Установка

Foundry Local CLI — ваш вход к управлению и запуску ИИ-моделей локально. Начнём с установки на вашу систему.

**Поддерживаемые платформы:** Windows и macOS

Для подробных инструкций по установке обратитесь к [официальной документации Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Шаг 2: Изучение доступных моделей

После установки Foundry Local CLI вы можете узнать, какие модели доступны для вашего случая. Эта команда покажет все поддерживаемые модели:


```bash
foundry model list
```

### Шаг 3: Понимание моделей семейства Phi

Семейство Phi предлагает ряд моделей, оптимизированных для разных случаев использования и конфигураций оборудования. Вот доступные модели Phi в Foundry Local:

**Доступные модели Phi:** 

- **phi-3.5-mini** — компактная модель для базовых задач
- **phi-3-mini-128k** — версия с расширенным контекстом для более длинных диалогов
- **phi-3-mini-4k** — стандартная модель для общего использования
- **phi-4** — усовершенствованная модель с улучшенными возможностями
- **phi-4-mini** — облегчённая версия Phi-4
- **phi-4-mini-reasoning** — специализированная для сложных задач рассуждений

> **Совместимость с оборудованием:** Каждая модель может быть настроена для разного аппаратного ускорения (CPU, GPU) в зависимости от возможностей вашей системы.

### Шаг 4: Запуск первой модели Phi

Начнём с практического примера. Запустим модель `phi-4-mini-reasoning`, которая отлично подходит для пошагового решения сложных задач.


**Команда для запуска модели:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Первый запуск:** При первом запуске модель автоматически загрузится на ваше локальное устройство. Время загрузки зависит от скорости вашего интернета, поэтому будьте терпеливы при начальной настройке.

### Шаг 5: Тестирование модели на реальной задаче

Теперь протестируем нашу модель на классической логической задаче, чтобы увидеть, как она выполняет пошаговое рассуждение:

**Пример задачи:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Ожидаемое поведение:** Модель должна разбить задачу на логические шаги, используя факт, что у фазанов 2 ноги, а у кроликов — 4, чтобы решить систему уравнений.

## Часть 2: Создание приложений с Foundry Local SDK

### Зачем использовать SDK?

В то время как CLI отлично подходит для тестирования и быстрого взаимодействия, SDK позволяет программно интегрировать Foundry Local в ваши приложения. Это открывает возможности для:

- Создания специализированных ИИ-приложений
- Автоматизации рабочих процессов
- Интеграции ИИ-возможностей в существующие системы
- Разработки чат-ботов и интерактивных инструментов

### Поддерживаемые языки программирования

Foundry Local предоставляет SDK с поддержкой нескольких языков программирования для удобства вашей разработки:

**📦 Доступные SDK:**

- **C# (.NET):** [Документация и примеры SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [Документация и примеры SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [Документация и примеры SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [Документация и примеры SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### Следующие шаги

1. **Выберите предпочитаемый SDK** в зависимости от вашей среды разработки
2. **Следуйте документации SDK** для подробных руководств по реализации
3. **Начните с простых примеров** перед созданием сложных приложений
4. **Изучите примерный код,** предоставленный в каждом репозитории SDK
