# **Использование Microsoft Foundry для оценки**

![aistudo](../../../../../translated_images/ru/AIFoundry.9e0b513e999a1c5a.webp)

Как оценить ваше приложение генеративного ИИ с помощью [Microsoft Foundry](https://ai.azure.com?WT.mc_id=aiml-138114-kinfeylo). Независимо от того, оцениваете ли вы односторонние или многократные диалоги, Microsoft Foundry предоставляет инструменты для оценки производительности модели и безопасности.

![aistudo](../../../../../translated_images/ru/AIPortfolio.69da59a8e1eaa70f.webp)

## Как оценить генеративные приложения ИИ с помощью Microsoft Foundry
Подробные инструкции смотрите в [документации Microsoft Foundry](https://learn.microsoft.com/azure/ai-studio/how-to/evaluate-generative-ai-app?WT.mc_id=aiml-138114-kinfeylo)

Вот шаги для начала работы:

## Оценка генеративных моделей ИИ в Microsoft Foundry

**Требования**

- Тестовый набор данных в формате CSV или JSON.
- Развернутая генеративная модель ИИ (такая как Phi-3, GPT 3.5, GPT 4 или модели Davinci).
- Среда выполнения с вычислительным экземпляром для запуска оценки.

## Встроенные метрики оценки

Microsoft Foundry позволяет оценивать как односторонние, так и сложные многократные диалоги.
Для сценариев Retrieval Augmented Generation (RAG), где модель базируется на конкретных данных, вы можете оценить производительность с помощью встроенных метрик.
Кроме того, можно оценивать общие сценарии односторонних ответов на вопросы (без RAG).

## Создание задачи оценки

В интерфейсе Microsoft Foundry перейдите на страницу Evaluate или Prompt Flow.
Следуйте мастеру создания оценки, чтобы настроить задачу оценки. При необходимости укажите имя для оценки.
Выберите сценарий, соответствующий целям вашего приложения.
Выберите одну или несколько метрик оценки для анализа вывода модели.

## Пользовательский процесс оценки (опционально)

Для большей гибкости вы можете настроить пользовательский процесс оценки, адаптировав его под свои конкретные требования.

## Просмотр результатов

После запуска оценки вы можете просматривать, журналировать и анализировать подробные метрики оценки в Microsoft Foundry. Получайте представление о возможностях и ограничениях вашего приложения.

**Note** Microsoft Foundry в настоящее время находится в публичном превью, поэтому используйте его для экспериментов и разработки. Для производственных задач рассмотрите другие варианты. Изучите официальную [документацию AI Foundry](https://learn.microsoft.com/azure/ai-studio/?WT.mc_id=aiml-138114-kinfeylo) для получения дополнительной информации и пошаговых инструкций.
