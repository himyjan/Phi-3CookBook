## Following extension guidelines

## Working with Markdown

អ្នកអាចសរសេរ README របស់អ្នកដោយប្រើ Visual Studio Code។ ខាងក្រោមជាប៊ូតុង shortcuts គ្រប់គ្រងកែសម្រួលដែលមានប្រយោជន៏៖

* បែងចែកកម្មវិធីកែសម្រួល (`Cmd+\` នៅលើ macOS ឬ `Ctrl+\` នៅលើ Windows និង Linux)។
* បិទបង្ហាញមើលមុន (`Shift+Cmd+V` នៅលើ macOS ឬ `Shift+Ctrl+V` នៅលើ Windows និង Linux)។
* ចុច `Ctrl+Space` (Windows, Linux, macOS) ដើម្បីមើលបញ្ជី snippets Markdown។
