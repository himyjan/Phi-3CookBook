## Following extension guidelines

## Working with Markdown

អ្នកអាចប្រើ Visual Studio Code សម្រាប់សរសេរ README របស់អ្នក។ នេះជាការបង្ហាញកីប៊តកាត់អត្តបទដែលមានប្រយោជន៍៖

* បំបែកកម្មវិធីកែសម្រួល (`Cmd+\` នៅលើ macOS ឬ `Ctrl+\` នៅលើ Windows និង Linux)។
* ប្ដូរមើល 미리 보기 (`Shift+Cmd+V` នៅលើ macOS ឬ `Shift+Ctrl+V` នៅលើ Windows និង Linux)។
* ចុច `Ctrl+Space` (Windows, Linux, macOS) ដើម្បីមើលបញ្ជីសន្លឹកកូដ Markdown។
