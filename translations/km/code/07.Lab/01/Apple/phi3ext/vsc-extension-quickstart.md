# бһҹбһјбһҳбһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚвҖӢбһҳбһҖбһҖбһ¶бһ“бҹӢвҖӢбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ VS Code бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ

## бһҳбһ¶бһ“бһўбҹ’бһңбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ

* бһҗбһҸбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҜбһҖбһҹбһ¶бһҡбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҠбҹӮбһӣбһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
* `package.json` - бһӮбһәбһҮбһ¶бһҜбһҖбһҹбһ¶бһҡбһҸбҹҶбһҺбһ¶бһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҖбһ¶бһҹбһўбҹҶбһ–бһёбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ бһ“бһ·бһ„бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
  * бһ”бҹ’бһӣбһҖбҹӢбһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ…бһ»бҹҮбһ”бһүбҹ’бһҮбһёбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҳбһҪбһҷ бһ“бһ·бһ„бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ„бһҮбһҫбһ„ бһ“бһ·бһ„бһҲбҹ’бһҳбҹ„бҹҮбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бҹ” бһҮбһ¶бһҳбһҪбһҷбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ“бҹҒбҹҮ VS Code бһўбһ¶бһ…бһ”бһ„бҹ’бһ бһ¶бһүбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹүаёІа№ҖаёҘа№Үаё•бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бҹ” бһңбһ¶бһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ•бҹ’бһ‘бһ»бһҖбһ”бҹ’бһӣбһҖбҹӢбһ“бҹ…бһЎбһҫбһҷбһ‘бҹҒбҹ”
* `src/extension.ts` - бһӮбһәбһҮбһ¶бһҜбһҖбһҹбһ¶бһҡбһҠбһҫбһҳ бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ•бҹ’бһҸбһӣбҹӢбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”
  * бһҜбһҖбһҹбһ¶бһҡбһ“бҹҒбҹҮбһ–бһёбһӣбһ¶бһҖбҹӢбһ‘бһҳ function бһҳбһҪбһҷ, `activate`, бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бҹҒбһӣбһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһўбһ“бһ»бһңбһҸбҹ’бһҸ (бһҖбҹ’бһ“бһ»бһ„бһҖбһҡбһҺбһёбһ“бҹҒбҹҮбһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҠбһ·бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶)бҹ” бһ“бҹ…бһҒбһ¶бһ„бһҖбҹ’бһ“бһ»бһ„ function `activate` бһҷбһҫбһ„бһ бҹ… `registerCommand`бҹ”
  * бһҷбһҫбһ„бһ•бҹ’бһҸбһӣбҹӢ function бһҳбһ¶бһ“бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҮбһ¶бһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡбһ‘бһёбһ–бһёбһҡбһ‘бҹ… `registerCommand`бҹ”

## бһҖбһ¶бһҡбһҸбҹҶбһЎбһҫбһ„

* бһҸбҹҶбһЎбһҫбһ„бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҠбһӣбҹӢбһўбһ“бһ»бһҹбһ¶бһҹбһ“бҹҚ (amodio.tsl-problem-matcher, ms-vscode.extension-test-runner, бһ“бһ·бһ„ dbaeumer.vscode-eslint)


## бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡвҖӢбһӮбҹ’бһҡбһ”бҹӢбһ–бҹҒбһӣ

* бһ…бһ»бһ… `F5` бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҫбһҖбһ”бһ„бҹ’бһўбһҪбһ…бһҗбҹ’бһҳбһёбһҳбһҪбһҷбһҠбҹӮбһӣбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ•бҹ’бһ‘бһ»бһҖбҹ”
* бһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ–бһёбһ”бҹүбһ¶а№ҖаёҘа№Үаё•бһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһҠбҹ„бһҷбһ…бһ»бһ… (`Ctrl+Shift+P` бһ¬ `Cmd+Shift+P` бһ“бҹ…бһӣбһҫ Mac) бһ бһҫбһҷбһңбһ¶бһҷ `Hello World`бҹ”
* бһҖбҹҶбһҺбһҸбҹӢбһ…бҹҶбһҺбһ»бһ…бһ•бҹ’бһўбһҫбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҖбҹ’бһ“бһ»бһ„ `src/extension.ts` бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”
* бһҡбһҖбһҳбһҫбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһ…бҹҒбһүбһ–бһёбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ»бһ„бһҹбһјбһӣбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бҹ”

## бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ

* бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸбһ–бһёбһҡбһ”бһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҖбһјбһҠбһҖбҹ’бһ“бһ»бһ„ `src/extension.ts`бҹ”
* бһўбҹ’бһ“бһҖбһҖбҹҸбһўбһ¶бһ…бһ•бҹ’бһ‘бһ»бһҖбһЎбһҫбһ„бһңбһ·бһүбһ”бһ„бҹ’бһўбһҪбһ… VS Code бһҠбҹ„бһҷбһ…бһ»бһ… (`Ctrl+R` бһ¬ `Cmd+R` бһ“бҹ…бһӣбһҫ Mac) бһҮбһ¶бһҳбһҪбһҷбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”


## бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһўбҹҶбһ–бһё API

* бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһҫбһҖбһҹбҹҶбһҺбһ»бҹҶ API бһӮбҹ’бһҡбһ”бҹӢбһӣбҹҶбһҠбһ¶бһ”бҹӢбһ–бҹҒбһүбһӣбҹҒбһүбһҡбһ”бһҹбҹӢбһҷбһҫбһ„бһ“бҹ…бһ–бҹҒбһӣбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ”бһҫбһҖбһҜбһҖбһҹбһ¶бһҡ `node_modules/@types/vscode/index.d.ts`бҹ”

## бһҡбһҸбҹӢбһҸбҹҒбһҹбҹ’бһҸ

* бһҸбҹҶбһЎбһҫбһ„ [Extension Test Runner](https://marketplace.visualstudio.com/items?itemName=ms-vscode.extension-test-runner)
* бһҡбһҸбҹӢбһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡ "watch" бһҸбһ¶бһҳбһҡбһҷбҹҲбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ **Tasks: Run Task**бҹ” бһҹбһјбһҳбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҖбҹҶбһ–бһ»бһ„бһҡбһҸбҹӢбһҳбһ·бһ“бһҠбһјбһ…бҹ’бһ“бҹ„бҹҮбһҳбһ¶бһ“бһ бһ¶бһ“бһ·бһ—бҹҗбһҷбһҗбһ¶бһҸбҹҒбһҹбҹ’бһҸбһ“бһ№бһ„бһҳбһ·бһ“бһҸбҹ’бһҡбһјбһңбһҡбһҖбһғбһҫбһүбҹ”
* бһ”бһҫбһҖбһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ– Testing бһ–бһёбһҡбһ”бһ¶бһҡбһҠбҹ’бһӢбһ¶бһ“бһҖбһ·бһ…бҹ’бһ…бһҖбһ¶бһҡ бһ бһҫбһҷбһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ "Run Test" бһ¬бһ”бҹ’бһҡбһҫбһҖбҹ’бһҸбһ¶бһҡбһ…бһ»бһ…бһҡбһ бҹҗбһҹ `Ctrl/Cmd + ; A`
* бһҳбһҫбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһҸбҹҒбһҹбҹ’бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ‘бһ·бһҠбҹ’бһӢбһ—бһ¶бһ– Test Resultsбҹ”
* бһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһ•бҹ’бһ“бҹӮбһҖ `src/test/extension.test.ts` бһ¬бһ”бһ„бҹ’бһҖбһҫбһҸбһҜбһҖбһҹбһ¶бһҡбһҸбҹҒбһҹбҹ’бһҸбһҗбҹ’бһҳбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҗбһҸ `test`бҹ”
  * бһўбҹ’бһ“бһҖбһ”бһҫбһҖбһ…бҹҶбһ бһҡбһҸбҹҒбһҹбҹ’бһҸбһ“бһ№бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҸбҹӮбһҜбһҖбһҹбһ¶бһҡбһҠбҹӮбһӣбһ•бҹ’бһӮбһјбһ•бҹ’бһӮбһ„бһ“бһ№бһ„бһ“бһ·бһҷбһ¶бһҳбһҖбһҡбһ”бһҹбҹӢбһҲбҹ’бһҳбҹ„бҹҮ `**.test.ts`бҹ”
  * бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһҖбҹ’бһ“бһ»бһ„ `test` бһҠбһҫбһҳбҹ’бһ”бһёбһҡбҹҖбһ”бһ…бҹҶбһҸбҹҒбһҹбҹ’бһҸбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһўбһ¶бһ…бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбҹ”

## бһ‘бҹ…бһҶбҹ’бһ„бһ¶бһҷбһҮбһ¶бһ„бһ“бҹҒбҹҮ

* бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ‘бҹҶбһ бҹҶбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһ“бһ–бҹҒбһӣбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷ [бһ”бҹүбҹҮбһ‘бһ„бҹ’бһӮбһ·бһ…бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](https://code.visualstudio.com/api/working-with-extensions/bundling-extension)бҹ”
* [бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ](https://code.visualstudio.com/api/working-with-extensions/publishing-extension) бһ“бҹ…бһ‘бһёбһ•бҹ’бһҹбһ¶бһҡ VS Code extensionбҹ”
