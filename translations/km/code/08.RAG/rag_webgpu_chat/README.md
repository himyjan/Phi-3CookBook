## бһҖбһ¶бһҡбһҸбһ¶бҹҶбһ„бһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһ бһ¶бһү WebGPU бһ“бһ·бһ„ бһӮбҹҶбһҡбһј RAG
бһӮбҹҶбһҡбһј RAG бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣ Phi-3 Onnx Hosted бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹӮбһӣбһ”бһ„бҹ’бһҡбһҪбһҳбһ”бһ“бҹ’бһҗбҹӮбһҳбһҠбҹ„бһҷбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“ (Retrieval-Augmented Generation) бһҠбҹӮбһӣбһ”бҹҶбһ”бһ»бһҹбһҸбҹҶбһҺбһҫбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣ Phi-3 бһҮбһ¶бһҳбһҪбһҷбһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣ ONNX бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһү AI бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ” бһӮбҹҶбһҡбһјбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҸбһҪбһ“бһ¶бһ‘бһёбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбҹҖбһ”бһ…бҹҶбһҳбҹүбһјбһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһңбһ·бһҹбҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһ“бһјбһңбһӮбһ»бһҺбһ—бһ¶бһ– бһҗбҹ’бһӣбҹғбһҗбҹ„бһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһ–бһёбһ”бһҡбһ·бһ”бһ‘бһңбҹӮбһ„бҹ—бҹ” бһңбһ¶бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ Azure AI бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһ“бһјбһңбһҮбһҳбҹ’бһҡбһҫбһҹбһҳбҹүбһјбһҠбҹӮбһӣбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһҠбҹӮбһӣбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҡбһҖ, бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ–бҹҒбһүбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһ¶бһҡбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳбҹ” бһҳбҹүбһјбһҠбҹӮбһӣ Phi-3 бһҡбһҪбһҳбһҳбһ¶бһ“ Phi-3-mini, Phi-3-small, бһ“бһ·бһ„ Phi-3-medium бһҳбһ¶бһ“бһ“бҹ…бһӣбһҫ Azure AI Model Catalog бһ бһҫбһҷбһўбһ¶бһ…бһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҠбҹ„бһҷбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһ¬бһҸбһ¶бһҳбһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбһјбһ…бһҮбһ¶ HuggingFace бһ“бһ·бһ„ ONNX бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбҹ’бһҸбһёбһ”бҹ’бһҸбҹҒбһҮбҹ’бһүбһ¶бһҡбһ”бһҹбҹӢ Microsoft бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ•бҹ’бһҸбһӣбҹӢбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI бһҠбҹӮбһӣбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһ“бһ·бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”

## бһҸбһҫ WebGPU бһҮбһ¶бһўбҹ’бһңбһё
WebGPU бһӮбһәбһҮбһ¶ API бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһ…бһңбҹҒбһ”бһҹбһҳбҹҗбһҷбһҗбҹ’бһҳбһё бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһҹбһҫбһҡбһҠбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ GPU бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһ–бһёбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһңбҹҒбһ”бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһӮбҹҒбһҡбҹҶбһ–бһ№бһ„бһҗбһ¶бһ“бһ№бһ„бһҮбҹҶбһ“бһҪбһҹ WebGL бһҠбҹ„бһҷбһ•бҹ’бһҸбһӣбҹӢбһҖбҹӮбһ‘бһҳбҹ’бһҡбһ„бҹӢбһ…бһҳбҹ’бһ”бһ„бҹ—бһҠбһјбһ…бһҮбһ¶бҹ–

1. **бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ GPU бһҹбһҳбҹҗбһҷбһ‘бҹҶбһ“бһҫбһ”**: WebGPU бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“бһҮбһ¶бһҳбһҪбһҷбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ GPU бһҹбһҳбҹҗбһҷбһ‘бҹҶбһ“бһҫбһ” бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ API бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбһјбһ…бһҮбһ¶ Vulkan, Metal бһ“бһ·бһ„ Direct3D 12бҹ”
2. **бһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ“бҹ’бһҗбҹӮбһҳ**: бһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһӮбһҺбһ“бһ¶ GPU бһҹбһҖбһӣ бһ“бһ·бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһӣбһҝбһ“бһҮбһ¶бһ„бһҳбһ»бһ“ бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһ… бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ·бһҖбҹ’бһҹбһ¶вҖӢбһҷбһ“бҹ’бһҸбһҳбҹҒбһӮбһ»бһҺбҹ”
3. **бһӣбһҖбҹ’бһҒбһҺбҹҲбһӣбһҳбҹ’бһўбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ**: WebGPU бһ•бҹ’бһҸбһӣбҹӢбһҹбһ·бһ‘бҹ’бһ’бһ·бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ– GPU бһҠбҹӮбһӣбһҳбһ¶бһ“бһ—бһ¶бһ–бһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһү бһ“бһ·бһ„бһҳбһ¶бһ“бһҹбһ·бһ‘бҹ’бһ’бһ·еӢ• dynamic бһҖбҹ’бһҡбһ¶бһ бҹ’бһңбһ·бһ…бһ“бһ·бһ„бһҖбһ¶бһҡбһ„бһ¶бһҡбһӮбһҺбһ“бһ¶бһҠбҹҸбһ’бҹҶбһ’бҹҒбһ„бҹ”
4. **бһ”бһ“бҹ’бһҗбһҷбһ”бһ“бҹ’бһ‘бһ»бһҖ JavaScript**: бһҠбҹ„бһҷбһ•бҹ’бһ‘бҹҒбһҡбһ”бһ“бҹ’бһ‘бһ»бһҖбһ…бҹ’бһҡбһҫбһ“бһ‘бҹ…бһҖбһ¶бһ“бҹӢ GPU WebGPU бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ”бһ“бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ„бһ¶бһҡбһӣбһҫ JavaScript бһҷбҹүбһ¶бһ„бһҒбҹ’бһӣбһ¶бҹҶбһ„ бһ“бһ¶бҹҶбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһӣбҹ’бһў бһ“бһ·бһ„бһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһҮбһ¶бһ“бҹӢбһҒбҹ’бһ–бһҹбҹӢбҹ”

WebGPU бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҠбҹ„бһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҠбһјбһ…бһҮбһ¶ Google Chrome бһ бһҫбһҷбһҖбҹҶбһ–бһ»бһ„бһ”бһ“бҹ’бһҸбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҠбһҫбһҳбҹ’бһ”бһёбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҮбһ¶бһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбҹ”

#### бһ”бһҫбһҖбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ»бһҖбһҡбһҖбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

#### бһҶбҹӮбһҖбһ…бһјбһӣбһ‘бҹҶбһ–бҹҗбһҡбһ”бһүбҹ’бһ…бһ¶бҹҶбҹ–
бһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһ‘бһ„бҹӢбһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“ бһңбһ¶бһҷбһ”бһүбҹ’бһ…бһјбһӣ `chrome://flags` бһ“бһ·бһ„бһ…бһ»бһ… Enterбҹ”

#### бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһүбҹ’бһ…бһ¶бҹҶбҹ–
бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһўбһ”бҹӢбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ“бҹ…бһӣбһҫбһ…бҹҶбһҺбһ»бһ…бһҒбһ¶бһ„бһӣбһҫ бһ“бһ·бһ–бһ“бҹ’бһ’ 'enable-unsafe-webgpu' бҹ”

#### бһ”бһҫбһҖбһ”бһүбҹ’бһ…бһ¶бҹҶбҹ–
бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ”бһүбҹ’бһ…бһ¶бҹҶ #enable-unsafe-webgpu бһҖбҹ’бһ“бһ»бһ„бһ”бһүбҹ’бһҮбһёбһӣбһ‘бҹ’бһ’бһ•бһӣбҹ”
