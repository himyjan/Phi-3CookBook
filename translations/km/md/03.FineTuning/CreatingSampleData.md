# бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹ„бһҷбһ‘бһ¶бһүбһҷбһҖ DataSet бһ–бһё Hugging Face бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’


### бһҹбһ„бҹ’бһҒбҹҒбһ”

бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ“бҹҒбҹҮбһҡбҹҖбһ”бһ…бҹҶбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһҠбҹ„бһҷбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ, бһҳбһҫбһӣбһ…бҹҒбһүбһ–бһёбһҮбһҪбһҡбһҠбҹҒбһҖбһҠбҹӮбһӣбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһҡбһ¶бһҮбҹҗбһҷ, бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҜбһҖбһҹбһ¶бһҡ CSVбҹ”

### бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ

бһҳбһ»бһ“бһ–бҹҒбһӣбһҡбһҸбҹӢбһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ“бҹҒбҹҮ, бһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһ¶бһҖбһҠбһҗбһ¶бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбһ”бһ¶бһ“бһҠбҹҶбһЎбһҫбһ„бһҡбһҪбһ…: `Pandas`, `Datasets`, `requests`, `PIL`, бһ“бһ·бһ„ `io`бҹ” бһўбҹ’бһ“бһҖбһҖбҹҸбһҸбҹ’бһҡбһјбһңбһ”бһҳбҹ’бһӣбҹӮбһ„ `'Insert_Your_Dataset'` бһ“бҹ…бһҮбһҪбһҡбһҠбҹҒбһҖбһ‘бһё 2 бһҮбһ¶бһҳбһҪбһҷбһ“бһ¶бһҳбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ–бһё Hugging Faceбҹ”

бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбһјбһ…бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

### бһҳбһ»бһҒбһ„бһ¶бһҡ

бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ“бҹҒбҹҮбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҮбҹҶбһ бһ¶бһ“бһҠбһјбһ…бһҸбһ‘бҹ…бҹ–

1. бһ‘бһ¶бһүбһҷбһҖбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһё Hugging Face бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `load_dataset()`бҹ”
2. бһ”бһҳбҹ’бһӣбҹӮбһ„бһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһё Hugging Face бһ‘бҹ…бһҮбһ¶ Pandas DataFrame бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ¶бһ…бһҡбҹҖбһ”бһ…бҹҶбһ”бһ¶бһ“бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҮбһ¶бһ„бһ“бҹҒбҹҮбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ `to_pandas()`бҹ”
3. бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһҜбһҖбһҹбһ¶бһҡ бһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бҹ”
4. бһҸбҹ’бһҡбһ„бһҮбһҪбһҡбһҠбҹҒбһҖбһҠбҹӮбһӣбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһҡбһ¶бһҮбҹҗбһҷбһҠбҹ„бһҷбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһҸбһ¶бһҳбһҮбһҪбһҡбһҠбҹҒбһҖбһҖбҹ’бһ“бһ»бһ„ DataFrame, бһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһҳбһ»бһҒбһ„бһ¶бһҡ `download_image()` бһҠбһјбһ…бһҮбһ¶бһҖбҹҶбһҺбһҸбҹӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“, бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳбһҮбһҪбһҡбһҠбҹҒбһҖбһҸбҹ’бһҡбһ„бһ‘бҹ… DataFrame бһҗбҹ’бһҳбһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮ `filtered_rows`бҹ”
5. бһ”бһ„бҹ’бһҖбһҫбһҸ DataFrame бһҗбҹ’бһҳбһёбһҮбһ¶бһҳбһҪбһҷбһҮбһҪбһҡбһҠбҹҒбһҖбһҠбҹӮбһӣбһҸбҹ’бһҡбһ„бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһңбһ¶бһҮбһ¶бһҜбһҖбһҹбһ¶бһҡ CSV бһ‘бҹ…бһҖбҹ’бһҡбҹ…бһҗбһ¶бһҹбҹ”
6. бһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһҹбһ¶бһҡбһ”бһ„бҹ’бһ бһ¶бһүбһ‘бһёбһҸбһ¶бҹҶбһ„бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–бҹ”

### бһҳбһ»бһҒбһ„бһ¶бһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“

бһҳбһ»бһҒбһ„бһ¶бһҡ `download_image()` бһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһ–бһё URL бһҳбһҪбһҷ бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһңбһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ Pillow Image (PIL) бһ“бһ·бһ„бһҳбҹүбһјбһҢбһ»бһӣ `io`бҹ” бһңбһ¶бһ”бһүбҹ’бһҮбһјбһ“бһҸбһҳбҹ’бһӣбҹғ True бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷ бһ бһҫбһҷ False бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ·бһ“бһ”бһ¶бһ“бҹ” бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһҖбҹҸбһ”бһ„бҹ’бһ бһ¶бһүбһҖбһҳбҹ’бһҡбһ„бһҖбһҡбһҺбһёбһҖбһҫбһҸбһҖбҹҶбһ бһ»бһҹбһҮбһ¶бһҳбһҪбһҷбһҹбһ¶бһҡвҖӢбһҖбҹҶбһ бһ»бһҹбһ“бҹ…бһ–бҹҒбһӣбһҹбҹҶбһҺбһҫбһ”бһҡбһ¶бһҮбҹҗбһҷбҹ”

### бһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ

бһҳбһ»бһҒбһ„бһ¶бһҡ download_image бһ‘бһ‘бһҪбһӣбһ–бҹ’бһҡбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһ–бһёбһҡбҹҲбһӮбһә image_url бһҠбҹӮбһӣбһҮбһ¶бһўбһ¶бһҹбһҷбһҠбҹ’бһӢбһ¶бһ“ URL бһҡбһјбһ”бһ—бһ¶бһ–бһҠбһҫбһҳбҹ’бһ”бһёбһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„ save_path бһҠбҹӮбһӣбһҮбһ¶бһҸбҹҶбһ”бһ“бҹӢбһ•бҹ’бһӣбһјбһң бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҡбһҖбҹ’бһҹбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бһ¶бһүбһҷбһҖбҹ”

бһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбҹ–

бһңбһ¶бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҠбҹ„бһҷбһ’бҹ’бһңбһҫбһҹбҹҶбһҺбһҫ GET бһ‘бҹ… image_url бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ requests.getбҹ” бһңбһ¶бһҹбҹҶбһҺбһҫбһҡбһ‘бһ‘бһҪбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһјбһ”бһ—бһ¶бһ–бһ–бһё URL бһ“бҹҒбҹҮбҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢ response.raise_for_status() бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶ бһҹбҹҶбһҺбһҫбһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷбһ¬бһўбһҸбҹӢбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҖбһјбһҠбһҹбҹ’бһҗбһ¶бһ“бһҖбһ¶бһҡбһҺбҹҚбһҳбһ¶бһ“бһҖбҹҶбһ бһ»бһҹ (бһ§. 404 - бһҳбһ·бһ“бһҳбһ¶бһ“бһ‘бҹҶбһ–бҹҗбһҡ), бһңбһ¶бһ“бһ№бһ„бһ”бһүбҹ’бһ…бҹҒбһүбһҖбһҳбҹ’бһҡбһ„бһҖбһҡбһҺбһёбһҖбҹҶбһ бһ»бһҹбҹ” бһңбһ¶бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҷбһҫбһ„бһ‘бҹ…бһҡбһҖбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбһ“бҹ…бһ–бҹҒбһӣбһҹбҹҶбһҺбһҫбһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷбҹ”

бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһүбһҫбһ‘бҹ…бһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҡбҹ’бһҸ Image.open бһ–бһёбһҳбҹүбһјбһҢбһ»бһӣ PIL (Python Imaging Library)бҹ” бһңбһ¶бһӮбһәбһ”бһ„бҹ’бһҖбһҫбһҸбһңбһҸбҹ’бһҗбһ» Image бһ–бһёбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһјбһ”бһ—бһ¶бһ–бҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢ image.save(save_path) бһҡбһҖбҹ’бһҹбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ…бһ•бҹ’бһӣбһјбһң save_path бһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбҹ” save_path бһӮбһҪбһҡбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһҜбһҖбһҹбһ¶бһҡ бһ“бһ·бһ„бһ•бҹ’бһ“бҹӮбһҖбһ”бһ“бҹ’бһҗбҹӮбһҳбҹ”

бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһ”бһүбҹ’бһҮбһјбһ“бһҸбһҳбҹ’бһӣбҹғ True бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһ бһ¶бһүбһҗбһ¶бһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ¶бһ“бһ‘бһ¶бһүбһҷбһҖ бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҳбһ¶бһ“бһҖбһҡбһҺбһёбһҖбҹҶбһ бһ»бһҹбһҺбһ¶бһҳбһҪбһҷбһҖбһҫбһҸбһЎбһҫбһ„ бһңбһ¶бһ…бһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһҖбҹҶбһ бһ»бһҹ бһ бһҫбһҷбһ”бҹ„бҹҮбһ–бһ»бһҳбҹ’бһ–бһҹбһ¶бһҡбһ”бһҡбһ¶бһҮбҹҗбһҷ бһ бһҫбһҷбһҸбҹ’бһҡбһЎбһ”бҹӢбһ‘бҹ… Falseбҹ”

бһҳбһ»бһҒбһ„бһ¶бһҡбһ“бҹҒбҹҮбһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһ–бһё URL бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһңбһ¶бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бҹ” бһңбһ¶бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹҶбһ бһ»бһҹбһҠбҹӮбһӣбһўбһ¶бһ…бһҖбһҫбһҸбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ– бһ бһҫбһҷбһ•бҹ’бһҠбһӣбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҗбһ¶бһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһ”бһ¶бһ“бһҮбҹ„бһӮбһҮбҹҗбһҷбһ¬бһўбһҸбҹӢбҹ”

бһӮбһҪбһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһҗбһ¶бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ requests бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ’бҹ’бһңбһҫбһҹбҹҶбһҺбһҫ HTTP, бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ PIL бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҡбһјбһ”бһ—бһ¶бһ–, бһ бһҫбһҷбһҗбҹ’бһ“бһ¶бһҖбҹӢ BytesIO бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҠбһҫбһҳбҹ’бһ”бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҡбһјбһ”бһ—бһ¶бһ–бһҮбһ¶бһҹбҹ’бһ‘бҹ’бһҡбһёбһҳбһ”бҹғбҹ”

### бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“

бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһ“бҹҒбҹҮбһ•бҹ’бһҠбһӣбҹӢбһ“бһјбһңбһңбһ·бһ’бһёбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһҠбҹ„бһҷбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡ, бһҸбҹ’бһҡбһ„бһҮбһҪбһҡбһҠбҹҒбһҖбһҠбҹӮбһӣбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһҡбһ¶бһҮбҹҗбһҷ, бһ бһҫбһҷбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҹбҹҶбһҺбһ»бҹҶбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҮбһ¶бһҜбһҖбһҹбһ¶бһҡ CSVбҹ”

### бһҹбҹ’бһӮбҹ’бһҡбһёбһ”бһӮбҹҶбһҡбһј

```python
import os
import pandas as pd
from datasets import load_dataset
import requests
from PIL import Image
from io import BytesIO

def download_image(image_url, save_path):
    try:
        response = requests.get(image_url)
        response.raise_for_status()  # бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶ бһҹбҹҶбһҺбһҫбһҮбҹ„бһӮбһҮбҹҗбһҷ бһ¬бһўбһҸбҹӢ
        image = Image.open(BytesIO(response.content))
        image.save(save_path)
        return True
    except Exception as e:
        print(f"Failed to download {image_url}: {e}")
        return False


# бһ‘бһ¶бһүбһҷбһҖбһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ–бһё Hugging Face
dataset = load_dataset('Insert_Your_Dataset')


# бһ”бһҳбҹ’бһӣбҹӮбһ„бһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ Hugging Face бһ‘бҹ…бһҮбһ¶ Pandas DataFrame
df = dataset['train'].to_pandas()


# бһ”бһ„бҹ’бһҖбһҫбһҸбһҗбһҸбһҜбһҖбһҹбһ¶бһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһҡбһјбһ”бһ—бһ¶бһ–
dataset_dir = './data/DataSetName'
images_dir = os.path.join(dataset_dir, 'images')
os.makedirs(images_dir, exist_ok=True)


# бһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹӮбһӣбһҖбһ¶бһҡбһ‘бһ¶бһүбһҷбһҖбһҡбһјбһ”бһ—бһ¶бһ–бһ”бһ¶бһ“бһ”бһҡбһ¶бһҮбҹҗбһҷ
filtered_rows = []
for idx, row in df.iterrows():
    image_url = row['imageurl']
    image_name = f"{row['product_code']}.jpg"
    image_path = os.path.join(images_dir, image_name)
    if download_image(image_url, image_path):
        row['local_image_path'] = image_path
        filtered_rows.append(row)


# бһ”бһ„бҹ’бһҖбһҫбһҸ DataFrame бһҗбҹ’бһҳбһёбһҮбһ¶бһҳбһҪбһҷбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһҠбҹӮбһӣбһ”бһ¶бһ“бһҮбҹ’бһҡбҹ„бҹҮбһҡбһҫбһҹ
filtered_df = pd.DataFrame(filtered_rows)


# бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҜбһҖбһҹбһ¶бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбҹҒбһЎбҹҒбҹҮбһ‘бҹ…бһҗбһ¶бһҹбһҠбҹ’бһӢбһ¶бһ“
dataset_path = os.path.join(dataset_dir, 'Dataset.csv')
filtered_df.to_csv(dataset_path, index=False)


print(f"Dataset and images saved to {dataset_dir}")
```
