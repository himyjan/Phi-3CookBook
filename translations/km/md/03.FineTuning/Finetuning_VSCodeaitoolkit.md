### бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣ

### бһҖбһ¶бһҡбһҸбһ¶бҹҶбһ„бһҸбһ¶бҹҶбһ„бһӮбһҳбҹ’бһҡбҹ„бһ„бһўбһ—бһ·бһңбһҢбҹ’бһҚбһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷ

2. бһ‘бҹ…бһҖбһ¶бһ“бҹӢ *Model Fine-tuning* бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҖбһ¶бһЎбһ¶бһЎбһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣбҹ” бһ•бҹ’бһҸбһӣбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһӮбһҳбҹ’бһҡбҹ„бһ„ бһ“бһ·бһ„бһҮбҹ’бһҡбһҫбһҹбһ‘бһёбһҸбһ¶бҹҶбһ„бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ…бһ»бһ… *"Configure Project"*бҹ”
3. бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһӮбһҳбҹ’бһҡбҹ„бһ„
    1. бһҖбһ»бҹҶбһ”бһҫбһҖбһҮбһҳбҹ’бһҡбһҫбһҹ *"Fine-tune locally"*бҹ”
    2. бһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢ Olive бһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһҸбһҳбҹ’бһӣбҹғбһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҳбһ»бһ“бҹ” бһҹбһјбһҳбһҖбҹӮбһ”бҹ’бһҡбҹӮ бһ“бһ·бһ„бһ”бҹҶбһ–бҹҒбһүбһҖбһ¶бһҡбһҖбҹҶбһҺбһҸбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһҳбһҫбһӣбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„ console logs бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫ Azure portal бһҸбһ¶бһҳбһҸбҹҶбһҺбһҠбҹӮбһӣбһҳбһ¶бһ“бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ‘бһ¶бҹҶбһ„ output (бһҮбҹҶбһ бһ¶бһ“бһ”бһ“бҹ’бһҗбҹӮбһҳбһ“бҹ… [бһҳбһҫбһӣбһ“бһ·бһ„бһҹбҹҶбһҺбһҪбһҡбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ“бҹ…бһӣбһҫ Azure](https://aka.ms/ai-toolkit/remote-provision#view-and-query-logs-on-azure))бҹ” бһ¬ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣ console logs бһҸбҹ’бһҡбһ„бҹӢбһ•бҹ’бһ‘бһ¶бҹҶбһ„ output VSCode бһҠбҹ„бһҷбһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `AI Toolkit: Show the running fine-tuning job streaming logs`бҹ”  
> **бһ…бҹҶбһҺбһ¶бҹҶ:** бһҖбһ¶бһҡбһ„бһ¶бһҡбһўбһ¶бһ…бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ„бһ…бһ¶бҹҶбһҠбҹ„бһҷбһҹбһ¶бһҡбһҸбҹӮбһ’бһ“бһ’бһ¶бһ“бһҳбһ·бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ¶бһ“бҹӢбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбһҳбһ·бһ“бһ”бһ„бҹ’бһ бһ¶бһүбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ» бһҹбһјбһҳбһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `AI Toolkit: Show the running fine-tuning job streaming logs` бһҳбҹ’бһҸбһ„бһ‘бҹҖбһҸ бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҡбһ„бҹӢбһ…бһ¶бҹҶбһ–бҹҒбһӣбһҒбҹ’бһӣбһё бһ бһҫбһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶бһңбһ·бһүбһҠбһҫбһҳбҹ’бһ”бһёбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»бһ•бҹ’бһҹбһ¶бһҷбһ”бһ“бҹ’бһҸбҹ”

бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹҒбҹҮ QLoRA бһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһ бһҫбһҷбһ“бһ№бһ„бһ”бһ„бҹ’бһҖбһҫбһҸ LoRA adapters бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ“бҹ…бһ–бҹҒбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбҹ”
бһӣбһ‘бҹ’бһ’бһ•бһӣбһ“бҹғбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ“бһ№бһ„бһҸбҹ’бһҡбһјбһңбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Azure Filesбҹ”

### бһ•бҹ’бһҠбһӣбҹӢбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ
бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһё adapters бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҺбҹ’бһҠбһ»бҹҮбһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ…бһҳбҹ’бһ„бһ¶бһҷ бһҹбһјбһҳбһ”бҹ’бһҡбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё Gradio бһҹбһ¶бһҳбһүбҹ’бһүбһҠбһҫбһҳбҹ’бһ”бһёбһ–бһ·бһ—бһ¶бһҖбҹ’бһҹбһ¶бһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбҹ”
бһҠбһјбһ…бһҮбһ¶вҖӢбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·вҖӢбһҖбһ¶бһҡвҖӢбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҸбһ¶бҹҶбһ„бһҸбһ¶бҹҶбһ„бһ’бһ“бһ’бһ¶бһ“ Azure бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ–бһёбһ…бһҳбҹ’бһ„бһ¶бһҷ бһҠбҹ„бһҷбһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `AI Toolkit: Provision Azure Container Apps for inference` бһ–бһё command paletteбҹ”

бһӣбҹҶбһ“бһ¶бҹҶбһҠбһҫбһҳ бһҖбһ¶бһҡбһҮбһ¶бһң бһ“бһ·бһ„бһҖбҹ’бһҡбһ»бһҳбһ’бһ“бһ’бһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӮбһҪбһҡбһҸбҹӮбһҹбҹ’бһҡбһ”бһӮбҹ’бһ“бһ¶бһ“бһ№бһ„бһӮбҹҒбһ бһ‘бҹҶбһ–бҹҗбһҡбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ” бһҖбһ¶бһҡбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ Azure Container App бһҠбһјбһ…бһӮбҹ’бһ“бһ¶ бһ“бһ·бһ„бһ…бһјбһӣбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„ adapter бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„ Azure Files бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһЎбһҫбһ„бһҖбҹ’бһ“бһ»бһ„бһҮбҹҶбһ бһ¶бһ“бһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ”

### бһ•бҹ’бһҹбһ–бҹ’бһңбһ•бҹ’бһҹбһ¶бһҷбһ…бҹҶбһҺбһ»бһ…бһ”бҹ’бһҡбһ‘бһ¶бһҖбҹӢбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚ
бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ…бһ„бҹӢбһҹбҹҶбһҡбһ»бһ„бһҖбһјбһҠбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ¬бһ•бҹ’бһ‘бһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһЎбһҫбһ„бһңбһ·бһү бһҹбһјбһҳбһҡбһҸбҹӢбһ–бһ¶бһҖбҹ’бһҷбһ”бһүбҹ’бһҮбһ¶ `AI Toolkit: Deploy for inference`бҹ” бһңбһ¶бһ“бһ№бһ„бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһјбһҠбһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҮбһ¶бһҳбһҪбһҷ Azure Container App бһ бһҫбһҷбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһЎбһҫбһ„бһңбһ·бһү Replica бҹ”

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ”бһ„бҹ’бһ бһ¶бһүбһўбһҹбҹӢбһҮбҹ„бһӮбһҮбҹҗбһҷ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ…бһјбһӣбһ”бҹ’бһҡбһҫ API бһўбһ“бһ»бһңбһҸбҹ’бһҸ бһҠбҹ„бһҷбһ…бһ»бһ…бһ”бҹҠбһјбһҸбһ»бһ„ "*Go to Inference Endpoint*" бһҠбҹӮбһӣбһ”бһ„бҹ’бһ бһ¶бһүбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҮбһјбһ“бһҠбҹҶбһҺбһ№бһ„ VSCodeбҹ” бһ¬ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҡбһҖбһғбһҫбһүбһ…бҹҶбһҺбһ»бһ…бһ‘бһёбһҸбһ¶бҹҶбһ„ web API бһ“бҹ…бһҖбҹ’бһҡбҹ„бһҳ `ACA_APP_ENDPOINT` бһҖбҹ’бһ“бһ»бһ„ `./infra/inference.config.json` бһ“бһ·бһ„бһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ‘бһ¶бҹҶбһ„ outputбҹ” бһҘбһЎбһјбһңбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһҡбҹҖбһ”бһ…бҹҶбһҡбһҪбһ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҠбҹӮбһӣбһҸбһ¶бһҳбһ”бһ“бҹ’бһ‘бһ¶бһҸбҹӢбһ“бҹҒбҹҮбҹ”

### бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢ
