## бһҹбһҖбҹ’бһҸбһ¶бһ“бһ»бһ–бһӣбһ“бҹғбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһ„бҹӢбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡ

![FineTuning with MS Services](../../../../translated_images/km/FinetuningwithMS.3d0cec8ae693e094.webp)

бһ•бҹ’бһ“бҹӮбһҖбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҖбһ¶бһҡбһ–бһ·бһ–бһҺбҹҢбһ“бһ¶бһ–бһёбһҹбһҖбҹ’бһҸбһ¶бһ“бһ»бһ–бһӣбһ“бҹғбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһ„бҹӢбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһңбҹҒбһҺ Microsoft Foundry бһ“бһ·бһ„ Azure бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһӮбҹҶбһҡбһјбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ бһҒбҹ’бһ“бһ¶бһҸбһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’ бһ“бһ·бһ„бһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶бҹ”

**бһңбҹҒбһ‘бһ·бһҖбһ¶**  
бһ“бҹҒбҹҮбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбһјбһ…бһҮбһ¶ Microsoft Foundry (бһҠбҹӮбһӣбһҳбһ»бһ“бһ“бҹҒбҹҮбһ бҹ…бһҗбһ¶ Microsoft Foundry) бһ“бһ·бһ„ Azure Machine Learning бһҠбҹӮбһӣбһ•бҹ’бһҸбһӣбҹӢбһҖбҹҶбһҺбһҸбҹӢбһ“бһёбһҸбһ·бһңбһ·бһ’бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӮбҹҶбһҡбһј бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ deploymentбҹ”

**бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’**  
бһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһ„бҹӢбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҒбҹ’бһ–бһҹбҹӢбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ’бһ“бһ’бһ¶бһ“бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһ”бһ¶бһ“бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бһҡбһ·бһңбҹҒбһҺ Azure бһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶ бһҡбһҪбһҳбһҳбһ¶бһ“бһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһңбһҫбһҢбһ»бһ…бһҒбҹ’бһ–бһҹбҹӢбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫ GPU бһ“бһ·бһ„бһ’бһ“бһ’бһ¶бһ“ CPU бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ“бҹ’бһ‘бһ»бһҖбһҖбһ¶бһҡбһ„бһ¶бһҡбһҸбһ·бһ…бҹ— бһҮбһ¶бһҳбһҪбһҷбһ“бһ№бһ„бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһ–бһ„бҹ’бһҡбһёбһҖбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһ»бҹҶбһ“бһ·бһ„бһ…бҹҶбһҺбһ»бһ…бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ”

**бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҹбҹҠбһ»бһҳ**  
бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһ„бҹӢбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҷбҹүбһ¶бһ„бһҮбһ¶бһ’бһҳбҹ’бһҳбһҸбһ¶ бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһҹбҹҠбһ»бһҳбһ“бһ·бһ„бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҠбһјбһ…бһҮбһ¶ Hugging Face Transformers, DeepSpeed бһ“бһ·бһ„ PEFT (Parameter-Efficient Fine-Tuning)бҹ”

бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһҳбҹ’бһҡбһ„бҹӢбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҮбһ¶бһҳбһҪбһҷбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ Microsoft бһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбһ‘бҹ…бһӣбһҫбһҹбҹҒбһңбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶ бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ бһ“бһ·бһ„бһҹбҹҠбһ»бһҳбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбҹ” бһҠбҹ„бһҷбһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ”бһ…бҹ’бһ…бҹҒбһҖбһ‘бҹҒбһҹбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶ бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҸбһҳбҹ’бһҡбһјбһңбһӮбҹҶбһҡбһјбһҠбһҫбһҳбһұбҹ’бһҷбһҹбһҳбһҹбҹ’бһҡбһ”бһ‘бҹ…бһ“бһ№бһ„бһ—бһ¶бһҡбһҖбһ·бһ…бҹ’бһ…бһ“бһ·бһ„бһҹбҹҒбһҺбһ¶бһҡбһёбһҷбҹүбһјбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҮбһ¶бһҳбһҪбһҷбһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”

## бһӮбҹҶбһҡбһјбһҮбһ¶ бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ

бһҸбһҳбҹ’бһҡбһ„бҹӢбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҖбҹ’бһ“бһ»бһ„бһӮбҹҶбһҡбһјбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫ fine-tuning бһҮбһ¶бһҡбһ”бҹҖбһ”бһҹбҹҒбһңбһ¶бһҮбһјбһ“бһҠбҹ„бһҷбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбҹ”
