# **бһҺбҹӮбһ“бһ¶бҹҶбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure Machine Learning**

[Azure Machine Learning](https://ml.azure.com?WT.mc_id=aiml-138114-kinfeylo) бһӮбһәбһҮбһ¶бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҳбҹҒбһғбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһјбһ“бһӣбҹ’бһ”бһҝбһ“ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҮбһ¶бһҗбҹ’бһ“бһ¶бһҖбҹӢбһўбҹ’бһ“бһҖбһҹбҹ’бһҡбһёбһҖбҹ’бһҠбһёбһңбһёбһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһӮбһҳбҹ’бһҡбҹ„бһ„бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ (ML)бҹ”

бһўбҹ’бһ“бһҖбһҮбҹҶбһ“бһ¶бһү ML, бһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһңбһ·бһҹбҹ’бһңбһҖбһҡ бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ…бһҡбһ“бҹ’бһҸбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһҠбһҫбһҳбҹ’бһ”бһё:

- бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣ бһ“бһ·бһ„бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҳбҹүбһјбһҠбҹӮбһӣбҹ”
бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһҳбҹүбһ¶бһҹбҹҠбһёбһ“ (MLOps)бҹ”
- бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһҪбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Azure Machine Learning бһ¬бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бһёбһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһ”бҹ’бһҡбһ—бһ–бһ”бһҫбһҖбһҠбһјбһ…бһҮбһ¶ PyTorch, TensorFlow, бһ¬ scikit-learnбҹ”
- бһ§бһ”бһҖбһҡбһҺбҹҚ MLOps бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷ, бһ”бһҺбҹ’бһҸбһ»бҹҮбһ”бһҺбҹ’бһҸбһ¶бһӣбһЎбһҫбһ„бһңбһ·бһү, бһ“бһ·бһ„бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһЎбһҫбһ„бһңбһ·бһүбҹ”

## Azure Machine Learning бһҹбҹҶбһҡбһ¶бһ”бҹӢбһ“бһҡбһҺбһ¶?

**бһўбҹ’бһ“бһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһңбһ·бһҹбҹ’бһңбһҖбһҡ ML**

бһ–бһҪбһҖбһӮбҹҒбһ…бһ¶бһ”бҹӢбһўбһ¶бһҡбһҳбҹ’бһҳбһҺбҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһ“бһӣбҹ’бһ”бһҝбһ“ бһ“бһ·бһ„бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбҹ’бһ“бһ»бһ„бһ…бһҡбһ“бҹ’бһҸбһҖбһ¶бһҡбһ„бһ¶бһҡбһ”бҹ’бһҡбһ…бһ¶бҹҶбһҗбҹ’бһ„бҹғбҹ”
Azure ML бһ•бҹ’бһҸбһӣбҹӢбһҳбһ»бһҒбһ„бһ¶бһҡвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң, бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһӣбҹӢ, бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ”

**бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё:**
бһ–бһҪбһҖбһӮбҹҒбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ¬бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбҹ„бһҷбһҡбһӣбһјбһ“бҹ”

**бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһңбҹҒбһ‘бһ·бһҖбһ¶**

бһ–бһҪбһҖбһӮбҹҒбһҳбһ¶бһ“бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бҹ…бһ§бһ”бһҖбһҡбһҺбҹҚбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһҠбҹӮбһӣбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҠбҹ„бһҷ API бһҡбһ№бһ„бһҳбһ¶бҹҶбһҡбһ”бһҹбҹӢ Azure Resource Managerбҹ”
бһ§бһ”бһҖбһҡбһҺбҹҚбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбһҹбһ„бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ ML бһҖбһҳбҹ’бһҡбһ·бһҸбһҒбҹ’бһ–бһҹбҹӢбҹ”

**бһҹбһ бһӮбҹ’бһҡбһ¶бһҹ**

бһ’бҹ’бһңбһҫбһҖбһ¶бһҡвҖӢбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһғ Microsoft Azure, бһҹбһ бһӮбҹ’бһҡбһ¶бһҹбһ‘бһ‘бһҪбһӣбһ”бһ¶бһ“бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ–бһёбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҠбҹӮбһӣбһӮбһҪбһҡбһұбҹ’бһҷбһҹбҹ’бһӮбһ¶бһӣбҹӢ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҸбһҪбһ“бһ¶бһ‘бһёбҹ”
бһҡбҹҖбһ”бһ…бҹҶбһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбһ¶бһҡбһ…бһјбһӣбһ”бҹ’бһҡбһҫбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбһ¶бһҡбһ–бһ¶бһҡ бһ“бһ·бһ„бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ”

## бһ•бһӣбһ·бһҸбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбһ“бһ»бһҹбҹ’бһҹбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһ“бһ¶бһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳ
бһӮбһҳбҹ’бһҡбҹ„бһ„ ML бһ—бһ¶бһӮбһ…бҹ’бһҡбһҫбһ“бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбҹ’бһҡбһ»бһҳбһҠбҹӮбһӣбһҳбһ¶бһ“бһҮбҹҶбһ“бһ¶бһүбһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһҹбһ¶бһ„ бһ“бһ·бһ„бһҗбҹӮбһ‘бһ¶бҹҶбҹ”

Azure ML бһ•бҹ’бһҸбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһўбҹ„бһҷбһўбҹ’бһ“бһҖ:
- бһҹбһ бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбҹ’бһҡбһ»бһҳбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ¶бһҡбһҪбһҳ, бһ§бһ”бһҖбһҡбһҺбҹҚбһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡ, бһҖбһ»бҹҶбһ–бҹ’бһҷбһјбһ‘бҹҗбһҡбһҘбһҸбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҳбҹҒ, бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ бһ“бһ·бһ„бһ”бһҡбһ·бһңбҹҒбһҺбҹ”
- бһўбһ—бһ·бһңбһҢбҹ’бһҚбһҳбҹүбһјбһҠбҹӮбһӣбһҮбһ¶бһҳбһҪбһҷбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң, бһҖбһ¶бһҡбһ–бһ“бҹ’бһҷбһӣбҹӢ, бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ–бҹҒбһүбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҠбҹӮбһӣбһ‘бһ¶бһҖбҹӢбһ‘бһ„бһ“бһ№бһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ· бһ“бһ·бһ„бһҖбһ¶бһҡбһўбһ“бһ»бһӣбҹ„бһҳбһҸбһ¶бһҳбһҡбһҷбҹҲбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ”
- бһ…бҹҒбһүбһ•бҹ’бһҹбһ¶бһҷбһҳбҹүбһјбһҠбҹӮбһӣ ML бһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҡбһ бҹҗбһҹ бһ“бһ·бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ“бҹ…бһ…бҹҶбһҺбһ»бһ…бһңбһ¶бһҹбҹӢбһҖбһҳбҹ’бһҡбһ·бһҸ бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһ¶бһ“бһ·бһ„бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһ¶бһІбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҮбһ¶бһҳбһҪбһҷ MLOpsбҹ”
- бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҳбҹүбһ¶бһҹбҹҠбһёбһ“бһҡбҹҖбһ“бһ“бҹ…бһӮбҹ’бһҡбһ”бҹӢбһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹ„бһҷбһҳбһ¶бһ“бһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҖбҹ’бһ“бһ»бһ„бһҒбҹ’бһӣбһҪбһ“, бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–, бһ“бһ·бһ„бһўбһ“бһ»бһӣбҹ„бһҳбһҸбһ¶бһҳбһ…бҹ’бһ”бһ¶бһ”бҹӢбҹ”

## бһ§бһ”бһҖбһҡбһҺбҹҚбһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҶбҹ’бһӣбһјбһ„бһӮбҹ’бһ“бһ¶

бһ“бһҡбһҺбһ¶бһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҖбҹ’бһҡбһ»бһҳ ML бһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҡбһ”бһҹбҹӢбһ–бһҪбһҖбһӮбҹҒбһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһ„бһ¶бһҡбҹ”
