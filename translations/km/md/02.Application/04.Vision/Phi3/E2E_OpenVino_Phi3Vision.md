សាកល្បងនេះបង្ហាញពីរបៀបប្រើម៉ូឌែលដែលបានបណ្ដុះមុន ដើម្បីបង្កើតកូដ Python ដោយផ្អែកលើរូបភាព និងសារ (text prompt)។

[កូដគំរូ](../../../../code/06.E2E/E2E_OpenVino_Phi3-vision.ipynb)

នេះជាការពន្យល់ជាដំណាក់កាល៖

1. **ការនាំចូល និងការកំណត់**:
   - បណ្ណាល័យ និងម៉ូឌុលដែលចាំបាច់ត្រូវបាននាំចូល ដូចជា `requests`, `PIL` សម្រាប់ដំណើរការរូបភាព និង `transformers` សម្រាប់ការរៀបចំម៉ូឌែល និងដំណើរការ។

2. **កំពុងផ្ទុក និងបង្ហាញរូបភាព**:
   - ឯកសាររូបភាព (`demo.png`) ត្រូវបានបើក ដោយប្រើបណ្ណាល័យ `PIL` ហើយបានបង្ហាញ។

3. **កំណត់សារ (prompt)**:
   - បានបង្កើតសារ​ដែលរួមបញ្ចូលរូបភាព និងសំណើឲ្យបង្កើតកូដ Python ដើម្បីដំណើរការរូបភាព និងរក្សាទុកវា ដោយប្រើ `plt` (matplotlib)។

4. **កំពុងផ្ទុក Processor**:
   - `AutoProcessor` ត្រូវបានផ្ទុកពីម៉ូឌែលដែលបានបណ្ដុះមុន ដែលបានបញ្ជាក់ដោយថត `out_dir`។ Processor នេះនឹងដំណើរការការបញ្ចូលអត្ថបទ និងរូបភាព។

5. **បង្កើតសារ (prompt)**:
   - វិធីសាស្រ្ត `apply_chat_template` ត្រូវបានប្រើដើម្បីទ្រង់ទ្រាយសារឱ្យជាព្រឹត្តិការណ៍ prompt ដែលសមស្របសម្រាប់ម៉ូឌែល។

6. **ដំណើរការបញ្ចូលទិន្នន័យ**:
   - Prompt និងរូបភាពត្រូវបានដំណើរការជា tensors ដែលម៉ូឌែលអាចយល់បាន។

7. **កំណត់អាគុយម៉ង់សម្រាប់ការបង្កើត**:
   - តម្លៃអាគុយម៉ង់សម្រាប់ដំណើរការបង្កើតរបស់ម៉ូឌែលត្រូវបានកំណត់រួច រួមទាំងចំនួនអតិបរមានៃ tokens ថ្មីដែលត្រូវបង្កើត និងថាតើត្រូវធ្វើការគំរូ (sample) លទ្ធផលឬទេ។

8. **បង្កើតកូដ**:
   - ម៉ូឌែលបង្កើតកូដ Python ដោយផ្អែកលើបញ្ចូល និងអាគុយម៉ង់នៃការបង្កើត។ `TextStreamer` ត្រូវបានប្រើដើម្បីគ្រប់គ្រងលទ្ធផល ដោយរំលង prompt និង tokens ពិសេស។

9. **លទ្ធផល**:
   - កូដដែលបានបង្កើតត្រូវបានបោះពុម្ព ដែលគួរតែមានកូដ Python ដើម្បីដំណើរការរូបភាព និងរក្សាទុកវាដែរ ដូចដែលបានបញ្ជាក់ក្នុង prompt។

សាកល្បងនេះបង្ហាញពីរបៀបប្រើម៉ូឌែលដែលបានបណ្ដុះមុន ដោយប្រើ OpenVino ដើម្បីបង្កើតកូដដោយស្វ័យប្រវត្តិ និងបត់បែន ដោយផ្អែកលើការបញ្ចូលពីអ្នកប្រើ និងរូបភាព។
