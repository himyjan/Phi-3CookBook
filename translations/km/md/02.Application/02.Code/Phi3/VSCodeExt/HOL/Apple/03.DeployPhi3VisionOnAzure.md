# **бһӣбһ¶бһ” 3 - бһҠбһ¶бһҖбҹӢбһ…бҹҒбһү Phi-3-vision бһӣбһҫ Azure Machine Learning Service**

бһҷбһҫбһ„бһ”бҹ’бһҡбһҫ NPU бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһ“бҹғбһҖбһјбһҠбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ (local) бһ бһҫбһҷбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҷбһҫбһ„бһ…бһ„бҹӢбһ”бһ“бҹ’бһҗбҹӮбһҳбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡ бһ”бһүбҹ’бһ…бһјбһӣ PHI-3-VISION бһҸбһ¶бһҳбһҡбһҷбҹҲбһңбһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһҳбҹ’бһӣбҹӮбһ„бһҡбһјбһ”бһ—бһ¶бһ–бһ‘бҹ…бһҖбһјбһҠбҹ”

бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһ“бҹҒбҹҮ бһҷбһҫбһ„бһўбһ¶бһ…бһҹбһ¶бһ„бһҹбһ„бҹӢбһҹбҹҒбһңбһ¶ Phi-3 Vision бһҮбһ¶ бһҳбҹүбһјбһҠбҹӮбһӣбһҮбһ¶бһҹбҹҒбһңбһ¶ бһӣбһҝбһ“бҹ— бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Azure Machine Learning Serviceбҹ”

***бһ…бҹҶбһҺбһ¶бҹҶ***пјҡ Phi-3 Vision бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһӮбһҺбһ“бһ¶ бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶ бһұбҹ’бһҷбһ”бһ¶бһ“бһӣбһҝбһ“бҹ” бһҷбһҫбһ„бһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҖбһҳбҹ’бһӣбһ¶бҹҶбһ„бһӮбһҺбһ“бһ¶бһ–бһёбһ–бһ–бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһұбҹ’бһҷбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ”бһ¶бһ“бһ“бҹҒбҹҮбҹ”
