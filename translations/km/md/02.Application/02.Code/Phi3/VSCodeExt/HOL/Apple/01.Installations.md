# **មេរៀនទី 0 - ការដំឡើង**

ពេលយើងចូលទៅក្នុងមេរៀន យើងត្រូវកំណត់បរិយាកាសដែលទាក់ទង៖


### **1. Python 3.11+**

ណែនាំឱ្យប្រើ miniforge ដើម្បីកំណត់បរិយាកាស Python របស់អ្នក

ក្នុងការកំណត់ miniforge សូមយោងទៅកាន់ [https://github.com/conda-forge/miniforge](https://github.com/conda-forge/miniforge)

បន្ទាប់ពីកំណត់ miniforge បាន សូមរត់ពាក្យបញ្ជាខាងក្រោមនៅក្នុង Power Shell

```bash

conda create -n pyenv python==3.11.8 -y

conda activate pyenv

```


### **2. Install Prompt flow SDK**

នៅក្នុង Lab 1 យើងប្រើ Prompt flow ដូច្នេះអ្នកត្រូវកំណត់ Prompt flow SDK។

```bash

pip install promptflow --upgrade

```

អ្នកអាចពិនិត្យ promptflow sdk ដោយប្រើពាក្យបញ្ជានេះ


```bash

pf --version

```

### **4. ស៊ុម MLX របស់ Apple**

MLX គឺជា framework សម្រាប់ការស្រាវជ្រាវម៉ាស៊ីនរៀនលើ Apple silicon ដែលផ្តល់ដោយក្រុមស្រាវជ្រាវម៉ាស៊ីនរៀនរបស់ Apple។ អ្នកអាចប្រើ **Apple MLX framework** ដើម្បីលឿនដំណើរការ LLM / SLM ជាមួយ Apple Silicon។ ប្រសិនបើអ្នកចង់ដឹងបន្ថែម អ្នកអាចអាន [https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/03/MLX_Inference.md](https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/03/MLX_Inference.md).

ដំឡើង MLX framework Library ក្នុង bash


```bash

pip install mlx-lm

```



### **5. បណ្ណាល័យ Python ផ្សេងទៀត**


បង្កើត requirements.txt ហើយបន្ថែមមាតិកានេះ

```txt

notebook
numpy 
scipy 
scikit-learn 
matplotlib 
pandas 
pillow 
graphviz

```


### **6. ដំឡើង NVM**

ដំឡើង nvm នៅក្នុង Powershell 


```bash

brew install nvm

```

ដំឡើង nodejs 18.20


```bash

nvm install 18.20.0

nvm use 18.20.0

```

### **7. ដំឡើងការគាំទ្រការអភិវឌ្ឍន៍សម្រាប់ Visual Studio Code**


```bash

npm install --global yo generator-code

```

សូមអបអរសាទរ! អ្នកបានកំណត់ SDK ជោគជ័យរួចរាល់។ រួចបន្តទៅដំណាក់កាលអនុវត្តផ្ទាល់។
