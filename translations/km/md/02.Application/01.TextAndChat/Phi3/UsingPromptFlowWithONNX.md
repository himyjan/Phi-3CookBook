# бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ GPU бһӣбһҫ Windows бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Prompt flow бһҮбһ¶бһҳбһҪбһҷ Phi-3.5-Instruct ONNX 

бһҜбһҖбһҹбһ¶бһҡвҖӢбһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳвҖӢбһӮбһәбһҮбһ¶бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ–бһёбһҡбһ”бҹҖбһ”вҖӢбһ”бҹ’бһҡбһҫ PromptFlow бһҮбһ¶бһҳбһҪбһҷ ONNX (Open Neural Network Exchange) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҠбҹӮбһӣбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӣбһҫбһҳбҹүбһјбһҠбҹӮбһӣ Phi-3бҹ”

PromptFlow бһӮбһәбһҮбһ¶бһҹбҹҠбһ»бһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ—бһ·бһҡбһҖбҹ’бһҹбһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбһҪбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ–бһёбһҠбһҫбһҳбһҠбһӣбҹӢбһ…бһ”бҹӢ бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ LLM (Large Language Model) бһ…бһ¶бһ”бҹӢбһ–бһёбһҖбһ¶бһҡбһӮбһјбһҡбһӮбҹҶбһ“бһ·бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһҳбҹ’бһӣбһҫбһҳбһӮбҹҶбһҡбһј бһҡбһ бһјбһҸбһҠбһӣбҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҸбҹҒбһҹбҹ’бһҸ бһ“бһ·бһ„бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбҹ”

бһҠбҹ„бһҷбһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣ PromptFlow бһҮбһ¶бһҳбһҪбһҷ ONNX бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһўбһ¶бһ…бҹҲ

- бһҖбҹӮбһӣбһҳбҹ’бһўбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҳбҹүбһјбһҠбҹӮбһӣ: бһ”бҹ’бһҡбһҫ ONNX бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ inference бһ“бһ·бһ„ deployment бһұбҹ’бһҷбһҳбһ¶бһ“бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бҹ”
- бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣ: бһ”бҹ’бһҡбһҫ PromptFlow бһҠбһҫбһҳбҹ’бһ”бһёбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһӣбҹҶбһ бһјбһҡбһҖбһ¶бһҡбһ„бһ¶бһҡ бһ“бһ·бһ„бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹӮбһӣбһ’бҹ’бһңбһҫбһҡбҹҖбһ„бһҡбһ¶бһӣбҹӢбҹ”
- бһҖбҹӮбһӣбһҳбҹ’бһўбһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡ: бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбһ бһҖбһ¶бһҡбһӣбҹ’бһўбһ”бҹ’бһҡбһҹбһҫбһҡбһҡбһңбһ¶бһ„бһҹбһҳбһ¶бһҮбһ·бһҖбһҖбҹ’бһҡбһ»бһҳ бһҠбҹ„бһҷбһ•бҹ’бһҸбһӣбҹӢбһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҸбҹӮбһҳбһҪбһҷбҹ”

**Prompt flow** бһӮбһәбһҮбһ¶вҖӢбһҹбҹҠбһ»бһҳбһ§бһ”бһҖбһҡбһҺбҹҚбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбһҪбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ–бһёбһҠбһҫбһҳбһҠбһӣбҹӢбһ…бһ”бҹӢ бһ“бҹғбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ LLM бһҠбһјбһ…бһҮбһ¶ ideation, prototyping, testing, evaluation бһ‘бҹ…бһҠбһӣбҹӢ production deployment бһ“бһ·бһ„ monitoringбҹ” бһңбһ¶бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһ”бҹҗбһҺбҹ’бһҺ prompt аёҮбһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҮбһ¶бһ„бһҳбһ»бһ“ бһ“бһ·бһ„бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһҹбһ¶бһ„бһҹбһ„бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё LLM бһҠбҹ„бһҷбһҳбһ¶бһ“бһӮбһ»бһҺбһ—бһ¶бһ–бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбҹ”

Prompt flow бһўбһ¶бһ…бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ‘бҹ…бһҖбһ¶бһ“бҹӢ OpenAI, Azure OpenAI Service бһ“бһ·бһ„бһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҠбһјбһҡбһ”бһ¶бһ“ (Huggingface, local LLM/SLM)бҹ” бһҷбһҫбһ„бһҳбһ¶бһ“бһӮбһҳбҹ’бһҡбҹ„бһ„бһҠбһ¶бһҖбҹӢбһ…бҹҒбһүбһҳбҹүбһјбһҠбҹӮбһӣ ONNX бһҠбҹӮбһӣбһ”бһ¶бһ“ quantize бһҡбһ”бһҹбҹӢ Phi-3.5 бһҠбҹ„бһҷбһҹбҹҶбһҠбҹ…бһӣбһҫбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢбҹ” Prompt flow бһўбһ¶бһ…бһҮбһҪбһҷбһҷбһҫбһ„бһ•бҹӮбһ“бһҖбһ¶бһҡбһҮбһ¶бһ„бһӣбҹ’бһўбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҹбҹ’бһҡбһ»бһҖбһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ Phi-3.5бҹ” бһҖбҹ’бһ“бһ»бһ„бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ“бһ№бһ„бһ”бһ»бһҺбҹ’бһҷбһҖбһ¶бһҡбһ—бҹ’бһҮбһ¶бһ”бҹӢбһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷ ONNX Runtime GenAI бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ”бҹӢбһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Prompt flow бһҠбҹӮбһӣбһ•бҹ’бһўбҹӮбһҖбһӣбһҫ Windows GPUбҹ”
