- **Fairness and Inclusiveness**: бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһӮбһҪбһҡбһҸбҹӮбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһ“бһ¶ бһ бһҫбһҷбһ…бҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһҠбһӣбҹӢбһҖбҹ’бһҡбһ»бһҳбһҳбһ“бһ»бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶бһ‘бҹ…бһңбһ·бһүбһ‘бҹ…бһҳбһҖбҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһ•бҹ’бһҸбһӣбҹӢбһҷбҹ„бһ”бһӣбҹӢбһўбҹҶбһ–бһёбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһ”бһ¶бһӣвҖӢбһңбҹҮбһҖбһ¶бһҸбҹӢ бһҖбһ¶бһҡбһҠбһ¶бһҖбҹӢбһ–бһ¶бһҖбҹ’бһҷбһҖбһҳбҹ’бһ…бһё бһ¬бһҖбһ¶бһҡбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбһҖбһ¶бһҡбһ„бһ¶бһҡ бһңбһ¶бһӮбһҪбһҡбһ•бҹ’бһҸбһӣбҹӢбһҷбҹ„бһ”бһӣбҹӢбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ…бҹҶбһ–бҹ„бҹҮбһҳбһ“бһ»бһҹбҹ’бһҹбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбҹ„бһӮбһҹбһүбҹ’бһүбһ¶бһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶ бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ бһ·бһҡбһүбҹ’бһүбһңбһҸбҹ’бһҗбһ» бһ¬бһӣбһҖбҹ’бһҒбһҺбҹҲбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһңбһ·бһҮбҹ’бһҮбһ¶бһҮбһёбһңбҹҲбһҹбҹ’бһҡбһҠбҹҖбһ„бһӮбҹ’бһ“бһ¶бҹ”

- **Reliability and Safety**: бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһ‘бҹҶбһ“бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ бһңбһ¶бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҠбҹӮбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһ‘бҹҖбһ„бһ‘бһ¶бһҸбҹӢ бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһ“бһ·бһ„бһҘбһҸбһҒбҹ’бһӣбһ¶бһ…бҹ” бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһӮбһҪбһҡбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһҷ бһ”бһҸбҹӢбһ”бҹӮбһ“бһҷбҹүбһ¶бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ…бҹҶбһ–бҹ„бҹҮбһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҳбһ·бһ“бһ”бһ¶бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖ бһ“бһ·бһ„бһ”бҹ’бһҡбһҶбһ¶бҹҶбһ„бһ“бһ№бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҠбҹӮбһӣбһўбһ¶бһ…бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҳбһ¶бһ“бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢбҹ” бһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһ–бһҪбһҖбһңбһ¶бһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡ бһ“бһ·бһ„бһ…бҹҶбһ“бһҪбһ“бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһҠбҹӮбһӣбһ–бһҪбһҖбһңбһ¶бһўбһ¶бһ…бһ”бҹ’бһҡбһҲбһҳбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҮбһҪбһҡбһ‘бһёбһҸбһ¶бҹҶбһ„ бһ“бһ·бһ„бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһ¶бһ“бһҡбҹҶбһ–бһ№бһ„бһ‘бһ»бһҖбһ“бҹ…бһ–бҹҒбһӣбһҡбһ…бһ“бһ¶ бһ“бһ·бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбҹ”

- **Transparency**: бһ“бҹ…бһ–бҹҒбһӣбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҮбһҪбһҷбһҠбһӣбҹӢбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бһӣбһ”бҹүбҹҲбһ–бһ¶бһӣбҹӢбһҷбҹүбһ¶бһ„бһ…бҹ’бһҡбһҫбһ“бһ…бҹҶбһ–бҹ„бҹҮбһҮбһёбһңбһ·бһҸбһўбҹ’бһ“бһҖбһҳбһ“бһ»бһҹбҹ’бһҹ бһңбһ¶бһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҠбҹӮбһӣбһҳбһ“бһ»бһҹбҹ’бһҹбһҷбһӣбҹӢбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһҹбҹҒбһ…бһҖбҹ’бһҸбһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһ“бҹ„бҹҮбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ’бҹ’бһңбһҫбһЎбһҫбһ„бҹ” бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚ бһ’бһ“бһ¶бһӮбһ¶бһҡбһҳбһҪбһҷбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҠбһҫбһҳбҹ’бһ”бһёбһҹбһҳбҹ’бһҡбҹҒбһ…бһҗбһ¶бһҳбһ“бһ»бһҹбҹ’бһҹбһҳбҹ’бһ“бһ¶бһҖбҹӢбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҘбһҺбһ‘бһ¶бһ“ бһ¬бһ‘бҹҒбҹ” бһҖбҹ’бһҡбһ»бһҳбһ бҹҠбһ»бһ“бһҳбһҪбһҷбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ AI бһҠбһҫбһҳбҹ’бһ”бһёбһҖбҹҶбһҺбһҸбҹӢбһ”бҹҒбһҖбҹ’бһҒбһҮбһ“бһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҒбҹ’бһ–бһҹбҹӢбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹ’бһҡбһҫбһҹбһҡбһҫбһҹбҹ”

- **Privacy and Security**: бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ“бҹҒбҹҮбһ–бҹҒбһӣ AI бһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶вҖӢбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶бһҠбҹӮбһӣбһӮбҹҒбһ…бҹ’бһҡбһҫбһ“ бһҖбһ¶бһҡбһҖбһ¶бһҡбһ–бһ¶бһҡбһ—бһ¶бһ–бһҜбһҖбһҮбһ“ бһ“бһ·бһ„бһҖбһ¶бһҡбһҹбһ“бҹ’бһҹбҹҶбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ бһ“бһ·бһ„бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһўбһ¶бһҮбһёбһңбһҖбһҳбҹ’бһҳбһҖбҹ’бһӣбһ¶бһҷбһҮбһ¶вҖӢбһ”бһүбҹ’бһ бһ¶бһҹбҹ’бһҡбһҪбһӣбһҹбҹ’бһҳбһ»бһӮбһҹбҹ’бһҳбһ¶бһүбҹ” бһҮбһ¶бһҳбһҪбһҷ AI бһҖбһ¶бһҡбһҡбһҖбҹ’бһҹбһ¶бһ—бһ¶бһ–бһҜбһҖбһҮбһ“ бһ“бһ·бһ„бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҷбһҖбһ…бһ·бһҸбҹ’бһҸбһ‘бһ»бһҖбһҠбһ¶бһҖбҹӢбһ–бһ·бһҹбҹҒбһҹ бһ–бҹ’бһҡбҹ„бҹҮбһҖбһ¶бһҡбһ…бһјбһӣбһҠбһӣбҹӢбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһӮбһәбһҳбһ¶бһ“бһҹбһ¶бһҡбҹҲбһҹбҹҶбһҒбһ¶бһ“бҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһ–бҹ’бһҷбһ¶бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһҠбҹӮбһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ“бһ·бһ„бһҳбһ¶бһ“бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһӣбһҫбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бһёбһҳбһ“бһ»бһҹбҹ’бһҹбҹ”

- **Accountability**: бһҳбһ“бһ»бһҹбҹ’бһҹбһҠбҹӮбһӣбһҡбһ…бһ“бһ¶ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһ–бһ„бҹ’бһҡбһёбһҖбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбһ…бҹҶбһ–бҹ„бҹҮбһҡбһ”бҹҖбһ”бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһ–бһҪбһҖбһӮбҹҒбҹ” бһўбһ„бҹ’бһӮбһ—бһ¶бһ–бһӮбһҪбһҡбһҠбһҖбһҷбһҖбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡвҖӢбһ§бһҹбҹ’бһҹбһ¶бһ бһҖбһҳбҹ’бһҳбһҠбһҫбһҳбҹ’бһ”бһёбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“ norme бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбҹ” norme бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһўбһ¶бһ…бһ’бһ¶бһ“бһ¶бһҗбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһўбҹҶбһҺбһ¶бһ…бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷбһӣбһҫбһ…бҹҶбһҺбһ»бһ…бһҺбһ¶бһҳбһҪбһҷбһҠбҹӮбһӣбһҳбһ¶бһ“бһ•бһӣбһ”бҹүбҹҮбһ–бһ¶бһӣбҹӢбһ…бҹҶбһ–бҹ„бҹҮбһҮбһёбһңбһ·бһҸбһҳбһ“бһ»бһҹбҹ’бһҹбҹ” бһңбһ¶бһҖбҹҸбһўбһ¶бһ…бһ’бһ¶бһ“бһ¶бһҗбһ¶бһҳбһ“бһ»бһҹбҹ’бһҹбһ“бҹ…бһҸбҹӮбһҮбһ¶бһҳбҹ’бһ…бһ¶бһҹбҹӢбһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһўбһҸбҹ’бһҗбһ“бҹҗбһҷбһӣбһҫбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’ AI бһҠбҹӮбһӣбһҳбһ¶бһ“бһўбһҸбҹ’бһҡбһ¶бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҒбҹ’бһ–бһҹбҹӢ бҹ”

#### бһңбһ·бһҳбһ¶бһӣбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣ Phi-3 бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҳбҹ’бһҡбһ№бһҸбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһҳбһ¶бһӣбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ Microsoft Foundryбҹ” бһңбһ·бһҳбһ¶бһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһўбһ¶бһ…бһҳбһ¶бһ“бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢ бһ“бһ·бһ„бһ—бһ¶бһ–бһ„бһ¶бһҷбһҡбһ»бһҖбһҡбһҖбһ…бҹҒбһүбһ–бһёбһҖбһ¶бһҡбһҸбһ—бҹ’бһҮбһ¶бһ”бҹӢ (jailbreak)бҹ” бһңбһ·бһҳбһ¶бһӣбһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ–бһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- **Self-harm-related Content**: бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҗбһ¶бһҸбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ“бҹ„бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһ“бһ№бһ„бһҖбһ¶бһҡбһҒбһјбһ…бһҒбһ¶бһҸбһҒбҹ’бһӣбһҪбһ“бһҜбһ„ бһ¬бһўбһҸбҹӢбҹ”
- **Hateful and Unfair Content**: бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҗбһ¶бһҸбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ“бҹ„бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҹбһјбһҡбһ·бһҷбһ¶ (hateful) бһ¬бһҳбһ·бһ“бһҷбһ»бһҸбҹ’бһҸбһ·бһ’бһҳбҹҢбҹ”
- **Violent Content**: бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҗбһ¶бһҸбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ“бҹ„бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһўбҹҶбһ–бһёбһўбҹҶбһ–бһҫбһ бһ·бһ„бҹ’бһҹбһ¶бҹ”
- **Sexual Content**: бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҗбһ¶бһҸбһҫбһҳбҹүбһјбһҠбҹӮбһӣбһҳбһ¶бһ“бһ‘бҹҶбһ“бҹ„бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһ—бҹҒбһ‘бһҠбҹӮбһӣбһҳбһ·бһ“бһҹбһҳбһҡбһҳбҹ’бһҷбҹ”

бһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҮбҹҶбһ–бһјбһҖбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһ’бһ¶бһ“бһ¶бһҗбһ¶ бһҳбҹүбһјбһҠбҹӮбһӣ AI бһҳбһ·бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһҸбһ·бһҖбһ¶бһҠбҹӮбһӣбһҳбһ¶бһ“бһӮбҹ’бһҡбҹ„бҹҮбһҗбҹ’бһ“бһ¶бһҖбҹӢ бһ¬бһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһҫбһҸбһҖбһ¶бһҡбһҡбһўбҹҖбһ”бһҡбһўбһҪбһӣ бһҠбҹӮбһӣбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһ‘бҹ…бһ“бһ№бһ„бһҸбһҳбҹ’бһӣбҹғбһҹбһ„бҹ’бһӮбһҳ бһ“бһ·бһ„бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡвҖӢбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бҹ”

#### бһңбһ·бһҳбһ¶бһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–

бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹүбһ¶бһ“бҹӢбһ”бҹ’бһҡбһҳбһ¶бһҺбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣ Phi-3 / Phi-3.5 бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһҳбҹ’бһҡбһ№бһҸбһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһңбһ·бһҳбһ¶бһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢ Microsoft Foundryбҹ” бһңбһ·бһҳбһ¶бһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҮбһҪбһҷбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ‘бһ¶бһҖбҹӢбһ‘бһ„ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбһҪбһӣбҹ” бһңбһ·бһҳбһ¶бһӣбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһҪбһҳбһҳбһ¶бһ“бҹ–

- **Groundedness**: бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһҗбһ¶бһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҳбһ¶бһ“бһҖбһ¶бһҡбһ…бһ¶бһ”бҹӢбһҷбһҖбһҸбһ¶бһҳбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ–бһёбһ”бҹ’бһҡбһ—бһ–бһ”бһүбҹ’бһ…бһјбһӣбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбҹүбҹҒбһ…бҹ”
- **Relevance**: бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһ—бһ¶бһ–бһ‘бһ¶бһҖбҹӢбһ‘бһ„бһҡбһ”бһҹбҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ бһ‘бҹ…бһ“бһ№бһ„бһҹбҹҶбһҺбһҪбһҡбһҠбҹӮбһӣбһ”бһ¶бһ“бһ•бҹ’бһҸбһӣбҹӢбҹ”
- **Coherence**: бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҡбһ”бҹҖбһ”бһҠбҹӮбһӣбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһ бһјбһҡбһ”бҹ’бһҡбһҖбһ”бһҠбҹ„бһҷбһҡбһӣбһјбһ“ бһўбһ¶бһ“бһҲбһ¶бһ“бһ‘бҹ…бһ”бһ¶бһ“бһ’бһҳбҹ’бһҳбһҮбһ¶бһҸбһ· бһ бһҫбһҷбһҹбҹ’бһҡбһҠбҹҖбһ„бһ“бһ№бһ„бһ—бһ¶бһҹбһ¶бһҳбһ“бһ»бһҹбҹ’бһҹбҹ”
- **Fluency**: бһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҮбҹҶбһ“бһ¶бһүбһ—бһ¶бһҹбһ¶бһҡбһ”бһҹбҹӢбһўбһҸбҹ’бһҗбһ”бһ‘бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбҹ”
- **GPT Similarity**: бһ’бҹҖбһ”бһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҮбһ¶бһҳбһҪбһҷбһ•бҹ’бһ‘бҹғбһҹбҹ’бһҳбһ¶бһҸ (ground truth) бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ—бһ¶бһ–бһҹбҹ’бһҡбһҠбҹҖбһ„бҹ”
- **F1 Score**: бһӮбһҺбһ“бһ¶бһЎбһҫбһ„бһўбһҸбҹ’бһҡбһ¶бһ–бһ¶бһҖбҹ’бһҷбһҡбһҪбһҳбһҡбһңбһ¶бһ„бһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸ бһ“бһ·бһ„бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ”бҹ’бһҡбһ—бһ–бҹ”

бһңбһ·бһҳбһ¶бһӣбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ–бһёбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҡбһ”бһҹбҹӢбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһ„бҹ’бһҖбһҫбһҸбһ…бһҳбҹ’бһӣбһҫбһҷбһҠбҹӮбһӣбһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһң бһ‘бһ¶бһҖбҹӢбһ‘бһ„ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбһҪбһӣбҹ”

    import logging
    import requests
    from promptflow import tool
    from promptflow.connections import CustomConnection

    # бһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҖбҹҶбһҺбһҸбҹӢбһ бҹҒбһҸбһ»
    logging.basicConfig(
        format="%(asctime)s - %(levelname)s - %(name)s - %(message)s",
        datefmt="%Y-%m-%d %H:%M:%S",
        level=logging.DEBUG
    )
    logger = logging.getLogger(__name__)

    def query_phi3_model(input_data: str, connection: CustomConnection) -> str:
        """
        Send a request to the Phi-3 / Phi-3.5 model endpoint with the given input data using Custom Connection.
        """

        # "connection" бһӮбһәбһҮбһ¶вҖӢбһҲбҹ’бһҳбҹ„бҹҮвҖӢбһ“бҹғ Custom Connection, "endpoint" бһ“бһ·бһ„ "key" бһҮбһ¶бһңбһ¶бһӣвҖӢбһ“бҹ…вҖӢбһҖбҹ’бһ“бһ»бһ„ Custom Connection
        endpoint_url = connection.endpoint
        api_key = connection.key

        headers = {
            "Content-Type": "application/json",
            "Authorization": f"Bearer {api_key}"
        }
    data = {
        "input_data": [input_data],
        "params": {
            "temperature": 0.7,
            "max_new_tokens": 128,
            "do_sample": True,
            "return_full_text": True
            }
        }
        try:
            response = requests.post(endpoint_url, json=data, headers=headers)
            response.raise_for_status()
            
            # бһҖбһҸбҹӢбһ‘бһ»бһҖбһ…бһҳбҹ’бһӣбһҫбһҷ JSON бһ‘бһ¶бҹҶбһ„бһҳбһјбһӣ
            logger.debug(f"Full JSON response: {response.json()}")

            result = response.json()["output"]
            logger.info("Successfully received response from Azure ML Endpoint.")
            return result
        except requests.exceptions.RequestException as e:
            logger.error(f"Error querying Azure ML Endpoint: {e}")
            raise

    @tool
    def my_python_tool(input_data: str, connection: CustomConnection) -> str:
        """
        Tool function to process input data and query the Phi-3 / Phi-3.5 model.
        """
        return query_phi3_model(input_data, connection)
