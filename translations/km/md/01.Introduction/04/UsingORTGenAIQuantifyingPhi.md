# **бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғ Phi Family бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ–бһ„бҹ’бһҡбһ№бһ„ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ onnxruntime**

## **бһ–бһ„бҹ’бһҡбһ№бһ„ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ onnxruntime бһҮбһ¶бһўбҹ’бһңбһё**

бһ–бһ„бҹ’бһҡбһ№бһ„бһ“бҹҒбҹҮбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳ AI бһҮбһ¶бһҳбһҪбһҷ ONNX Runtime( [https://github.com/microsoft/onnxruntime-genai](https://github.com/microsoft/onnxruntime-genai))бҹ” бһңбһ¶бһ”бһ„бҹ’бһҖбһҫбһҸбһҡбһ„бҹ’бһңбһ·бһӣ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ ONNX бһҡбһҪбһҳбһҳбһ¶бһ“бһҖбһ¶бһҡбһҹбһ“бҹ’бһ“бһ·бһҠбҹ’бһӢбһ¶бһ“бһҮбһ¶бһҳбһҪбһҷ ONNX Runtime бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһӣбһёбһҖбһӣбһёбһҸ бһҖбһ¶бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖ бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹҶбһҡбһј бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ•бҹ’бһ‘бһ»бһҖ KV cacheбҹ” бһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһўбһ¶бһ…бһ бҹ…бһҳбҹҒбһ’бһҸ generate() бһӣбҹҶбһҠбһ¶бһ”бҹӢбһҒбҹ’бһ–бһҹбҹӢ бһ¬бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһҠбҹӮбһҖбһҳбһҪбһҷбһ“бҹғбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ„бҹ’бһңбһ·бһӣ бһҠбҹ„бһҷбһ”бһ„бҹ’бһҖбһҫбһҸбһҸбһјбһ…бһҺбһјбһңбһҳбһҪбһҷбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһ–бҹҒбһӣ бһ“бһ·бһ„бһҮбҹ’бһҡбһ¶бһ”бһҗбҹӮбһҳбһҮбҹҶбһ бһ¶бһ“бһ“бҹғбһҖбһ¶бһҡбһ•бһӣбһ·бһҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҡбһ„бҹ’бһңбһ·бһӣбҹ” бһңбһ¶бһҳбһ¶бһ“бһҖбһ¶бһҡбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһ–бһҪбһҖ greedy/beam бһ“бһ·бһ„бһҖбһ¶бһҡбһӮбҹҶбһҡбһј TopP, TopK бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһ„бҹ’бһҖбһҫбһҸбһӣбҹҶбһҠбһ¶бһ”бҹӢбһҸбһјбһ…бһҺбһјбһң бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһӣбһёбһҖбһӣбһёбһҸбһҠбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„бһҠбһјбһ…бһҮбһ¶бһ—бһ¶бһ–бһҮбһ¶бһ”бҹӢбһҲбҹ’бһҳбҹ„бҹҮбһЎбһҫбһ„бһңбһ·бһүбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһ“бҹ’бһҗбҹӮбһҳбһҖбһ¶бһҡбһңбһ¶бһҷбһҸбһҳбҹ’бһӣбҹғбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“бһҷбҹүбһ¶бһ„бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ•бһ„бһҠбҹӮбһҡбҹ”

бһ“бҹ…бһҖбһҳбҹ’бһҡбһ·бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ–бһ„бҹ’бһҡбһ№бһ„ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ onnxruntime бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһҹбһ¶бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ C++/ C# / Pythonбҹ” бһ“бҹ…бһҖбһҳбҹ’бһҡбһ·бһҸбһҳбҹүбһјбһҠбҹӮбһӣ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһңбһ¶бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһјбһӣбһҳбҹүбһјбһҠбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһ„бҹ’бһ бһ¶бһҸбҹӢбһӣбһҳбҹ’бһўбһ·бһҸ бһ“бһ·бһ„бһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ…бҹӮбһҖбһ…бһ¶бһҷбһ…бҹҶбһ“бһҪбһ“бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бҹ”

## **бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғ Phi-3.5 бһҮбһ¶бһҳбһҪбһҷбһ–бһ„бҹ’бһҡбһ№бһ„ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ onnxruntime**

### **бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҳбҹүбһјбһҠбҹӮбһӣ**

бһ–бһ„бҹ’бһҡбһ№бһ„ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ onnxruntime бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҠбҹӮбһӣ Microsoft Phi, Google Gemma, Mistral, Meta LLaMAбҹ”

### **бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ–бһ„бҹ’бһҡбһ№бһ„ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ onnxruntime**

бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣбһӣбһҝбһ“бһ бҹ’бһңбһ№бһҖбһ бһ¶бһҸбҹӢбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣ ONNX бһҠбҹӮбһӣбһ”бһ¶бһ“бһўбһ»бһ”бһ‘бһёбһҳбҹүбһ¶бһҷ бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһўбҹҠбһёбһ—бһё generate() бһ“бҹғ ONNX Runtimeбҹ”

бһҸбһ¶бһҳбһҡбһҷбҹҲбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҠбҹӮбһӣбһ‘бҹ… INT4, INT8, FP16, FP32 бһ бһҫбһҷбһ…бҹ’бһҡбҹ„бҹҮбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ“бҹ’бһ‘бҹҮбһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһҹбҹҒбһ„бҹ—бһҠбһјбһ…бһҮбһ¶ CPU, CUDA, DirectML, Mobile, бһӣбҹ”

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸбһҳбҹүбһјбһҠбҹӮбһӣ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҸбҹҶбһЎбһҫбһ„

```bash

pip install torch transformers onnx onnxruntime

pip install --pre onnxruntime-genai

```
  
бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҸбҹҶбһЎбһҫбһ„ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһңбһ·бһ’бһёбһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡ Model Builder бһ–бһё terminal бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ‘бҹ’бһҡбһ„бҹӢбһ‘бҹ’бһҡбһ„бҹӢбһҳбҹүбһјбһҠбҹӮбһӣ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғбҹ”

```bash

python3 -m onnxruntime_genai.models.builder -m model_name -o path_to_output_folder -p precision -e execution_provider -c cache_dir_to_save_hf_files

```
  
бһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһ”бҹүбһ¶бһҡбҹүбһ¶бһҳбҹүбҹӮбһҸбҹ’бһҡвҖӢбһҠбҹӮбһӣбһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’

1. **model_name** бһҳбҹүбһјбһҠбҹӮбһӣбһ“бҹҒбҹҮбһ“бҹ…бһӣбһҫ Hugging face бһҠбһјбһ…бһҮбһ¶ microsoft/Phi-3.5-mini-instruct, microsoft/Phi-3.5-vision-instruct, бһӣбҹ” бһңбһ¶бһҖбҹҸбһўбһ¶бһ…бһҮбһ¶бһ•бҹ’бһӣбһјбһңбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҳбҹүбһјбһҠбҹӮбһӣбһ•бһ„бһҠбҹӮбһҡ

2. **path_to_output_folder** бһ•бҹ’бһӣбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғ

3. **execution_provider** бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бһ“бҹ’бһ‘бҹҮбһ”бһ“бҹ’бһҗбҹӮбһҳбһ•бҹ’бһҹбҹҒбһ„бҹ— бһҠбһјбһ…бһҮбһ¶ cpu, cuda, DirectML

4. **cache_dir_to_save_hf_files** бһҷбһҫбһ„бһ‘бһ¶бһүбһҷбһҖбһҳбҹүбһјбһҠбҹӮбһӣбһ–бһё Hugging face бһ бһҫбһҷбһ”бһҳбҹ’бһҡбһ»бһ„бһҮбһ¶бһҖбһ¶бһҡбһ•бҹ’бһ‘бһ»бһҖбһҖбҹ’бһ“бһ»бһ„бһҸбҹҶбһ”бһ“бҹӢ


***бһ…бҹҶбһҺбһ¶бҹҶбҹ–*** <ul>бһ”бһҫбһ‘бҹ„бҹҮбһ”бһёбһҗбһ¶ бһ–бһ„бҹ’бһҡбһ№бһ„ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ onnxruntime бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһ¶бһҖбҹӢбһҖбһ¶бһӣбһҳбһҫбһӣбһҮбһ¶бһҳбһ»бһ“ бһңбһ¶бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһјбһӣбһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ Microsoft Olive бһ бһҫбһҷбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ бҹ…бһҳбһ»бһҒбһ„бһ¶бһҡ Model Builder бһ“бҹғбһ–бһ„бҹ’бһҡбһ№бһ„ AI бһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ onnxruntime бһҸбһ¶бһҳбһҡбһҷбҹҲ Microsoft Olive бһ•бһ„бһҠбҹӮбһҡбҹ”</ul>

## **бһңбһ·бһ’бһёбһ”бҹ’бһҡбһҫ Model Builder бһҠбһҫбһҳбҹ’бһ”бһёбһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғ Phi-3.5**

бһ–бҹҒбһӣбһ“бҹҒбҹҮ Model Builder бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғбһҳбҹүбһјбһҠбҹӮбһӣ ONNX бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ Phi-3.5 Instruct бһ“бһ·бһ„ Phi-3.5-Vision

### **Phi-3.5-Instruct**

**бһҖбһ¶бһҡбһ”бҹҶбһӣбҹӮбһ„бһ”бһ“бҹ’бһ‘бҹҮбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ CPU бһ‘бҹ… INT 4 бһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бһүбҹ’бһ…бһ»бҹҮбһҸбһҳбҹ’бһӣбҹғ**

