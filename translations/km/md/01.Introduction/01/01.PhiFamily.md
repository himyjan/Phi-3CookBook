# គ្រួសារ Phi របស់ Microsoft

ម៉ូដែល Phi គឺជាម៉ូដែលភាសាញឹកញាប់តូច (SLMs) ដែលមានសមត្ថភាពខ្ពស់ និងមានថ្លៃថោកបំផុតដែលអាចប្រើបាន ដោយអាចលើសម៉ូដែលដែលមានទំហំពីរដងដូចគ្នា និងទំហំបន្ទាប់ៗលើការវាស់វែងភាសា ការរៀនសូត្រ ការសរសេរកូដ អូឌីយ៉ូ ចក្ខុវិស័យ និងគណិតវិទ្យាជាច្រើន។ ការបញ្ចេញនេះបានពង្រីកជម្រើសម៉ូដែលដែលមានគុណភាពខ្ពស់សម្រាប់អតិថិជន ដោយផ្តល់ជម្រើស ដែលមានប្រយោជន៍បន្ថែមសម្រាប់ការសរសេរ និងសង់កម្មវិធី AI បង្កើត។

គ្រួសារ Phi បានចាប់ផ្ដើមជាមួយ Phi-1 សម្រាប់ការបង្កើតកូដ Python បន្តទៅ Phi-1.5 /2 ប្រកាន់ដោយការបញ្ចប់អត្ថបទ និងការជជែក Phi-3-mini/small/medium-instruct និង Phi-3.5/4-mini-instruct ហើយបានអភិវឌ្ឍទៅជា Phi-3/3.5-vision សម្រាប់ចក្ខុវិស័យ Phi-4 ដោយផ្អែកលើការពិចារណាដ៏មាំមួន និង Phi-3.5-MoE សម្រាប់ MoE ហើយឥឡូវនេះម៉ូដែលពហុម៉ូដាល Phi-4-multimodal ។ តាមរយៈសំណុំព័ត៌មានគុណភាពខ្ពស់ និទាឃរូបមន្តអាចបណ្តុះបណ្តាលឲ្យស្ទើរត្រូវបានប្រៀបធៀបនឹងម៉ូដែលដែលមានប៉ារ៉ាម៉ែត្របណ្តុះបណ្តាលធំបំផុត។
