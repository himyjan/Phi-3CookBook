# ចាប់ផ្តើមជាមួយ Phi-3 ក្នុងសៀវភៅមួយជាកន្លែង

គូសនីយកម្មនេះនឹងជួយអ្នកដាក់ការតំឡើងបរិយាកាសក្នុងសៀវភៅរបស់អ្នក ដើម្បីរត់ម៉ូដែល Phi-3 តាមរយៈ Ollama។ អ្នកអាចបើកម៉ូដែលនេះបានជាច្រើនវិធីផ្សេងគ្នា រួមមានការប្រើ GitHub Codespaces, VS Code Dev Containers ឬបរិយាកាសក្នុងសៀវភៅរបស់អ្នកផ្ទាល់។

## ការតំឡើងបរិយាកាស

### GitHub Codespaces

អ្នកអាចបើកទំរង់ប្លង់នេះបានជាការចៃដន្យដោយប្រើ GitHub Codespaces។ ប៊ូតុងនឹងបើក VS Code ឆ្លើមក្រោមម៉ូតខ្សែអ៊ីនធឺណេតក្នុងកម្មវិធីរុករករបស់អ្នក៖

1. បើកទំរង់ប្លង់ (នេះអាចចំណាយពេលពីរបីនាទី)៖

    [![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/microsoft/phi-3cookbook)

2. បើកវីនដូបន្ទប់ពាក្យបញ្ជា

### VS Code Dev Containers

⚠️ ជម្រើសនេះនឹងដំណើរការតែបើ Docker Desktop របស់អ្នកមាន RAM យ៉ាងតិច 16 GB។ ប្រសិនបើអ្នកមាន RAM តិចជាង 16 GB អ្នកអាចសាកល្បងជម្រើស [GitHub Codespaces](#github-codespaces) ឬ [ដាក់វាក្នុះផ្ទះ](#បរិយាកាសក្នុងសៀវភៅ)។

ជម្រើសដែលពាក់ព័ន្ធគឺ VS Code Dev Containers ដែលនឹងបើកគម្រោងក្នុង VS Code ផ្ទាល់របស់អ្នកដោយប្រើកម្មវិធីបន្ថែម [Dev Containers extension](https://marketplace.visualstudio.com/items?itemName=ms-vscode-remote.remote-containers)៖

1. ចាប់ផ្តើម Docker Desktop (តំឡើងបើមិនទាន់បានតំឡើង)
2. បើកគម្រោង៖

    [![Open in Dev Containers](https://img.shields.io/static/v1?style=for-the-badge&label=Dev%20Containers&message=Open&color=blue&logo=visualstudiocode)](https://vscode.dev/redirect?url=vscode://ms-vscode-remote.remote-containers/cloneInVolume?url=https://github.com/microsoft/phi-3cookbook)

3. ក្នុងវីនដូ VS Code ដែលបើកខាងលើ អន្ទះឯកសារគម្រោងបង្ហាញ (នេះអាចចំណាយពេលពីរបីនាទី), បើកបន្ទប់ពាក្យបញ្ជា។
4. បន្តជាមួយ [ជំហានប្រតិបត្តិការ](#deployment)

### បរិយាកាសក្នុងសៀវភៅ

1. ត្រូវប្រាកដថាឧបករណ៍ខាងក្រោមត្រូវបានតំឡើង៖

    * [Ollama](https://ollama.com/)
    * [Python 3.10+](https://www.python.org/downloads/)
    * [OpenAI Python SDK](https://pypi.org/project/openai/)

## សាកល្បងម៉ូដែល

1. សូមទុកឲ្យ Ollama ទាញយក និងរត់ម៉ូដែល phi3:mini៖

    ```shell
    ollama run phi3:mini
    ```

    វានឹងចំណាយពេលពីរបីនាទីដើម្បីទាញយកម៉ូដែល។

2. បន្ទាប់ពីអ្នកឃើញ "success" នៅក្នុងលទ្ធផល អ្នកអាចផ្ញើសារទៅម៉ូដែលនោះពីបន្ទាត់បញ្ជា។

    ```shell
    >>> Write a haiku about hungry hippos
    ```

3. បន្ទាប់ពីជិតបញ្ចប់ចំនួនវិនាទី អ្នកគួរតែឃើញចរន្តចម្លើយមកពីម៉ូដែល។

4. ដើម្បីស្វែងយល់អំពីបច្ចេកទេសផ្សេងៗដែលប្រើប្រាស់ជាមួយម៉ូដែលភាសា សូមបើកកំណត់ហេតុ Python [ollama.ipynb](../../../code/01.Introduce/ollama.ipynb) ហើយបើកដំណើរការពីរូចក្នុងមួយកោសិកា។ ប្រសិនបើអ្នកប្រើម៉ូដែលផ្សេងពី 'phi3:mini', សូមផ្លាស់ប្ដូរ `MODEL_NAME` នៅក្នុងកោសិកាដំបូង។

5. ដើម្បីនិយាយជាមួយម៉ូដែល phi3:mini ពី Python សូមបើកឯកសារ Python [chat.py](../../../../../code/01.Introduce/chat.py) ហើយរត់វា។ អ្នកអាចផ្លាស់ប្ដូរ `MODEL_NAME` នៅលើឯកសារដូចប្រើចាំបាច់ ហើយអាចក៏កែសម្រួលសារប្រព័ន្ធ ឬបញ្ចូលឧទាហរណ៍មួយចំនួនថែមទៀតបានប្រសិនបើចង់បាន។
