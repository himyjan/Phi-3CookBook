## бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҠбҹӮбһӣбһҖбҹ’бһҡбһ»бһҳбһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ Phi бһҖбҹ’бһ“бһ»бһ„ Foundry Local

### бһҖбһ¶бһҡбһҺбҹӮбһ“бһ¶бҹҶбһўбҹҶбһ–бһё Foundry Local

Foundry Local бһӮбһәбһҮбһ¶бһ•бһӣбһ·бһҸбһ•бһӣбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһҸбҹӢбһҹбһүбҹ’бһүбһ¶ AI бһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ бһҠбҹӮбһӣбһ“бһ¶бҹҶбһҳбһ»бһҒбһ„бһ¶бһҡ AI бһҳбһ¶бһ“бҹӢбһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбҹ’бһҡбһҳбһҹбһ·бһ‘бҹ’бһ’бһ·бһ‘бҹ…бһҖбһ¶бһ“бҹӢбһҡбһ№бһ„бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ“бһ№бһ„бһҺбҹӮбһ“бһ¶бҹҶбһўбҹ’бһ“бһҖбһ–бһёбһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҠбҹӮбһӣ Phi-Family бһҮбһ¶бһҳбһҪбһҷ Foundry Local, бһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһ“бһјбһңбһҖбһ¶бһҡбһҸбҹ’бһҡбһҪбһҸбһҸбҹ’бһҡбһ„бҹӢбһӣбһҫбһ”бһ“бҹ’бһ‘бһ»бһҖ AI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҒбһҺбҹҲбһҠбҹӮбһӣбһҡбһҖбҹ’бһҹбһ¶бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҜбһҖбһҮбһ“ бһ“бһ·бһ„бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ…бҹҶбһҺбһ¶бһҷбҹ”

Foundry Local бһ•бҹ’бһҸбһӣбҹӢбһ“бһјбһңбһӣбҹ’бһ”бһҝбһ“ бһҹбһҳбҹ’бһ„бһ¶бһҸбҹӢбһ—бһ¶бһ– бһҖбһ¶бһҡбһ”бҹ’бһҠбһјбһҡбһ…бҹ’бһ“бҹғ бһ“бһ·бһ„бһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҸбһҳбҹ’бһӣбҹғбһҠбҹ„бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҳбҹүбһјбһҢбҹӮбһӣ AI бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ” бһңбһ¶бһ”бһүбҹ’бһ…бһјбһӣбһҷбҹүбһ¶бһ„бһҡбһӣбһјбһ“бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ“бһ·бһ„бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҪбһ…бһҸбһ¶бһҳбһҡбһҷбҹҲ CLI, SDK бһ“бһ·бһ„ REST API бһҠбҹӮбһӣбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ”бҹ’бһҡбһҫбҹ”

### бһ бҹҒбһҸбһ»бһўбҹ’бһңбһёбһҮбҹ’бһҡбһҫбһҹ Foundry Local?

бһҖбһ¶бһҡбһҷбһӣбҹӢбһҠбһ№бһ„бһўбҹҶбһ–бһёбһўбһҸбҹ’бһҗбһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ“бҹғ Foundry Local бһ“бһ№бһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһІбҹ’бһҷбһ‘бһ‘бһҪбһӣбһҖбһ¶бһҡбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһўбҹҶбһ–бһёбһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбһ¶бһҖбҹӢбһ”бһүбҹ’бһ…бһјбһӣ AI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ–

- **Inference бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ:** бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҳбҹүбһјбһҢбҹӮбһӣ бһ‘бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖ бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ…бҹҶбһҺбһ¶бһҷ бһ–бҹ’бһҡбһҳбһ‘бһ¶бҹҶбһ„бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһ“бҹ…бһӣбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбҹ”

- **бһҖбһ¶бһҡбһҖбҹӮбһ”бҹ’бһҡбҹӮбһҳбҹүбһјбһҢбҹӮбһӣ:** бһҮбҹ’бһҡбһҫбһҹбһ–бһёбһҳбҹүбһјбһҢбҹӮбһӣбһҠбҹӮбһӣбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҡбһҪбһ… бһ¬бһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбҹӮбһӣбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹҶбһ–бҹҒбһүбһҸбһ¶бһҳбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡ бһ“бһ·бһ„бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһҮбһ¶бһҖбҹӢбһӣбһ¶бһҖбҹӢбҹ”

- **бһ”бҹ’бһҡбһҹбһ·бһ‘бҹ’бһ’бһ—бһ¶бһ–бһҗбҹ’бһӣбҹғбһҠбһҫбһҳ:** бһӣбһ»бһ”бһ…бҹ„бһӣбһ…бҹҶбһҺбһ¶бһҷбһҹбҹҒбһңбһ¶бһ–бһ–бһҖбһҠбҹӮбһӣбһҳбһ¶бһ“бһ…бһ„бҹ’бһңбһ¶бһҖбҹӢбһҠбҹ„бһҷбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣбһҳбһ¶бһ“бһҡбһҪбһ… бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһІбҹ’бһҷ AI бһҳбһ¶бһ“бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһ…бһјбһӣбһҠбһӣбҹӢбҹ”

- **бһҖбһ¶бһҡбһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһҠбҹ„бһҷбһҡбһӣбһјбһ“:** бһ—бҹ’бһҮбһ¶бһ”бҹӢбһҮбһ¶бһҳбһҪбһҷбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһҸбһ¶бһҳ SDK, endpoints API бһ¬ CLI, бһ“бһ·бһ„бһҳбһ¶бһ“бһ—бһ¶бһ–бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ–бһ„бҹ’бһҡбһёбһҖбһ‘бҹ… Microsoft Foundry бһҒбһҺбҹҲбһҠбҹӮбһӣбһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһІбҹ’бһҷбһҖбһҫбһ“бһЎбһҫбһ„бҹ”

> **бһҖбҹҶбһҺбһҸбҹӢбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбҹ–** бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһ•бҹ’бһҸбҹ„бһҸбһӣбһҫбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ Foundry Local бһҸбһ¶бһҳбһҡбһҷбҹҲбһ…бҹҶбһҺбһ»бһ… CLI бһ“бһ·бһ„ SDKбҹ” бһўбҹ’бһ“бһҖбһ“бһ№бһ„бһҡбҹҖбһ“бһ–бһёбһңбһ·бһ’бһёбһ‘бһ¶бҹҶбһ„бһ–бһёбһҡбһҠбһҫбһҳбҹ’бһ”бһёбһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҮбҹ’бһҡбһҫбһҹбһңбһ·бһ’бһёбһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

- **phi-3-mini-4k** - бһҳбҹүбһјбһҢбҹӮбһӣбһ”бһҡбһ·бһ”бһ‘бһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ‘бһјбһ‘бҹ…
