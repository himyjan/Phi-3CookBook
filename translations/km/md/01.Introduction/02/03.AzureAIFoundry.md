## бһӮбҹ’бһҡбһҪбһҹбһ¶бһҡ Phi бһҖбҹ’бһ“бһ»бһ„ Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) бһӮбһәбһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶бһҠбҹӮбһӣбһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸ бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚбһҮбҹҶбһҡбһ»бһүбһҖбһ¶бһҡбһ…бҹ’бһ“бҹғбһ”бҹ’бһҡбһҢбһ·бһҸ бһ“бһ·бһ„бһҡбҹҖбһ”бһ…бҹҶбһўбһ“бһ¶бһӮбһҸбһҮбһ¶бһҳбһҪбһҷ AI бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһңбһ·бһ’бһёбһҠбҹӮбһӣбһҳбһ¶бһ“бһҹбһ»бһңбһҸбҹ’бһҗбһ·бһ—бһ¶бһ– бһҳбһ¶бһ“бһҹбһ“бҹ’бһҸбһ·бһҹбһ»бһҒ бһ“бһ·бһ„бһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбҹ”



[Microsoft Foundry](https://ai.azure.com) бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ“бҹҚ бһҠбһҫбһҳбҹ’бһ”бһёбҹ–

- бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢ бһ”бһ„бҹ’бһҖбһҫбһҸ бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҠбһ¶бһҖбҹӢбһІбҹ’бһҷбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡ бһҠбҹ„бһҷбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚ AI бһ“бһ·бһ„бһҳбҹүбһјбһҢбҹӮбһӣ ML бһҗбҹ’бһҳбһёбҹ— бһҠбҹҸбһ‘бҹҶбһ“бһҫбһ” бһҠбҹӮбһӣбһўбһ“бһ»бһңбһҸбҹ’бһҸбһ‘бҹ…бһӣбһҫбһҖбһ¶бһҡбһўбһ—бһ·бһ”бһ¶бһӣ AI бһҠбҹ„бһҷбһ‘бһ‘бһҪбһӣбһҒбһ»бһҹбһҸбҹ’бһҡбһјбһңбҹ”
- бһҹбһ бһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһҖбҹ’бһҡбһ»бһҳбһҖбҹ’бһ“бһ»бһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бҹҒбһүбһӣбҹ’бһңбҹӮбһ„бһ“бҹғбһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбҹ”

бһҮбһ¶бһҳбһҪбһҷ Microsoft Foundry бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҷбһӣбҹӢбһҳбҹүбһјбһҢбҹӮбһӣ бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ бһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ бһ“бһ·бһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҸбһҫбһҳбһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё AI бһҠбҹӮбһӣбһ”бһҳбҹ’бһҡбһҫбһӮбҹ„бһӣбһҠбҹ…бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ”бһ¶бһ“бһӣбҹ’бһўбһ”бҹҶбһ•бһ»бһҸбҹ” бһңбҹҒбһ‘бһ·бһҖбһ¶ Microsoft Foundry бһҮбһҪбһҷбһұбҹ’бһҷбһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһ”бһ„бҹ’бһҖбһҫбһ“бһ‘бҹҶбһ бҹҶ бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бһҳбҹ’бһӣбҹӮбһ„бһҖбһ¶бһҡбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһӮбҹҶбһ“бһ·бһҸбһ‘бҹ…бһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһ–бҹҒбһүбһӣбҹҒбһүбһҠбҹ„бһҷбһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбҹ” бһҖбһ¶бһҡбһҸбһ¶бһҳбһҠбһ¶бһ“ бһ“бһ·бһ„бһҹбһҳбҹ’бһҡбһҪбһӣбһ”бһ“бҹ’бһҸбһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҮбҹ„бһӮбһҮбҹҗбһҷбһҡбһҷбҹҲбһ–бҹҒбһӣбһңбҹӮбһ„бҹ”

### **бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ Phi-4 бһҖбҹ’бһ“бһ»бһ„ Microsoft Foundry Playground**

![бһ‘бһёбһӣбһ¶бһ“бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„](../../../../../translated_images/km/AIFoundryPlayground.193b81a9e472c5d1.webp)

### **бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҖбһјбһҠ Python бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ… Microsoft Foundry Phi-4**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# бһҡбһҪбһҳбһ”бһүбҹ’бһ…бһјбһӣбһӣбһ‘бҹ’бһ’бһ•бһӣбһҹбҹҶбһЎбҹҒбһ„ бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһҹбҹҶбһЎбҹҒбһ„бһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһҫбһҖ
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
