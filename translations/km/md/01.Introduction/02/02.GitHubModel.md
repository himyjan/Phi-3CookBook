### Run a basic code sample

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһ бҹ…бһҳбһҪбһҷбһҷбҹүбһ¶бһ„бһҹбһ¶бһҳбһүбҹ’бһүбһ‘бҹ… API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһҮбһҮбҹӮбһҖ (chat completion)бҹ” бһңбһ¶бһҖбҹҶбһ–бһ»бһ„бһ”бҹ’бһҡбһҫбһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢ GitHub AI model inference бһ“бһ·бһ„ token GitHub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҖбһ¶бһҡбһ бҹ…бһ“бҹҒбҹҮбһӮбһәбһҹбһҳбҹҗбһҖбһҒбҹ’бһӣбһҪбһ“ (synchronous)бҹ”

```python
import os
from azure.ai.inference import ChatCompletionsClient
from azure.ai.inference.models import SystemMessage, UserMessage
from azure.core.credentials import AzureKeyCredential

endpoint = "https://models.inference.ai.azure.com"
model_name = "Phi-4"
token = os.environ["GITHUB_TOKEN"]

client = ChatCompletionsClient(
    endpoint=endpoint,
    credential=AzureKeyCredential(token),
)

response = client.complete(
    messages=[
        UserMessage(content="I have $20,000 in my savings account, where I receive a 4% profit per year and payments twice a year. Can you please tell me how long it will take for me to become a millionaire? Also, can you please explain the math step by step as if you were explaining it to an uneducated person?"),
    ],
    temperature=0.4,
    top_p=1.0,
    max_tokens=2048,
    model=model_name
)

print(response.choices[0].message.content)
```

### Run a multi-turn conversation

бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһ“бҹҒбҹҮбһ”бһ„бҹ’бһ бһ¶бһүбһ–бһёбһҖбһ¶бһҡбһҹбһ“бҹ’бһ‘бһ“бһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“бһҮбһ¶бһҳбһҪбһҷ API бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ…бһ”бҹӢбһҖбһ¶бһҡбһҮбһҮбҹӮбһҖбҹ” бһ–бҹҒбһӣбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбҹӮбһӣбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҮбһҮбҹӮбһҖ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ“бҹ’бһ‘бһ“бһ¶вҖӢбһ“бҹ„бҹҮ бһ“бһ·бһ„бһ•бҹ’бһүбһҫбһҹбһ¶бһҡбһҗбҹ’бһҳбһёбҹ—бһ‘бҹ…бһҳбҹүбһјбһҢбҹӮбһӣбҹ”

```
import os
from azure.ai.inference import ChatCompletionsClient
from azure.ai.inference.models import AssistantMessage, SystemMessage, UserMessage
from azure.core.credentials import AzureKeyCredential

token = os.environ["GITHUB_TOKEN"]
endpoint = "https://models.inference.ai.azure.com"
# Replace Model_Name
model_name = "Phi-4"

client = ChatCompletionsClient(
    endpoint=endpoint,
    credential=AzureKeyCredential(token),
)

messages = [
    SystemMessage(content="You are a helpful assistant."),
    UserMessage(content="What is the capital of France?"),
    AssistantMessage(content="The capital of France is Paris."),
    UserMessage(content="What about Spain?"),
]

response = client.complete(messages=messages, model=model_name)

print(response.choices[0].message.content)
```

### бһ•бҹ’бһҹбһ¶бһҷбһ…бҹҒбһүбһӣбһ‘бҹ’бһ’бһ•бһӣ

бһҠбһҫбһҳбҹ’бһ”бһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бҹ’бһҡбһҹбһҫбһҡбһЎбһҫбһ„ бһҹбһјбһҳбһ”бһүбҹ’бһ…бһјбһ“бһ…бһҡбһ“бҹ’бһҸбһ“бҹғбһ…бһҳбҹ’бһӣбһҫбһҷбһ–бһёбһҳбҹүбһјбһҢбҹӮбһӣ бһҠбһҫбһҳбҹ’бһ”бһёбһўбҹ„бһҷ token бһҠбҹҶбһ”бһјбһ„бһ…бҹҒбһүбһҳбһҖбһҶбһ¶бһ”бҹӢ бһ“бһ·бһ„бһҮбҹҖбһҹбһңбһ¶бһ„бһҖбһ¶бһҡбһҡбҹҶбһ–бһ№бһ„бһҡбһҷбҹҲбһ–бҹҒбһӣбһҷбһјбһҡвҖӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ…бһҳбҹ’бһӣбһҫбһҷбһ’бҹҶбҹ—бҹ”

```
import os
from azure.ai.inference import ChatCompletionsClient
from azure.ai.inference.models import SystemMessage, UserMessage
from azure.core.credentials import AzureKeyCredential

token = os.environ["GITHUB_TOKEN"]
endpoint = "https://models.inference.ai.azure.com"
# Replace Model_Name
model_name = "Phi-4"

client = ChatCompletionsClient(
    endpoint=endpoint,
    credential=AzureKeyCredential(token),
)

response = client.complete(
    stream=True,
    messages=[
        SystemMessage(content="You are a helpful assistant."),
        UserMessage(content="Give me 5 good reasons why I should exercise every day."),
    ],
    model=model_name,
)

for update in response:
    if update.choices:
        print(update.choices[0].delta.content or "", end="")

client.close()
```

The [бһўбһҸбҹ’бһҡбһ¶бһҖбҹҶбһҺбһҸбҹӢбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ“бҹ’бһӣбҹӮбһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ бһ“бһ·бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ API бһҠбҹ„бһҷбһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғ](https://docs.github.com/en/github-models/prototyping-with-ai-models#rate-limits) бһӮбһәбһҳбһ¶бһ“бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҮбһҪбһҷбһўбҹ’бһ“бһҖбһҹбһ¶бһҖбһӣбҹ’бһ”бһ„бһҳбҹүбһјбһҢбҹӮбһӣ бһ“бһ·бһ„бһ”бһ„бҹ’бһҖбһҫбһҸбһ–бҹҗбһ‘бҹ’бһ’бһӮбһҳбҹ’бһҡбҹ„бһ„ AI бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһӣбһҫбһҹбһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮ бһ“бһ·бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһ“бһ¶бҹҶбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһІбҹ’бһҷбһҳбһ¶бһ“бһҹбһҖбҹ’бһҸбһ¶бһ“бһ»бһ–бһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҡбһёбһҖбһ…бһҳбҹ’бһҡбһҫбһ“ бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ•бҹ’бһҸбһӣбҹӢбһ’бһ“бһ’бһ¶бһ“бһ–бһёбһӮбһҺбһ“бһё Azure бһ“бһ·бһ„бһўбһ’бһ·бһ”бҹ’бһ”бһ¶бһҷбһ…бһјбһӣбһ–бһёбһ‘бһёбһ“бҹ„бҹҮбһҮбҹҶбһ“бһҪбһҹ token бһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҒбҹ’бһӣбһҪбһ“ GitHub бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ” бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ•бҹ’бһӣбһ¶бһҹбҹӢбһ”бҹ’бһҸбһјбһҡбһўбҹ’бһңбһёбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸбһҖбҹ’бһ“бһ»бһ„бһҖбһјбһҠбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһ‘бҹҒбҹ” бһ”бҹ’бһҡбһҫбһҸбҹҶбһҺбһ“бҹҒбҹҮбһҠбһҫбһҳбҹ’бһ”бһёбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһңбһ·бһ’бһёбһ…бһјбһӣбһӣбһҫбһҹбһ–бһёбһҖбҹҶбһҺбһҸбҹӢбһҗбҹ’бһ“бһ¶бһҖбҹӢбһҘбһҸбһӮбһ·бһҸбһҗбҹ’бһӣбҹғбһҖбҹ’бһ“бһ»бһ„ Azure AIбҹ”

### бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ

бһ…бһјбһҡбһҡбҹҶбһ–бһ№бһ„бһҗбһ¶бһ–бҹҒбһӣбһўбһ“бҹ’бһҸбһҡбһ¶бһӮбһҳбһ“бҹҚбһҮбһ¶бһҳбһҪбһҷбһҳбҹүбһјбһҢбҹӮбһӣ бһўбҹ’бһ“бһҖбһҖбҹҶбһ–бһ»бһ„бһҹбһ¶бһҖбһӣбҹ’бһ”бһ„ AI бһҠбһјбһ…бҹ’бһ“бҹҒбҹҮбһҒбһ»бһҹбһ”бҹ’бһҡбһҗбһ»бһҷбһҡбһ”бһҹбҹӢбһҳбһ¶бһҸбһ·бһҖбһ¶бһўбһ¶бһ…бһҖбһҫбһҸбһҳбһ¶бһ“бһ”бһ¶бһ“бҹ”

бһӣбһҖбҹ’бһҒбһҒбһҺбҹ’бһҢбһ“бҹҒбҹҮбһҳбһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“ (бһҡбһҪбһҳбһ‘бһ¶бҹҶбһ„бһҹбҹҶбһҺбһҫбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһ“бһ¶бһ‘бһё, бһҹбҹҶбһҺбһҫбһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҗбҹ’бһ„бҹғ, tokens бһҖбҹ’бһ“бһ»бһ„бһҳбһҪбһҷбһҹбҹҶбһҺбһҫ бһ“бһ·бһ„бһҹбҹҶбһҺбһҫбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһҳбһҪбһҷбһӮбҹ’бһ“бһ¶) бһ бһҫбһҷбһҳбһ·бһ“бһ”бһ¶бһ“бһҡбһ…бһ“бһ¶бһЎбһҫбһ„бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ•бһӣбһ·бһҸбһҖбһҳбҹ’бһҳбһ‘бҹҒбҹ”

GitHub Models бһ”бҹ’бһҡбһҫ Azure AI Content Safetyбҹ” бһҖбһ¶бһҡбһҸбҹ’бһҡбһ„бҹӢбһҳбһ¶бһҸбһ·бһҖбһ¶бһ‘бһ¶бҹҶбһ„бһ“бҹҒбҹҮбһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ·бһ‘бһ”бһ¶бһ“бһҮбһ¶бһ•бҹ’бһ“бҹӮбһҖбһ“бҹғбһ”бһ‘бһ–бһ·бһҹбҹ„бһ’бһ“бҹҚ GitHub Modelsбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбҹҒбһ…бһ…бһ·бһҸбҹ’бһҸбһ”бҹ’бһҡбһҫбһҳбҹүбһјбһҢбҹӮбһӣбһҸбһ¶бһҳбһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҠбҹӮбһӣбһ…бҹӮбһҖбһӮбҹ’бһ“бһ¶ бһҷбһҫбһ„бһҒбҹ’бһүбһ»бҹҶбһҹбҹ’бһ“бһҫбһІбҹ’бһҷбһҖбҹҶбһҺбһҸбҹӢбһҸбҹ’бһҡбһ„бҹӢбһҳбһ¶бһҸбһ·бһҖбһ¶бһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбһІбҹ’бһҷбһҹбһҳбһ“бһ№бһ„бһҸбһҳбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһҡбһ”бһҹбҹӢбһўбҹ’бһ“бһҖбҹ”

бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһ“бҹҒбҹҮбһҹбҹ’бһҗбһ·бһҸбһҖбҹ’бһҡбҹ„бһҳбһӣбҹҗбһҖбҹ’бһҒбһҒбҹҗбһҺбҹ’бһҢ Pre-release бһҡбһ”бһҹбҹӢ GitHubбҹ”
