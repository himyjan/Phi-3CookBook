## **1. бһҖбһ¶бһҡбһҠбҹҶбһЎбһҫбһ„**

Ollama бһӮбһ¶бҹҶбһ‘бҹ’бһҡбһҖбһ¶бһҡбһҡбһҸбҹӢбһӣбһҫ Windows, macOS, бһ“бһ·бһ„ Linuxбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҠбҹҶбһЎбһҫбһ„ Ollama бһҸбһ¶бһҳбһҖбҹҶбһҺбһҸбҹӢбһҸбҹҶбһҺбһ“бҹҒбҹҮ ([https://ollama.com/download](https://ollama.com/download))бҹ” бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһ–бһёбһҠбҹҶбһЎбһҫбһ„бһҠбҹ„бһҷбһҮбҹ„бһӮбһҮбҹҗбһҷ бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһҹбҹ’бһӮбҹ’бһҡбһёбһ” Ollama бһҠбһҫбһҳбҹ’бһ”бһёбһ бҹ… Phi-3 бһҠбҹ„бһҷбһ•бҹ’бһ‘бһ¶бһӣбҹӢбһҸбһ¶бһҳбһ”бһ„бҹ’бһўбһҪбһ… terminalбҹ” бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһҳбһҫбһӣ [бһ”бһҺбҹ’бһҺбһ¶бһӣбҹҗбһҷбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһ”бһ¶бһ“бһҖбҹ’бһ“бһ»бһ„ Ollama](https://ollama.com/library) бһ”бһ¶бһ“бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбҹ” бһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһўбҹ’бһ“бһҖбһ”бһҫбһҖ repository бһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Codespace бһңбһ¶бһ“бһ№бһ„бһҳбһ¶бһ“ Ollama бһҠбҹҶбһЎбһҫбһ„бһҡбһҪбһ…бһ бһҫбһҷбҹ”

## бһҖбһ¶бһҡбһ бҹ… Ollama бһ–бһё Python

бһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫ `requests` бһ¬ `urllib3` бһҠбһҫбһҳбҹ’бһ”бһёбһ’бҹ’бһңбһҫбһҹбҹҶбһҺбһҫбһ‘бҹ… endpoints бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳбһҖбҹ’бһ“бһ»бһ„бһ•бҹ’бһ‘бҹҮбһҠбҹӮбһӣбһ”бһ¶бһ“бһ”бҹ’бһҡбһҫбһҒбһ¶бһ„бһӣбһҫбҹ” бһ‘бҹ„бҹҮбһ”бһёбһҮбһ¶бһҷбҹүбһ¶бһ„бһҺбһ¶ бһңбһ·бһ’бһёбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫ Ollama бһҖбҹ’бһ“бһ»бһ„ Python бһӮбһәбһҸбһ¶бһҳбһҡбһҷбҹҲ SDK [openai](https://pypi.org/project/openai/), бһ–бҹ’бһҡбҹ„бҹҮ Ollama бһ•бҹ’бһҸбһӣбҹӢ endpoints бһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһҮбһ¶бһҳбһҪбһҷ OpenAI бһ•бһ„бһҠбҹӮбһҡбҹ”

бһ“бҹҒбҹҮбһӮбһәбһҮбһ¶ бһ§бһ‘бһ¶бһ бһҡбһҺбҹҚбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ phi3-mini:

```python
import openai

client = openai.OpenAI(
    base_url="http://localhost:11434/v1",
    api_key="nokeyneeded",
)

response = client.chat.completions.create(
    model="phi4",
    temperature=0.7,
    n=1,
    messages=[
        {"role": "system", "content": "You are a helpful assistant."},
        {"role": "user", "content": "Write a haiku about a hungry cat"},
    ],
)

print("Response:")
print(response.choices[0].message.content)
```

## бһҖбһ¶бһҡбһ бҹ… Ollama бһ–бһё C#

бһ”бһ„бҹ’бһҖбһҫбһҸбһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё C# Console бһҗбҹ’бһҳбһёбһҳбһҪбһҷ бһ бһҫбһҷбһ”бһ“бҹ’бһҗбҹӮбһҳ NuGet package бһҒбһ¶бһ„бһҖбҹ’бһҡбҹ„бһҳбҹ–

```bash
dotnet add package Microsoft.SemanticKernel --version 1.34.0
```

бһ”бһ“бҹ’бһ‘бһ¶бһ”бҹӢбһҳбһҖбһҮбҹҶбһ“бһҪбһҹбһҖбһјбһҠбһ“бҹҒбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҜбһҖбһҹбһ¶бһҡ `Program.cs`

```csharp
using Microsoft.SemanticKernel;
using Microsoft.SemanticKernel.ChatCompletion;

// add chat completion service using the local ollama server endpoint
#pragma warning disable SKEXP0001, SKEXP0003, SKEXP0010, SKEXP0011, SKEXP0050, SKEXP0052
builder.AddOpenAIChatCompletion(
    modelId: "phi4",
    endpoint: new Uri("http://localhost:11434/"),
    apiKey: "non required");

// invoke a simple prompt to the chat service
string prompt = "Write a joke about kittens";
var response = await kernel.InvokePromptAsync(prompt);
Console.WriteLine(response.GetValue<string>());
```
