# AITK'da Phi Ailesi

[VS Code için AI Araç Seti](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio), Microsoft Foundry Kataloğu ve Hugging Face gibi diğer kataloglardan en son AI geliştirme araçları ve modellerini bir araya getirerek üretken AI uygulama geliştirmeyi basitleştirir. GitHub Modelleri ve Microsoft Foundry Model Katalogları tarafından desteklenen AI modelleri kataloğunu gezebilir, yerel veya uzak olarak indirebilir, ince ayar yapabilir, test edebilir ve uygulamanızda kullanabilirsiniz.

AI Araç Seti Önizlemesi yerel olarak çalışacaktır. Yerel çıkarım veya ince ayar, seçtiğiniz modele bağlıdır, NVIDIA CUDA GPU gibi bir GPU'ya sahip olmanız gerekebilir. GitHub Modellerini doğrudan AITK ile de çalıştırabilirsiniz.

### AI Araç Setini Yükleyin

AI Araç Seti [Visual Studio Code Uzantısı](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions) olarak sunulur, bu yüzden önce [VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo) yüklemelisiniz ve AI Araç Setini [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) üzerinden indirmelisiniz.
[AI Araç Seti Visual Studio Marketplace'de mevcuttur](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) ve diğer VS Code uzantıları gibi yüklenebilir.

VS Code uzantılarının nasıl yüklendiğine aşina değilseniz, bu adımları izleyin:

### Oturum Açma

1. VS Code'da Aktivite Çubuğunda **Uzantılar**ı seçin
1. Uzantılar Arama çubuğuna "AI Toolkit" yazın
1. "Visual Studio Code için AI Toolkit"i seçin
1. **Yükle**yi seçin

Artık uzantıyı kullanmaya hazırsınız!

GitHub'a giriş yapmanız istenecek, devam etmek için "İzin Ver"e tıklayın. GitHub giriş sayfasına yönlendirileceksiniz.

Lütfen oturum açın ve süreç adımlarını takip edin. Başarıyla tamamlandıktan sonra VS Code'a yönlendirileceksiniz.

Uzantı yüklendikten sonra AI Toolkit simgesini Aktivite Çubuğunuzda göreceksiniz.

Mevcut işlemleri keşfedelim!

### Mevcut İşlemler

AI Toolkit'in ana kenar çubuğu şu şekilde düzenlenmiştir:

- **Modeller**
- **Kaynaklar**
- **Oyun Alanı**  
- **İnce Ayar**
- **Değerlendirme**

Kaynaklar bölümünde mevcuttur. Başlamak için **Model Kataloğu**nu seçin.

### Katalogdan bir model indirin

AI Toolkit'i VS Code kenar çubuğundan başlattığınızda aşağıdaki seçeneklerden birini seçebilirsiniz:

![AI toolkit model catalog](../../../../../translated_images/tr/AItoolkitmodel_catalog.7a7be6a7d8468d31.webp)

- **Model Kataloğu**ndan desteklenen bir modeli bulun ve yerel olarak indirin
- Model çıkarımını **Model Oyun Alanı**nda test edin
- Modeli yerel veya uzak olarak **Model İnce Ayar**da ince ayar yapın
- İnce ayarlı modelleri AI Toolkit için komut paleti ile buluta dağıtın
- Modelleri değerlendirin

> [!NOTE]
>
> **GPU ve CPU**
>
> Model kartlarında model boyutu, platform ve hızlandırıcı türü (CPU, GPU) gösterilir. **En az bir GPU'ya sahip Windows cihazlarında**, yalnızca Windows hedefleyen model sürümlerini seçerek optimize edilmiş performans elde edersiniz.
>
> Bu, DirectML hızlandırıcı için optimize edilmiş bir model kullandığınızdan emin olmanızı sağlar.
>
> Model adları aşağıdaki biçimdedir:
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
> Windows cihazınızda GPU'nuz olup olmadığını kontrol etmek için **Görev Yöneticisini** açın ve ardından **Performans** sekmesini seçin. GPU'nuz varsa, "GPU 0" veya "GPU 1" gibi adlar altında listelenecektir.

### Modeli oyun alanında çalıştırın

Tüm parametreler ayarlandıktan sonra **Proje Oluştur**a tıklayın.

Modeliniz indirildikten sonra kataloğundaki model kartında **Oyun Alanında Yükle**yi seçin:

- Model indirilmeye başlar
- Tüm önkoşullar ve bağımlılıklar yüklenir
- VS Code çalışma alanı oluşturulur

### REST API'yi uygulamanızda kullanın

AI Araç Seti, [OpenAI chat completions formatını](https://platform.openai.com/docs/api-reference/chat/create) kullanan yerel bir REST API web sunucusu **5272 numaralı portta** ile birlikte gelir.

Bu, bulut AI model servislerine bağlı kalmadan uygulamanızı yerel olarak test etmenizi sağlar. Örneğin, aşağıdaki JSON dosyası isteğin gövdesini nasıl yapılandıracağınızı gösterir:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```

REST API'yi (örneğin) [Postman](https://www.postman.com/) veya CURL (Client URL) aracı kullanarak test edebilirsiniz:

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```

### Python için OpenAI istemci kitaplığı kullanarak

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # API için gerekli ancak kullanılmıyor
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```

### .NET için Azure OpenAI istemci kitaplığı kullanarak

Projeye NuGet kullanarak [Azure OpenAI istemci kitaplığı](https://www.nuget.org/packages/Azure.AI.OpenAI/) ekleyin:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```

Projeye **OverridePolicy.cs** adında bir C# dosyası ekleyin ve aşağıdaki kodu yapıştırın:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```

Ardından, aşağıdaki kodu **Program.cs** dosyanıza yapıştırın:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
