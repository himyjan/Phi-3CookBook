## Phi-pere mudelitega alustamine Foundry Local’is

### Sissejuhatus Foundry Local’i

Foundry Local on võimas seadmesisene tehisintellekti järelduslahendus, mis toob ettevõtte standarditele vastava tehisintellekti võimsuse otse teie kohalikule riistvarale. See juhend aitab teil seadistada ja kasutada Phi-pere mudeleid Foundry Local’is, pakkudes täielikku kontrolli oma tehisintellekti töökoormuste üle, säilitades privaatsuse ja vähendades kulusid.

Foundry Local pakub jõudlust, privaatsust, kohandatavust ja kuluefektiivsust, käivitades tehisintellekti mudeleid kohapeal teie seadmes. See integreerub sujuvalt teie olemasolevatesse töökäikudesse ja rakendustesse intuitiivse CLI, SDK ja REST API kaudu.


![arch](../../../../../translated_images/et/foundry-local-arch.8823e321dd8258d7.webp)

### Miks valida Foundry Local?

Foundry Local’i eeliste mõistmine aitab teil teha teadlikke otsuseid oma tehisintellekti juurutamise strateegia osas:

- **Seadmesisene järeldus:** Käivitage mudeleid lokaalselt oma riistvaral, vähendades kulusid ja hoides kogu oma andmed seadmes.

- **Mudelipõhine kohandamine:** Valige eelmääratud mudelite seast või kasutage oma mudeleid, et täita spetsiifilisi nõudeid ja kasutusjuhtumeid.

- **Kuluefektiivsus:** Vältige korduvaid pilveteenuste kulusid, kasutades olemasolevat riistvara, muutes AI kättesaadavamaks.

- **Sujuv integreerimine:** Ühenduge oma rakendustega SDK, API otspunktide või CLI kaudu ning saate vajadusel hõlpsasti Microsoft Foundry’le skaleerida.

> **Alustamisnõuanne:** See juhend keskendub Foundry Local’i kasutamisele CLI ja SDK liideste kaudu. Õpite mõlemaid lähenemisviise, et leida oma kasutusjuhtumile sobivaim meetod.

## Osa 1: Foundry Local CLI seadistamine

### Samm 1: Paigaldamine

Foundry Local CLI on teie värav tehisintellekti mudelite kohalikuks haldamiseks ja käivitamiseks. Alustame selle paigaldamisest teie süsteemi.

**Toetatud platvormid:** Windows ja macOS

Üksikasjalike paigaldusjuhiste jaoks vaadake [ametlikku Foundry Local dokumentatsiooni](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Samm 2: Saadavalolevate mudelite uurimine

Kui olete Foundry Local CLI paigaldanud, saate avastada, millised mudelid on teie kasutusjuhtumi jaoks saadaval. See käsk näitab kõiki toetatud mudeleid:


```bash
foundry model list
```

### Samm 3: Phi pere mudelite mõistmine

Phi pere pakub mitmesuguseid mudeleid, mis on optimeeritud erinevateks kasutusjuhtudeks ja riistvarakonfiguratsioonideks. Siin on Foundry Local’is saadaval olevad Phi mudelid:

**Saadaval Phi mudelid:** 

- **phi-3.5-mini** - Kompaktne mudel põhitöödeks
- **phi-3-mini-128k** - Pikema konteksti versioon pikemate vestluste jaoks
- **phi-3-mini-4k** - Standardne kontekstimudel üldiseks kasutuseks
- **phi-4** - Täiustatud mudel parandatud võimekusega
- **phi-4-mini** - Kergem versioon Phi-4-st
- **phi-4-mini-reasoning** - Spetsiaalne keerukate loogikate ülesannete lahendamiseks

> **Riistvara ühilduvus:** Iga mudelit saab konfigureerida erineva riistvarakiirenduse (CPU, GPU) jaoks vastavalt teie süsteemi võimalustele.

### Samm 4: Esimese Phi mudeli käivitamine

Alustame praktilise näitega. Käivitame mudeli `phi-4-mini-reasoning`, mis on suurepärane samm-sammult keerukate probleemide lahendamisel.


**Käsk mudeli käivitamiseks:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Esimese korra seadistamine:** Mudelit esmakordsel käivitamisel laadib Foundry Local automaatselt alla teie kohalikku seadmesse. Allalaadimise aeg sõltub teie võrgu kiirusest, seega olge esialgse seadistuse ajal kannatlik.

### Samm 5: Mudeli testimine reaalse probleemiga

Nüüd testime meie mudelit klassikalise loogikaprobleemiga, et näha, kuidas see samm-sammult loogikat rakendab:

**Näide Probleemist:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Oodatav käitumine:** Mudel peaks selle probleemi jagama loogilisteks sammudeks, kasutades fakti, et faasanitel on 2 jalga ja küülikutel 4 jalga, et lahendada võrrandisüsteem.

## Osa 2: Rakenduste ehitamine Foundry Local SDK-ga

### Miks kasutada SDK-d?

Kuigi CLI sobib testimiseks ja kiireks suhtlemiseks, võimaldab SDK teil integreerida Foundry Local oma rakendustesse programmeeritult. See avab võimalused:

- Kohandatud AI-põhiste rakenduste loomiseks
- Automaatsete töökäikude ehitamiseks
- Tehisintellekti võimekuse integreerimiseks olemasolevatesse süsteemidesse
- Vestlusrobotite ning interaktiivsete tööriistade arendamiseks

### Toetatud programmeerimiskeeled

Foundry Local pakub SDK tuge mitmele programmeerimiskeelele, mis sobivad teie arendusstiiliga:

**📦 Saadaval SDK-d:**

- **C# (.NET):** [SDK dokumentatsioon ja näited](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK dokumentatsioon ja näited](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK dokumentatsioon ja näited](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK dokumentatsioon ja näited](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### Järgmised sammud

1. **Valige eelistatud SDK** oma arenduskeskkonna põhjal
2. **Järgige SDK-spetsiifilist dokumentatsiooni** üksikasjalike rakendamisjuhiste jaoks
3. **Alustage lihtsate näidetega** enne keerukate rakenduste ehitamist
4. **Uurige iga SDK hoidlas olevaid näitekoodi**
