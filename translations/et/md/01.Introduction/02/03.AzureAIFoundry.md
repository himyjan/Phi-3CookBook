## Phi perekond Microsoft Foundrys

[Microsoft Foundry](https://ai.azure.com) on usaldusväärne platvorm, mis võimaldab arendajatel innovatsiooni juhtida ja kujundada tulevikku tehisintellekti abil turvalisel, kindlal ja vastutustundlikul viisil.



[Microsoft Foundry](https://ai.azure.com) on loodud arendajatele järgmiseks:

- Ehita generatiivseid tehisintellekti rakendusi ettevõtte tasemel platvormil.
- Uuri, ehita, testi ja juuruta kasutades tipptasemel tehisintellekti tööriistu ja masinõppemudeleid, lähtudes vastutustundlikest tehisintellekti põhimõtetest.
- Tee koostööd meeskonnaga rakenduste arenduse kogu elutsükli vältel.

Microsoft Foundry abil saad uurida mitmesuguseid mudeleid, teenuseid ja võimalusi ning hakata ehitama tehisintellekti rakendusi, mis kõige paremini teenivad sinu eesmärke. Microsoft Foundry platvorm hõlbustab skaleerimist, muutes kontseptsioonide tõestamise lihtsaks täisväärtuslikeks tootmisrakendusteks. Jätkuv jälgimine ja täiustamine toetavad pikaajalist edu.

### **Python-koodi käitamine Microsoft Foundry Phi-4 kutsumiseks**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Lisa kõnetulemus, kui kõne on sisse lülitatud
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
