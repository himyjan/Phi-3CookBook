# Phi riistvara tugi

Microsoft Phi on optimeeritud ONNX Runtime’i jaoks ja toetab Windows DirectML-i. See töötab hästi erinevatel riistvaratüüpidel, sealhulgas GPU-del, CPU-del ja isegi mobiilseadmetel.

## Seadme riistvara
Täpsustatult hõlmab toetatud riistvara:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPU-d, 128 GiB mälu)

## Mobiili SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 või uuem A16/A17 protsessor

## Phi riistvara spetsifikatsioon

- Nõutav minimaalne konfiguratsioon.
- Windows: DirectX 12-toega GPU ja vähemalt 4 GB kogumälu

CUDA: NVIDIA GPU arvutusvõimekusega >= 7.02

## onnxruntime’i käitamine mitmel GPU-l

Praegu on saadaval Phi ONNX mudelid ainult 1 GPU jaoks. Multi-GPU toetamine Phi mudelitele on võimalik, kuid ORT kahe GPU-ga ei taga suuremat läbilaskevõimet võrreldes kahe ort eksemplariga. Lisateabe saamiseks vaadake palun [ONNX Runtime](https://onnxruntime.ai/) viimaseid uuendusi.

Build 2024-l teatas GenAI ONNX meeskond [Build 2024 the GenAI ONNX Team](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC), et nad on lubanud Phi mudelitele multi-instance’i, mitte multi-GPU kasutamise.

Praegu võimaldab see teil käitada ühte onnxruntime või onnxruntime-genai eksemplari koos CUDA_VISIBLE_DEVICES keskkonnamuutujaga järgmiselt.

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

Julgelt uurige Phi kohta lähemalt [Microsoft Foundry’s](https://ai.azure.com)
