## התחלה עם דגמי משפחת Phi ב-Foundry Local

### מבוא ל-Foundry Local

Foundry Local היא פתרון חזק להפעלת בינה מלאכותית במכשיר שמביא יכולות AI ברמת ארגונית ישירות לחומרה המקומית שלך. מדריך זה ינחה אותך בהגדרה ובשימוש בדגמי משפחת Phi עם Foundry Local, ומציע לך שליטה מלאה על עומסי העבודה של ה-AI שלך תוך שמירה על פרטיות והפחתת עלויות.

Foundry Local מספקת יתרונות בביצועים, פרטיות, התאמה אישית וחיסכון בעלויות על ידי הרצת דגמי AI בצורה מקומית במכשיר שלך. היא משתלבת ללא תקלות עם זרמי העבודה והיישומים הקיימים שלך באמצעות CLI אינטואיטיבי, SDK ו-REST API.


![arch](../../../../../translated_images/he/foundry-local-arch.8823e321dd8258d7.webp)

### מדוע לבחור ב-Foundry Local?

הבנה של היתרונות ב-Foundry Local תעזור לך לקבל החלטות מושכלות לגבי אסטרטגיית הפעלת ה-AI שלך:

- **הפעלת AI במכשיר:** הפעל דגמים באופן מקומי על החומרה שלך, וצמצם את העלויות תוך שמירת כל הנתונים במכשיר שלך.

- **התאמת דגמים:** בחר מתוך דגמים מוגדרים מראש או השתמש בדגם משלך כדי לעמוד בדרישות ובמקרי שימוש ספציפיים.

- **יעילות בעלות:** ביטול עלויות חוזרות של שירותי ענן על ידי שימוש בחומרה הקיימת שלך, מה שהופך את ה-AI לזמין יותר.

- **אינטגרציה חלקה:** התחבר ליישומיך דרך SDK, נקודות קצה API, או ה-CLI, עם אפשרות להרחבה ל-Microsoft Foundry ככל שהצרכים גדלים.

> **הערת התחלה:** מדריך זה מתמקד בשימוש ב-Foundry Local דרך ממשקי CLI ו-SDK. תלמד את שתי הגישות כדי לעזור לך לבחור את השיטה הטובה ביותר למקרה השימוש שלך.

## חלק 1: הגדרת Foundry Local CLI

### שלב 1: התקנה

Foundry Local CLI הוא שער הכניסה שלך לניהול והפעלת דגמי AI מקומית. נתחיל בהתקנתו במערכת שלך.

**פלטפורמות נתמכות:** Windows ו-macOS

להוראות התקנה מפורטות, אנא עיין ב-[התיעוד הרשמי של Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### שלב 2: חקר דגמים זמינים

לאחר שהתקנת את Foundry Local CLI, תוכל לגלות אילו דגמים זמינים למקרה השימוש שלך. הפקודה הזו תציג את כל הדגמים הנתמכים:


```bash
foundry model list
```

### שלב 3: הבנת דגמי משפחת Phi

משפחת Phi מציעה מגוון דגמים מותאמים למקרי שימוש שונים וקונפיגורציות חומרה שונות. הנה הדגמים של Phi הזמינים ב-Foundry Local:

**דגמי Phi זמינים:** 

- **phi-3.5-mini** - דגם קומפקטי למשימות בסיסיות
- **phi-3-mini-128k** - גרסת הקשר מורחבת לשיחות ארוכות יותר
- **phi-3-mini-4k** - דגם הקשר סטנדרטי לשימוש כללי
- **phi-4** - דגם מתקדם עם יכולות משופרות
- **phi-4-mini** - גרסה קלה של Phi-4
- **phi-4-mini-reasoning** - מיוחד למשימות חשיבה מורכבות

> **התאמה לחומרה:** כל דגם ניתן לקונפיגורציה להאצה חומרתית שונה (CPU, GPU) בהתאם ליכולות המערכת שלך.

### שלב 4: הרצת דגם Phi ראשון שלך

נתחיל עם דוגמה פרקטית. נפעיל את דגם `phi-4-mini-reasoning`, שמצטיין בפתרון בעיות מורכבות שלב אחר שלב.


**פקודה להרצת הדגם:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **הגדרה בפעם הראשונה:** בעת הרצת דגם בפעם הראשונה, Foundry Local יוריד אוטומטית את הדגם למכשיר המקומי שלך. זמן ההורדה משתנה בהתאם למהירות הרשת שלך, נא להמתין בסבלנות בזמן ההגדרה הראשונית.

### שלב 5: בדיקת הדגם עם בעיה אמיתית

עכשיו נבדוק את הדגם עם בעיה לוגית קלאסית כדי לראות כיצד הוא מבצע חשיבה שלב אחר שלב:

**דוגמת בעיה:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**התנהגות מצופה:** הדגם אמור לפרק את הבעיה הזו לשלבים לוגיים, תוך שימוש בעובדה שלצפורים יש 2 רגליים ולארנבים יש 4 רגליים כדי לפתור את מערכת המשוואות.

## חלק 2: בניית יישומים עם Foundry Local SDK

### מדוע להשתמש ב-SDK?

בעוד שה-CLI מושלם לבדיקה ואינטראקציות מהירות, ה-SDK מאפשר לך לשלב את Foundry Local ביישומיך בתכנות. זה פותח אפשרויות ל:

- בניית יישומים מותאמים עם יכולות AI
- יצירת זרמי עבודה אוטומטיים
- אינטגרציה של יכולות AI במערכות קיימות
- פיתוח צ׳אטבוטים וכלים אינטראקטיביים

### שפות תכנות נתמכות

Foundry Local מספקת תמיכה ב-SDK עבור שפות תכנות רבות כדי להתאים להעדפות הפיתוח שלך:

**📦 SDKs זמינים:**

- **C# (.NET):** [תיעוד ודוגמאות ל-SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [תיעוד ודוגמאות ל-SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [תיעוד ודוגמאות ל-SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [תיעוד ודוגמאות ל-SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### צעדים הבאים

1. **בחר את ה-SDK המועדף עליך** על בסיס סביבת הפיתוח שלך
2. **עקוב אחרי תיעוד ה-SDK הספציפי** להנחיות יישום מפורטות
3. **התחל עם דוגמאות פשוטות** לפני בניית יישומים מורכבים
4. **חקור את קוד הדוגמה** המסופק בכל מחסן SDK
