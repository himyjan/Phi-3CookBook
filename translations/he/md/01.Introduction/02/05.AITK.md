# משפחת פאי ב-AITK

[ערכת כלי AI ל-VS Code](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) מפשטת את פיתוח אפליקציות AI גנרטיביות על ידי איחוד כלי פיתוח AI מתקדמים ומודלים מקטלוג Microsoft Foundry ומקטלוגים נוספים כמו Hugging Face. תוכל לגלוש בקטלוג מודלי ה-AI המופעל על ידי GitHub Models וקטלוגי מודלים של Microsoft Foundry, להוריד אותם למחשב המקומי או מרחוק, לכוונן, לבדוק ולהשתמש בהם באפליקציה שלך.

גרסת התצוגה המקדימה של ערכת הכלים תרוץ מקומית. ההסקה המקומית או הכיוונון המדויק, תלוי במודל שבחרת, ייתכן שתזדקק ל-GPU כמו NVIDIA CUDA GPU. ניתן להריץ גם את דגמי GitHub ישירות עם AITK.

[ערכת הכלים AI זמינה בשוק Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) וניתנת להתקנה כמו כל תוסף VS Code אחר.

### התחברות

1. בבר פעילות ב-VS Code בחר **תוספים**
1. בשורת חיפוש התוספים הקלד "AI Toolkit"
1. בחר את "AI Toolkit for Visual Studio code"
1. לחץ על **התקן**

כעת, אתה מוכן להשתמש בהרחבה!

תתבקש להתחבר ל-GitHub, לכן לחץ על "Allow" כדי להמשיך. תועבר לעמוד ההתחברות של GitHub.

התחבר ופעל על פי שלבי התהליך. לאחר סיום בהצלחה, תחזור ל-VS Code.

לאחר התקנת ההרחבה תראה את סמל ערכת הכלים AI מופיע בבר הפעילות.

בוא נחקור את הפעולות הזמינות!

### פעולות זמינות

סרגל הצד הראשי של ערכת הכלים AI מאורגן ל-

- **מודלים**
- **משאבים**
- **מגרש משחקים**  
- **כוונון מדויק**
- **הערכה**

נמצאים בחלק המשאבים. כדי להתחיל בחר **קטלוג מודלים**.

### הורדת מודל מהקטלוג

בהפעלת ערכת הכלים AI מבר הצד של VS Code, תוכל לבחור באפשרויות הבאות:

![קטלוג המודלים של ערכת הכלים AI](../../../../../translated_images/he/AItoolkitmodel_catalog.7a7be6a7d8468d31.webp)

- מצא מודל נתמך ב-**קטלוג מודלים** והורד למחשב המקומי
- בדוק את ההסקה של המודל ב-**מגרש המשחקים של המודל**
- כוונן את המודל במחשב המקומי או מרחוק ב-**כוונון מדויק של מודל**
- פרוס מודלים מכווננים לענן דרך לוח הפקודות של ערכת הכלים AI
- הערכת מודלים

> [!NOTE]
>
> **GPU מול CPU**
>
> שמים לב שכרטיסי המודל מציגים את גודל המודל, הפלטפורמה וסוג המואץ (CPU, GPU). לביצועים מיטביים על **מכשירי Windows עם לפחות GPU אחד**, בחר גרסאות מודל המיועדות רק ל-Windows.
>
> זה מבטיח שמודל מותאם למואץ DirectML.
>
> שמות המודלים נמצאים בפורמט של
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
>כדי לבדוק אם יש לך GPU במכשיר Windows שלך, פתח את **מנהל המשימות** ואז בחר בלשונית **ביצועים**. אם יש לך GPU(s), הם יופיעו תחת שמות כמו "GPU 0" או "GPU 1".

### שימוש ב-REST API באפליקציה שלך

ערכת הכלים AI מגיעה עם שרת אינטרנט REST API מקומי **בפורט 5272** המשתמש ב[פורמט השלמות צ'אט של OpenAI](https://platform.openai.com/docs/api-reference/chat/create).

זה מאפשר לך לבדוק את האפליקציה שלך מקומית מבלי להסתמך על שירות מודל AI בענן. לדוגמה, קובץ JSON הבא מציג כיצד להגדיר את גוף הבקשה:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```
  
ניתן לבדוק את REST API באמצעות (למשל) [Postman](https://www.postman.com/) או כלי CURL (Client URL):

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```
  
### שימוש בספריית הלקוח של OpenAI לפייתון

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # נדרש עבור ה-API אך לא בשימוש
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```
  
### שימוש בספריית הלקוח Azure OpenAI ל-.NET

הוסף את [ספריית הלקוח Azure OpenAI ל-.NET](https://www.nuget.org/packages/Azure.AI.OpenAI/) לפרויקט שלך באמצעות NuGet:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```
  
הוסף קובץ C# בשם **OverridePolicy.cs** לפרויקט והדבק בו את הקוד הבא:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```
  
לאחר מכן, הדבק את הקוד הבא בקובץ **Program.cs** שלך:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
