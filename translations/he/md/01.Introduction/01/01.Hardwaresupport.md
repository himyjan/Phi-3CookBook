# תמיכה בחומרת Phi

Microsoft Phi אופטימיזציה ל-ONNX Runtime ותומך ב-Windows DirectML. הוא עובד היטב על סוגי חומרה שונים, כולל GPUs, CPUs ואפילו מכשירים ניידים.

## חומרת המכשיר  
החומרה הנתמכת כוללת במיוחד:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPUs, 128 GiB זיכרון)

## Mobile SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 או מעבד גבוה יותר A16/A17

## מפרט החומרה ל-Phi

- דרישת תצורה מינימלית.
- Windows: GPU התומך ב-DirectX 12 ומינימום 4GB של זיכרון RAM משולב

CUDA: GPU של NVIDIA עם Compute Capability >= 7.02

![HardwareSupport](../../../../../translated_images/he/01.phihardware.5d51b2377cba18af.webp)

## הרצת onnxruntime על מספר GPUs

כרגע, דגמי Phi ONNX זמינים רק עבור GPU אחד. ניתן לתמוך בריבוי GPU לדגם Phi, אך ORT עם 2 GPU אינו מבטיח שייתן יותר תעבורה לעומת 2 מופעי ort. אנא עיין ב-[ONNX Runtime](https://onnxruntime.ai/) לעדכונים האחרונים.

ב(Build 2024 the GenAI ONNX Team)[https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC] הוכרז שהם אפשרו multi-instance במקום multi-GPU לדגמי Phi.

נכון לעכשיו, זה מאפשר לך להריץ מופע אחד של onnnxruntime או onnxruntime-genai עם משתנה הסביבה CUDA_VISIBLE_DEVICES כך.

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

אל תהסס לחקור את Phi לעומק ב-[Microsoft Foundry](https://ai.azure.com)
