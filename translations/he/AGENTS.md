# AGENTS.md

## סקירת הפרויקט

PhiCookBook הוא מאגר מתכונים מקיף הכולל דוגמאות מעשיות, מדריכים ותיעוד לעבודה עם משפחת Phi של מודלי שפה קטנים (SLMs) של מיקרוסופט. המאגר מדגים שימושים שונים כולל הסקה, כוונון עדין, כימות, יישומי RAG, ויישומים מולטימודליים בפלטפורמות ומסגרות עבודה שונות.

**טכנולוגיות מרכזיות:**
- **שפות:** Python, C#/.NET, JavaScript/Node.js
- **מסגרות עבודה:** ONNX Runtime, PyTorch, Transformers, MLX, OpenVINO, Semantic Kernel
- **פלטפורמות:** Microsoft Foundry, GitHub Models, Hugging Face, Ollama
- **סוגי מודלים:** Phi-3, Phi-3.5, Phi-4 (גרסאות טקסט, ראייה, מולטימודל, הסקה)

**מבנה המאגר:**
- `/code/` - דוגמאות קוד ועבודות דגמה
- `/md/` - תיעוד מפורט, מדריכים, ומדריכי שימוש  
- `/translations/` - תרגומים רב-לשוניים (50+ שפות באמצעות תהליך אוטומטי)
- `/.devcontainer/` - קונפיגורציית מיכל פיתוח (Python 3.12 עם Ollama)

## הגדרת סביבת פיתוח

### שימוש ב-GitHub Codespaces או מיכלי פיתוח (מומלץ)

1. פתיחה ב-GitHub Codespaces (המהירה ביותר):
   - לחצו על סמל "Open in GitHub Codespaces" ב-README
   - המיכל מוגדר אוטומטית עם Python 3.12 ו-Ollama עם Phi-3

2. פתיחה במיכלי פיתוח ב-VS Code:
   - השתמשו בסמל "Open in Dev Containers" מתוך README
   - נדרש לפחות 16GB זיכרון מארח במיכל

### התקנה מקומית

**דרישות מקדימות:**
- Python 3.12 ומעלה
- .NET 8.0 SDK (לדוגמאות ב-C#)
- Node.js 18+ ו-npm (לדוגמאות ב-JavaScript)
- מומלץ לפחות 16GB זיכרון RAM

**התקנה:**
```bash
git clone https://github.com/microsoft/PhiCookBook.git
cd PhiCookBook
```

**לדוגמאות ב-Python:**
נווטו לתיקיות דוגמאות ספציפיות והתקינו את התלויות:
```bash
cd code/<example-directory>
pip install -r requirements.txt  # אם קובץ requirements.txt קיים
```

**לדוגמאות ב-.NET:**
```bash
cd md/04.HOL/dotnet/src
dotnet restore LabsPhi.sln
dotnet build LabsPhi.sln
```

**לדוגמאות ב-JavaScript/ווב:**
```bash
cd code/08.RAG/rag_webgpu_chat
npm install
npm run dev  # הפעל שרת פיתוח
npm run build  # בנייה לפרודקשן
```

## ארגון המאגר

### דוגמאות קוד (`/code/`)

- **01.Introduce/** - דוגמאות היכרות בסיסיות ודוגמאות התחלה מהירה
- **03.Finetuning/** ו-**04.Finetuning/** - דוגמאות לכוונון עדין בשיטות שונות
- **03.Inference/** - דוגמאות להסקה על חומרה שונה (AIPC, MLX)

- **09.UpdateSamples/** - דוגמאות מעודכנות אחרונות

### תיעוד (`/md/`)

- **01.Introduction/** - מדריכי פתיחה, הגדרת סביבה, מדריך לפלטפורמות
- **02.Application/** - דוגמאות אפליקציה ממוינות לפי סוג (טקסט, קוד, ראייה, אודיו וכו')
- **02.QuickStart/** - מדריכים להתחלה מהירה עם Microsoft Foundry ו-GitHub Models
- **03.FineTuning/** - תיעוד ומדריכים לכוונון עדין
- **04.HOL/** - מעבדות מעשיות (כולל דוגמאות ב-.NET)

### פורמטי קבצים

- **Jupyter Notebooks (`.ipynb`)** - מדריכי Python אינטראקטיביים המסומנים ב-📓 ב-README
- **סקריפטים ב-Python (`.py`)** - דוגמאות Python עצמאיות
- **פרויקטים ב-C# (`.csproj`, `.sln`)** - אפליקציות ודוגמאות ב-.NET
- **JavaScript (`.js`, `package.json`)** - דוגמאות ווב ו-Node.js
- **Markdown (`.md`)** - תיעוד ומדריכים

**טעינת מודלים:**
