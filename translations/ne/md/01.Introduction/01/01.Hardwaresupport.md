# Phi हार्डवेयर समर्थन

Microsoft Phi ONNX Runtime का लागि अनुकूल गरिएको छ र Windows DirectML समर्थन गर्दछ। यो विभिन्न हार्डवेयर प्रकारहरूमा राम्रोसँग काम गर्छ, जसमा GPU, CPU, र यहाँसम्म कि मोबाइल उपकरणहरू पनि समावेश छन्।

## उपकरण हार्डवेयर 
विशेषगरी, समर्थित हार्डवेयरहरू समावेश छन्:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPUs, 128 GiB मेमोरी)

## मोबाइल SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 वा माथिको A16/A17 प्रोसेसर

## Phi हार्डवेयर विनिर्देश

- न्यूनतम कन्फिगरेसन आवश्यक।
- Windows: DirectX 12-सक्षम GPU र संयोजित 4GB मेमोरी कम्तिमा

CUDA: NVIDIA GPU कम्प्युट क्षमता >= 7.02

![HardwareSupport](../../../../../translated_images/ne/01.phihardware.5d51b2377cba18af.webp)

## एक भन्दा बढी GPU मा onnxruntime सञ्चालन

हाल उपलब्ध Phi ONNX मोडेलहरू केवल 1 GPU का लागि मात्र हुन्। Phi मोडेलका लागि मल्टि-GPU समर्थन सम्भव छ, तर 2 GPU सहितको ORT ले 2 ORT इन्स्ट्यान्स भन्दा बढी थ्रूपुट दिने ग्यारेन्टी गर्दैन। कृपया [ONNX Runtime](https://onnxruntime.ai/) मा नवीनतम अपडेटहरू हेर्नुहोस्।

[Build 2024 मा GenAI ONNX टोली](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) ले Phi मोडेलहरूका लागि मल्टि-GPU सट्टा मल्टि-इन्स्ट्यान्स सक्षम गरेको घोषणा गरेको थियो।

हाल यसले तपाईंलाई CUDA_VISIBLE_DEVICES वातावरण भेरिएबलसँग एउटै onnxruntime वा onnxruntime-genai इन्स्ट्यान्स चलाउन अनुमति दिन्छ।

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

[Microsoft Foundry](https://ai.azure.com) मा Phi लाई थप अन्वेषण गर्न स्वतन्त्र महसुस गर्नुहोस्।
