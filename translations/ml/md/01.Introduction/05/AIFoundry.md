# **Microsoft Foundry ഉപയോഗിച്ച് മൂല്യനിർണ്ണയം ചെയ്യൽ**

![aistudo](../../../../../translated_images/ml/AIFoundry.9e0b513e999a1c5a.webp)

[Microsoft Foundry](https://ai.azure.com?WT.mc_id=aiml-138114-kinfeylo) ഉപയോഗിച്ച് നിങ്ങളുടെ ജനറേറ്റീവ് AI അപ്ലിക്കേഷൻ എങ്ങനെ മൂല്യനിർണ്ണയം ചെയ്യാമെന്ന് കണ്ടു നോക്കുക. നിങ്ങൾ ഏകമായ സംഭാഷണമോ, ബഹു-സംഭാഷണമോ വിലയിരുത്തുകയാണെങ്കിൽ, Microsoft Foundry മോഡൽ പെർഫോർമൻസ് һәм സുരക്ഷ മൂല്യനിർണ്ണയത്തിനുള്ള ഉപകരണങ്ങൾ നൽകുന്നു.

![aistudo](../../../../../translated_images/ml/AIPortfolio.69da59a8e1eaa70f.webp)

## Microsoft Foundry ഉപയോഗിച്ച് ജനറേറ്റീവ് AI ആപ്പുകൾ എങ്ങനെ വിലയിരുത്താം  
കൂടുതൽ വിശദമായ നിർദ്ദേശങ്ങൾക്കായി [Microsoft Foundry ഡോക്യുമെന്റേഷൻ](https://learn.microsoft.com/azure/ai-studio/how-to/evaluate-generative-ai-app?WT.mc_id=aiml-138114-kinfeylo) നോക്കുക

തുടക്കാൻ ഈ ഘട്ടങ്ങൾ ആണ്:

## Microsoft Foundry-ൽ ജനറേറ്റീവ് AI മോഡലുകൾ മൂല്യനിർണ്ണയം ചെയ്യൽ

**ആവശ്യമായത്**

- CSV അല്ലെങ്കിൽ JSON ഫോർമാറ്റിലുള്ള ടെസ്റ്റ് ഡാറ്റാസെറ്റ്.
- ഡിപ്പ്ലോയിംഗ് ചെയ്ത ജനറേറ്റീവ് AI മോഡൽ (Phi-3, GPT 3.5, GPT 4, അല്ലെങ്കിൽ Davinci മോഡലുകൾ പോലുള്ളവ).
- മൂല്യനിർണ്ണയം നടത്താൻ ഒരു കംപ്യൂട്ട് ഇൻസ്റ്റാൻസുമായ റൺടൈം.

## ബിൽറ്റ്-ഇൻ മൂല്യനിർണ്ണയ മാനദണ്ഡങ്ങൾ

Microsoft Foundry ഒറ്റ ചിരിയിലോ സങ്കീർണ്ണമായ ബഹുചിരിയിലോ സംഭാഷണങ്ങൾ മൂല്യനിർണ്ണയം ചെയ്യാൻ അനുവദിക്കുന്നു. 
Retrieval Augmented Generation (RAG) സീനാരിയോകളിൽ, മോഡൽ പ്രത്യേക ഡാറ്റയിൽ അടിസ്ഥാനമാക്കിയതിനാൽ, നിങ്ങൾ ബിൽറ്റ്-ഇൻ മൂല്യനിർണ്ണയ മാനദണ്ഡങ്ങൾ ഉപയോഗിച്ച് പെർഫോർമൻസ് വിലയിരുത്താം.  
അതിനോടൊപ്പം, ജനറൽ ഒറ്റചിരി ചോദ്യോത്തര സീനാരിയോകളെ (non-RAG) നിങ്ങൾ വിലയിരുത്താം.

## ഒരു മൂല്യനിർണ്ണയ റൺ സൃഷ്ടിക്കൽ

Microsoft Foundry UI-യിൽ നിന്നും Evaluate പേജ് അല്ലെങ്കിൽ Prompt Flow പേജിലേക്ക് ചെല്ലുക.  
മൂല്യനിർണ്ണയ സൃഷ്ടിക്കൽ വിസാർഡ് പിന്തുടർന്ന് ഒരു മൂല്യനിർണ്ണയ റൺ സെറ്റപ്പ് ചെയ്യുക. നിങ്ങളുടെ മൂല്യനിർണ്ണയത്തിന് ഇച്ഛാനുസരണം ഒരു പേര് നൽകാം.  
നിങ്ങളുടെ അപ്ലിക്കേഷന്റെ ലക്ഷ്യങ്ങൾക്ക് പൊരുത്തപ്പെടുന്ന സീനാരിയോ തിരഞ്ഞെടുക്കുക.  
മോഡലിന്റെ ഔട്ട്പുട്ട് വിലയിരുത്തുന്നതിനായി ഒരു അല്ലെങ്കിൽ കൂടുതൽ മൂല്യനിർണ്ണയ മാനദണ്ഡങ്ങൾ തിരഞ്ഞെടുക്കുക.

## കസ്റ്റം മൂല്യനിർണ്ണയ ഫ്ലോ (ഐച്ഛികം)

കൂടുതൽ സൗകര്യത്തിനായി, നിങ്ങൾക്ക് കസ്റ്റം മൂല്യനിർണ്ണയ ഫ്ലോ സ്ഥാപിക്കാം. നിങ്ങളുടെ പ്രത്യേക ആവശ്യകതകൾ അടിസ്ഥാനമാക്കി മൂല്യനിർണ്ണയ പ്രക്രിയ ഇഷ്ടാനുസൃതമാക്കാം.

## ഫലങ്ങൾ കാണൽ

മൂല്യനിർണ്ണയം നടത്തുമ്പോൾ, Microsoft Foundry-ൽ വിശദമായ മൂല്യനിർണ്ണയ മാനദണ്ഡങ്ങൾ ലോഗ് ചെയ്യുകയും, കാണുകയും, വിശ്ലേഷണം ചെയ്യുകയും ചെയ്യുക. നിങ്ങളുടെ അപ്ലിക്കേഷന്റെ ശേഷികളും പരിധികളും മനസ്സിലാക്കുക.

**Note** Microsoft Foundry നിലവിൽ പൊതു മുൻകാട്ടി ഘട്ടത്തിലാണ്, അതിനാൽ പരീക്ഷണത്തിനും ഡെവലപ്‌മെന്റിനും മാത്രമേ ഉപയോഗിക്കൂ. പ്രൊഡക്ഷൻ വർക്ക്ലോഡുകൾക്കായി മറ്റൊരു ഓപ്ഷനുകൾ പരിഗണിക്കുക. കൂടുതൽ വിശദാംശങ്ങൾക്കും ഘട്ടം ഘട്ടമായ നിർദ്ദേശങ്ങൾക്കുമായി ഔദ്യോഗിക [AI Foundry ഡോക്യുമെന്റേഷൻ](https://learn.microsoft.com/azure/ai-studio/?WT.mc_id=aiml-138114-kinfeylo) കാണുക.
