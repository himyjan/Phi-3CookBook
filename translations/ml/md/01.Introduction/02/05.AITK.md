# AITK-ൽ Phi കുടുംബം

[AI Toolkit for VS Code](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) മൈക്രോസോഫ്റ്റ് ഫൗണ്ട്രി കാറ്റലോഗ് എന്നിടത്തും Hugging Face പോലുള്ള മറ്റു കാറ്റലോഗുകളിലെയും പുതിയ തലമുറ AI വികസന ഉപകരണങ്ങളും മോഡലുകളും ഒരുമിച്ചുകൊണ്ട് ജനറേറ്റീവ് AI ആപ്പ് വികസനത്തെ എളുപ്പമാക്കുന്നു. GitHub മോഡലുകളും Microsoft Foundry Model Catalogs-ഉം മുന്നികൊണ്ടുള്ള AI മോഡൽ കാറ്റലോഗ് ബ്രൗസുചെയ്യാനും, അവയിടെ നിന്നുള്ള മോഡലുകൾ ലോക്കലായി അല്ലെങ്കിൽ റിമോട്ട് ആയി ഡൗൺലോഡ് ചെയ്യാനും, ഫൈൻ-ട്യൂൺ ചെയ്‌തെടുക്കാനും, ടെസ്റ്റ് നടത്താനും, നിങ്ങളുടെ ആപ്പ്ലിക്കേഷനിൽ ഉപയോഗിക്കാനും കഴിയും.

AI Toolkit പ്രിവ്യു ലോക്കലായി പ്രവർത്തിക്കും. ലോക്കൽ ഇൻഫെറൻസ് അല്ലെങ്കിൽ ഫൈൻ-ട്യൂൺ ചെയ്യുന്നത് നിങ്ങൾ തിരഞ്ഞെടുത്ത മോഡലിൽ ആശ്രയിച്ചിരിക്കും, നിങ്ങൾക്കു NVIDIA CUDA GPU പോലുള്ള GPU വേണം. GitHub മോഡലുകൾ AITK ഉപയോഗിച്ച് നേരിട്ട് ഓടിപ്പോകാനും കഴിയും.

### AI Toolkit ഇൻസ്റ്റാൾ ചെയ്യുക

AI Toolkit [Visual Studio Code ഇക്സ്റ്റൻഷനായി](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions) എത്തുന്നു, അതുകൊണ്ട് ആദ്യം [VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo) ഇൻസ്റ്റാൾ ചെയ്യണം, തുടർന്ന് AI Toolkit [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) നിന്ന് ഡൗൺലോഡ് ചെയ്യുക.
[AI Toolkit Visual Studio Marketplace-ൽ ലഭ്യമാണ്](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) മറ്റേതൊരു VS Code ഇക്സ്റ്റൻഷനു പോലെ ഇൻസ്റ്റാൾ ചെയ്യാം.

VS Code ഇക്സ്റ്റൻഷനുകൾ ഇൻസ്റ്റാൾ ചെയ്യുന്നതിൽ പരിചയക്കുറവ് ഉണ്ടെങ്കിൽ താഴെ കൊടുത്തിരിക്കുന്ന ഘട്ടങ്ങൾ പാലിക്കുക:

### സയിൻ ഇൻ ചെയ്യുക

1. VS Code-ൽ Activity Bar-ൽ **Extensions** തിരഞ്ഞെടുക്കുക
1. Extensions Search ബാറിൽ "AI Toolkit" എന്ന് ടൈപ്പ് ചെയ്യുക
1. "AI Toolkit for Visual Studio code" തിരഞ്ഞെടുക്കുക
1. **Install** തിരഞ്ഞെടുക്കുക

ഇപ്പോൾ, നിങ്ങൾക്ക് ഈ ഇക്സ്റ്റൻഷൻ ഉപയോഗിക്കാൻ തയ്യാറാണ്!

GitHub-ലേക്ക് സയിൻ ഇൻ ചെയ്യാൻ പ്രോംപ്റ്റ് ലഭിക്കും, തുടർന്നുകൊണ്ടുപോകാൻ "Allow" ക്ലിക്ക് ചെയ്യുക. GitHub സയൻ ഇൻ പേജ് തുറക്കും.

ദയവായി സയിൻ ഇൻ ചെയ്ത് പ്രോസസ് ഘട്ടങ്ങൾ പിന്തുടരുക. വിജയകരമായി പൂർത്തിയാക്കിയ ശേഷം നിങ്ങൾ വീണ്ടും VS Code-ലേക്ക് തിരികെ പോകും.

> [!NOTE]
>
> **GPU Vs CPU**
>
> മോഡൽ കാർഡുകളിൽ മോഡലിന്റെ വലുപ്പം, പ്ലാറ്റ്ഫോം, ആക്സിലറേറ്റർ തരം (CPU, GPU) എന്നിവ കാണിക്കും. **കുറഞ്ഞതും ഒന്ന് 이상의 GPU ഉള്ള Windows ഉപകരണങ്ങളിൽ,** Windows മാത്രം ലക്ഷ്യമാക്കിയ മോഡൽ വേർഷനുകൾ തിരഞ്ഞെടുക്കുക.
>
> ഇത് DirectML ആക്സിലറേറ്ററിനായി മെച്ചപ്പെടുത്തിയ മോഡൽ ലഭ്യമാക്കും.
>
> മോഡൽ നാമങ്ങളുടെ ഫോർമാറ്റ്:
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
> Windows ഉപകരണത്തിൽ GPU ഉണ്ടോ എന്ന് പരിശോധിക്കാൻ, **Task Manager** തുറന്ന് **Performance** ടാബ് തിരഞ്ഞെടുക്കുക. GPU ഉണ്ടെങ്കിൽ, "GPU 0" അല്ലെങ്കിൽ "GPU 1" പോലുള്ള പേരുകളിൽ കാണപ്പെടും.

### നിങ്ങളുടെ ആപ്പ്ലിക്കേഷനിൽ REST API ഉപയോഗിക്കുക

AI Toolkit-ന് ഒരു ലോക്കൽ REST API വെബ് സർവർ **പോർട്ട് 5272-ൽ** ഉണ്ട്, ഇത് [OpenAI ചാറ്റ് കോംപ്ലീഷൻസ് ഫോർമാറ്റ്](https://platform.openai.com/docs/api-reference/chat/create) ഉപയോഗിക്കുന്നു.

ഇതുവഴി നിങ്ങളുടെ ആപ്പ്ലിക്കേഷൻ ക്ലൗഡ് AI മോഡൽ സർവീസിൽ ആശ്രയിക്കാതെ ലോക്കലായി ടെസ്റ്റ് ചെയ്യാനാകും. ഉദാഹരണത്തിന്, താഴെ വിവരിച്ചിരിക്കുന്ന JSON ഫയൽ റിക്ക്വസ്റ്റ് ബോഡി എങ്ങനെ കോൺഫിഗർ ചെയ്യാമെന്ന് കാണിക്കുന്നു:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```


REST API പരീക്ഷിക്കാൻ [Postman](https://www.postman.com/) പോലുള്ള ടൂളുകളോ CURL (Client URL) യൂട്ടിലിറ്റിയോ ഉപയോഗിക്കാം:

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```


### Python നുള്ള OpenAI ക്ലയന്റ് ലൈബ്രറി ഉപയോഗിക്കൽ

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # API-യ്ക്കായി ആവശ്യമുണ്ട്, പക്ഷേ ഉപയോഗിക്കപ്പെട്ടിട്ടില്ല
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```


### .NET നുള്ള Azure OpenAI ക്ലയന്റ് ലൈബ്രറി ഉപയോഗിക്കൽ

നിങ്ങളുടെ പ്രോജക്റ്റിൽ [Azure OpenAI client library for .NET](https://www.nuget.org/packages/Azure.AI.OpenAI/) NuGet ഉപയോഗിച്ച് ചേർക്കുക:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```


**OverridePolicy.cs** എന്ന പേരിൽ ഒരു C# ഫയൽ ചേർത്തു താഴെ കൊടുത്തിരിക്കുന്നത് പിരിച്ചുചേർക്കുക:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```


അടുത്തതിനായി നിങ്ങളുടെ **Program.cs** ഫയലിൽ താഴെ കൊടുത്തിരിക്കുന്ന കോഡ് പകർത്തുക:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
