## Microsoft Foundry ലെ Phi കുടുംബം

[Microsoft Foundry](https://ai.azure.com) വികസിപ്പിക്കുന്നവര്‍ക്ക് സുരക്ഷിതമായ, വിശ്വസനീയമായ, ഉത്തരവാദിത്വത്തോടെ AI ഉപയോഗിച്ച് നവസംവേചനം നയിക്കാനും ഭവിയെ രൂപപ്പെടുത്താനും ശേഷിയുള്ള ഒരു വിശ്വാസയോഗ്യമായ പ്ലാറ്റ്‌ഫോമാണ്.

[Microsoft Foundry] (https://ai.azure.com) വികസിപ്പിക്കുന്നവര്‍ക്കായി രൂപകല്‍പന ചെയ്തതാണ്:

- എന്റര്‍പ്രൈസ്-ഗ്രേഡ് പ്ലാറ്റ്‌ഫോമില്‍ ജനറേറ്റീവ് AI ആപ്ലിക്കേഷനുകള്‍ സൃഷ്ടിക്കുക.
- Cutting-edge AI ഉപകരണങ്ങളും ML മോഡലുകളും ഉപയോഗിച്ച് അന്വേഷിക്കുക, നിര്‍മിക്കുക, പരീക്ഷിക്കുക, പ്രയോഗിക്കുക; ഉത്തരവാദിത്വമുള്ള AI പ്രക്രിയകളില്‍ ആധരിച്ചുള്ളവ.
- ആപ്ലിക്കേഷന്‍ വികസനത്തിന്റെ പൂര്‍ണ ജീവവൃത്താക്രമത്തില്‍ ഒരു ടീമിനൊപ്പം സഹകരിക്കുക.

Microsoft Foundry ഉപയോഗിച്ച്, നിങ്ങള്‍ക്ക് വ്യത്യസ്ത മോഡലുകളും സേവനങ്ങളും കഴിവുകളും പരിശോധിച്ച്, നിങ്ങളുടെ ലക്ഷ്യങ്ങള്‍ക്ക് ഏറ്റവും അനുയോജ്യമായ AI ആപ്ലിക്കേഷനുകള്‍ നിര്‍മിക്കാവുന്നതാണ്. Microsoft Foundry പ്ലാറ്റ്‌ഫോം പ്രൂഫ് ഓഫ് കണ്‍സെപ്റ്റുകള്‍ പൂര്‍ണ്ണതയുള്ള ഉത്പാദന ആപ്ലിക്കേഷനുകളായി രൂപാന്തരപ്പെടുത്തുന്നതിനുള്ള സ്‌കെയിലബിലിറ്റി സുഗമമാക്കുന്നു. നിരന്തരം നിയന്ത്രണവും മെച്ചപ്പെടുത്തലും ദൈര്‍ഘ്യമുള്ള വിജയത്തിനായി പിന്തുണ നല്‍കുന്നു.

Microsoft Foundry-യില്‍ Azure AOAI Service ഉപയോഗിക്കുന്നതിന്റെ പുറമേ, Microsoft Foundry മോഡല്‍ കാറ്റലോഗില്‍ തൃतीयപക്ഷ മോഡലുകളും നിങ്ങള്‍ ഉപയോഗിക്കാം. AI സൊല്യൂഷന്‍ പ്ലാറ്റ്‌ഫോമായി Microsoft Foundry ഉപയോഗിക്കുവാന്‍ ഇത് സ дзяржаўവുമാണ്.

### **Microsoft Foundry Phi-4 വിളിക്കാന്‍ Python കോഡ് പ്രവര്‍ത്തിപ്പിക്കല്‍**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# പ്രസംഗം സജ്ജമാക്കിയിട്ടുണ്ടെങ്കിൽ പ്രസംഗഫലം ഉൾപ്പെട യാക്കുക
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
