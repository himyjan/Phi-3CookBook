## ഫൗണ്ട്രി ലൊക്കലിൽ ഫൈ-ഫാമിലി മോഡലുകളുമായി തുടങ്ങുന്നു

### ഫൗണ്ട്രി ലൊക്കലിന്റെ മുഴുവൻ പരിചയം

ഫൗണ്ട്രി ലൊക്കൽ നിങ്ങളുടെ ലോക്കൽ ഹാർഡ്‌വെയർ നേരിട്ട് സംരംഭ-നിലവാരമുള്ള AI ശേഷികൾ കൊണ്ടുവരുന്ന ശക്തമായ ഓൺ-ഡിവൈസ് AI ഇൻഫറൻസ് സൊല്യൂഷനാണ്. ഫൗണ്ട്രി ലൊക്കലിൽ ഫൈ-ഫാമിലി മോഡലുകൾ സെറ്റ് ചെയ്തു ഉപയോഗിക്കുന്നതിൽ ഈ ടൂട്ടോറിയൽ നിങ്ങളെ ഗൈഡ് ചെയ്യും, സ്വകാര്യത നിലനിർത്തുകയും ചെലവുകൾ കുറയ്ക്കുകയും ചെയ്യുക കൂടാതെ നിങ്ങളുടെ AI ജോലിഭാരങ്ങൾക്കുള്ള പൂർണ നിയന്ത്രണം നൽകുന്നു.

ഫൗണ്ട്രി ലൊക്കൽ നിങ്ങളുടെ ഉപകരണത്തിൽ AI മോഡലുകൾ സദൃശമായി പ്രവർത്തിപ്പിച്ച് പ്രകടനം, സ്വകാര്യത, ഇഴക്കുറവ്, ചെലവ് ലാഭങ്ങൾ നൽകുന്നു. ഇത് ഒരു സുഗമ CLI, SDK, REST API വഴിയാണ് നിങ്ങളുടെ നിലവിലുള്ള പ്രവൃത്തിപാതകളിലും അപ്ലിക്കേഷനുകളിലും ലളിതമായി ഇന്റഗ്രേറ്റ് ചെയ്യുന്നത്.


![arch](../../../../../translated_images/ml/foundry-local-arch.8823e321dd8258d7.webp)

### ഫൗണ്ട്രി ലൊക്കൽ തിരഞ്ഞെടുക്കേണ്ടതിന്റെ കാരണങ്ങൾ

ഫൗണ്ട്രി ലൊക്കലിന്റെ ലാഭങ്ങൾ മനസിലാക്കുന്നത് നിങ്ങളുടെ AI വിന്യാസ തന്ത്രം ബോധപൂർവം തീർക്കാൻ സഹായിക്കും:

- **ഓൺ-ഡിവൈസ് ഇൻഫറൻസ്:** നിങ്ങളുടെ സ്വന്തം ഹാർഡ്‌വെയറിൽ മോഡലുകൾ ലോക്കലായി ഓടിക്കുക, ഉപയോഗിക്കുന്ന ഡാറ്റ മുഴുവൻ ഉപകരണത്തിൽ സൂക്ഷിച്ച് ചെലവുകൾ കുറക്കുക.

- **മോഡൽ ഇഷ്ടാനുസൃതം:** മുൻകൂർ സെറ്റ് ചെയ്ത മോഡലുകൾ നിന്ന് തിരഞ്ഞെടുക്കുക അല്ലെങ്കിൽ നിങ്ങളുടെ സ്വന്തം മോഡലുകൾ ഉപയോഗിച്ച് പ്രത്യേക ആവശ്യകതകൾ പാലിക്കുക.

- **ചെലവ് കാര്യക്ഷമത:** നിങ്ങളുടെ നിലവിലുള്ള ഹാർഡ്‌വെയർ ഉപയോഗിച്ച് ക്ലൗഡ് സേവന ചെലവുകൾ ഒഴിവാക്കി AI കൂടുതൽ പ്രാപ്യമായാക്കുക.

- **സുഗമമായ ഇന്റഗ്രേഷൻ:** SDK, API എൻഡ്പോയിന്റുകൾ അല്ലെങ്കിൽ CLI മുഖേന നിങ്ങളുടെ അപ്ലിക്കേഷനുമായി ബന്ധിപ്പിക്കുകയും ആവശ്യത്തിനു വളർച്ചയുണ്ടായാൽ Microsoft Foundry-യിലേക്ക് എളുപ്പത്തിൽ സ്കെയിൽ ചെയ്യുകയുമാണ്.

> **ആരംഭക സൂചനം:** ഈ ടൂട്ടോറിയൽ CLI, SDK ഇന്റർഫേസുകൾ വഴി ഫൗണ്ട്രി ലോക്കലിന്റെ ഉപയോഗത്തെ കവരുന്നു. രണ്ടും പഠിച്ചു നിങ്ങളുടെ ഉപയോഗത്തിനു ഏറ്റവും അനുയോജ്യമായ പാത തിരഞ്ഞെടുക്കാൻ സഹായിക്കും.

## ഭാഗം 1: ഫൗണ്ട്രി ലൊക്കൽ CLI സജ്ജമാക്കൽ

### ഘട്ടം 1: ഇൻസ്റ്റാളേഷൻ

ഫൗണ്ട്രി ലൊക്കൽ CLI നിങ്ങളുടെ AI മോഡലുകൾ ലോക്കലായി കൈകാര്യം ചെയ്ത് ഓടിക്കാൻ പ്രവേശന ദ്വാരമാണ്. ഇത് നിങ്ങളുടെ സിസ്റ്റത്തിൽ ഇൻസ്റ്റാൾ ചെയ്യുന്നതിൽ നിന്നു തുടങ്ങി.

**സഹായിക്കുന്ന പ്ലാറ്റ്ഫോമുകൾ:** Windows, macOS

കൃത്യമായ ഇൻസ്റ്റാളേഷൻ നിർദ്ദേശങ്ങൾക്കായി [അധികൃത ഫൗണ്ട്രി ലൊക്കൽ ഡോക്യുമെന്റേഷൻ](https://github.com/microsoft/Foundry-Local/blob/main/README.md) കാണുക.

### ഘട്ടം 2: ലഭ്യമായ മോഡലുകൾ പരിശോധിക്കൽ

ഫൗണ്ട്രി ലൊക്കൽ CLI ഇൻസ്റ്റാൾ ചെയ്ത ശേഷം നിങ്ങളുടെ ഉപയോഗത്തിനു അനുയോജ്യമായ മോഡലുകൾ കണ്ടെത്താവുന്നതാണ്. ഈ കമാൻഡ് നിങ്ങളെ എല്ലാ പിന്തുണയുള്ള മോഡലുകളും കാണിക്കും:


```bash
foundry model list
```

### ഘട്ടം 3: ഫൈ ഫാമിലി മോഡലുകൾ അടിസ്ഥാനപരമായി മനസിലാക്കൽ

ഫൈ ഫാമിലി വ്യത്യസ്ത ഉപയോഗ കേസുകൾക്കും ഹാർഡ്‌വെയർ കോൺഫിഗറേഷനുകൾക്കുമായി ഒപ്റ്റിമൈസ്ഡ് മോഡലുകളുടെ ഒരു നിര നൽകുന്നു. ഫൗണ്ട്രി ലൊക്കലിൽ ലഭ്യമായ ഫൈ മോഡലുകൾ ഇവയാണ്:

**ലഭ്യമായ ഫൈ മോഡലുകൾ:** 

- **phi-3.5-mini** - അടിസ്ഥാന ജോലികൾക്കായുള്ള കൊച്ചു മോഡൽ
- **phi-3-mini-128k** - നീണ്ട സംഭാഷണങ്ങൾക്ക് വിപുലീകരിച്ച കോണ്ടക്സ്റ്റ് പതിപ്പ്
- **phi-3-mini-4k** - പൊതുവായ ഉപയോഗത്തിനുള്ള സ്റ്റാൻഡേർഡ് കോണ്ടക്സ്റ്റ് മോഡൽ
- **phi-4** - മെച്ചപ്പെട്ട കഴിവുകളുള്ള പുരോഗമിത മോഡൽ
- **phi-4-mini** - Phi-4 ന്റെ ലഘു പതിപ്പ്
- **phi-4-mini-reasoning** - സങ്കീർണമായ തർക്കാന്വേഷണ ജോലികൾക്കായി പ്രത്യേകിച്ചുള്ളത്

> **ഹാർഡ്‌വെയർ അനുയോജ്യത:** ഓരോ മോഡലും നിങ്ങളുടെ സിസ്റ്റം കഴിവ് അനുസരിച്ചു വ്യത്യസ്ത ഹാർഡ്‌വെയർ ആക്സിലറേഷൻ (CPU, GPU) ആയി കോൺഫിഗർ ചെയ്യാവുന്നതാണ്.

### ഘട്ടം 4: നിങ്ങളുടെ ആദ്യ ഫൈ മോഡൽ ഓടിക്കുക

പ്രായോഗിക ഉദാഹരണത്തോടെ ആരംഭിക്കാം. സങ്കീർണ പ്രശ്നങ്ങൾ ഘട്ടം ഘട്ടമായി പരിഹരിക്കാൻ മികവുള്ള `phi-4-mini-reasoning` മോഡൽ ഉപയോഗിച്ച് പ്രവർത്തിപ്പിക്കാം.


**മോഡൽ ഓടിക്കുന്ന കമാൻഡ്:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **ആദ്യം സ്ഥാപിക്കൽ:** ഒരു മോഡൽ ആദ്യമായി ഓടിക്കുമ്പോൾ, ഫൗണ്ട്രി ലൊക്കൽ അത് നിങ്ങളുടെ ലോക്കൽ ഉപകരണത്തിൽ സ്വയം ഡൗൺലോഡ് ചെയ്യും. ഡൗൺലോഡ് സമയം നിങ്ങളുടെ നെറ്റ്‌വർക്ക് വേഗത അനുസരിച്ചു വ്യത്യാസപ്പെടും, അതിനാൽ തുടക്ക സജ്ജീകരണത്തിൽ ദയവായി ക്ഷമിക്കണം.

### ഘട്ടം 5: യഥാർത്ഥ പ്രശ്നത്തോടെ മോഡൽ തുടരുക

ഇപ്പോൾ ഒരു ക്ലാസിക് ലജിക് പ്രശ്നം ഉപയോഗിച്ച് മോഡലിന് ഘട്ടം ഘട്ടമായ തർക്കാന്വേഷണത്തിൽ എങ്ങനെയാണ് പ്രവര്‍ത്തിക്കുന്നത് പരിശോധിക്കും:

**ഉദാഹരണ പ്രശ്നം:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**പ്രതീക്ഷിക്കപ്പെടുന്ന പെരുമാറ്റം:** ഈ പ്രശ്നം ലജിക് ഘട്ടങ്ങളിലായി മോഡൽ വിഭാഗപ്പെടുത്തണം, ഫീസാന്റുകൾക്ക് 2 കാലുകളുള്ളതു, രാപ്പിറ്റുകൾക്ക് 4 കാലുകൾ ഉള്ളതു ഉപയോഗിച്ച് സമവാക്യ സിസ്റ്റം പരിഹരിക്കുക.

## ഭാഗം 2: ഫൗണ്ട്രി ലൊക്കൽ SDK ഉപയോഗിച്ച് അപ്ലിക്കേഷനുകൾ നിർമ്മാണം

### SDK ഉപയോഗിക്കുന്നതിന് കാരണങ്ങൾ

CLI ടെസ്റ്റിംഗിനും വേഗ പ്രയോഗങ്ങൾക്കുമായി അനുയോജ്യമാണ്, SDK ഫൗണ്ട്രി ലൊക്കൽ നിങ്ങളുടെ അപ്ലിക്കേഷനുകളിൽ പ്രോഗ്രാമാറ്റിക്കായി സംയോജിപ്പിക്കാൻ സഹായിക്കുന്നു. ഇതിലൂടെ സാധ്യമായ മാറ്റങ്ങൾ:

- ഇഷ്ടാനുസൃത AI-അധികൃത അപ്ലിക്കേഷനുകൾ നിർമ്മാണം
- ഓട്ടോമേറ്റഡ് ജോലിപ്രവാഹങ്ങൾ സൃഷ്ടിക്കൽ
- നിലവിലുള്ള സംവിധാനങ്ങളിൽ AI കഴിവുകൾ കൂട്ടിച്ചേർക്കൽ
- ചാറ്റ്‌ബോട്ടുകൾ, ഇന്ററാക്ടീവ് ടൂളുകൾ വികസിപ്പിക്കൽ

### പിന്തുണയുള്ള പ്രോഗ്രാമിങ്ങ് ഭാഷകൾ

ഫൗണ്ട്രി ലൊക്കൽ നിങ്ങളുടെ വികസന ഇഷ്ടാനുസരിച്ച് നിരവധി പ്രോഗ്രാമിങ്ങ് ഭാഷകളിൽ SDK പിന്തുണ നൽകുന്നു:

- **C# (.NET):** [SDK ഡോക്യുമെന്റേഷൻ & ഉദാഹരണങ്ങൾ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK ഡോക്യുമെന്റേഷൻ & ഉദാഹരണങ്ങൾ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK ഡോക്യുമെന്റേഷൻ & ഉദാഹരണങ്ങൾ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK ഡോക്യുമെന്റേഷൻ & ഉദാഹരണങ്ങൾ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### അടുത്ത ഘട്ടങ്ങൾ

1. നിങ്ങളുടെ വികസന പരിതസ്ഥിതിക്ക് അനുസൃത SDK തിരഞ്ഞെടുക്കുക
2. വിശദമായ നിര്‍വഹണ മാർഗ്ഗനിർദ്ദേശങ്ങൾക്ക് SDK-നിഷ്ട ഡോക്യുമെന്റേഷൻ പാലിക്കുക
3. സങ്കീർണ അപ്ലിക്കേഷനുകൾ നിർമ്മിക്കുന്നതിന് മുമ്പ് ലളിതമായ ഉദാഹരണങ്ങളിൽ ആരംഭിക്കുക
4. ഓരോ SDK റിപ്പോസിറ്ററിയിലും നൽകിയ സാംപിള് കോഡ് പരിശോധിക്കുക

- ✅ ഫൈ ഫാമിലി മോഡലുകൾ കണ്ടെത്തി ഓടിക്കാൻ
- ✅ യഥാർത്ഥ പ്രശ്നങ്ങളിലൂടെ മോഡലുകൾ പരീക്ഷിക്കാൻ
- ✅ അപ്ലിക്കേഷൻ വികസനത്തിന് SDK തിരഞ്ഞെടുക്കാനുള്ള ഓപ്ഷനുകൾ മനസിലാക്കാൻ

ഫൗണ്ട്രി ലൊക്കൽ നിങ്ങളുടെ ലോക്കൽ പരിതസ്ഥിതിയിലേക്ക് AI കഴിവുകൾ നേരിട്ട് കൊണ്ടുവരാൻ ശക്തമായ അടിസ്ഥാനമൊരുക്കുന്നു, പ്രകടനം, സ്വകാര്യത, ചെലവുകൾ എന്നിവയിൽ നിയന്ത്രണം നൽകുകയും ആവശ്യത്തിന് ക്ലൗഡ് പരിഹാരങ്ങളിലേക്ക് സ്കെയിൽ ചെയ്യാനുള്ള സൗകര്യം അനുവദിക്കുകയും ചെയ്യുന്നു.
