# Keluarga Phi Microsoft

Model Phi adalah Model Bahasa Kecil (SLM) yang paling berupaya dan kos efektif yang tersedia, mengatasi model dengan saiz yang sama dan saiz seterusnya merangkumi pelbagai penanda aras bahasa, penaakulan, pengkodan, audio, visi, dan matematik. Pelepasan ini mengembangkan pilihan model berkualiti tinggi untuk pelanggan, menawarkan lebih banyak pilihan praktikal untuk mengarang dan membina aplikasi AI generatif.

Keluarga Phi bermula dengan Phi-1 untuk penjanaan Kod Python, diteruskan ke Phi-1.5 /2 berdasarkan teks dan pelengkap sembang, Phi-3-mini/kecil/sederhana-instruksi dan Phi-3.5/4-mini-instruksi, dan dibangunkan ke Phi-3/3.5-vision untuk visi, Phi-4 berdasarkan penaakulan kuat, dan Phi-3.5-MoE untuk MoE, dan kini model modal penuh Phi-4-multimodal. Melalui set data berkualiti tinggi, penanda aras boleh dilatih agar setara dengan model yang mempunyai parameter latihan yang lebih besar.
