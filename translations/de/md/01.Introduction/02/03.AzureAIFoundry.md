## Phi-Familie in Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) ist eine vertrauenswürdige Plattform, die Entwickler befähigt, Innovationen voranzutreiben und die Zukunft mit KI auf sichere, geschützte und verantwortungsvolle Weise zu gestalten.



[Microsoft Foundry](https://ai.azure.com) ist für Entwickler konzipiert, um:

- Generative KI-Anwendungen auf einer Unternehmensplattform zu erstellen.
- Zu erforschen, zu entwickeln, zu testen und bereitzustellen mit modernen KI-Tools und ML-Modellen, basierend auf verantwortungsvollen KI-Praktiken.
- Mit einem Team für den gesamten Lebenszyklus der Anwendungsentwicklung zusammenzuarbeiten.

Mit Microsoft Foundry können Sie eine Vielzahl von Modellen, Diensten und Fähigkeiten erkunden und KI-Anwendungen entwickeln, die am besten Ihren Zielen dienen. Die Microsoft Foundry Plattform erleichtert die Skalierbarkeit, um Prototypen mühelos in vollwertige Produktionsanwendungen zu transformieren. Kontinuierliches Monitoring und Verfeinerung unterstützen den langfristigen Erfolg.

### **Ausführen von Python-Code zum Aufrufen von Microsoft Foundry Phi-4**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Sprachresultat einschließen, wenn Sprache aktiviert ist
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
