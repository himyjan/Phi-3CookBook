## Erste Schritte mit Phi-Family-Modellen in Foundry Local

### Einführung in Foundry Local

Foundry Local ist eine leistungsstarke On-Device-AI-Inferenzlösung, die KI-Funktionen in Unternehmensqualität direkt auf Ihre lokale Hardware bringt. Dieses Tutorial führt Sie durch die Einrichtung und Nutzung von Phi-Family-Modellen mit Foundry Local und bietet Ihnen vollständige Kontrolle über Ihre KI-Arbeitslasten bei gleichzeitigem Schutz der Privatsphäre und Kostensenkung.

Foundry Local bietet Leistungs-, Datenschutz-, Anpassungs- und Kostenvorteile, indem KI-Modelle lokal auf Ihrem Gerät ausgeführt werden. Es integriert sich nahtlos in Ihre bestehenden Workflows und Anwendungen durch eine intuitive CLI, SDK und REST API.


![arch](../../../../../translated_images/de/foundry-local-arch.8823e321dd8258d7.webp)

### Warum Foundry Local wählen?

Das Verständnis der Vorteile von Foundry Local hilft Ihnen, fundierte Entscheidungen für Ihre KI-Bereitstellungsstrategie zu treffen:

- **On-Device-Inferenz:** Führen Sie Modelle lokal auf Ihrer eigenen Hardware aus, senken Sie Ihre Kosten und behalten Sie alle Ihre Daten auf Ihrem Gerät.

- **Modellanpassung:** Wählen Sie aus voreingestellten Modellen oder verwenden Sie eigene, um spezifische Anforderungen und Anwendungsfälle zu erfüllen.

- **Kostenersparnis:** Eliminieren Sie wiederkehrende Cloud-Service-Kosten, indem Sie Ihre vorhandene Hardware nutzen und KI so zugänglicher machen.

- **Nahtlose Integration:** Verbinden Sie sich mit Ihren Anwendungen über ein SDK, API-Endpunkte oder die CLI mit einfacher Skalierung zu Microsoft Foundry, wenn Ihr Bedarf wächst.

> **Hinweis zum Einstieg:** Dieses Tutorial konzentriert sich auf die Nutzung von Foundry Local über CLI- und SDK-Schnittstellen. Sie lernen beide Ansätze kennen, um die beste Methode für Ihren Anwendungsfall zu wählen.

## Teil 1: Einrichtung der Foundry Local CLI

### Schritt 1: Installation

Die Foundry Local CLI ist Ihr Tor zur Verwaltung und Ausführung von KI-Modellen lokal. Beginnen wir mit der Installation auf Ihrem System.

**Unterstützte Plattformen:** Windows und macOS

Für detaillierte Installationsanweisungen lesen Sie bitte die [offizielle Foundry Local Dokumentation](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Schritt 2: Verfügbare Modelle erkunden

Nachdem Sie die Foundry Local CLI installiert haben, können Sie entdecken, welche Modelle für Ihren Anwendungsfall verfügbar sind. Dieser Befehl zeigt Ihnen alle unterstützten Modelle:


```bash
foundry model list
```

### Schritt 3: Die Phi-Family-Modelle verstehen

Die Phi-Family bietet eine Reihe von Modellen, die für unterschiedliche Anwendungsfälle und Hardwarekonfigurationen optimiert sind. Hier sind die in Foundry Local verfügbaren Phi-Modelle:

**Verfügbare Phi-Modelle:** 

- **phi-3.5-mini** - Kompaktes Modell für grundlegende Aufgaben
- **phi-3-mini-128k** - Erweiterte Kontextversion für längere Gespräche
- **phi-3-mini-4k** - Standardkontextmodell für allgemeine Verwendung
- **phi-4** - Fortgeschrittenes Modell mit verbesserten Fähigkeiten
- **phi-4-mini** - Leichtgewichtige Version von Phi-4
- **phi-4-mini-reasoning** - Spezialisiert auf komplexe Denkaufgaben

> **Hardware-Kompatibilität:** Jedes Modell kann je nach Ihren Systemfähigkeiten für verschiedene Hardwarebeschleunigungen (CPU, GPU) konfiguriert werden.

### Schritt 4: Ihr erstes Phi-Modell ausführen

Beginnen wir mit einem praktischen Beispiel. Wir werden das Modell `phi-4-mini-reasoning` starten, das hervorragend darin ist, komplexe Probleme Schritt für Schritt zu lösen.


**Befehl zum Ausführen des Modells:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Ersteinrichtung:** Beim ersten Ausführen eines Modells wird Foundry Local dieses automatisch auf Ihr lokales Gerät herunterladen. Die Downloadzeit variiert je nach Netzwerkgeschwindigkeit, bitte haben Sie während der anfänglichen Einrichtung Geduld.

### Schritt 5: Das Modell mit einem realen Problem testen

Testen wir nun unser Modell mit einem klassischen Logikproblem, um zu sehen, wie es schrittweise Schlüsse zieht:

**Beispielproblem:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Erwartetes Verhalten:** Das Modell sollte dieses Problem in logische Schritte zerlegen und unter Verwendung der Tatsache, dass Fasane 2 Beine und Kaninchen 4 Beine haben, das Gleichungssystem lösen.

## Teil 2: Anwendungen mit Foundry Local SDK entwickeln

### Warum das SDK verwenden?

Während die CLI perfekt für Tests und schnelle Interaktionen ist, ermöglicht Ihnen das SDK, Foundry Local programmgesteuert in Ihre Anwendungen zu integrieren. Das eröffnet Möglichkeiten für:

- Entwicklung maßgeschneiderter KI-gestützter Anwendungen
- Erstellung automatisierter Workflows
- Integration von KI-Funktionen in bestehende Systeme
- Entwicklung von Chatbots und interaktiven Tools

### Unterstützte Programmiersprachen

Foundry Local bietet SDK-Unterstützung für mehrere Programmiersprachen, um Ihren Entwicklungspräferenzen gerecht zu werden:

**📦 Verfügbare SDKs:**

- **C# (.NET):** [SDK-Dokumentation & Beispiele](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK-Dokumentation & Beispiele](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK-Dokumentation & Beispiele](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK-Dokumentation & Beispiele](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### Nächste Schritte

1. **Wählen Sie Ihr bevorzugtes SDK** basierend auf Ihrer Entwicklungsumgebung

3. **Beginnen Sie mit einfachen Beispielen**, bevor Sie komplexe Anwendungen entwickeln
4. **Erkunden Sie den Beispielcode**, der in jedem SDK-Repository bereitgestellt wird
