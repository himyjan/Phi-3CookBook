## ਮਾਈਕ੍ਰੋਸਾਫਟ ਫਾਉਂਡਰੀ ਵਿੱਚ ਫਾਈ ਪਰਿਵਾਰ

[Microsoft Foundry](https://ai.azure.com) ਇੱਕ ਭਰੋਸੇਮੰਦ ਪਲੇਟਫਾਰਮ ਹੈ ਜੋ ਵਿਕਾਸਕਾਰਾਂ ਨੂੰ ਸੁਸ਼ਕਤ, ਸੁਰੱਖਿਅਤ ਅਤੇ ਜਿਮੇਵਾਰ ਤਰੀਕੇ ਨਾਲ ਏਆਈ ਨਾਲ ਨਵੀਂ ਸੋਚ ਚਲਾਉਣ ਅਤੇ ਭਵਿੱਖ ਦਾ ਰੂਪ ਦੇਣ ਲਈ ਸਸ਼ਕਤ ਕਰਦਾ ਹੈ।



[Microsoft Foundry](https://ai.azure.com) ਵਿਕਾਸਕਾਰਾਂ ਲਈ ਤਿਆਰ ਕੀਤਾ ਗਿਆ ਹੈ:

- ਇੱਕ ਐਂਟਰਪ੍ਰਾਈਜ਼-ਗ੍ਰੇਡ ਪਲੇਟਫਾਰਮ 'ਤੇ ਜਨਰੇਟਿਵ ਏਆਈ ਐਪਲੀਕੇਸ਼ਨ ਬਣਾਉਣਾ।
- ਜੁਮੇਵਾਰ ਏਆਈ ਅਮਲਾਂ ਵਿੱਚ ਅਧਾਰਿਤ, ਅਗੇਤਰ ਏਆਈ ਸੰਦਾਂ ਅਤੇ ਐੱਮਐਲ ਮਾਡਲਾਂ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਖੋਜ, ਬਣਾਉ, ਟੈਸਟ ਅਤੇ ਡਿਪਲੋਏ ਕਰਨਾ।
- ਐਪਲੀਕੇਸ਼ਨ ਵਿਕਾਸ ਦੇ ਪੂਰੇ ਜੀਵਨ ਚੱਕਰ ਲਈ ਟੀਮ ਦੇ ਨਾਲ ਸਹਿਕਾਰ ਕਰਨਾ।

ਮਾਈਕ੍ਰੋਸਾਫਟ ਫਾਉਂਡਰੀ ਨਾਲ, ਤੁਸੀਂ ਵੱਖ-ਵੱਖ ਮਾਡਲਾਂ, ਸੇਵਾਵਾਂ ਅਤੇ ਸਮਰੱਥਾਵਾਂ ਦੀ ਖੋਜ ਕਰ ਸਕਦੇ ਹੋ ਅਤੇ ਆਪਣੇ ਲਕੜਿਆਂ ਦੇ ਲਈ ਸਭ ਤੋਂ ਚੰਗੀਆਂ ਏਆਈ ਐਪਲੀਕੇਸ਼ਨਾਂ ਬਣਾਉਣ ਲਈ ਜਾ ਸਕਦੇ ਹੋ। ਮਾਈਕ੍ਰੋਸਾਫਟ ਫਾਉਂਡਰੀ ਪਲੇਟਫਾਰਮ ਸਪ੍ਰੂਤਾਂ ਨੂੰ ਪੂਰੇ ਪੈਮਾਨੇ 'ਤੇ ਉਤਪਾਦਨ ਵਾਲੀਆਂ ਐਪਲੀਕੇਸ਼ਨਾਂ ਵਿੱਚ ਬਦਲਣ ਲਈ ਸਕੇਲ ਕਰਨ ਵਿੱਚ ਸਹੂਲਤ ਦਿੰਦਾ ਹੈ। ਲੰਮੇ ਸਮੇਂ ਦੀ ਸਫਲਤਾ ਲਈ ਲਗਾਤਾਰ ਨਿਗਰਾਨੀ ਅਤੇ ਸੁਧਾਰ ਸਮਰਥਿਤ ਹਨ।

### **ਮਾਈਕ੍ਰੋਸਾਫਟ ਫਾਉਂਡਰੀ Phi-4 ਨੂੰ ਕਾਲ ਕਰਨ ਲਈ ਪਾਇਥਨ ਕੋਡ ਚਲਾ ਰਹੇ ਹਾਂ**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# ਜੇ ਭਾਸ਼ਣ ਚਾਲੂ ਹੈ ਤਾਂ ਭਾਸ਼ਣ ਨਤੀਜੇ ਸ਼ਾਮਲ ਕਰੋ
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
