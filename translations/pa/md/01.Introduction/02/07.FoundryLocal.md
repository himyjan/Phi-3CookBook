## ਫਾਉਂਡਰੀ ਲੋਕਲ ਵਿੱਚ ਫਾਈ-ਫੈਮਿਲੀ ਮਾਡਲਾਂ ਨਾਲ ਸ਼ੁਰੂਆਤ

### ਫਾਉਂਡਰੀ ਲੋਕਲ ਦਾ ਪਰਿਚਯ

ਫਾਉਂਡਰੀ ਲੋਕਲ ਇੱਕ ਸ਼ਕਤੀਸ਼ਾਲੀ ਔਨ-ਡਿਵਾਈਸ ਏਆਈ ਇੰਫਰੈਂਸ ਸੌਲਿਊਸ਼ਨ ਹੈ ਜੋ ਅਡਾਣੇ ਦਰਜੇ ਦੀ ਏਆਈ ਸਮਰੱਥਾ ਸਿੱਧੀ ਤੌਰ 'ਤੇ ਤੁਹਾਡੇ ਲੋਕਲ ਹਾਰਡਵੇਅਰ ਤੱਕ ਲਿਆਉਂਦਾ ਹੈ। ਇਹ ਟਿਊਟੋਰਿਯਲ ਤੁਹਾਨੂੰ ਫਾਉਂਡਰੀ ਲੋਕਲ ਨਾਲ ਫਾਈ-ਫੈਮਿਲੀ ਮਾਡਲਾਂ ਦੀ ਸੈੱਟਅਪ ਅਤੇ ਵਰਤੋਂ ਵਿੱਚ ਰਹਿਨੁਮਾ ਕਰੇਗਾ, ਜਿਸ ਨਾਲ ਤੁਸੀਂ ਆਪਣੀਆਂ ਏਆਈ ਵਰਕਲੋਡਾਂ ਉੱਤੇ ਪੂਰਾ ਨਿਯੰਤਰਣ ਪ੍ਰਾਪਤ ਕਰੋਂਗੇ ਅਤੇ ਗੋਪਨੀਯਤਾ ਬਣਾਈ ਰੱਖਦੇ ਹੋਏ ਲਾਗਤਾਂ ਘਟਾ ਸਕੋਂਗੇ।

ਫਾਉਂਡਰੀ ਲੋਕਲ ਤੁਹਾਡੇ ਡਿਵਾਈਸ 'ਤੇ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਏਆਈ ਮਾਡਲ ਚਲਾਉਣ ਦੁਆਰਾ ਪ੍ਰਦਰਸ਼ਨ, ਗੋਪਨੀਯਤਾ, ਕਸਟਮਾਈਜੇਸ਼ਨ ਅਤੇ ਲਾਗਤ ਵਿਚ ਫਾਇਦੇ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ। ਇਹ ਇੰਟੂਐਟਿਵ CLI, SDK ਅਤੇ REST API ਰਾਹੀਂ ਤੁਹਾਡੇ ਮੌਜੂਦਾ ਵਰਕਫਲੋ ਅਤੇ ਐਪਲੀਕੇਸ਼ਨਾਂ ਨਾਲ ਬਿਨਾਂ ਕਿਸੇ ਰੁਕਾਵਟ ਦੇ ਇੰਟੀਗ੍ਰੇਟ ਹੁੰਦਾ ਹੈ।


![arch](../../../../../translated_images/pa/foundry-local-arch.8823e321dd8258d7.webp)

### ਫਾਉਂਡਰੀ ਲੋਕਲ ਕਿਉਂ ਚੁਣੀਏ?

ਫਾਉਂਡਰੀ ਲੋਕਲ ਦੇ ਫਾਇਦੇ ਸਮਝ ਕੇ ਤੁਸੀਂ ਆਪਣੀ ਏਆਈ ਡਿਪਲੋਇਮੈਂਟ ਰਣਨੀਤੀ ਬਾਰੇ ਜਾਣੂ ਫੈਸਲੇ ਕਰ ਸਕੋਗੇ:

- **ਔਨ-ਡਿਵਾਈਸ ਇੰਫਰੈਂਸ:** ਮਾਡਲ ਆਪਣੀ ਵਰਤੋਂ ਵਾਲੇ ਹਾਰਡਵੇਅਰ 'ਤੇ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਚਲਾਓ, ਲਾਗਤਾਂ ਘਟਾਓ ਅਤੇ ਸਾਰਾ ਡਾਟਾ ਆਪਣੇ ਡਿਵਾਈਸ 'ਤੇ ਰੱਖੋ।

- **ਮਾਡਲ ਕਸਟਮਾਈਜ਼ੇਸ਼ਨ:** ਪੂਰਵ ਨਿਰਧਾਰਤ ਮਾਡਲਾਂ ਵਿੱਚੋਂ ਚੁਣੋ ਜਾਂ ਆਪਣਾ ਆਪਣੇ ਖਾਸ ਲੋੜਾਂ ਅਤੇ ਇਸਤੇਮਾਲਾਂ ਲਈ ਵਰਤੋਂ।

- **ਲਾਗਤ ਸਾਫਲਤਾ:** ਮੌਜੂਦਾ ਹਾਰਡਵੇਅਰ ਦੀ ਵਰਤੋਂ ਕਰਕੇ ਕਲਾਉਡ ਸੇਵਾਵਾਂ ਦੀ ਦੁਹਰਾਈ ਜਾਂਦੀ ਲਾਗਤਾਂ ਕੱਟੋ, ਜਿਸ ਨਾਲ ਏਆਈ ਹੋਰ ਪਹੁੰਚਯੋਗ ਬਣਦਾ ਹੈ।

- **ਸਰਲ ਇੰਟੀਗ੍ਰੇਸ਼ਨ:** ਐਸਡੀਕੇ, ਏਪੀਆਈ ਐਂਡਪੌਇੰਟਸ ਜਾਂ CLI ਰਾਹੀਂ ਆਪਣੇ ਐਪਲੀਕੇਸ਼ਨਾਂ ਨਾਲ ਜੁੜੋ, ਅਤੇ ਜੇ ਲੋੜਵੇਂ ਤਾਂ ਮਾਈਕਰੋਸਾਫਟ ਫਾਉਂਡਰੀ ਵੱਲ ਆਸਾਨੀ ਨਾਲ ਸਕੇਲ ਕਰੋ।

> **ਸ਼ੁਰੂਆਤ ਨੋਟ:** ਇਹ ਟਿਊਟੋਰਿਯਲ ਫਾਉਂਡਰੀ ਲੋਕਲ ਨੂੰ CLI ਅਤੇ SDK ਇੰਟਰਫੇਸਾਂ ਰਾਹੀਂ ਵਰਤਣ 'ਤੇ ਧਿਆਨ ਕੇਂਦਰਿਤ ਕਰਦਾ ਹੈ। ਤੁਸੀਂ ਦੋਹਾਂ ਤਰੀਕਿਆਂ ਨੂੰ ਸਿੱਖੋਗੇ ਤਾਂ ਜੋ ਆਪਣੀ ਵਰਤੋਂ ਲਈ ਸਭ ਤੋਂ ਵਧੀਆ ਤਰੀਕਾ ਚੁਣ ਸako।

## ਭਾਗ 1: ਫਾਉਂਡਰੀ ਲੋਕਲ CLI ਸੈੱਟਅਪ

### ਛੇਪ 1: ਇੰਸਟਾਲੇਸ਼ਨ

ਫਾਉਂਡਰੀ ਲੋਕਲ CLI ਤੁਹਾਡੇ ਮਾਡਲਾਂ ਨੂੰ ਸਥਾਨਕ ਤੌਰ 'ਤੇ ਚਲਾਉਣ ਅਤੇ ਪ੍ਰਬੰਧਿਤ ਕਰਨ ਦਾ ਦਰਵਾਜ਼ਾ ਹੈ। ਆਓ ਆਪਣੇ ਸਿਸਟਮ 'ਤੇ ਇਸਨੂੰ ਇੰਸਟਾਲ ਕਰੀਏ।

**ਸਮਰਥਿਤ ਪਲੇਟਫਾਰਮ:** Windows ਅਤੇ macOS

ਵਿਸਥਾਰ ਨਾਲ ਇੰਸਟਾਲੇਸ਼ਨ ਲਈ, ਕਿਰਪਾ ਕਰਕੇ [ਆਧਿਕਾਰਿਕ ਫਾਉਂਡਰੀ ਲੋਕਲ ਦਸਤਾਵੇਜ਼](https://github.com/microsoft/Foundry-Local/blob/main/README.md) ਪੜ੍ਹੋ।

### ਛੇਪ 2: ਉਪਲਬਧ ਮਾਡਲਾਂ ਦੀ ਖੋਜ

ਜਦੋਂ ਤੁਸੀਂ ਫਾਉਂਡਰੀ ਲੋਕਲ CLI ਇੰਸਟਾਲ ਕਰ ਲੈਂਦੇ ਹੋ, ਤਾਂ ਤੁਸੀਂ ਦੇਖ ਸਕਦੇ ਹੋ ਕਿ ਤੁਹਾਡੇ ਵਰਤੋਂ ਮਾਮਲੇ ਲਈ ਕਿਹੜੇ ਮਾਡਲ ਉਪਲਬਧ ਹਨ। ਇਹ ਕਮਾਂਡ ਤੁਹਾਨੂੰ ਸਾਰੇ ਸਮਰਥਿਤ ਮਾਡਲ ਦਿਖਾਏਗੀ:


```bash
foundry model list
```

### ਛੇਪ 3: ਫਾਈ ਪਰਿਵਾਰਕ ਮਾਡਲਾਂ ਦੀ ਸਮਝ

ਫਾਈ ਪਰਿਵਾਰ ਵੱਖ-ਵੱਖ ਵਰਤੋਂ ਮਾਮਲਿਆਂ ਅਤੇ ਹਾਰਡਵੇਅਰ ਸੰਰਚਨਾਵਾਂ ਲਈ ਅਨੁਕੂਲਿਤ ਕਈ ਮਾਡਲ ਪੇਸ਼ ਕਰਦਾ ਹੈ। ਫਾਉਂਡਰੀ ਲੋਕਲ ਵਿੱਚ ਉਪਲਬਧ ਫਾਈ ਮਾਡਲਾਂ ਹਨ:

**ਉਪਲਬਧ ਫਾਈ ਮਾਡਲ:**

- **phi-3.5-mini** - ਬੇਸਿਕ ਕਾਰਜਾਂ ਲਈ ਸੰਕੁਚਿਤ ਮਾਡਲ
- **phi-3-mini-128k** - ਲੰਬੀਆਂ ਗੱਲਬਾਤਾਂ ਲਈ ਵਧੀਆ ਸੰਦਰਭ ਸੰਜੋਖਿਤ ਵਰਜਨ
- **phi-3-mini-4k** - ਆਮ ਵਰਤੋਂ ਲਈ ਮਿਆਰੀ ਸੰਦਰਭ ਮਾਡਲ
- **phi-4** - ਸੁਧਾਰਿਆ ਗਿਆ ਸਮਰੱਥਾ ਵਾਲਾ ਅਗਿਆਤ ਮਾਡਲ
- **phi-4-mini** - phi-4 ਦਾ ਹਲਕਾ ਸੰਸਕਰਨ
- **phi-4-mini-reasoning** - ਕੁਠਰੀਚ ਸੂਚਨਾ ਨੂੰ ਹੱਲ ਕਰਨ ਦੇ ਲਈ ਖਾਸ ਮਾਡਲ

> **ਹਾਰਡਵੇਅਰ ਅਨੁਕੂਲਤਾ:** ਹਰ ਮਾਡਲ ਨੂੰ ਤੁਹਾਡੇ ਸਿਸਟਮ ਦੀ ਸਮਰੱਥਾ ਮੁਤਾਬਕ ਵੱਖ-ਵੱਖ ਹਾਰਡਵੇਅਰ ਤਹਿਲਾ (CPU, GPU) ਲਈ ਵਿਵਸਥਿਤ ਕੀਤਾ ਜਾ ਸਕਦਾ ਹੈ।

### ਛੇਪ 4: ਆਪਣਾ ਪਹਿਲਾ ਫਾਈ ਮਾਡਲ ਚਲਾਉਣਾ

ਆਓ ਇੱਕ ਵਿਹਾਰਕ ਉਦਾਹਰਨ ਨਾਲ ਸ਼ੁਰੂਆਤ ਕਰੀਏ। ਅਸੀਂ `phi-4-mini-reasoning` ਮਾਡਲ ਚਲਾਵਾਂਗੇ, ਜੋ ਮੌੜਦਾਰ ਮੁਸ਼ਕਲਾਂ ਨੂੰ ਕਦਮ-ਦਰ-কਦਮ ਹੱਲ ਕਰਨ ਵਿੱਚ ਮਹਿਰਤ ਰੱਖਦਾ ਹੈ।


**ਮਾਡਲ ਚਲਾਉਣ ਲਈ ਕਮਾਂਡ:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **ਪਹਿਲੀ ਵਾਰ ਸੈੱਟਅਪ:** ਜਦੋਂ ਤੁਸੀਂ ਪਹਿਲੀ ਵਾਰ ਕੋਈ ਮਾਡਲ ਚਲਾਉਂਦੇ ਹੋ, ਫਾਉਂਡਰੀ ਲੋਕਲ ਉਹ ਮਾਡਲ ਆਪਣੇ ਆਪ ਤੁਹਾਡੇ ਸਥਾਨਕ ਡਿਵਾਈਸ 'ਤੇ ਡਾਊਨਲੋਡ ਕਰ ਲੈਂਦਾ ਹੈ। ਡਾਊਨਲੋਡ ਸਮਾਂ ਤੁਹਾਡੇ ਨੈਟਵਰਕ ਦੀ ਗਤੀ ਤੇ ਨਿਰਭਰ ਕਰਦਾ ਹੈ, ਇਸ ਲਈ ਪ੍ਰਾਰੰਭਿਕ ਸੈੱਟਅਪ ਦੌਰਾਨ ਕਿਰਪਾ ਕਰਕੇ ਧੈਰਜ ਧਰੋ।

### ਛੇਪ 5: ਅਸਲੀ ਸਮੱਸਿਆ ਨਾਲ ਮਾਡਲ ਦੀ ਜਾਂਚ

ਹੁਣ ਅਸੀਂ ਆਪਣੇ ਮਾਡਲ ਨੂੰ ਇੱਕ ਮਸ਼ਹੂਰ ਤਰਕ ਸਮੱਸਿਆ ਨਾਲ ਪਰਖਦੇ ਹਾਂ ਕਿ ਇਹ ਕਦਮ-ਦਰ-कਦਮ ਤਰਕਸ਼ੀਲ ਹੱਲ ਕਿਵੇਂ ਕਰਦਾ ਹੈ:

**ਉਦਾਹਰਨ ਸਮੱਸਿਆ:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**ਉਮੀਦ ਕੀਤੀ ਗਈ ਵਰਤੋਂ:** ਮਾਡਲ ਨੂੰ ਇਸ ਸਮੱਸਿਆ ਨੂੰ ਤਰਕ ਸੂਚਕ ਕਦਮਾਂ ਵਿੱਚ ਵੰਡਣਾ ਚਾਹੀਦਾ ਹੈ, ਜਿਸ ਵਿੱਚ ਇਹ ਜਾਣਨਾ ਜਰੂਰੀ ਹੈ ਕਿ ਤਿਤਲੀ ਦੇ 2 ਪੈਰ ਹੁੰਦੇ ਹਨ ਅਤੇ ਖਰਗੋਸ਼ ਦੇ 4 ਪੈਰ, ਤਾਂ ਜੋ ਸਮੀਕਰਨਾਂ ਦਾ ਹੱਲ ਕੀਤਾ ਜਾ ਸਕੇ।

## ਭਾਗ 2: ਫਾਉਂਡਰੀ ਲੋਕਲ SDK ਨਾਲ ਐਪਲੀਕੇਸ਼ਨਾਂ ਬਣਾਉਣਾ

### SDK ਵਰਤਣ ਦਾ ਕਾਰਣ

ਜਿਵੇਂ ਕਿ CLI ਟੈਸਟ ਕਰਨ ਅਤੇ ਤੇਜ਼ ਇੰਟਰਐਕਸ਼ਨਾਂ ਲਈ ਉੱਤਮ ਹੈ, SDK ਤੁਹਾਨੂੰ ਫਾਉਂਡਰੀ ਲੋਕਲ ਨੂੰ ਆਪਣੇ ਐਪਲੀਕੇਸ਼ਨਾਂ ਵਿੱਚ ਪ੍ਰੋਗ੍ਰਾਮਮੈਟਿਕ ਤੌਰ 'ਤੇ ਸ਼ਾਮਿਲ ਕਰਨ ਦੀ ਸਮਰੱਥਾ ਦਿੰਦਾ ਹੈ। ਇਹ ਖੁਲ੍ਹੇ ਦਰਵਾਜੇ ਖੋਲ੍ਹਦਾ ਹੈ:

- ਕਸਟਮ ਏਆਈ-ਚਲਿਤ ਐਪਲੀਕੇਸ਼ਨਾਂ ਦੀ ਬਣਤਰ
- ਆਟੋਮੇਟਿਕ ਵਰਕਫਲੋਜ਼ ਬਣਾਉਣਾ
- ਮੌਜੂਦਾ ਸਿਸਟਮਾਂ ਵਿੱਚ ਏਆਈ ਸਮਰੱਥਾਵਾਂ ਦੀ ਸ਼ਾਮਿਲਤਾ
- ਚੈਟਬੋਟ ਅਤੇ ਇੰਟਰਐਕਟਿਵ ਟੂਲ ਵਿਕਾਸ

### ਸਮਰਥਿਤ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ

ਫਾਉਂਡਰੀ ਲੋਕਲ ਤੁਹਾਡੇ ਵਿਕਾਸ ਪ੍ਰਫਰੰਸਾਂ ਲਈ ਕਈ ਪ੍ਰੋਗ੍ਰਾਮਿੰਗ ਭਾਸ਼ਾਵਾਂ ਵਿੱਚ SDK ਸਹਾਇਤਾ ਪ੍ਰਦਾਨ ਕਰਦਾ ਹੈ:

**📦 ਉਪਲਬਧ SDK:**

- **C# (.NET):** [SDK ਦਸਤਾਵੇਜ਼ ਅਤੇ ਉਦਾਹਰਨ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK ਦਸਤਾਵੇਜ਼ ਅਤੇ ਉਦਾਹਰਨ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK ਦਸਤਾਵੇਜ਼ ਅਤੇ ਉਦਾਹਰਨ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK ਦਸਤਾਵੇਜ਼ ਅਤੇ ਉਦਾਹਰਨ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### ਅੱਗੇ ਦੇ ਕਦਮ

1. ਆਪਣੀ ਵਿਕਾਸ ਮਾਹੌਲ ਅਨੁਸਾਰ ਪਸੰਦੀਦਾ SDK ਚੁਣੋ  
2. ਵਿਸਥਾਰਿਤ ਲਾਗੂ ਕਰਨ ਲਈ SDK-ਖਾਸ ਦਸਤਾਵੇਜ਼ਾਂ ਦੀ ਪਾਲਣਾ ਕਰੋ  
3. ਜਟਿਲ ਐਪਲੀਕੇਸ਼ਨਾਂ ਦੇ ਨਿਰਮਾਣ ਤੋਂ ਪਹਿਲਾਂ ਸਾਦੇ ਉਦਾਹਰਨਾਂ ਨਾਲ ਸ਼ੁਰੂ ਕਰੋ  
4. ਹਰ SDK ਰਿਪੋਜ਼ਟਰੀ ਵਿੱਚ ਦਿੱਤਾ ਗਿਆ ਨਮੂਨਾ ਕੋਡ ਖੋਜੋ
