# Microsoft Foundry ನಲ್ಲಿ Prompt flow ಜೊತೆಗೆ ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಗಳನ್ನು ಫೈನ್-ಟ್ಯೂನ್ ಮಾಡಿ ಮತ್ತು ಏಕರೂಪಗೊಳಿಸಿ

ಈ ಎಂಡ್-ಟು-ಎಂಡ್ (E2E) ಮಾದರಿ "[Microsoft Foundry ನಲ್ಲಿ Prompt Flow ಜೊತೆಗೆ ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಗಳನ್ನು ಫೈನ್-ಟ್ಯೂನ್ ಹಾಗೂ ಏಕರೂಪಗೊಳಿಸುವುದು](https://techcommunity.microsoft.com/t5/educator-developer-blog/fine-tune-and-integrate-custom-phi-3-models-with-prompt-flow-in/ba-p/4191726?WT.mc_id=aiml-137032-kinfeylo)" ಎಂಬ , ಮೈಕ್ರೋಸಾಫ್ಟ್ ಟೆಕ್ ಕಾಮ್ಯುನಿಟಿಯಿಂದ ಪಡೆದ ಮಾರ್ಗದರ್ಶಿಕೆಯ ಆಧಾರದ ಮೇಲೆ ಬರೆದಿದೆ. ಇದರಲ್ಲಿ Phi-3 ಮಾದರಿಗಳನ್ನು ಫೈನ್-ಟ್ಯೂನ್ ಮಾಡುವಿಕೆ, ನಿಯೋಜಿಸುವಿಕೆ ಮತ್ತು Microsoft Foundry ಯಲ್ಲಿ Prompt flow ಜೊತೆ ಏಕರೂಪಗೊಳಿಸುವಿಕೆ ಪ್ರಕ್ರಿಯೆಗಳ ಪರಿಚಯ ನೀಡಲಾಗಿದೆ. ಸ್ಥಳೀಯವಾಗಿ ಕೋಡ್ ಅನ್ನು ಓಡಿಸುವ E2E ಮಾದರಿ "[Prompt Flow ಜೊತೆ ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಗಳನ್ನು ಫೈನ್-ಟ್ಯೂನ್ ಮಾಡಿ ಮತ್ತು ಏಕರೂಪಗೊಳಿಸುವುದು](./E2E_Phi-3-FineTuning_PromptFlow_Integration.md)" ನಿಂದ ವಿಭಿನ್ನವಾಗಿ, ಈ ಟ್ಯೂಟೋರಿಯಲ್ ಸಂಪೂರ್ಣವಾಗಿ Azure AI / ML ಸ್ಟುಡಿಯೋದಲ್ಲಿ ನಿಮ್ಮ ಮಾದರಿಯನ್ನು ಫೈನ್-ಟ್ಯೂನ್ ಮತ್ತು ಏಕರೂಪಗೊಳಿಸುವಿಕೆ ಮೇಲೆಯೇ ಕೇಂದ್ರೀಕರಿಸಿದೆ.

## ಅವಲೋಕನ

ಈ E2E ಮಾದರಿಯಲ್ಲಿ, ನೀವು Phi-3 ಮಾದರಿಯನ್ನು ಫೈನ್-ಟ್ಯೂನ್ ಮಾಡುವುದು ಮತ್ತು Microsoft Foundry ನಲ್ಲಿ Prompt flow ಜೊತೆಗೆ ಏಕರೂಪಗೊಳಿಸುವಿಕೆಯನ್ನು ಕಲಿಯುತ್ತೀರಿ. Azure AI / ML ಸ್ಟುಡಿಯೋಕನ್ನು ಬಳಸಿಕೊಂಡು, ಕಸ್ಟಮ್ AI ಮಾದರಿಗಳನ್ನು ನಿಯೋಜಿಸಿ ಬಳಸಲು ಕಾರ್ಯಪದರವನ್ನು ಸ್ಥಾಪಿಸುತ್ತೀರಿ. ಈ E2E ಮಾದರಿ ಮೂರು ಕೇಸ್‌ಗಳಾಗಿ ವಿಭಾಗಿಸಲಾಗಿದೆ:

**ಕೇಸ್ 1: Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ಸ್ಥಾಪಿಸಿ ಮತ್ತು ಫೈನ್-ಟ್ಯೂನಿಂಗ್ ಸಿದ್ಧತೆಮಾಡಿ**

**ಕೇಸ್ 2: Phi-3 ಮಾದರಿಯನ್ನು ಫೈನ್-ಟ್ಯೂನಿಂಗ್ ಮಾಡಿ ಮತ್ತು Azure Machine Learning Studio ಯಲ್ಲಿ ನಿಯೋಜಿಸಿ**

**ಕೇಸ್ 3: Prompt flow ಜೊತೆ ಏಕರೂಪಗೊಳಿಸಿ ಮತ್ತು Microsoft Foundry ಯಲ್ಲಿ ನಿಮ್ಮ ಕಸ್ಟಮ್ ಮಾದರಿಯಿಂದ ಚಾಟ್ ಮಾಡಿ**

## ಕೇಸ್ 1: Azure ಸಂಪನ್ಮೂಲಗಳನ್ನು ಸ್ಥಾಪಿಸಿ ಮತ್ತು ಫೈನ್-ಟ್ಯೂನಿಂಗ್ ಸಿದ್ಧತೆಮಾಡಿ

### Azure Machine Learning ಕಾರ್ಯಸ್ಥಳವನ್ನು ಸೃಷ್ಟಿ ಮಾಡುವುದು

1. ಪೋರ್ಟಲ್ ಪುಟದ ಮೇಲ್ಭಾಗದಲ್ಲಿರುವ **ಸರ್ಚ್ ಬಾರ್** ನಲ್ಲಿ *azure machine learning* ಎಂದು ಟೈಪ ಮಾಡಿ ಮತ್ತು ತೋರಿಸಿದ ಆಯ್ಕೆಯಲ್ಲಿಂದ **Azure Machine Learning** ಆಯ್ಕೆಮಾಡಿ.

    ![Type azure machine learning.](../../../../../../translated_images/kn/01-01-type-azml.acae6c5455e67b4b.webp)

2. ನ್ಯಾವಿಗೇಶನ್ ಮೆನುದಿಂದ **+ Create** ಆಯ್ಕೆಮಾಡಿ.

3. ನ್ಯಾವಿಗೇಶನ್ ಮೆನುದಿಂದ **New workspace** ಆಯ್ಕೆಮಾಡಿ.

    ![Select new workspace.](../../../../../../translated_images/kn/01-02-select-new-workspace.cd09cd0ec4a60ef2.webp)

4. ಕೆಳಗಿನ ಕಾರ್ಯಗಳನ್ನು ಮಾಡಿ:

    - ನಿಮ್ಮ Azure **Subscription** ಆಯ್ಕೆಮಾಡಿ.
    - ಬಳಕೆ ಮಾಡಲು **Resource group** ಆಯ್ಕೆಮಾಡಿ (ಬೇಕಿದ್ದಲ್ಲಿ ಹೊಸದಾಗಿ ಸೃಷ್ಟಿಸಿ).
    - **Workspace Name** ನಮೂದಿಸಿ. ಇದು ವಿಶಿಷ್ಟ ಮೌಲ್ಯವಾಗಿರಬೇಕು.
    - ನೀವು ಬಳಸಲು ಬಯಸುವ **Region** ಆಯ್ಕೆಮಾಡಿ.
    - ಬಳಸುವ **Storage account** ಆಯ್ಕೆಮಾಡಿ (ಹೊಸದಾಗಿ ಸೃಷ್ಟಿಸಬಹುದಾಗಿದೆ).
    - ಬಳಸುವ **Key vault** ಆಯ್ಕೆಮಾಡಿ (ಹೊಸದಾಗಿ ಸೃಷ್ಟಿಸಬಹುದಾಗಿದೆ).
    - ಬಳಸಲು ಬಯಸುವ **Application insights** ಆಯ್ಕೆಮಾಡಿ (ಹೊಸದಾಗಿ ಸೃಷ್ಟಿಸಬಹುದಾಗಿದೆ).
    - ಬಳಸಿವ **Container registry** ಆಯ್ಕೆಮಾಡಿ (ಹೊಸದಾಗಿ ಸೃಷ್ಟಿಸಬಹುದಾಗಿದೆ).

    ![Fill azure machine learning.](../../../../../../translated_images/kn/01-03-fill-AZML.a1b6fd944be0090f.webp)

5. **Review + Create** ಆಯ್ಕೆಮಾಡಿ.

6. **Create** ಆಯ್ಕೆಮಾಡಿ.

### ಪಾತ್ರ ನಿಯೋಜನೆ ಸೇರಿಸುವುದು

ನಿಮ್ಮ ಮಾದರಿಗಳನ್ನು ಫೈನ್-ಟ್ಯೂನ್ ಮತ್ತು ನಿಯೋಜಿಸಲು, ಮೊದಲು ನೀವು User Assigned Managed Identity (UAI) ಅನ್ನು ಸೃಷ್ಟಿಸಿ ಅದಕ್ಕೆ ಸರಿಯಾದ ಅನುಮತಿಗಳನ್ನು ಆದೇಶಿಸಬೇಕು. ನಿಯೋಜಿಸುವಿಕೆ دوران ಈ UAI ನಮೂದಿಸುವಿಕೆಗಾಗಿ ಬಳಸಲಾಗುತ್ತದೆ.

#### User Assigned Managed Identity (UAI) ಅನ್ನು ಸೃಷ್ಟಿ ಮಾಡುವುದು

1. ಪೋರ್ಟಲ್ ಪುಟದ ಮೇಲ್ಭಾಗದಲ್ಲಿ **managed identities** ಎಂದು ಟೈಪ್ ಮಾಡಿ ಮತ್ತು ತೋರಿಸಿದ ಆಯ್ಕೆಯಲ್ಲಿಂದ **Managed Identities** ಆಯ್ಕೆಮಾಡಿ.

    ![Type managed identities.](../../../../../../translated_images/kn/03-01-type-managed-identities.24de763e0f1f37e5.webp)

1. **+ Create** ಆಯ್ಕೆಮಾಡಿ.

    ![Select create.](../../../../../../translated_images/kn/03-02-select-create.92bf8989a5cd98f2.webp)

1. ಕೆಳಗಿನ ಕಾರ್ಯಗಳನ್ನು ಮಾಡಿ:

    - ನಿಮ್ಮ Azure **Subscription** ಆಯ್ಕೆಮಾಡಿ.

#### *download_dataset.py* ಬಳಸಿ ನಿಮ್ಮ ಡೇಟಾಸೆಟ್‌ ಅನ್ನು ಡೌನ್ಲೋಡ್ ಮಾಡಿಕೊಳ್ಳಿ

1. ಕೆಳಗಿನ ಕೋಡ್ ಅನ್ನು *download_dataset.py* ಫೈಲ್‌ಗೆ ಸೇರಿಸಿ.

    ```python
    import json
    import os
    from datasets import load_dataset

    def load_and_split_dataset(dataset_name, config_name, split_ratio):
        """
        Load and split a dataset.
        """
        # ನಿರ್ದಿಷ್ಟ ಹೆಸರು, ಸಂರಚನೆ ಮತ್ತು ವಿಭಾಜನೆ ಅನುಪಾತದೊಂದಿಗೆ ಡೇಟಾಸೆಟ್ ಅನ್ನು ಲೋಡ್ ಮಾಡಿ
        dataset = load_dataset(dataset_name, config_name, split=split_ratio)
        print(f"Original dataset size: {len(dataset)}")
        
        # ಡೇಟಾಸೆಟ್ ಅನ್ನು ತರಬೇತಿ ಮತ್ತು ಪರೀಕ್ಷಾ ಸೆಟ್‌ಗಳಾಗಿ ವಿಭಜಿಸಿ (80% ತರಬೇತಿ, 20% ಪರೀಕ್ಷೆ)
        split_dataset = dataset.train_test_split(test_size=0.2)
        print(f"Train dataset size: {len(split_dataset['train'])}")
        print(f"Test dataset size: {len(split_dataset['test'])}")
        
        return split_dataset

    def save_dataset_to_jsonl(dataset, filepath):
        """
        Save a dataset to a JSONL file.
        """
        # ಡೈರೆಕ್ಟರಿ ಇದ್ದಿಲ್ಲದಿದ್ದರೆ ಸೃಷ್ಟಿಸಿ
        os.makedirs(os.path.dirname(filepath), exist_ok=True)
        
        # ಫೈಲನ್ನು ಬರವಣಿಗೆ ಮೋಡ್‌ನಲ್ಲಿ ತೆರೆದುಕೊಳ್ಳಿ
        with open(filepath, 'w', encoding='utf-8') as f:
            # ಡೇಟಾಸೆಟ್‌ನ ಪ್ರತಿ ದಾಖಲೆ ಮೇಲೆ ಪುನರಾವರ್ತಿಸಿ
            for record in dataset:
                # ದಾಖಲೆ ಅನ್ನು JSON ವಸ್ತುವಾಗಿ ಡಂಪ್ ಮಾಡಿ ಮತ್ತು ಅದನ್ನು ಫೈಲಿಗೆ ಬರೆಯಿರಿ
                json.dump(record, f)
                # ದಾಖಲಗಳನ್ನು ವಿಭಜಿಸಲು ಹೊಸ ಸಾಲಿನ ಅಕ್ಷರವನ್ನು ಬರೆಯಿರಿ
                f.write('\n')
        
        print(f"Dataset saved to {filepath}")

    def main():
        """
        Main function to load, split, and save the dataset.
        """
        # ವಿಶಿಷ್ಟ ಸಂರಚನೆ ಮತ್ತು ವಿಭಜನೆ ಅನುಪಾತದೊಂದಿಗೆ ULTRACHAT_200k ಡೇಟಾಸೆಟ್ ಅನ್ನು ಲೋಡ್ ಮತ್ತು ವಿಭಜಿಸಿ
        dataset = load_and_split_dataset("HuggingFaceH4/ultrachat_200k", 'default', 'train_sft[:1%]')
        
        # ವಿಭಜನೆಯಿಂದ ತರಬೇತಿ ಮತ್ತು ಪರೀಕ್ಷಾ ಡೇಟಾಸೆಟ್‌ಗಳನ್ನು ತೆಗೆದುಕೊಳ್ಳಿ
        train_dataset = dataset['train']
        test_dataset = dataset['test']

        # ತರಬೇತಿ ಡೇಟಾಸೆಟ್ ಅನ್ನು JSONL ಫೈಲಿಗೆ ಉಳಿಸಿ

        # ಪರೀಕ್ಷಾ ಡೇಟಾಸೆಟ್ ಅನ್ನು ಬೇರೆ JSONL ಫೈಲಿಗೆ ಉಳಿಸಿ
        save_dataset_to_jsonl(test_dataset, "data/test_data.jsonl")

    if __name__ == "__main__":
        main()

    ```

## ಘಟನೆ 3: ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋ ಜೊತೆ ಏಕೀಕೃತಗೊಳ್ಳಿ ಮತ್ತು ನಿಮ್ಮ ಕಸ್ಟಮ್ ಮಾದರితో ಮೈಕ್ರೋಸಾಫ್ಟ್ ಫೌಂಡ್ರಿಯಲ್ಲಿಚಾಟ್ ಮಾಡಿ

### ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಯನ್ನು ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋ ಜೊತೆ ಏಕೀಕೃತಗೊಳಿಸಿ

ನೀವು ಯಶಸ್ವಿಯಾಗಿ ಫೈನ್-ಟ್ಯೂನ್ ಮಾಡಿದ ಮಾದರಿಯನ್ನು ನಿಯೋಜಿಸಿದ ನಂತರ, ಈಗ ಅದನ್ನು ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋ ಜೊತೆ ಏಕೀಕೃತಗೊಳಿಸಿ ನಿಮ್ಮ ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಯನ್ನು ನೇರ ಕಾಲ ಬಳಕೆಯ ಅಪ್ಲಿಕೇಶನ್‌ಗಳಲ್ಲಿ ಬಳಸಬಹುದು, ಬಹುಮಾನಕರ ಪರಸ್ಪರ ಕ್ರಿಯಾತ್ಮಕ ಕಾರ್ಯಗಳನ್ನು ಸಾಧ್ಯವಾಗಿಸುತ್ತದೆ.

ಈ ಅಭ್ಯಾಸದಲ್ಲಿ, ನೀವು:

- ಮೈಕ್ರೋಸಾಫ್ಟ್ ಫೌಂಡ್ರಿ ಹಬ್ ರಚಿಸಿ.
- ಮೈಕ್ರೋಸಾಫ್ಟ್ ಫೌಂಡ್ರಿ ಪ್ರಾಜೆಕ್ಟ್ ರಚಿಸಿ.
- ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋ ರಚಿಸಿ.
- ಫೈನ್-ಟ್ಯೂನ್ ಮಾಡಿದ Phi-3 ಮಾದರಿಗಾಗಿ ಕಸ್ಟಮ್ ಸಂಪರ್ಕ ಸೇರಿಸಿ.
- ನಿಮ್ಮ ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಯೊಂದಿಗೆ ಚಾಟ್ ಮಾಡಲು ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋ ಕಾನ್ಫಿಗರ್ ಮಾಡಿ.

#### ನಿಮ್ಮ ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಯೊಂದಿಗೆ ಚಾಟ್ ಮಾಡಲು ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋ ಸೆಟ್ ಮಾಡಿ

ನೀವು ಸೂಕ್ಷ್ಮತೆಗೊಂಡ Phi-3 ಮಾದರಿಯನ್ನು ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋನಲ್ಲಿ ಸಂಯೋಜಿಸಬೇಕಾಗಿದೆ. ಆದರೆ ಹಳೆಯ ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋ ಇದೀಗ ಇದರಿಗಾಗಿ ವಿನ್ಯಾಸಗೊಳಿಸಲಾಗಿಲ್ಲ. ಆದ್ದರಿಂದ, ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋವನ್ನು ಮತ್ತೆ ವಿನ್ಯಾಸಗೊಳಿಸಿ ಕಸ್ಟಮ್ ಮಾದರಿಯನ್ನು ಸಂಯೋಜಿಸಲು ಸಿದ್ಧಪಡಿಸಬೇಕು.

1. ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋನಲ್ಲಿ, ಹಳೆಯ ಫ್ಲೋವನ್ನು ಪುನರ್ ನಿರ್ಮಿಸಲು ಕೆಳಗಿನ ಕೆಲಸಗಳನ್ನು ನಿಭಾಯಿಸಿ:

    - **ಕಚ್ಚಾ ಫೈಲ್ ಮೋಡ್** ಆಯ್ಕೆಮಾಡಿ.
    - *flow.dag.yml* ಫೈಲ್‌ನ ಎಲ್ಲಾ ಹಳೆಯ ಕೋಡ್ ಅನ್ನು ಅಳಿಸಿ.
    - ಕೆಳಗಿನ ಕೋಡ್ ಅನ್ನು *flow.dag.yml* ಫೈಲ್‌ಗೆ ಸೇರಿಸಿ.

        ```yml
        inputs:
          input_data:
            type: string
            default: "Who founded Microsoft?"

        outputs:
          answer:
            type: string
            reference: ${integrate_with_promptflow.output}

        nodes:
        - name: integrate_with_promptflow
          type: python
          source:
            type: code
            path: integrate_with_promptflow.py
          inputs:
            input_data: ${inputs.input_data}
        ```

    - **ಉಳಿಸಿ** ಆಯ್ಕೆಮಾಡಿ.

    ![Select raw file mode.](../../../../../../translated_images/kn/08-15-select-raw-file-mode.61d988b41df28985.webp)

1. ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋನಲ್ಲಿ ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಯನ್ನು ಬಳಸಲು *integrate_with_promptflow.py* ಫೈಲ್‌ಗೆ ಕೆಳಗಿನ ಕೋಡ್ ಅನ್ನು ಸೇರಿಸಿ.

    ```python
    import logging
    import requests
    from promptflow import tool
    from promptflow.connections import CustomConnection

    # ಲಾಗಿಂಗ್ ಸ್ಥಾಪನೆ
    logging.basicConfig(
        format="%(asctime)s - %(levelname)s - %(name)s - %(message)s",
        datefmt="%Y-%m-%d %H:%M:%S",
        level=logging.DEBUG
    )
    logger = logging.getLogger(__name__)

    def query_phi3_model(input_data: str, connection: CustomConnection) -> str:
        """
        Send a request to the Phi-3 model endpoint with the given input data using Custom Connection.
        """

        # "ಕನೆಕ್ಷನ್" ಕಸ್ಟಮ್ ಕನೆಕ್ಷನ್‌ನ ಹೆಸರು, "ಎಂಡ್‌ಪಾಯಿಂಟ್", "ಕೀ"ಗಳು ಕಸ್ಟಮ್ ಕನೆಕ್ಷನ್‌ನ ಕೀಗಳು
        endpoint_url = connection.endpoint
        api_key = connection.key

        headers = {
            "Content-Type": "application/json",
            "Authorization": f"Bearer {api_key}"
        }
        data = {
            "input_data": {
                "input_string": [
                    {"role": "user", "content": input_data}
                ],
                "parameters": {
                    "temperature": 0.7,
                    "max_new_tokens": 128
                }
            }
        }
        try:
            response = requests.post(endpoint_url, json=data, headers=headers)
            response.raise_for_status()
            
            # ಪೂರ್ಣ JSON ಪ್ರತಿಕ್ರಿಯೆಯನ್ನು ಲಾಗ್ ಮಾಡಿ
            logger.debug(f"Full JSON response: {response.json()}")

            result = response.json()["output"]
            logger.info("Successfully received response from Azure ML Endpoint.")
            return result
        except requests.exceptions.RequestException as e:
            logger.error(f"Error querying Azure ML Endpoint: {e}")
            raise

    @tool
    def my_python_tool(input_data: str, connection: CustomConnection) -> str:
        """
        Tool function to process input data and query the Phi-3 model.
        """
        return query_phi3_model(input_data, connection)

    ```

### ನಿಮ್ಮ ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಯೊಂದಿಗೆ ಚಾಟ್ ಮಾಡಿ

ನೀವು ಈಗ ಸೂಕ್ಷ್ಮತೆಮಾಡಿ ಮತ್ತು ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋನಲ್ಲಿ ನಿಮ್ಮ ಕಸ್ಟಮ್ Phi-3 ಮಾದರಿಯನ್ನು ಸಂಯೋಜಿಸಿದ್ದೀರಿ, ಇದರಿಂದ ನೀವು ಅದರೊಂದಿಗೆ ಸಂವಹನ ಪ್ರಾರಂಭಿಸಲು ಸಿದ್ಧರಿದ್ದೀರಿ. ಈ ಅಭ್ಯಾಸವು ನಿಮ್ಮ ಮಾದರಿಯೊಂದಿಗೆ ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋ ಬಳಸಿ ಚಾಟ್ ಅನ್ನು ಹೇಗೆ ಸ್ಥಾಪಿಸುವುದು ಮತ್ತು ಪ್ರಾರಂಭಿಸುವುದು ಎಂಬುದನ್ನು ಮಾರ್ಗದರ್ಶನ ಮಾಡುತ್ತದೆ. ಈ ಹೆಜ್ಜೆಗಳನ್ನು ಅನುಸರಿಸುವ ಮೂಲಕ, ನೀವು ವಿವಿಧ ಕಾರ್ಯಗಳು ಮತ್ತು ಸಂವಾದಗಳಿಗೆ ನಿಮ್ಮ ಸೂಕ್ಷ್ಮತೆಗೊಂಡ Phi-3 ಮಾದರಿಯ ಶಕ್ತಿಗಳನ್ನು ಸಂಪೂರ್ಣವಾಗಿ ಬಳಸಬಹುದು.
