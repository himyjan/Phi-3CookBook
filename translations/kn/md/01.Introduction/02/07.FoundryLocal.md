## Foundry Local ನಲ್ಲಿ Phi-ಪರಿವಾರ ಮಾದರಿಗಳೊಂದಿಗೆ ಪ್ರಾರಂಭಿಸುವುದು

### Foundry Local ಪರಿಚಯ

Foundry Local ನಂತೆ ಆನ್-ಡಿವೈಸ್ AI ಇನ್ಫರೆನ್ಸ್ ಪರಿಹಾರವು ಎಂಟರ್ಪ್ರೈಸ್-ಗ್ರೇಡ್ AI ಸಾಮರ್ಥ್ಯಗಳನ್ನು ನೇರವಾಗಿ ನಿಮ್ಮ ಸ್ಥಳೀಯ ಹಾರ್ಡ್‌ವೇರ್‌ಗೆ ತರುತ್ತದೆ. ಈ ಟ್ಯೂಟೋರಿಯಲ್ ನಿಮಗೆ Foundry Local ಸಹಿತ Phi-ಪರಿವಾರ ಮಾದರಿಗಳನ್ನು ಸೆಟ್ ಅಪ್ ಮಾಡಿ ಬಳಸುವ ಮೂಲಕ ನಿಮ್ಮ AI ಕೆಲಸಗಳನ್ನು ಸಂಪೂರ್ಣ ನಿಯಂತ್ರಣದಲ್ಲಿಟ್ಟುಕೊಳ್ಳುವುದರೊಂದಿಗೆ ಗೌಪ್ಯತೆಯನ್ನು ಕಾಯ್ದುಕೊಳ್ಳಲು ಮತ್ತು ವೆಚ್ಚಗಳನ್ನು ಕಡಿಮೆ ಮಾಡಲು ಮಾರ್ಗದರ್ಶನ ನೀಡುತ್ತದೆ.

Foundry Local ನಿಮ್ಮ ಸಾಧನದ ಮೇಲೆ ಸ್ಥಳೀಯವಾಗಿ AI ಮಾದರಿಗಳನ್ನು ರನ್ನಿಂಗ್ ಮಾಡುವ ಮೂಲಕ ಕಾರ್ಯಕ್ಷಮತೆ, ಗೌಪ್ಯತೆ, ಕಸ್ಟಮೈಜೆಷನ್ ಮತ್ತು ವೆಚ್ಚದ ಲಾಭಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ. ಇದು ನಿಮ್ಮ ಇತ್ತೀಚಿನ ವರ್ಕ್‌ಫ್ಲೋಗಳು ಮತ್ತು ಅಪ್ಲಿಕೇಶನ್‌ಗಳಲ್ಲಿ ಸುಲಭವಾಗಿ CLI, SDK ಮತ್ತು REST API ಮುಖಾಂತರ ಸಮಗ್ರೀಕರಿಸಲಾಗುತ್ತದೆ.


![arch](../../../../../translated_images/kn/foundry-local-arch.8823e321dd8258d7.webp)

### Foundry Local ಯನ್ನು ಏಕೆ ಆರಿಸಿಕೊಳ್ಳಬೇಕು?

Foundry Local ನ ಪ್ರಯೋಜನಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು ನಿಮ್ಮ AI ನಿಯೋಜನಾ ತಂತ್ರಜ್ಞಾನವನ್ನು ತಿಳಿವಳಿಕೆ ಪಡೆಯುವಲ್ಲಿ ಸಹಾಯ ಮಾಡುತ್ತದೆ:

- **ಒನ್-ಡಿವೈಸ್ ಇನ್ಫರೆನ್ಸ್:** ನಿಮ್ಮದೇ ಹಾರ್ಡ್‌ವೇರ್‌ನಲ್ಲಿ ಸ್ಥಳೀಯವಾಗಿ ಮಾದರಿಗಳನ್ನು ರನ್ನಿಂಗ್ ಮಾಡಿ, ನಿಮ್ಮ ವೆಚ್ಚಗಳನ್ನು ಕಡಿಮೆ ಮಾಡಿ ಮತ್ತು ನಿಮ್ಮ ಎಲ್ಲಾ ಡೇಟಾವನ್ನು ನಿಮ್ಮ ಸಾಧನದಲ್ಲಿಡಿ.

- **ಮಾದರಿ ಕಸ್ಟಮೈಜೆಷನ್:** ಪೂರ್ವನಿರ್ಧಾರಿತ ಮಾದರಿಗಳಲ್ಲಿ ಆಯ್ಕೆ ಮಾಡಿ ಅಥವಾ ನಿಮ್ಮ ಸ್ವಂತವನ್ನು ಬಳಸಿಕೊಂಡು ನಿರ್ದಿಷ್ಟ ಅಗತ್ಯಗಳು ಮತ್ತು ಬಳಕೆ ಪ್ರಕರಣಗಳಿಗೆ ಹೊಂದಿಸಿಕೊಳ್ಳಿ.

- **ವೆಚ್ಚ ಕಾರ್ಯಕ್ಷಮತೆ:** ನಿಮ್ಮ ಇರುವ ಹಾರ್ಡ್‌ವೇರ್ ಬಳಸಿ ಆಗತಕ್ಕ ಇದೀಗಪ್ರದಾಯಮಾಡುವ ಕ್ಲೌಡ್ ಸೇವೆಗಳ ವೆಚ್ಚಗಳನ್ನು ನಿವಾರಣೆ ಮಾಡಿ AI ಅನ್ನು ಹೆಚ್ಚು ಪ್ರವೇಶಾತಿಯಾಗಿಸಿ.

- **ಸ್ಮೂತ್ ಇಂಟಿಗ್ರೇಶನ್:** ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್‌ಗಳೊಂದಿಗಿನ SDK, API ಎಂಟಿಪಾಯಿಂಟ್‌ಗಳು ಅಥವಾ CLI ಮೂಲಕ ಸಂಪರ್ಕಿಸುವ ಸುಗಮ ವಿಧಾನ, ಅಗತ್ಯ ಹಾಗು ಹೆಚ್ಚಿದಂತೆ Microsoft Foundry ಗೆ ಸುಲಭ ವಿಸ್ತರಣೆ.

> **ಪ್ರಾರಂಭಿಕ ಟಿಪ್ಪಣಿ:** ಈ ಟ್ಯೂಟೋರಿಯಲ್ Foundry Local ಅನ್ನು CLI ಮತ್ತು SDK ಇಂಟರ್ಫೇಸ್‌ಗಳ ಮೂಲಕ ಬಳಸುವುದರ ಮೇಲೆ ಕೇಂದ್ರೀಕರಿಸುತ್ತದೆ. ನಿಮ್ಮ ಬಳಕಾ ಪ್ರಕರಣಕ್ಕೆ ಅತ್ಯುತ್ತಮ ವಿಧಾನ ಆಯ್ಕೆ ಮಾಡಿಕೊಳ್ಳಲು ಇಬ್ಬರನ್ನೂ ತಿಳಿಯುತ್ತೀರಿ.

## ಭಾಗ 1: Foundry Local CLI ಸೆಟ್ ಅಪ್ ಮಾಡುವುದು

### ಹಂತ 1: ಇನ್‌ಸ್ಟಾಲೇಶನ್

Foundry Local CLI ನಿಮ್ಮ ಸ್ಥಳೀಯವಾಗಿ AI ಮಾದರಿಗಳನ್ನು ನಿರ್ವಹಿಸಿ ರನ್ನಿಂಗ್ ಮಾಡುವ ಪ್ರವೇಶದ್ವಾರವಾಗಿದೆ. ನಿಮ್ಮ ಸಿಸ್ಟಂನಲ್ಲಿ ಇದನ್ನು ಸ್ಥಾಪಿಸುವುದರಿಂದ ಪ್ರಾರಂಭಿಸೋಣ.

**ಬೆಂಬಲಿತ ವೇದಿಕೆಗಳು:** ವಿಂಡೋಸ್ ಮತ್ತು ಮ್ಯಾಕ್‌ಒಎಸ್

ವಿಸ್ತೃತ ಇನ್‌ಸ್ಟಾಲೇಶನ್ ಸೂಚನೆಗಾಗಿ ದಯವಿಟ್ಟು [ಅಧಿಕೃತ Foundry Local ಡಾಕ್ಯುಮೆಂಟೇಶನ್](https://github.com/microsoft/Foundry-Local/blob/main/README.md) ಅನ್ನು ನೋಡಿ.

### ಹಂತ 2: ಲಭ್ಯವಿರುವ ಮಾದರಿಗಳನ್ನು ಅನ್ವೇಷಿಸುವುದು

Foundry Local CLI ಇನ್‌ಸ್ಟಾಲ್ ಮಾಡಿದ ಮೇಲೆ, ನಿಮ್ಮ ಬಳಕೆಗೆ ಲಭ್ಯವಿರುವ ಮಾದರಿಗಳನ್ನು ಕಂಡುಹಿಡಿಯಬಹುದು. ಈ ಆಜ್ಞೆ ಎಲ್ಲಾ ಬೆಂಬಲಿತ ಮಾದರಿಗಳನ್ನು ತೋರಿಸುತ್ತದೆ:


```bash
foundry model list
```

### ಹಂತ 3: Phi ಪರಿವಾರ ಮಾದರಿಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು

Phi Family ವಿವಿಧ ಬಳಕೆ ಪ್ರಕರಣಗಳು ಮತ್ತು ಹಾರ್ಡ್‌ವೇರ್ ಸಂರಚನೆಗಳಿಗೆ ಸೂಕ್ತವಾದ ಮಾದರಿಗಳ ವ್ಯಾಪ್ತಿಯನ್ನು ನೀಡುತ್ತದೆ. ಇಲ್ಲಿವೆ Foundry Local ನಲ್ಲಿ ಲಭ್ಯವಿರುವ Phi ಮಾದರಿಗಳು:

**ಲಭ್ಯವಿರುವ Phi ಮಾದರಿಗಳು:**

- **phi-3.5-mini** - ಮೂಲಕಾರ್ಯಗಳಿಗಾಗಿ ಸಣ್ಣ ಮಾದರಿ
- **phi-3-mini-128k** - ದೀರ್ಘ ಸಂಭಾಷಣೆಗಳಿಗಾಗಿ ವಿಸ್ತೃತ ಸੰਦਰಭ ಆವೃತ್ತಿ
- **phi-3-mini-4k** - ಸಾಮಾನ್ಯ ಬಳಕೆಗೆ ಸ್ಟ್ಯಾಂಡರ್ಡ್ ಸੰਦਰಭ ಮಾದರಿ
- **phi-4** - ಸುಧಾರಿತ ಸಾಮರ್ಥ್ಯಗಳೊಂದಿಗೆ ಪ್ರಗತಿಯ ಮಾೆಾದರಿ
- **phi-4-mini** - Phi-4 ನ ತೂಕದಲ್ಲಿ ಕಡಿಮೆ ಆವೃತ್ತಿ
- **phi-4-mini-reasoning** - ಸಂಕೀರ್ಣ ತರ್ಕ ಇಡುವ ಕಾರ್ಯಗಳಿಗಾಗಿ ವಿಶೇಷೀಕೃತ

> **ಹಾರ್ಡ್‌ವೇರ್ ಪರಿಶುದ್ಧತೆ:** ಪ್ರತಿ ಮಾದರಿಯನ್ನು ನಿಮ್ಮ ಸಿಸ್ಟಂ ಸಾಮರ್ಥ್ಯಗಳ ಮೇಲೆ ಆಧಾರಿತವಾಗಿ ವಿವಿಧ ಹಾರ್ಡ್‌ವೇರ್ ವಿಖರಣ (CPU, GPU) ಗೆ ಹೊಂದಿಸಬಹುದು.

### ಹಂತ 4: ನಿಮ್ಮ ಮೊದಲ Phi ಮಾದರಿಯನ್ನು ರನ್ನಿಂಗ್ ಮಾಡುವುದು

ಪ್ರಾಯೋಗಿಕ ಉದಾಹರಣೆಯಿಂದ ಪ್ರಾರಂಭಿಸೋಣ. ನಾವು `phi-4-mini-reasoning` ಮಾದರಿಯನ್ನು ರನ್ನಿಂಗ್ ಮಾಡೋಣ, ಇದು ಹಂತ ಹಂತವಾಗಿ ಸಂಕೀರ್ಣ ಸಮಸ್ಯೆಗಳನ್ನು ಪರಿಹರಿಸುವಲ್ಲಿ ನಿಪುಣವಾಗಿದೆ.


**ಮಾದರಿಯನ್ನು ರನ್ನಿಂಗ್ ಮಾಡಲು ಆಜ್ಞೆ:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **ಮೊದಲ ಬಾರಿಗೆ ಸೆಟ್ ಅಪ್:** ಮಾದರಿಯನ್ನು ಮೊದಲ ಬಾರಿಗೆ ರನ್ನಿಂಗ್ ಮಾಡುತ್ತಿರುವಾಗ, Foundry Local ಸ್ವಯಂಚಾಲಿತವಾಗಿ ಅದನ್ನು ನಿಮ್ಮ ಸ್ಥಳೀಯ ಸಾಧನಕ್ಕೆ ಡೌನ್‌ಲೋಡ್ ಮಾಡುತ್ತದೆ. ನಿಮ್ಮ ನೆಟ್ವರ್ಕ್ ವೇಗದ ಮೇಲೆ ಅವಲಂಬಿತವಾಗಿಯೇ ಡೌನ್‌ಲೋಡ್ ಸಮಯ ಬದಲಾಗಬಹುದು, ಆದ್ದರಿಂದ ಪ್ರಾರಂಭಿಕ ಸೆಟ್ ಅಪ್ ಸಮಯದಲ್ಲಿ ಸಹನೆ ಇರಲಿ.

### ಹಂತ 5: ವಾಸ್ತವ ಸಮಸ್ಯೆಯಿಂದ ಮಾದರಿಯನ್ನು ಪರೀಕ್ಷಿಸುವುದು

ಈಗ ಹಂತ ಹಂತವಾಗಿ ತર્ક ಇಡುತ್ತಿರುವುದನ್ನು ನೋಡಲು ನಮ್ಮ ಮಾದರಿಯನ್ನು ಕ್ಲಾಸಿಕ್ ಲಾಜಿಕ್ ಸಮಸ್ಯೆಯಿಂದ ಪರೀಕ್ಷಿಸೋಣ:

**ಉದಾಹರಣೆ ಸಮಸ್ಯೆ:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**ನಿರೀಕ್ಷಿತ ವರ್ತನೆ:** pheasants ಗೆ ಎರಡು ಕಾಲುಗಳು ಮತ್ತು ಹಂದಿಗಳಿಗೆ ನಾಲ್ಕು ಕಾಲುಗಳು ಎಂದುFACTನನ್ನು ಬಳಸಿಕೊಂಡು ಈ ಸಮಸ್ಯೆಯನ್ನು ತರ್ಕದ ಹಂತಗಳಾಗಿ ವಿಭಜಿಸಿ ಸಮೀಕರಣ ವ್ಯವಸ್ಥೆ ಪರಿಹರಿಸಬೇಕು.

## ಭಾಗ 2: Foundry Local SDK ಮೂಲಕ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವುದು

### SDK ಯನ್ನು ಏಕೆ ಬಳಸಬೇಕು?

CLI ಪರೀಕ್ಷೆ ಮತ್ತು ವೇಗದ ಸಂವಹನಗಳಿಗೆ ಅತ್ಯುತ್ತಮವಾದರೂ, SDK ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್‌ಗಳಲ್ಲಿ Foundry Local ಅನ್ನು ಪ್ರೋಗ್ರಾಮ್ಯಾಟಿಕವಾಗಿ ಸಮಗ್ರೀಕರಿಸಲು ಸಹಾಯ ಮಾಡುತ್ತದೆ. ಇದು ಸಾಧ್ಯತೆಗಳನ್ನು ತೆರೆಯುತ್ತದೆ:

- ಕಸ್ಟಮ್ AI ಚಾಲಿತ ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವುದು
- ಸ್ವಯಂಚಾಲಿತ ವರ್ಕ್‌ಫ್ಲೋಗಳನ್ನು ರಚಿಸುವುದು
- ಇತ್ತೀಚಿನ ವ್ಯವಸ್ಥೆಗಳಿಗೆ AI ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಸೇರಿಸುವುದು
- ಚಾಟ್‌ಬಾಟ್‌ಗಳು ಮತ್ತು ಪರಸ್ಪರ ಸಂವಹನೋಪಕರಣಗಳನ್ನು ಅಭಿವೃದ್ಧಿಪಡಿಸುವುದು

### ಬೆಂಬಲಿತ ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಭಾಷೆಗಳು

Foundry Local ನಿಮ್ಮ ಅಭಿವೃದ್ಧಿ ಆದ್ಯತೆಯಗಳಿಗೆ ಹೊಂದಿಕೊಳ್ಳುವ ಹಲವು ಪ್ರೋಗ್ರಾಮಿಂಗ್ ಭಾಷೆಗಳ SDK ಬೆಂಬಲವನ್ನು ಒದಗಿಸುತ್ತದೆ:

**📦 ಲಭ್ಯವಿರುವ SDKಗಳು:**

- **C# (.NET):** [SDK ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಮತ್ತು ಉದಾಹರಣೆಗಳು](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಮತ್ತು ಉದಾಹರಣೆಗಳು](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಮತ್ತು ಉದಾಹರಣೆಗಳು](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಮತ್ತು ಉದಾಹರಣೆಗಳು](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### ಮುಂದಿನ ಹಂತಗಳು

1. ನಿಮ್ಮ ಅಭಿವೃದ್ಧಿ ಪರಿಸರ ಆಧರಿಸಿ **ನಿಮ್ಮ ಪ್ರಾರ್ಥಿಯ SDK ಆಯ್ಕೆಮಾಡಿ**  
2. ವಿವರವಾದ ಅನುಷ್ಠಾನ ಮಾರ್ಗದರ್ಶನಕ್ಕಾಗಿ **SDK-ನಿರ್ದಿಷ್ಟ ಡಾಕ್ಯುಮೆಂಟೇಶನ್ ಅನುಸರಿಸಿ**  
3. ಸಂಕೀರ್ಣ ಅಪ್ಲಿಕೇಶನ್ ನಿರ್ಮಿಸುವ ಮೊದಲು **ಸರಳ ಉದಾಹರಣೆಗಳಿಂದ ಪ್ರಾರಂಭಿಸಿ**  
4. ಪ್ರತಿ SDK ರೆಪೊದಲ್ಲಿ ಒದಗಿಸಿರುವ **ಉದಾಹರಣಾ ಕೋಡ್ ಅನ್ವೇಷಿಸಿ**

- ✅ Phi ಪರಿವಾರ ಮಾದರಿಗಳನ್ನು ಕಂಡುಹಿಡಿದು ರನ್ನಿಂಗ್ ಮಾಡುವುದು
- ✅ ವಾಸ್ತವ ಸಮಸ್ಯೆಗಳಿಗೆ ಮಾದರಿಗಳನ್ನು ಪರೀಕ್ಷಿಸುವುದು
- ✅ ಅಪ್ಲಿಕೇಶನ್ ಅಭಿವೃದ್ಧಿಗಾಗಿ SDK ಆಯ್ಕೆಗಳನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವುದು
