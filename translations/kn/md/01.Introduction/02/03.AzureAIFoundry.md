## ಮೈಕ್ರೋಸಾಫ್ಟ್ ಫೌಂಡ್ರಿಯಲ್ಲಿ ಫೈ ಕುಟುಂಬ

[Microsoft Foundry](https://ai.azure.com) ಅಭಿವೃದ್ಧಿಪಡಿಸುವವರಿಗೆ ಸುರಕ್ಷಿತ, ಭದ್ರ, ಮತ್ತು ಜವಾಬ್ದಾರಿಯುತ ರೀತಿಯಲ್ಲಿ ಕೃತಕ ಬುದ್ಧಿಮತ್ತೆ ಆಧಾರಿತ ಹೊಸತನವನ್ನು ಚಾಲನೆ ನೀಡಲು ಮತ್ತು ಭವಿಷ್ಯವನ್ನು ರೂಪಿಸಲು ಸಾಮರ್ಥ್ಯ ನೀಡುವ ವಿಶ್ವಾಸಾರ್ಹ ವೇದಿಕೆ ಆಗಿದೆ.



[Microsoft Foundry](https://ai.azure.com) ಅಭಿವೃದ್ಧಿಪಡಿಸುವವರಿಗಾಗಿ ವಿನ್ಯಾಸಗೊಳ್ಳಲಾಗಿದೆ:

- ಎಂಟರ್ಪ್ರೈಸ್-ಗ್ರೇಡ್ ವೇದಿಕೆಯಲ್ಲಿ ಜನರೇಟಿವ್ AI ಅನ್ವಯಿಕಾಗಳನ್ನು ನಿರ್ಮಿಸಲು.
- ಸುಧಾರಿತ AI ಉಪಕರಣಗಳು ಮತ್ತು ಯಂತ್ರಾಧಾರಿತ ಮಾದರಿಗಳನ್ನು ಬಳಸಿಕೊಂಡು ಅನ್ವೇಷಣೆ, ನಿರ್ಮಾಣ, ಪರೀಕ್ಷೆ, ಮತ್ತು ನಿಯೋಜನೆ ಮಾಡುವುದು, ಜವಾಬ್ದಾರಿಯುತ AI ಅಭ್ಯಾಸಗಳ ಮೇಲ್ಪಡಿಸಿ.
- ಸಮಗ್ರ ಅಪ್ಲಿಕೇಶನ್ ಅಭಿವೃದ್ಧಿ ಸಂಪೂರ್ಣ ಜೀವನಚಕ್ರಕ್ಕಾಗಿ ತಂಡದೊಡನೆ ಸಹಕಾರ ಮಾಡುವುದು.

Microsoft Foundry ಮೂಲಕ, ನೀವು ವಿವಿಧ ಮಾದರಿಗಳು, ಸೇವೆಗಳು ಮತ್ತು ಸಾಮರ್ಥ್ಯಗಳನ್ನು ಪರಿಶೀಲಿಸಿ ನಿಮ್ಮ ಗುರಿಗಳನ್ನು ಅತ್ಯುತ್ತಮವಾಗಿ ಪೂರೈಸುವ AI ಅನ್ವಯಿಕೆಗಳನ್ನು ನಿರ್ಮಿಸಬಹುದು. Microsoft Foundry ವೇದಿಕೆ ಸಿದ್ಧಾಂತಗಳ ಪರೀಕ್ಷೆಗಳನ್ನು ಸಂಪೂರ್ಣ ಉತ್ಪಾದನಾ ಅನ್ವಯಿಕೆಗಳಾಗಿ ಪರಿವರ್ತಿಸಲು ಸುಗಮವಾಗಿ ವಿಸ್ತರಣೆಯನ್ನು ಪ್ರೇರೇಪಿಸುತ್ತದೆ. ನಿರಂತರ ನಿಗಾದಾರಣೆ ಮತ್ತು ಪರಿಷ್ಕರಣೆ ದೀರ್ಘಕಾಲೀನ ಷಡ್ಜೇತವನ್ನು ಬೆಂಬಲಿಸುತ್ತದೆ.

Microsoft Foundry ನಲ್ಲಿ Azure AOAI ಸೇವೆಯನ್ನು ಬಳಸುವುದರ ಜೊತೆಗೆ, ನೀವು Microsoft Foundry ಮಾದರಿ ಕ್ಯಾಟಲಾಗ್ನಲ್ಲಿ ಮೂವರುಮಟ್ಟದ ಮಾದರಿಗಳನ್ನು ಕೂಡ ಬಳಸಬಹುದು. ನೀವು Microsoft Foundry ಅನ್ನು ನಿಮ್ಮ AI ಪರಿಹಾರ ವೇದಿಕೆಯಾಗಿ ಬಳಸಲು ಬಯಸಿದರೆ ಇದು ಉತ್ತಮ ಆಯ್ಕೆ.

ನಾವು Microsoft Foundry ನಲ್ಲಿ ಮಾದರಿ ಕ್ಯಾಟಲಾಗ್ ಮೂಲಕ ಫೈ ಕುಟುಂಬದ ಮಾದರಿಗಳನ್ನು ತ್ವರಿತವಾಗಿ ನಿಯೋಜಿಸಬಹುದು

### **Microsoft Foundry Phi-4 ಅನ್ನು ಕರೆ ಮಾಡಲು Python ಕೋಡ್ ಅನ್ನು ರನ್ ಮಾಡುವುದು**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# ಮಾತನಾಡುವಿಕೆ ಸಕ್ರಿಯವಾಗಿದ್ದರೆ ಮಾತು ಫಲಿತಾಂಶವನ್ನು ಸೇರಿಸಿ
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
