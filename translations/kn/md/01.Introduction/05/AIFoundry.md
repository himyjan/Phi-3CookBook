# **Microsoft Foundry ಬಳಸಿ ಮೌಲ್ಯಮಾಪನ**

![aistudo](../../../../../translated_images/kn/AIFoundry.9e0b513e999a1c5a.webp)

ನೀವು [Microsoft Foundry](https://ai.azure.com?WT.mc_id=aiml-138114-kinfeylo) ಬಳಸಿ ನಿಮ್ಮ ಜನರೇಟಿವ್ AI ಅಪ್ಲಿಕೇಶನ್ ಅನ್ನು ಹೇಗೆ ಮೌಲ್ಯಮಾಪನ ಮಾಡಬಹುದು. ನೀವು ಒಂದೇ ಸಂಭಾಷಣೆ ಅಥವಾ ಬಹು-ಸಂಭಾಷಣೆಗಳನ್ನು ಅಂದಾಜಿಸುವಾಗ, Microsoft Foundry ಮಾದರಿ ಪ್ರದರ್ಶನ ಮತ್ತು ಸುರಕ್ಷತೆಗಾಗಿ ಮೌಲ್ಯಮಾಪನ ಸಾಧನಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ.

![aistudo](../../../../../translated_images/kn/AIPortfolio.69da59a8e1eaa70f.webp)

## Microsoft Foundry ಮೂಲಕ ಜನರೇಟಿವ್ AI ಅಪ್ಲಿಕೇಶನ್‌ಗಳನ್ನು ಹೇಗೆ ಮೌಲ್ಯಮಾಪನ ಮಾಡುವುದು
ಹೆಚ್ಚಿನ ವಿವರಗಳಿಗೆ [Microsoft Foundry ಡಾಕ್ಯುಮೆಂಟೇಷನ್](https://learn.microsoft.com/azure/ai-studio/how-to/evaluate-generative-ai-app?WT.mc_id=aiml-138114-kinfeylo) ನೋಡಿ

ನೀವು ಪ್ರಾರಂಭಿಸಲು ಈ ಹಂತಗಳು ಇಲ್ಲಿವೆ:

## Microsoft Foundry ನಲ್ಲಿ ಜನರೇಟಿವ್ AI ಮಾದರಿಗಳನ್ನು ಮೌಲ್ಯಮಾಪನ ಮಾಡುವುದು

**ಪೂರ್ವಶರತ್ತುಗಳು**

- CSV ಅಥವಾ JSON ಸ್ವರೂಪದಲ್ಲಿ ಒಂದು ಪರೀಕ್ಷಾ ಡೇಟಾಸೆಟ್.
- ನಿಯೋಜಿತ ಜನರೇಟಿವ್ AI ಮಾದರಿ (ಉದಾಹರಣೆಗೆ Phi-3, GPT 3.5, GPT 4, ಅಥವಾ Davinci ಮಾದರಿಗಳು).
- ಮೌಲ್ಯಮಾಪನ ನಡೆಸಲು ಗಣನೆ ಘಟಕದೊಂದಿಗೆ রান್ಟೈಮ್.

## ಒಳಗೊಂಡಿರುವ ಮೌಲ್ಯಮಾಪನ ನಿಯಾಮಕಗಳು

Microsoft Foundry ಒಂದೇ ಸಂಭಾಷಣೆ ಹಾಗೂ ಸಾಂಕೀರ್ಣ, ಬಹು-ಸಂಭಾಷಣೆಗಳನ್ನು ಮೌಲ್ಯಮಾಪನ ಮಾಡಲು ಅನುಮತಿಸುತ್ತದೆ.
Retrieval Augmented Generation (RAG) ಪರಿಸ್ಥಿತಿಗಳಲ್ಲಿ, ಮಾದರಿ ನಿರ್ದಿಷ್ಟ ಡೇಟಾದ ಮೇಲೆ ಆಧಾರಿತವಾಗಿರುವಾಗ, ಒಳಗೊಂಡಿರುವ ಮೌಲ್ಯಮಾಪನ ನಿಯಾಮಕಗಳನ್ನು ಬಳಸಿ ಪ್ರದರ್ಶನವನ್ನು ಅಂದಾಜಿಸಬಹುದು.
ಇದೆಯೊಂದಿಗೆ, ಸಾಮಾನ್ಯ ಒಂದೇ ಸಂಭಾಷಣೆ ಪ್ರಶ್ನೋತ್ತರ ಪರಿಸ್ಥಿತಿಗಳನ್ನು (ನಾನ್-RAG) ಮೌಲ್ಯಮಾಪನ ಮಾಡಬಹುದು.

## ಮೌಲ್ಯಮಾಪನ ರನ್ ರಚನೆ

Microsoft Foundry UI ಯಿಂದ, Evaluate ಪುಟ ಅಥವಾ Prompt Flow ಪುಟಕ್ಕೆ ಹೋಗಿ.
ಮೌಲ್ಯಮಾಪನ ರನ್ ಅನ್ನು ಸ್ಥಾಪಿಸಲು ಮೌಲ್ಯಮಾಪನ ರಚನಾ ವಿಸಾರ್ಡ್ ಅನುಸರಿಸಿ. ನಿಮ್ಮ ಮೌಲ್ಯಮಾಪನಕ್ಕಾಗಿ ಐಚ್ಛಿಕ ಹೆಸರು ನೀಡಿ.
ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್ ಗುರಿಗಳ ಜೊತೆಗೆ ಹೊಂದಿಕೆ ಹೊಂದಿರುವ ಪರಿಸ್ಥಿತಿಯನ್ನು ಆಯ್ಕೆಮಾಡಿ.
ಮಾದರಿಯ ಫಲಿತಾಂಶವನ್ನು ಅಂದಾಜಿಸಲು ಒಂದು ಅಥವಾ ಹೆಚ್ಚು ಮೌಲ್ಯಮಾಪನ ನಿಯಾಮಕಗಳನ್ನು ಆಯ್ಕೆಮಾಡಿ.

## ಸಹಜ ಮೌಲ್ಯಮಾಪನ ಪ್ರವಾಹ (ಐಚ್ಛಿಕ)

ಹೆಚ್ಚಿನ ಸ್ವಾತಂತ್ರ್ಯಕ್ಕೆ, ನೀವು ಕಸ್ಟಮ್ ಮೌಲ್ಯಮಾಪನ ಪ್ರವಾಹವನ್ನು ರಚಿಸಬಹುದು. ನಿಮ್ಮ ನಿರ್ದಿಷ್ಟ ಅಗತ್ಯಗಳ ಆಧಾರದ ಮೇಲೆ ಮೌಲ್ಯಮಾಪನ ಪ್ರಕ್ರಿಯೆಯನ್ನು ಕಸ್ಟಮೈಸ್ ಮಾಡಿ.

## ಫಲಿತಾಂಶಗಳನ್ನು ವೀಕ್ಷಣೆ

ಮೌಲ್ಯಮಾಪನ ನಡೆಸಿದ ನಂತರ, Microsoft Foundry ನಲ್ಲಿ ವಿವರವಾದ ಮೌಲ್ಯಮಾಪನ ನಿಯಾಮಕಗಳನ್ನು ಲಾಗ್, ವೀಕ್ಷಿಸಿ ಮತ್ತು ವಿಶ್ಲೇಷಿಸಿರಿ. ನಿಮ್ಮ ಅಪ್ಲಿಕೇಶನ್ ಸಾಮರ್ಥ್ಯಗಳು ಮತ್ತು ಮಿತಿ ತಿಳಿದುಕೊಳ್ಳಿ.

**Note** Microsoft Foundry ಸದ್ಯದಲ್ಲೇ ಸಾರ್ವಜನಿಕ ಪೂರ್ವದರ್ಶನದಲ್ಲಿದೆ, ಆದ್ದರಿಂದ ಅದನ್ನು ಪ್ರಯೋಗ ಮತ್ತು ಅಭಿವೃದ್ಧಿ ಉದ್ದೇಶಗಳಿಗಾಗಿ ಬಳಸಿರಿ. ಉತ್ಪಾದನಾ ಕೆಲಸಗಳಿಗೆ, ಇತರ ಆಯ್ಕೆಗಳನ್ನು ಪರಿಗಣಿಸಿ. ಅಧಿಕ ಮಾಹಿತಿಗೆ ಮತ್ತು ನೀಡಿಕೆ ಹಂತದ ಸೂಚನೆಗಳಿಗಾಗಿ ಅಧಿಕೃತ [AI Foundry ಡಾಕ್ಯುಮೆಂಟೇಷನ್](https://learn.microsoft.com/azure/ai-studio/?WT.mc_id=aiml-138114-kinfeylo) ಅನ್ನು ಪರಿಶೀಲಿಸಿ.
