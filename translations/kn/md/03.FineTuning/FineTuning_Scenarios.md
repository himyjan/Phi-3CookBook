## ಫೈನ್ ಟ್ಯೂನಿಂಗ್ ದೃಶ್ಯಗಳು

![FineTuning with MS Services](../../../../translated_images/kn/FinetuningwithMS.3d0cec8ae693e094.webp)

ಈ ಭಾಗವು Microsoft Foundry ಮತ್ತು Azure ಪರಿಸರಗಳಲ್ಲಿನ ಫೈನ್-ಟ್ಯೂನಿಂಗ್ ದೃಶ್ಯಗಳ ಅವಲೋಕನವನ್ನು ನೀಡುತ್ತದೆ, ಇದರಲ್ಲಿ despleo ಮಾದರಿಗಳು, ಇನ್‌ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್ ಪದರಗಳು ಮತ್ತು ಸಾಮಾನ್ಯವಾಗಿ ಬಳಸಲಾಗುವ ಆಪ್ಟಿಮೈಜೆಶನ್ ತಂತ್ರಗಳು ಸೇರಿವೆ.

**ಮೈದಾನ**  
ಇದು Microsoft Foundry (ಹಿಂದಿನ Microsoft Foundry) ಮತ್ತು Azure ಮಷೀನ್ ಲರ್ನಿಂಗ್ ಸೇರಿದಂತೆ ನಿರ್ವಹಿತ ಸೇವೆಗಳನ್ನು ಒಳಗೊಂಡಿದ್ದು, ಮಾದರಿ ನಿರ್ವಹಣೆ, ಸಮನ್ವಯ, ಪ್ರಯೋಗ ಟ್ರ್ಯಾಕಿಂಗ್ ಮತ್ತು despleo ವರ್ಕ್‌ಫ್ಲೋಗಳನ್ನು ಒದಗಿಸುತ್ತದೆ.

**ಇನ್‌ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್**  
ಫೈನ್-ಟ್ಯೂನಿಂಗ್‌ಗೆ ಸ್ಕೇಲಬಲ್ ಕಂಪ್ಯೂಟ್ ಸಂಪನ್ಮೂಲಗಳು ಅಗತ್ಯವಿರುತ್ತವೆ. Azure ಪರಿಸರಗಳಲ್ಲಿ, ಇದರಲ್ಲಿ ಸಾಮಾನ್ಯವಾಗಿ GPU ಆಧಾರಿತ ವರ್ಚುವಲ್ ಮಶೀನಿಗಳು ಮತ್ತು ಲಘು-ಬೋঝೋಡಿಕೆಗಳಿಗೆ CPU ಸಂಪನ್ಮೂಲಗಳು, ಜೊತೆಗೆ ಡೇಟಾಸೆಟ್‌ಗಳು ಮತ್ತು ಚೆಕ್ಪಾಯಿಂಟ್‌ಗಳಿಗೆ ಸ್ಕೇಲಬಲ್ ಸಂಗ್ರಹಣೆ ಸೇರಿವೆ.

**ಪ ಶಕ್ತಿಗಳು ಮತ್ತು ಫ್ರೇ ಮುವ ದಳ**  
ಫೈನ್-ಟ್ಯೂನಿಂಗ್ ವರ್ಕ್‌ಫ್ಲೋಗಳು ಸಾಮಾನ್ಯವಾಗಿ Hugging Face Transformers, DeepSpeed ಮತ್ತು PEFT (ಪ್ಯಾರಾಮೀಟರ್-ದಕ್ಷ ಫೈನ್-ಟ್ಯೂನಿಂಗ್) ಮುಂತಾದ ಫ್ರೇಮುವರ್ಗಳು ಮತ್ತು ಆಪ್ಟಿಮೈಜೆಶನ್ ಲೈಬ್ರರಿಗಳನ್ನು ಅವಲಂಬಿಸುತ್ತವೆ.

Microsoft ತಂತ್ರಜ್ಞಾನಗಳೊಂದಿಗೆ ಫೈನ್-ಟ್ಯೂನಿಂಗ್ ಪ್ರಕ್ರಿಯೆ ಮೈದಾನ ಸೇವೆಗಳು, ಕಂಪ್ಯೂಟ್ ಇನ್‌ಫ್ರಾಸ್ಟ್ರಕ್ಚರ್, ಮತ್ತು ತರಬೇತಿ ಫ್ರೇಮುವರ್ಗಳನ್ನು ಒಳಗೊಂಡಿದೆ. ಈ ಅಂಶಗಳು ಒಟ್ಟಾಗಿ ಹೇಗೆ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತವೆ ಎಂಬುದನ್ನು ಅರ್ಥಮಾಡಿಕೊಳ್ಳುವ ಮೂಲಕ, ಅಭಿವೃದ್ಧಿಪಡಿಸುವವರು ಆಧಾರ ಮಾದರಿಗಳನ್ನು ನಿರ್ದಿಷ್ಟ ಕಾರ್ಯಗಳು ಮತ್ತು ಉತ್ಪಾದನಾ ದೃಶ್ಯಗಳಿಗೆ ತ್ವರಿತವಾಗಿ ಹೊಂದಿಸಬಹುದು.

## ಮಾದರಿ ಜತೆಗೆ ಸೇವೆ

ಹೋಸ್ಟು ಮಾಡಲಾದ ಫೈನ್-ಟ್ಯೂನಿಂಗ್ ಬಳಸಿ ಮಾದರಿಯನ್ನು ಫೈನ್-ಟ್ಯೂನ್ ಮಾಡಿ, ಕಂಪ್ಯೂಟ್ ರಚಿಸಲು ಮತ್ತು ನಿರ್ವಹಿಸಲು ಅವಶ್ಯಕತೆ ಇಲ್ಲದೆ.

![MaaS Fine Tuning](../../../../translated_images/kn/MaaSfinetune.3eee4630607aff0d.webp)

ಸರ್ವರ್‌ಲೆಸ್ ಫೈನ್-ಟ್ಯೂನಿಂಗ್ ಈಗ Phi-3, Phi-3.5 ಮತ್ತು Phi-4 ಮಾದರಿ ಕುಟುಂಬಗಳಿಗೆ ಲಭ್ಯವಿದೆ, ಅಭಿವೃದ್ಧಿಪಡಿಸುವವರು ಕಂಪ್ಯೂಟ್ ವ್ಯವಸ್ಥೆ ಮಾಡದೇ ಮೋಡ ಮತ್ತು ಎಡ್ಜ್ ದೃಶ್ಯಗಳಿಗೆ ಮಾದರಿಗಳನ್ನು ಸರಳವಾಗಿ ಮತ್ತು ತ್ವರಿತವಾಗಿ ಕಸ್ಟಮೈಸ್ ಮಾಡಲು ಇದನ್ನು ಬಳಸಬಹುದು.

## ಮಾದರಿ ಜತೆಗೆ ಮೈದಾನ

ಬಳಕೆದಾರರು ತಮ್ಮ ಸ್ವಂತ ಕಂಪ್ಯೂಟ್ ನಿರ್ವಹಣೆ ಮಾಡಿ ತಮ್ಮ ಮಾದರಿಗಳ ಫೈನ್-ಟ್ಯೂನಿಂಗ್ ಮಾಡುತ್ತಾರೆ.
