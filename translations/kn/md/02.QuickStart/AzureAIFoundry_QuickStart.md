# **Microsoft Foundry ನಲ್ಲಿ Phi-3 ಬಳಸುವುದು**

Generative AI ಅಭಿವೃದ್ಧಿಯೊಂದಿಗೆ, ನಾವು ವಿವಿಧ LLM ಮತ್ತು SLM, ಎಂಟರ್‌ಪ್ರೈಸ್ ಡೇಟಾ ಸಮ್ಮೇಳನ, ಫೈನ್-ಟ್ಯೂನಿಂಗ್/RAG ಕಾರ್ಯಾಚರಣೆಗಳು ಮತ್ತು LLM ಮತ್ತು SLM ಅನ್ನು ಸೇರಿಸಿದ ನಂತರ ವಿವಿಧ ಎಂಟರ್‌ಪ್ರೈಸ್ ವ್ಯವಹಾರಗಳ ಮೌಲ್ಯಮಾಪನವನ್ನು ನಿರ್ವಹಿಸಲು ಏಕೀಕೃತ ವೇದಿಕೆಯನ್ನು ಬಳಸುವ ನಿರೀಕ್ಷೆಯನ್ನು ಹೊಂದಿದ್ದೇವೆ, ಇದರಿಂದ ಗಣನೀಯ AI ಸ್ಮಾರ್ಟ್ ಅಪ್ಲಿಕೇಶನ್ಗಳು ಉತ್ತಮವಾಗಿ ಜಾರಿಗೊಳ್ಳಬಹುದು. [Microsoft Foundry](https://ai.azure.com) ಒಂದು ಎಂಟರ್‌ಪ್ರೈಸ್ ಮಟ್ಟದ ಗಣನೀಯ AI ಅಪ್ಲಿಕೇಶನ್ ವೇದಿಕೆ ಆಗಿದೆ.

![aistudo](../../../../translated_images/kn/aifoundry_home.f28a8127c96c7d93.webp)

Microsoft Foundry ಮೂಲಕ, ನೀವು ದೊಡ್ಡ ಭಾಷಾ ಮಾದರಿ (LLM) ಪ್ರತಿಕ್ರಿಯೆಗಳನ್ನು ಮೌಲ್ಯಮಾಪನ ಮಾಡಬಹುದು ಮತ್ತು ಉತ್ತಮ ಕಾರ್ಯಕ್ಷಮತೆಗೆ ಪ್ರಾಂಪ್ಟ್ ಫ್ಲೋ ಮೂಲಕ ಪ್ರಾಂಪ್ಟ್ ಅನ್ವಯ ಘಟಕಗಳನ್ನು ಸಂಯೋಜಿಸಬಹುದು. ಈ ವೇದಿಕೆ ತ್ಯಾಗವಾದ ಅಂಶಗಳನ್ನು ಸಂಪೂರ್ಣ ಉತ್ಪಾದನೆಯಲ್ಲಿ ಪರಿವರ್ತಿಸಲು ಪ್ರಮಾಣಮಿತಿ ಯೋಗ್ಯತೆಯನ್ನು ಸೌಲಭ್ಯಗೊಳಿಸುತ್ತದೆ. ನಿರಂತರ ನಿಗಾರಿಕೆ ಮತ್ತು ಪರಿಷ್ಕರಣೆ ದೀರ್ಘಕಾಲಿಕ ಯಶಸ್ಸಿಗೆ ಬೆಂಬಲ ನೀಡುತ್ತವೆ.

ನಾವು ಸರಳ ಹಂತಗಳ ಮೂಲಕ Microsoft Foundry ನಲ್ಲಿ Phi-3 ಮಾದರಿಯನ್ನು ಶೀಘ್ರವಾಗಿ ನಿಯೋಜಿಸಬಹುದು, ನಂತರ Microsoft Foundry ಅನ್ನು ಬಳಸಿಕೊಂಡು Phi-3 ಸಂಬಂಧಿತ ಆಟದ ಮೈದಾನ/ಚಾಟ್, ಫೈನ್-ಟ್ಯೂನಿಂಗ್, ಮೌಲ್ಯಮಾಪನ ಮತ್ತು ಇತರ ಸಂಬಂಧಿತ ಕಾರ್ಯಗಳನ್ನು ಪೂರ್ಣಗೊಳಿಸಬಹುದು.

## **1. ಸಿದ್ಧತೆ**

ನಿಮ್ಮ ಯಂತ್ರದಲ್ಲಿ ಈಗಾಗಲೇ [Azure Developer CLI](https://learn.microsoft.com/azure/developer/azure-developer-cli/overview?WT.mc_id=aiml-138114-kinfeylo) ಇನ್‌ಸ್ಟಾಲ್ ಆಗಿದ್ದರೆ, ಈ ಟೆಂಪ್ಲೇಟನ್ನು ಬಳಸಿ ಹೊಸ ಡೈರೆಕ್ಟರಿಯಲ್ಲಿ ಈ ಕಮಾಂಡ್ ಅನ್ನು ಚಲಿಸುವುದು ಎಷ್ಟು ಸುಲಭವಾಗಿದೆ.

## ಕೈಯಿಂದ ರಚನೆ

Microsoft Foundry ಪ್ರಾಜೆಕ್ಟ್ ಮತ್ತು ಹಬ್ ರಚಿಸುವುದು ನಿಮ್ಮ AI ಕೆಲಸವನ್ನು ಸಂಘಟಿಸಲು ಮತ್ತು ನಿರ್ವಹಿಸಲು ಅತ್ಯುತ್ತಮ ಮಾರ್ಗವಾಗಿದೆ. ಪ್ರಾರಂಭಿಸಲು ಹಂತ-ಹಂತ ಮಾರ್ಗದರ್ಶನ ಇಲ್ಲಿದೆ:

### Microsoft Foundry ನಲ್ಲಿ ಪ್ರಾಜೆಕ್ಟ್ ರಚಿಸುವುದು

1. **Microsoft Foundry ಗೆ ಹೋಗಿ**: Microsoft Foundry ಪೋರ್ಟಲ್ ನಲ್ಲಿ ಸೈನ್ ಇನ್ ಆಗಿ.
2. **ಪ್ರಾಜೆಕ್ಟ್ ರಚಿಸಿ**:
   - ನೀವು ಒಂದು ಪ್ರಾಜೆಕ್ಟ್ ಒಳಗೆ ಇದ್ದರೆ, ಪುಟದ ಮೇಲ್ಕೈ ಬಲದಲ್ಲಿ "Microsoft Foundry" ಆಯ್ಕೆಮಾಡಿ ಹೋಮ್ ಪುಟಕ್ಕೆ ಹೋಗಿ.
   - "+ Create project" ಆಯ್ಕೆಮಾಡಿ.
   - ಪ್ರಾಜೆಕ್ಟ್ ಗೆ ಹೆಸರು ನಮೂದಿಸಿ.
   - ನೀವು ಒಂದು ಹಬ್ ಹೊಂದಿದ್ದರೆ, ಅದು ಡೀಫಾಲ್ಟ್ ಆಗಿ ಆಯ್ಕೆ ಮಾಡಲಾಗುತ್ತದೆ. ನೀವು ಒಂದುಕ್ಕಿಂತ ಹೆಚ್ಚಿನ ಹಬ್ ಗಳಿಗೆ ಪ್ರವೇಶ ಇರುವಿದ್ದರೆ, ಡ್ರಾಪ್‌‌ಡೌನ್ ನಿಂದ ಇತರ ಹಬ್ ಅನ್ನು ಆಯ್ಕೆ ಮಾಡಬಹುದು. ಹೊಸ ಹಬ್ ರಚಿಸಲು "Create new hub" ಆಯ್ಕೆ ಮಾಡಿ ಮತ್ತು ಹೆಸರು ನೀಡಿರಿ.
   - "Create" ಆಯ್ಕೆಮಾಡಿರಿ.

### Microsoft Foundry ನಲ್ಲಿ ಹಬ್ ರಚಿಸುವುದು

1. **Microsoft Foundry ಗೆ ಹೋಗಿ**: ನಿಮ್ಮ Azure ಖಾತೆಯನ್ನು ಬಳಸಿ ಸೈನ್ ಇನ್ ಆಗಿ.
2. **ಹಬ್ ರಚಿಸಿ**:
   - ಎಡ ಮೆನು ನಿಂದ ಮ್ಯಾನೇಜ್‌ಮೆಂಟ್ ಸೆಂಟರ್ ಆಯ್ಕೆಮಾಡಿ.
   - "All resources" ಆಯ್ಕೆಮಾಡಿ, ನಂತರ "+ New project" ಅಡ್ಡArrow ಬಳಸಿ "+ New hub" ಆಯ್ಕೆಮಾಡಿ.
   - "Create a new hub" ಸಂವಾದ ಬಾಕ್ಸಿನಲ್ಲಿ ನಿಮ್ಮ ಹಬ್ ಗೆ ಹೆಸರು ನಮೂದಿಸಿ (ಉದಾಹರಣೆಗೆ, contoso-hub) ಮತ್ತು ಇತರ ಕ್ಷೇತ್ರಗಳನ್ನು ಅಗತ್ಯದಂತೆ ಬದಲಾಯಿಸಿ.
   - "Next" ಆಯ್ಕೆ ಮಾಡಿ ತಥ್ಯಗಳನ್ನು ಪರಿಶೀಲಿಸಿ ಮತ್ತು "Create" ಆಯ್ಕೆಮಾಡಿ.

ವಿಸ್ತೃತ ಮಾರ್ಗಸೂಚಿಗಾಗಿ ಅಧಿಕೃತ [Microsoft ಡಾಕ್ಯುಮೆಂಟೇಶನ್](https://learn.microsoft.com/azure/ai-studio/how-to/create-projects) ಅನ್ನು ನೋಡಿ.

ಯಶಸ್ವಿಯಾಗಿ ರಚಿಸಿದ ನಂತರ, ನೀವು [ai.azure.com](https://ai.azure.com/) ಮೂಲಕ ನೀವು ರಚಿಸಿದ ಸ್ಟುಡಿಯೋವನ್ನು ಪ್ರವೇಶಿಸಬಹುದು.

## **4. Microsoft Foundry ನಿಂದ ಮಾದರಿಯನ್ನು ನಿಯೋಜಿಸುವುದು**

Azure Model Catalog ನಿಂದ ಮಾದರಿಯನ್ನು ನಿಯೋಜಿಸಲು, ಈ ಹಂತಗಳನ್ನು ಅನುಸರಿಸಬಹುದು:

- Microsoft Foundry ಗೆ ಸೈನ್ ಇನ್ ಆಗಿ.
- Microsoft Foundry ಮಾದರಿ ಕ್ಯಾಟಲಾಗ್ ನಿಂದ ನೀವು ನಿಯೋಜಿಸಲು ಇಚ್ಛಿಸುವ ಮಾದರಿಯನ್ನು ಆಯ್ಕೆಮಾಡಿ.
- ಮಾದರಿಯ "Details" ಪುಟದಲ್ಲಿ Deploy ಆಯ್ಕೆಮಾಡಿ ಮತ್ತು ನಂತರ Azure AI Content Safety ಹೊಂದಿರುವ Serverless API ಆಯ್ಕೆಮಾಡಿ.
- ನೀವು ನಿಮ್ಮ ಮಾದರಿಗಳನ್ನು ನಿಯೋಜಿಸಲು ಇಚ್ಛಿಸುವ ಪ್ರಾಜೆಕ್ಟ್ ಆಯ್ಕೆಮಾಡಿ. Serverless API ಸೇವೆಯನ್ನು ಬಳಸಲು, ನಿಮ್ಮ ವರ್ಕ್‌ಸ್ಪೇಸ್ ಈಸ್ಟ್ US 2 ಅಥವಾ ಸ್ವೀಡನ್ ಸೆಂಟ್ರಲ್ ಪ್ರಾಂತ್ಯಕ್ಕೆ ಸೇರಿದಿರಬೇಕು. Deployment ಹೆಸರು ಕಸ್ಟಮೈಸ್ ಮಾಡಬಹುದು.
- ನಿಯೋಜನೆ ಮಾರ್ಗದರ್ಶಿಯಲ್ಲಿ, ಬೆಲೆ ಮತ್ತು ನಿಯಮಾವಳಿಗಳನ್ನು ತಿಳಿಯಲು Pricing and terms ಆಯ್ಕೆಮಾಡಿ.
- Deploy ಆಯ್ಕೆಮಾಡಿ. ನಿಯೋಜನೆ ಸಿದ್ಧವಾಗುವವರೆಗೆ ಕಾಯಿರಿ ಮತ್ತು ನೀವು Deployments ಪುಟಕ್ಕೆ ಮಾರ್ಗದರ್ಶಿಸಲ್ಪಡುವಿರಿ.
- ಮಾದರಿಯೊಂದಿಗೆ ಸಂವಹನ ಪ್ರಾರಂಭಿಸಲು Open in playground ಆಯ್ಕೆಮಾಡಿ.
- Deployments ಪುಟಕ್ಕೆ ಮರಳಿ ಹೋಗಿ, ನಿಯೋಜನೆಯನ್ನು ಆಯ್ಕೆಮಾಡಿ ಮತ್ತು ಗುರಿ URL ಹಾಗೂ Secret Key ಅನ್ನು ಗಮನಿಸಿ, ನೀವು ನಿಯೋಜನೆಯನ್ನು ಕರೆಮಾಡಲು ಮತ್ತು ಪೂರ್ಣಗೊಳಿಸುವಿಕೆಗಳನ್ನು ಉತ್ಪಾದಿಸಲು ಬಳಸಬಹುದು.
- ನೀವು ಯಾವಾಗಲೂ Build ಟ್ಯಾಬ್ ಗೆ ಹೋಗಿ Components ವಿಭಾಗದಿಂದ Deployments ಆಯ್ಕೆಮಾಡಿ ಗುರಿ ವಿವರಗಳು, URL ಮತ್ತು ಪ್ರವೇಶ ಕೀಗಳನ್ನು ಪಡೆಯಬಹುದು.

> [!NOTE]
> ದಯವಿಟ್ಟು ಗಮನಿಸಿ ನಿಮ್ಮ ಖಾತೆಗೆ Resource Group ನಲ್ಲಿ Azure AI Developer ಪಾತ್ರ ಅನುಮತಿಗಳು ಇರಬೇಕು ಈ ಹಂತಗಳನ್ನು ನೆರವೇರಿಸಲು.

## **5. Microsoft Foundry ನಲ್ಲಿ Phi API ಬಳಕೆ**
