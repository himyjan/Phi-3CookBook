## Kom igång med Phi-Family-modeller i Foundry Local

### Introduktion till Foundry Local

Foundry Local är en kraftfull AI-inferenslösning på enheten som ger företagsklassade AI-funktioner direkt till din lokala hårdvara. Denna handledning guidar dig genom att sätta upp och använda Phi-Family-modeller med Foundry Local, vilket ger dig fullständig kontroll över dina AI-arbetsbelastningar samtidigt som sekretess upprätthålls och kostnader minskas.

Foundry Local erbjuder prestanda, integritet, anpassning och kostnadsfördelar genom att köra AI-modeller lokalt på din enhet. Den integreras sömlöst i dina befintliga arbetsflöden och applikationer via ett intuitivt CLI, SDK och REST API.


![arch](../../../../../translated_images/sv/foundry-local-arch.8823e321dd8258d7.webp)

### Varför välja Foundry Local?

Att förstå fördelarna med Foundry Local hjälper dig att fatta välgrundade beslut om din AI-distributionsstrategi:

- **Inferens på enheten:** Kör modeller lokalt på din egen hårdvara, vilket minskar dina kostnader samtidigt som all din data stannar på din enhet.

- **Modellanpassning:** Välj bland förinställda modeller eller använd dina egna för att möta specifika krav och användningsfall.

- **Kostnadseffektivitet:** Eliminera återkommande molntjänstkostnader genom att använda din befintliga hårdvara, vilket gör AI mer tillgängligt.

- **Sömlös integration:** Anslut till dina applikationer via SDK, API-endpoints eller CLI, med enkel skalning till Microsoft Foundry när dina behov växer.

> **Notering för igångsättning:** Denna handledning fokuserar på att använda Foundry Local via CLI och SDK-gränssnitt. Du får lära dig båda metoder för att kunna välja den bästa lösningen för ditt användningsfall.

## Del 1: Installera Foundry Local CLI

### Steg 1: Installation

Foundry Local CLI är din port till att hantera och köra AI-modeller lokalt. Vi börjar med att installera det på ditt system.

**Stödda plattformar:** Windows och macOS

För detaljerade installationsinstruktioner, vänligen se den [officiella Foundry Local-dokumentationen](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Steg 2: Utforska tillgängliga modeller

När du har Foundry Local CLI installerat kan du upptäcka vilka modeller som finns tillgängliga för ditt användningsfall. Detta kommando visar alla stödjade modeller:


```bash
foundry model list
```

### Steg 3: Förstå Phi Family-modeller

Phi Family erbjuder en rad modeller optimerade för olika användningsfall och hårdvarukonfigurationer. Här är Phi-modellerna som finns i Foundry Local:

**Tillgängliga Phi-modeller:** 

- **phi-3.5-mini** - Kompakt modell för grundläggande uppgifter
- **phi-3-mini-128k** - Version med förlängt kontext för längre konversationer
- **phi-3-mini-4k** - Standardmodell med normal kontext för allmänt bruk
- **phi-4** - Avancerad modell med förbättrade möjligheter
- **phi-4-mini** - Lättviktsversion av Phi-4
- **phi-4-mini-reasoning** - Specialiserad för komplexa resonemangsuppgifter

> **Hårdvarukompatibilitet:** Varje modell kan konfigureras för olika hårdvaruacceleration (CPU, GPU) beroende på din systems kapacitet.

### Steg 4: Kör din första Phi-modell

Låt oss börja med ett praktiskt exempel. Vi kör `phi-4-mini-reasoning`-modellen, som utmärker sig i att lösa komplexa problem steg för steg.


**Kommando för att köra modellen:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Första gången-setup:** När du kör en modell första gången kommer Foundry Local automatiskt att ladda ner den till din enhet. Nedladdningstiden varierar beroende på din nätverkshastighet, så var vänlig och ha tålamod under den initiala installationen.

### Steg 5: Testa modellen med ett verkligt problem

Nu testar vi vår modell med ett klassiskt logikproblem för att se hur den presterar med steg-för-steg-resonemang:

**Exempelproblem:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Förväntat beteende:** Modellen bör dela upp problemet i logiska steg, använda faktumet att fasaner har 2 ben och kaniner 4 ben för att lösa ekvationssystemet.

## Del 2: Bygga applikationer med Foundry Local SDK

### Varför använda SDK?

Medan CLI är perfekt för tester och snabba interaktioner, möjliggör SDK att integrera Foundry Local i dina applikationer programmässigt. Detta öppnar möjligheter för:

- Att bygga skräddarsydda AI-drivna applikationer
- Skapa automatiserade arbetsflöden
- Integrera AI-funktionalitet i befintliga system
- Utveckla chatbots och interaktiva verktyg

### Stödda programmeringsspråk

Foundry Local erbjuder SDK-stöd för flera programmeringsspråk för att passa dina utvecklingspreferenser:

**📦 Tillgängliga SDK:er:**

- **C# (.NET):** [SDK-dokumentation & exempel](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK-dokumentation & exempel](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK-dokumentation & exempel](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK-dokumentation & exempel](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### Nästa steg

1. **Välj din föredragna SDK** baserat på din utvecklingsmiljö
2. **Följ SDK-specifik dokumentation** för detaljerade implementationsguider
3. **Börja med enkla exempel** innan du bygger komplexa applikationer
4. **Utforska exempelkoden** som finns i varje SDK-förråd
