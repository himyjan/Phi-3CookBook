# Phi 硬體支援

Microsoft Phi 已針對 ONNX Runtime 進行優化，並支援 Windows DirectML。它在各種硬體類型上運作良好，包括 GPU、CPU，甚至是行動裝置。

## 裝置硬體  
具體支援的硬體包括：

- GPU SKU：RTX 4090（DirectML）
- GPU SKU：1 個 A100 80GB（CUDA）
- CPU SKU：Standard F64s v2（64 vCPU，128 GiB 記憶體）

## 行動版 SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 或更高版本 A16/A17 處理器

## Phi 硬體規格

- 最低配置需求。
- Windows：支援 DirectX 12 的 GPU，及至少 4GB 的合併記憶體

CUDA：計算能力 >= 7.02 的 NVIDIA GPU

![HardwareSupport](../../../../../translated_images/zh-TW/01.phihardware.5d51b2377cba18af.webp)

## 在多 GPU 上執行 onnxruntime

目前可用的 Phi ONNX 模型僅支援單 GPU。Phi 模型是有可能支援多 GPU，但使用 2 GPU 的 ORT 不保證會比執行 2 個獨立的 ORT 實例提供更高的效能。請參考 [ONNX Runtime](https://onnxruntime.ai/) 以取得最新消息。

在 [Build 2024 的 GenAI ONNX 團隊](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) 宣布，他們已為 Phi 模型啟用多實例而非多 GPU。

目前這讓你能以 CUDA_VISIBLE_DEVICES 環境變數，執行一個 onnnxruntime 或 onnxruntime-genai 實例，如以下範例。

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

歡迎在 [Microsoft Foundry](https://ai.azure.com) 深入探索 Phi。
