# 微軟的 Phi 系列

Phi 模型是目前最具能力且具性價比的小型語言模型（SLMs），在各種語言、推理、程式碼、音訊、視覺與數學基準測試中，表現勝過同尺寸及下一尺寸的模型。此次發布擴展了高品質模型的選擇，為客戶提供更多實用的選擇，以組合和構建生成式 AI 應用。

Phi 系列始於用於 Python 程式碼生成的 Phi-1，接續推出基於文本與聊天完成的 Phi-1.5 / 2、Phi-3-mini/small/medium-instruct 及 Phi-3.5/4-mini-instruct，並發展到專注視覺的 Phi-3/3.5-vision、基於強推理的 Phi-4、採用 MoE 的 Phi-3.5-MoE，現在則推出全模態模型 Phi-4-multimodal。透過高品質數據集，基準可訓練出與訓練參數更多的模型相當的效能。

## 模型選擇範例

| | | | |
|-|-|-|-|
|客戶需求|任務|起始模型|更多細節|
|需要一個能簡單摘要訊息串的模型|對話摘要|Phi-3 / 3.5 文本模型|此處決定因素是客戶有明確且直白的語言任務|
