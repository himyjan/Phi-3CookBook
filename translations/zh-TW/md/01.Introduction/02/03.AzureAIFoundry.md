## Microsoft Foundry 中的 Phi 系列

[Microsoft Foundry](https://ai.azure.com) 是一個值得信賴的平台，讓開發者能夠以安全、可靠且負責任的方式推動創新並塑造 AI 的未來。



[Microsoft Foundry](https://ai.azure.com) 旨在讓開發者能夠：

- 在企業級平台上構建生成式 AI 應用程式。
- 利用尖端 AI 工具和機器學習模型進行探索、建構、測試和部署，並以負責任的 AI 實踐為基礎。
- 與團隊協作完成應用程式開發的整個生命週期。

透過 Microsoft Foundry，您可以探索各式各樣的模型、服務和功能，並著手打造最符合您目標的 AI 應用程式。Microsoft Foundry 平台促進了將概念驗證輕鬆轉變為成熟生產應用程式的可擴充性。持續的監控和改進有助於長期成功。

### **使用 Python 代碼呼叫 Microsoft Foundry Phi-4**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# 如果語音已啟用，則包含語音結果
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
