# Phi Family in AITK

[AI Toolkit for VS Code](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) 簡化了生成式 AI 應用程式的開發，將微軟 Foundry 目錄及 Hugging Face 等其他目錄的尖端 AI 開發工具與模型整合在一起。您將能夠瀏覽由 GitHub Models 與 Microsoft Foundry Model Catalogs 支援的 AI 模型目錄，並可將它們下載到本地或遠端，進行微調、測試並在您的應用程式中使用。

AI Toolkit 預覽版將在本地執行。根據所選模型，本地推論或微調可能需要具有 NVIDIA CUDA GPU 的 GPU。您也可以直接使用 AITK 執行 GitHub Models。

### 安裝 AI Toolkit

AI Toolkit 作為[Visual Studio Code 擴充套件](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions)發布，因此您需要先安裝[VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo)，然後從[VS Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio)下載 AI Toolkit。
[AI Toolkit 可在 Visual Studio Marketplace 中取得](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio)，並可像其他 VS Code 擴充套件一樣進行安裝。

如果您不熟悉如何安裝 VS Code 擴充套件，請依照以下步驟：

### 登入

1. 在 VS Code 活動欄中選擇 **Extensions**
1. 在 Extensions 搜尋欄輸入 "AI Toolkit"
1. 選取 "AI Toolkit for Visual Studio code"
1. 選擇 **Install**

現在，您已準備好使用此擴充套件！

系統會提示您登入 GitHub，請點選「Allow」繼續。您會被導向到 GitHub 登入頁面。

請登入並依指示完成流程。成功完成後，您將被重新導向回 VS Code。

安裝完成後，您會在活動欄中看到 AI Toolkit 的圖示。

### 在您的應用程式中使用 REST API

AI Toolkit 附帶本地 REST API 網路伺服器 **在 5272 端口**，並採用[OpenAI chat completions 格式](https://platform.openai.com/docs/api-reference/chat/create)。

這讓您得以在本地測試應用程式，無需依賴雲端 AI 模型服務。例如，以下 JSON 文件展示如何配置請求主體：

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```

您可以使用（例如）[Postman](https://www.postman.com/) 或 CURL（用戶端 URL）工具測試 REST API：

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```

### 使用 Python 的 OpenAI 客戶端庫

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # API 所必需，但未使用
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```

### 使用 .NET 的 Azure OpenAI 客戶端庫

透過 NuGet 將 [.NET Azure OpenAI 客戶端庫](https://www.nuget.org/packages/Azure.AI.OpenAI/) 加入您的專案：

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```

新增一個名為 **OverridePolicy.cs** 的 C# 檔案到您的專案，並貼上以下程式碼：

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```

接著，將以下程式碼貼到您的 **Program.cs** 檔案：

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
