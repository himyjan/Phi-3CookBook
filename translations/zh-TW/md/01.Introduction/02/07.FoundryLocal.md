## 在 Foundry Local 中開始使用 Phi-Family 模型

### Foundry Local 介紹

Foundry Local 是一個強大的裝置端 AI 推論解決方案，能將企業級 AI 能力直接帶到您的本地硬體中。本教學將引導您設定和使用 Phi-Family 模型搭配 Foundry Local，讓您完全掌控 AI 工作負載，同時維護隱私並降低成本。

Foundry Local 透過在您的裝置上本地運行 AI 模型，提供效能、隱私、自訂化和成本優勢。它通過直觀的 CLI、SDK 和 REST API 無縫整合到您現有的工作流程和應用程式中。


![arch](../../../../../translated_images/zh-TW/foundry-local-arch.8823e321dd8258d7.webp)

### 為什麼選擇 Foundry Local？

理解 Foundry Local 的優點將幫助您做出明智的 AI 部署策略決策：

- **裝置端推論：** 在您自己的硬體上本地運行模型，降低成本，同時所有資料均留在您的裝置上。

- **模型自訂化：** 從預設模型中選擇或使用您自己的模型，以符合特定需求和使用案例。

- **成本效益：** 利用現有硬體，消除反覆的雲端服務費用，讓 AI 更加普及。

- **無縫整合：** 透過 SDK、API 端點或 CLI 連接您的應用程式，隨著需求成長輕鬆擴展至 Microsoft Foundry。

> **入門提醒：** 本教學聚焦使用 Foundry Local 透過 CLI 和 SDK 介面，您將學習兩種方法，協助選擇最適合您使用案例的方式。

## 第 1 部分：設定 Foundry Local CLI

### 步驟 1：安裝

Foundry Local CLI 是您管理和本地運行 AI 模型的門戶。讓我們先在系統上安裝它。

**支援平台：** Windows 和 macOS

詳細安裝說明，請參閱[官方 Foundry Local 文件](https://github.com/microsoft/Foundry-Local/blob/main/README.md)。

### 步驟 2：瀏覽可用模型

安裝 Foundry Local CLI 後，您可以查看有哪些模型適用於您的使用案例。此指令會顯示所有支援的模型：


```bash
foundry model list
```

### 步驟 3：認識 Phi 家族模型

Phi Family 提供一系列針對不同使用案例和硬體配置優化的模型。以下為 Foundry Local 中可用的 Phi 模型：

**可用的 Phi 模型：**

- **phi-3.5-mini** - 用於基本任務的緊湊型模型
- **phi-3-mini-128k** - 延伸上下文版本，適合較長對話
- **phi-3-mini-4k** - 一般用途的標準上下文模型
- **phi-4** - 擁有改良能力的進階模型
- **phi-4-mini** - phi-4 的輕量版本
- **phi-4-mini-reasoning** - 專門用於複雜推理任務

> **硬體相容性：** 每款模型可根據您的系統配置，設定不同的硬體加速（CPU、GPU）。

### 步驟 4：運行您的第一個 Phi 模型

讓我們來一個實際範例。將運行擅長逐步解決複雜問題的 `phi-4-mini-reasoning` 模型。


**運行模型指令：**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **首次設定：** 第一次運行模型時，Foundry Local 會自動下載模型至您的本地裝置。因網路速度不同，下載時間會有所差異，請耐心等待初次設定完成。
