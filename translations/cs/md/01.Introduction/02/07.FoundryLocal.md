## Začínáme s modely Phi-Family ve Foundry Local

### Úvod do Foundry Local

Foundry Local je výkonné řešení pro AI inference na zařízení, které přináší podnikové AI schopnosti přímo do vašeho lokálního hardwaru. Tento návod vás provede nastavením a používáním modelů Phi-Family s Foundry Local, nabízí vám úplnou kontrolu nad vašimi AI úlohami při zachování soukromí a snížení nákladů.

Foundry Local poskytuje výhody v oblasti výkonu, soukromí, přizpůsobení a nákladové efektivity díky lokalnímu provozu AI modelů na vašem zařízení. Integruje se hladce do vašich stávajících pracovních postupů a aplikací pomocí intuitivního CLI, SDK a REST API.


![arch](../../../../../translated_images/cs/foundry-local-arch.8823e321dd8258d7.webp)

### Proč zvolit Foundry Local?

Pochopení výhod Foundry Local vám pomůže činit informovaná rozhodnutí o strategii nasazení AI:

- **Inference na zařízení:** Spouštějte modely přímo na vašem hardwaru, čímž snížíte náklady a zároveň uchováte veškerá data na svém zařízení.

- **Přizpůsobení modelů:** Vyberte si z přednastavených modelů nebo použijte své vlastní pro specifické požadavky a scénáře použití.

- **Nákladová efektivita:** Odstraňte opakující se náklady na cloudové služby tím, že využijete svůj existující hardware, což činí AI dostupnější.

- **Bezproblémová integrace:** Připojte se k aplikacím přes SDK, API endpointy nebo CLI, s jednoduchým škálováním do Microsoft Foundry podle vašich potřeb.

> **Poznámka k začátkům:** Tento návod se zaměřuje na používání Foundry Local přes rozhraní CLI a SDK. Naučíte se oba přístupy, abyste mohli vybrat nejlepší metodu pro svůj případ použití.

## Část 1: Nastavení Foundry Local CLI

### Krok 1: Instalace

Foundry Local CLI je vaším vstupem pro správu a spuštění AI modelů lokálně. Začněme jeho instalací na vašem systému.

**Podporované platformy:** Windows a macOS

Podrobné instrukce k instalaci naleznete v [oficiální dokumentaci Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Krok 2: Prozkoumání dostupných modelů

Jakmile máte Foundry Local CLI nainstalované, můžete zjistit, jaké modely jsou dostupné pro vaše použití. Tento příkaz vám ukáže všechny podporované modely:


```bash
foundry model list
```

### Krok 3: Pochopení modelů Phi Family

Rodina Phi nabízí řadu modelů optimalizovaných pro různé případy použití a konfigurace hardwaru. Zde jsou Phi modely dostupné ve Foundry Local:

**Dostupné Phi modely:** 

- **phi-3.5-mini** - Kompaktní model pro základní úlohy
- **phi-3-mini-128k** - Verze s rozšířeným kontextem pro delší konverzace
- **phi-3-mini-4k** - Standardní model s kontextem pro obecné použití
- **phi-4** - Pokročilý model s vylepšenými schopnostmi
- **phi-4-mini** - Lehká verze Phi-4
- **phi-4-mini-reasoning** - Specializovaný na složité úlohy uvažování

> **Kompatibilita hardwaru:** Každý model lze nakonfigurovat pro různé hardwarové akcelerace (CPU, GPU) podle schopností vašeho systému.

### Krok 4: Spuštění prvního Phi modelu

Začněme praktickým příkladem. Spustíme model `phi-4-mini-reasoning`, který vyniká v řešení složitých problémů krok za krokem.


**Příkaz ke spuštění modelu:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **První nastavení:** Při prvním spuštění modelu jej Foundry Local automaticky stáhne na vaše lokální zařízení. Doba stahování závisí na rychlosti vaší sítě, buďte prosím trpěliví během počátečního nastavení.

### Krok 5: Testování modelu na reálném problému

Teď otestujme náš model na klasickém logickém problému, abychom viděli, jak provádí krok za krokem uvažování:

**Ukázkový problém:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Očekávané chování:** Model by měl rozdělit tento problém na logické kroky a využít fakt, že bažanti mají 2 nohy a králíci 4 nohy, k vyřešení soustavy rovnic.
