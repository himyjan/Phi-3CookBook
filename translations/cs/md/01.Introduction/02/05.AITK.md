# Rodina Phi v AITK

[AI Toolkit pro VS Code](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) usnadňuje vývoj aplikací generativní AI tím, že přináší špičkové nástroje pro vývoj AI a modely z Microsoft Foundry Catalog a dalších katalogů jako Hugging Face. Budete moci procházet katalog AI modelů poháněný GitHub Models a Microsoft Foundry Model Catalogs, stahovat je lokálně nebo vzdáleně, doladit, testovat a používat je ve své aplikaci.

Preview AI Toolkitu poběží lokálně. Lokální inference nebo doladění závisí na vybraném modelu, pro některé můžete potřebovat GPU jako NVIDIA CUDA GPU. GitHub Modely můžete spouštět přímo také přes AITK.

### Instalace AI Toolkit

AI Toolkit je distribuován jako [rozšíření Visual Studio Code](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions), proto musíte nejdříve nainstalovat [VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo) a stáhnout AI Toolkit z [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio).
[AI Toolkit je dostupný ve Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) a lze jej nainstalovat jako kterákoliv jiná rozšíření VS Code.

Pokud neznáte instalaci rozšíření VS Code, postupujte podle těchto kroků:

### Přihlášení

1. V panelu aktivit VS Code vyberte **Rozšíření**
1. Do vyhledávacího pole zadejte "AI Toolkit"
1. Vyberte "AI Toolkit for Visual Studio code"
1. Klikněte na **Instalovat**

Nyní jste připraveni rozšíření používat!

Budete vyzváni k přihlášení na GitHub, klikněte prosím na "Povolit" pro pokračování. Budete přesměrováni na přihlašovací stránku GitHubu.

Přihlaste se a postupujte podle kroků procesu. Po úspěšném dokončení budete přesměrováni zpět do VS Code.

Po instalaci rozšíření se v panelu aktivit objeví ikona AI Toolkitu.

Pojďme prozkoumat dostupné akce!

### Dostupné akce

Primární boční panel AI Toolkitu je organizován do:

- **Modely**
- **Zdroje**
- **Hřiště**  
- **Doladění**
- **Hodnocení**

Jsou dostupné v sekci Zdroje. Pro začátek vyberte **Model Catalog**.

### Stažení modelu z katalogu

Po spuštění AI Toolkit z postranního panelu VS Code můžete vybrat z následujících možností:

![AI toolkit model catalog](../../../../../translated_images/cs/AItoolkitmodel_catalog.7a7be6a7d8468d31.webp)

- Najít podporovaný model v **Model Catalog** a stáhnout lokálně
- Otestovat inference modelu v **Model Playground**
- Doladit model lokálně nebo vzdáleně v **Model Fine-tuning**
- Nasadit doladěné modely do cloudu přes příkazovou paletu AI Toolkitu
- Hodnocení modelů

> [!NOTE]
>
> **GPU vs CPU**
>
> Modelové karty zobrazují velikost modelu, platformu a typ akcelerátoru (CPU, GPU). Pro optimalizovaný výkon na **zařízeních Windows, která mají alespoň jedno GPU**, vyberte modelové verze cílené pouze na Windows.
>
> To zajistí, že máte model optimalizovaný pro akcelerátor DirectML.
>
> Názvy modelů mají formát
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
> Chcete-li zkontrolovat, zda máte GPU na svém zařízení Windows, otevřete **Správce úloh** a přejděte na kartu **Výkon**. Pokud máte GPU, budou uvedena pod názvy jako "GPU 0" nebo "GPU 1".

### Použití REST API ve vaší aplikaci

AI Toolkit obsahuje lokální webový server REST API **na portu 5272**, který používá [formát OpenAI chat completions](https://platform.openai.com/docs/api-reference/chat/create).

To umožňuje testovat vaši aplikaci lokálně, bez potřeby spoléhat se na AI model v cloudu. Například níže uvedený JSON soubor ukazuje, jak nakonfigurovat tělo požadavku:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```

REST API můžete testovat pomocí (například) [Postman](https://www.postman.com/) nebo nástroje CURL (Client URL):

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```

### Použití klientské knihovny OpenAI pro Python

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # vyžadováno pro API, ale nepoužívá se
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```

### Použití Azure OpenAI klientské knihovny pro .NET

Přidejte do svého projektu [Azure OpenAI klientskou knihovnu pro .NET](https://www.nuget.org/packages/Azure.AI.OpenAI/) pomocí NuGet:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```

Do projektu přidejte soubor C# s názvem **OverridePolicy.cs** a vložte následující kód:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```

Následně vložte následující kód do souboru **Program.cs**:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
