# 微軟的 Phi 系列

Phi 模型是目前最強大且具成本效益的小型語言模型（SLMs），在多種語言、推理、程式碼、音訊、視覺及數學基準測試中，表現超越同尺寸及更大尺寸的模型。這次發布擴充了高品質模型的選擇，為客戶提供了更多實用的選擇，以便組合並構建生成式 AI 應用。

Phi 系列始於用於 Python 程式碼生成的 Phi-1，接續推出基於文本與聊天完成的 Phi-1.5 / 2，Phi-3-mini/small/medium-instruct 及 Phi-3.5/4-mini-instruct，並發展出用於視覺的 Phi-3/3.5-vision、基於強大推理的 Phi-4、MoE 的 Phi-3.5-MoE，以及現在的全模態模型 Phi-4-multimodal。透過高質量的數據集，訓練後的基準測試表現可媲美擁有更多訓練參數的模型。

## 模型選擇範例

| | | | |
|-|-|-|-|
|客戶需求|任務|建議起點|更多資料|
|需要一個模型簡單總結一串訊息|對話摘要|Phi-3 / 3.5 文字模型|決策因素是客戶的語言任務明確且直接 |
