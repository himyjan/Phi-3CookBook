# <strong>介紹負責任的人工智能</strong>

[Microsoft Responsible AI](https://www.microsoft.com/ai/responsible-ai?WT.mc_id=aiml-138114-kinfeylo) 是一項旨在幫助開發人員和組織建立透明、值得信賴及具問責性的人工智能系統的計劃。該計劃提供指導和資源，協助開發符合倫理原則（如隱私、公平性和透明度）的負責任人工智能解決方案。我們還將探討一些與建立負責任人工智能系統相關的挑戰和最佳實踐。

## Microsoft 負責任人工智能概述

![RAIPrinciples](../../../../../translated_images/zh-HK/RAIPrinciples.bf9c9bc6ca160d33.webp)

<strong>倫理原則</strong>

Microsoft 負責任人工智能以一套倫理原則為指導，例如隱私、公平性、透明度、問責性和安全性。這些原則旨在確保人工智能系統以倫理且負責任的方式開發。

<strong>透明的人工智能</strong>

Microsoft 負責任人工智能強調人工智能系統透明度的重要性。這包括提供清晰的人工智能模型運作解釋，以及確保資料來源和算法公開可得。

<strong>具問責性的人工智能</strong>

[Microsoft Responsible AI](https://www.microsoft.com/ai/responsible-ai?WT.mc_id=aiml-138114-kinfeylo) 推動開發具問責性的人工智能系統，這些系統可提供人工智能模型決策過程的見解，幫助使用者理解和信任人工智能系統的輸出。

<strong>包容性</strong>

人工智能系統應設計為惠及所有人。Microsoft 致力於創造包容性的人工智能，考慮多元觀點並避免偏見或歧視。

<strong>可靠性與安全</strong>

確保人工智能系統可靠和安全至關重要。Microsoft 專注於建立穩健的模型，使其表現穩定且避免產生有害結果。

<strong>人工智能的公平性</strong>

Microsoft 負責任人工智能認識到，如果使用偏見的資料或算法訓練人工智能系統，該系統可能延續偏見。該計劃提供指導，協助開發不基於種族、性別或年齡等因素歧視的公平人工智能系統。

<strong>隱私與安全</strong>

Microsoft 負責任人工智能強調保護使用者隱私和資料安全的重要性。這包括實施強大的資料加密和存取控制，並定期審核人工智能系統的漏洞。

<strong>問責與責任</strong>

Microsoft 負責任人工智能推廣人工智能開發和部署中的問責和責任，確保開發者和組織了解人工智能系統相關的潛在風險，並採取措施降低這些風險。

## 建立負責任人工智能系統的最佳實踐

<strong>使用多元資料集開發人工智能模型</strong>

為避免人工智能系統中的偏見，重要的是使用代表多元觀點和經驗的多元資料集。

<strong>使用可解釋的人工智能技術</strong>

可解釋的人工智能技術可幫助使用者理解人工智能模型如何做決策，從而提升對系統的信任。

<strong>定期審核人工智能系統的漏洞</strong>

定期審核人工智能系統可以幫助識別需要處理的潛在風險和漏洞。

<strong>實施強大的資料加密和存取控制</strong>

資料加密和存取控制有助於保護人工智能系統中的使用者隱私和安全。

<strong>在人工智能開發中遵循倫理原則</strong>

遵循公平、透明和問責等倫理原則有助於建立對人工智能系統的信任，確保其以負責任的方式開發。

## 使用 AI Foundry 實現負責任的人工智能

[Microsoft Foundry](https://ai.azure.com?WT.mc_id=aiml-138114-kinfeylo) 是一個強大的平台，讓開發人員和組織能夠快速創建智能、尖端、市場就緒且負責任的應用程序。以下是 Microsoft Foundry 的一些主要功能和能力：

**即用型 API 和模型**

Microsoft Foundry 提供預建且可定制的 API 和模型，涵蓋廣泛人工智能任務，包括生成式人工智能、用於對話的自然語言處理、搜索、監控、翻譯、語音、視覺及決策等。

<strong>提示流程</strong>

Microsoft Foundry 中的提示流程讓您能夠創建對話式人工智能體驗。它允許您設計和管理對話流程，便於構建聊天機器人、虛擬助理和其他互動應用程序。

**檢索增強生成（RAG）**

RAG 是一種結合檢索式和生成式方法的技術，通過利用既有知識（檢索）和創造性生成（生成）提升生成回應的質量。

<strong>生成式人工智能的評估與監控指標</strong>

Microsoft Foundry 提供評估和監控生成式人工智能模型的工具。您可以評估其性能、公平性及其他重要指標，以確保負責任的部署。此外，如果您建立了儀表板，可使用 Azure Machine Learning Studio 中的無代碼 UI 定制並生成基於[Responsible AI Toolbox](https://responsibleaitoolbox.ai/?WT.mc_id=aiml-138114-kinfeylo) Python 庫的負責任人工智能儀表板和相關分數卡。這個分數卡有助於與技術和非技術持份者分享與公平性、特徵重要性及其他負責任部署考量相關的主要見解。

使用 AI Foundry 推動負責任人工智能時，您可以遵循以下最佳實踐：

<strong>明確定義人工智能系統的問題與目標</strong>

在開始開發過程前，明確定義人工智能系統旨在解決的問題或目標，這將有助於識別構建有效模型所需的資料、算法和資源。

<strong>蒐集並預處理相關資料</strong>

用於訓練人工智能系統的資料質量及數量對其性能有重大影響。因此，蒐集相關資料、清理並預處理資料，確保其能代表您欲解決的人群或問題非常重要。

<strong>選擇適當的評估算法</strong>

市面上有多種評估算法，根據您的資料和問題選擇最合適的算法非常重要。

<strong>評估並解釋模型</strong>

建立人工智能模型後，使用適當的指標評估其性能並以透明的方式解釋結果，這將幫助您識別模型中任何偏見或限制，並在必要時加以改進。

<strong>確保透明性和可解釋性</strong>

人工智能系統應該具備透明性和可解釋性，讓使用者理解其運作方式及決策過程。這對於對人類生活有重大影響的應用，如醫療、金融和法律系統尤為重要。

<strong>監控並更新模型</strong>

人工智能系統應持續監控和更新，確保其隨時間保持準確和有效。這需要不斷的維護、測試和重新訓練模型。
