## Getting Started with Phi-Family Models in Foundry Local

### Introduction to Foundry Local

Foundry Local 是一款強大的本地 AI 推理解決方案，將企業級 AI 能力直接帶到您的本地硬件。這個教程將指導您如何使用 Foundry Local 設置和使用 Phi-Family 模型，讓您完全掌控 AI 工作負載，同時保持隱私並降低成本。

Foundry Local 通過在您的設備上本地運行 AI 模型，提供性能、隱私、客製化和成本優勢。它通過直覺的 CLI、SDK 和 REST API 無縫整合到您現有的工作流程和應用中。


![arch](../../../../../translated_images/zh-MO/foundry-local-arch.8823e321dd8258d7.webp)

### Why Choose Foundry Local?

了解 Foundry Local 的優勢，有助於您對 AI 部署策略做出明智的選擇：

- **On-Device Inference:** 在您自己的硬件上本地運行模型，降低成本，同時確保所有數據留在您的設備上。

- **Model Customization:** 從預設模型中選擇或使用您自己的模型，以滿足特定需求和使用場景。

- **Cost Efficiency:** 使用您現有的硬件，消除重複雲端服務成本，使 AI 更易得。

- **Seamless Integration:** 通過 SDK、API 端點或 CLI 與您的應用連接，並可根據需求輕鬆擴展至 Microsoft Foundry。

> **Getting Started Note:** 本教程聚焦於通過 CLI 和 SDK 介面使用 Foundry Local。您將學習這兩種方法，幫助您選擇最適合您使用場景的方式。

## Part 1: Setting Up Foundry Local CLI

### Step 1: Installation

Foundry Local CLI 是您管理和本地運行 AI 模型的入口。讓我們先在您的系統上安裝它。

**Supported Platforms:** Windows 和 macOS

詳細的安裝說明，請參閱 [官方 Foundry Local 文件](https://github.com/microsoft/Foundry-Local/blob/main/README.md)。

### Step 2: Exploring Available Models

安裝好 Foundry Local CLI 後，您可以查看可用於您使用需求的模型。這條命令將顯示所有支援的模型：


```bash
foundry model list
```

### Step 3: Understanding Phi Family Models

Phi Family 提供了一系列針對不同使用場景和硬件配置優化的模型。以下是在 Foundry Local 可用的 Phi 模型：

**Available Phi Models:** 

- **phi-3.5-mini** - 用於基本任務的精簡模型
- **phi-3-mini-128k** - 長對話擴展上下文版本
- **phi-3-mini-4k** - 一般用途的標準上下文模型
- **phi-4** - 具備提升能力的進階模型
- **phi-4-mini** - Phi-4 的輕量版本
- **phi-4-mini-reasoning** - 專門用於複雜推理任務

> **Hardware Compatibility:** 每個模型可依系統能力配置不同硬件加速（CPU、GPU）。

### Step 4: Running Your First Phi Model

讓我們從一個實際示例開始。我們將運行在步驟推導複雜問題上表現優異的 `phi-4-mini-reasoning` 模型。


**Command to run the model:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **First-Time Setup:** 初次運行模型時，Foundry Local 會自動下載模型到您的本地設備。根據您的網絡速度，下載時間會有所不同，請耐心等待初始設置完成。

### Step 5: Testing the Model with a Real Problem

現在讓我們用一個經典邏輯問題測試模型，看看它如何逐步推理：

**Example Problem:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Expected Behavior:** 模型應把問題拆解成邏輯步驟，利用野雞有 2 條腿，兔子有 4 條腿的事實來解這個方程組。
