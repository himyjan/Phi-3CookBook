# Phi 硬件支援

Microsoft Phi 已針對 ONNX Runtime 進行優化，並支援 Windows DirectML。它在各種硬件類型上表現良好，包括 GPU、CPU，甚至是行動裝置。

## 裝置硬件  
具體支援的硬件包括：

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPUs, 128 GiB 記憶體)

## 行動裝置 SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 或更高版本 A16/A17 處理器

## Phi 硬件規格

- 最低配置需求。
- Windows：支援 DirectX 12 的 GPU 及至少 4GB 的合併記憶體

CUDA：NVIDIA GPU，運算能力 >= 7.02

## 在多 GPU 上運行 onnxruntime

目前可用的 Phi ONNX 模型僅支援 1 GPU。Phi 模型有可能支援多 GPU，但使用 2 GPU 的 ORT 並不保證其吞吐量會優於運行 2 個 ORT 實例。請參閱 [ONNX Runtime](https://onnxruntime.ai/) 以獲取最新資訊。

於 [Build 2024 the GenAI ONNX Team](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) 宣布，他們已對 Phi 模型啟用多實例而非多 GPU。

目前，您可以使用 CUDA_VISIBLE_DEVICES 環境變數，運行一個 onnnxruntime 或 onnxruntime-genai 實例，如下所示。

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

歡迎於 [Microsoft Foundry](https://ai.azure.com) 進一步探索 Phi。
