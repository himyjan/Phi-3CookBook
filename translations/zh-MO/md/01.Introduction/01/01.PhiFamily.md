# 微軟的 Phi 系列

Phi 模型是目前最具能力且具成本效益的小型語言模型（SLM），在多種語言、推理、編碼、音訊、視覺及數學基準測試中，表現優於相同規模及下一個規模以上的模型。此次發佈擴展了高質量模型的選擇，為客戶提供更多實用選擇，用於撰寫和構建生成式 AI 應用程序。

Phi 系列始於用於 Python 代碼生成的 Phi-1，進而發展到基於文本和聊天補全的 Phi-1.5 / 2，接著是 Phi-3-mini/small/medium-instruct 及 Phi-3.5/4-mini-instruct，又發展至用於視覺的 Phi-3/3.5-vision、具強力推理能力的 Phi-4、搭載混合專家（MoE）架構的 Phi-3.5-MoE，現在則有全模態模型 Phi-4-multimodal。通過高質量數據集，這些基準模型能訓練到與具有更大訓練參數模型相當的水平。

## 模型選擇範例

| | | | |
|-|-|-|-|
|客戶需求|任務|起始模型|更多細節|
|需要一款簡單總結訊息串的模型|對話總結|Phi-3 / 3.5 文字模型|決定因素是客戶有明確且直白的語言任務|
|一款免費的兒童數學輔導應用|數學與推理|Phi-3 / 3.5 / 4 文字模型|因應應用免費，客戶希望方案不會有持續性支出|
|自駕警車攝影機|視覺分析|Phi-3 /3.5 -Vision 或 Phi-4-multimodal|需要能在邊緣裝置上無網絡下運作的解決方案|
|想要打造 AI 旅遊訂票代理|需要複雜計劃、函數調用及協調|GPT 模型|需要具備規劃能力，調用 API 蒐集資訊並執行|
|想打造員工輔助駕駛員（copilot）|檢索式生成（RAG）、多領域、複雜及開放式|GPT 模型 + Phi 系列 |開放式場景，需要廣泛的世界知識，故較大型模型更合適。需將知識內容切塊，SLM 可能適合您|
