# Família Phi da Microsoft

Os modelos Phi são os Modelos de Linguagem Pequenos (SLMs) mais capazes e rentáveis disponíveis, superando modelos do mesmo tamanho e do tamanho seguinte em vários benchmarks de linguagem, raciocínio, codificação, áudio, visão e matemática. Esta versão expande a seleção de modelos de alta qualidade para os clientes, oferecendo escolhas mais práticas para compor e construir aplicações de IA generativa.

A Família Phi começou com o Phi-1 para geração de código Python, continuou para o Phi-1.5 /2 baseado em texto e conclusão de chat, Phi-3-mini/pequeno/médio-instrutivo e Phi-3.5/4-mini-instrutivo, e evoluiu para Phi-3/3.5-visão para visão, Phi-4 baseado em raciocínio forte, e Phi-3.5-MoE para MoE, e agora para o modelo multimodal completo Phi-4-multimodal. Através de conjuntos de dados de alta qualidade, o benchmark pode ser treinado para ser comparável a modelos com parâmetros de treino maiores.

## Exemplo de Seleção de Modelo

| | | | |
|-|-|-|-|
|Necessidade do Cliente|Tarefa|Começar com|Mais Detalhes|
|Precisa de um modelo que simplesmente resuma uma sequência de mensagens|Sumarização de Conversa|Modelo de texto Phi-3 / 3.5|O fator decisivo aqui é que o cliente tem uma tarefa de linguagem bem definida e simples|
|Uma aplicação gratuita de tutor de matemática para crianças|Matemática e Raciocínio|Modelos de texto Phi-3 / 3.5 / 4|Como a aplicação é gratuita, os clientes querem uma solução que não lhes custe de forma recorrente |
|Câmara de vigilância para automóvel|Análise de visão|Phi-3 /3.5 -Visão ou Phi-4-multimodal|Necessita de uma solução que funcione em edge sem internet|
|Quer construir um agente de reserva de viagens baseado em IA|Necessita de planeamento complexo, chamada de funções e orquestração|Modelos GPT|Necessita de capacidade para planear, chamar APIs para obter informação e executar |
|Quer construir um copiloto para os seus funcionários|RAG, múltiplos domínios, complexa e aberta|Modelos GPT + Família Phi|Cenário aberto, necessita de conhecimento amplo do mundo, portanto um modelo maior é mais adequado.
