## Começar com Modelos da Família Phi no Foundry Local

### Introdução ao Foundry Local

O Foundry Local é uma solução poderosa de inferência AI no dispositivo que traz capacidades AI de nível empresarial diretamente para o seu hardware local. Este tutorial irá guiá-lo na configuração e utilização dos modelos da Família Phi com o Foundry Local, oferecendo-lhe controlo total sobre as suas cargas de trabalho AI, mantendo a privacidade e reduzindo custos.

O Foundry Local oferece vantagens de desempenho, privacidade, personalização e custo ao executar modelos AI localmente no seu dispositivo. Integra-se perfeitamente nos seus fluxos de trabalho e aplicações existentes através de uma CLI intuitiva, SDK e API REST.


![arch](../../../../../translated_images/pt-PT/foundry-local-arch.8823e321dd8258d7.webp)

### Por que Escolher o Foundry Local?

Compreender os benefícios do Foundry Local ajudar-vos-á a tomar decisões informadas sobre a sua estratégia de implementação de AI:

- **Inferência no Dispositivo:** Execute modelos localmente no seu próprio hardware, reduzindo custos enquanto mantém todos os seus dados no seu dispositivo.

- **Personalização de Modelos:** Selecione entre modelos pré-definidos ou use os seus próprios para atender a requisitos e casos de uso específicos.

- **Eficiência de Custo:** Elimine custos recorrentes de serviços na cloud utilizando o seu hardware existente, tornando a AI mais acessível.

- **Integração Sem Esforço:** Conecte-se às suas aplicações através de um SDK, endpoints API, ou a CLI, com fácil escalabilidade para o Microsoft Foundry à medida que as suas necessidades crescem.

> **Nota de Início:** Este tutorial foca-se no uso do Foundry Local através das interfaces CLI e SDK. Aprenderá ambas as abordagens para ajudá-lo a escolher o melhor método para o seu caso de uso.

## Parte 1: Configurar o Foundry Local CLI

### Passo 1: Instalação

O Foundry Local CLI é a sua porta de entrada para gerir e executar modelos AI localmente. Vamos começar por instalá-lo no seu sistema.

**Plataformas Suportadas:** Windows e macOS

Para instruções detalhadas de instalação, por favor consulte a [documentação oficial do Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Passo 2: Explorar Modelos Disponíveis

Assim que tiver o Foundry Local CLI instalado, pode descobrir quais os modelos disponíveis para o seu caso de uso. Este comando mostrará todos os modelos suportados:


```bash
foundry model list
```

### Passo 3: Compreender os Modelos da Família Phi

A Família Phi oferece uma gama de modelos otimizados para diferentes casos de uso e configurações de hardware. Aqui estão os modelos Phi disponíveis no Foundry Local:

**Modelos Phi Disponíveis:** 

- **phi-3.5-mini** - Modelo compacto para tarefas básicas
- **phi-3-mini-128k** - Versão com contexto alargado para conversas mais longas
- **phi-3-mini-4k** - Modelo com contexto standard para uso geral
- **phi-4** - Modelo avançado com capacidades melhoradas
- **phi-4-mini** - Versão leve do Phi-4
- **phi-4-mini-reasoning** - Especializado para tarefas complexas de raciocínio

> **Compatibilidade de Hardware:** Cada modelo pode ser configurado para diferentes acelerações de hardware (CPU, GPU) dependendo das capacidades do seu sistema.

### Passo 4: Executar o Seu Primeiro Modelo Phi

Vamos começar com um exemplo prático. Iremos executar o modelo `phi-4-mini-reasoning`, que se destaca na resolução passo a passo de problemas complexos.


**Comando para executar o modelo:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Configuração Inicial:** Ao executar um modelo pela primeira vez, o Foundry Local irá descarregá-lo automaticamente para o seu dispositivo local. O tempo de download varia consoante a velocidade da sua rede, por isso, por favor, tenha paciência durante a configuração inicial.

### Passo 5: Testar o Modelo com um Problema Real

Vamos agora testar o nosso modelo com um problema clássico de lógica para ver como ele executa o raciocínio passo a passo:

**Problema Exemplo:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Comportamento Esperado:** O modelo deverá decompor este problema em passos lógicos, usando o facto de que faisões têm 2 pernas e coelhos têm 4 pernas para resolver o sistema de equações.
