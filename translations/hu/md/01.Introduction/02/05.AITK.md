# Phi Family in AITK

[Az AI Toolkit a VS Code-hoz](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) egyszerűsíti a generatív MI alkalmazásfejlesztést azáltal, hogy összefogja a Microsoft Foundry katalógusának és más katalógusoknak, például a Hugging Face-nek a legkorszerűbb MI fejlesztőeszközeit és modelljeit. Böngészhet a GitHub Modellek és a Microsoft Foundry modell katalógusai által támogatott MI modellek között, letöltheti azokat helyileg vagy távolról, finomhangolhatja, tesztelheti és használhatja alkalmazásában.

Az AI Toolkit előnézet helyileg fut. A helyi következtetés vagy finomhangolás a kiválasztott modelltől függően GPU-t igényelhet, például NVIDIA CUDA GPU-t. A GitHub modelleket közvetlenül az AITK-val is futtathatja.

### AI Toolkit telepítése

Az AI Toolkit [Visual Studio Code bővítményként](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions) érkezik, ezért először telepítenie kell a [VS Code-ot](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo), majd letöltheti az AI Toolkitet a [VS Marketplace-ről](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio).
Az [AI Toolkit elérhető a Visual Studio Marketplace-ben](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio), és úgy telepíthető, mint bármely más VS Code bővítmény.

Ha nem ismeri a VS Code bővítmények telepítését, kövesse az alábbi lépéseket:

### Bejelentkezés

1. Válassza az **Extensions** lehetőséget a VS Code tevékenységi sávjában
1. Az Extensions keresőmezőjébe írja be: "AI Toolkit"
1. Válassza az "AI Toolkit for Visual Studio code" bővítményt
1. Kattintson a **Telepítés** gombra

Most készen áll a bővítmény használatára!

A rendszer felkéri, hogy jelentkezzen be GitHub-fiókjába, ezért kattintson az "Engedélyezés" gombra a folytatáshoz. Átirányítják a GitHub bejelentkezési oldalra.

Jelentkezzen be, és kövesse a lépéseket. A sikeres bejelentkezés után visszairányítják a VS Code-ba.

A telepítés után megjelenik az AI Toolkit ikon a tevékenységi sávban.

Fedezzük fel az elérhető lehetőségeket!

> [!NOTE]
>
> **GPU vagy CPU**
>
> Az észre fogja venni, hogy a modellkártyákon megjelenik a modell mérete, a platform és a gyorsító típusa (CPU, GPU). Az optimalizált teljesítmény érdekében **Windows eszközökön, amelyek rendelkeznek legalább egy GPU-val**, válasszon olyan modellverziókat, amelyek csak Windows platformra céloznak.

> Ez biztosítja, hogy olyan modellje legyen, amely a DirectML gyorsítóhoz van optimalizálva.
>
> A modellek elnevezése a következő formátumban van:
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
>A Windows eszközön lévő GPU meglétének ellenőrzéséhez nyissa meg a **Feladatkezelőt**, majd válassza a **Teljesítmény** fület. Ha van GPU-ja, azok neve például "GPU 0" vagy "GPU 1" lesz.

### Használja a REST API-t az alkalmazásában

Az AI Toolkit tartalmaz egy helyi REST API web szervert **5272-es porton**, amely a [OpenAI csevegés befejezési formátumát](https://platform.openai.com/docs/api-reference/chat/create) használja.

Ez lehetővé teszi, hogy helyben tesztelje alkalmazását anélkül, hogy felhőalapú MI modell szolgáltatásra kellene hagyatkoznia. Például az alábbi JSON fájl mutatja, hogyan konfigurálja a kérés törzsét:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```

A REST API-t tesztelheti például a [Postman](https://www.postman.com/) vagy a CURL (Client URL) segédprogram segítségével:

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```

### Az OpenAI ügyfélkönyvtár használata Pythonhoz

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # szükséges az API-hoz, de nincs használatban
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```

### Azure OpenAI ügyfélkönyvtár használata .NET-hez

Adja hozzá az [Azure OpenAI ügyfélkönyvtárat .NET-hez](https://www.nuget.org/packages/Azure.AI.OpenAI/) a projektjéhez a NuGet segítségével:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```

Adjon hozzá a projekthez egy **OverridePolicy.cs** nevű C# fájlt, és illessze be a következő kódot:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```

Ezután illessze be a következő kódot a **Program.cs** fájlba:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
