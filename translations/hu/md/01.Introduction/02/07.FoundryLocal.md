## Phi-család modellek használatának megkezdése a Foundry Local-ban

### Bevezetés a Foundry Local-ba

A Foundry Local egy erőteljes, eszközön futó mesterséges intelligencia következtetési megoldás, amely vállalati szintű AI képességeket hoz közvetlenül a helyi hardverére. Ez az útmutató végigvezeti a Phi-család modellek beállításán és használatán a Foundry Local segítségével, teljes irányítást biztosítva az AI munkaterhelései felett, miközben megőrzi a magánszférát és csökkenti a költségeket.

A Foundry Local teljesítményt, adatvédelmet, testreszabhatóságot és költséghatékonyságot biztosít azáltal, hogy AI modelleket helyben, az eszközön futtat. Zökkenőmentesen integrálódik a meglévő munkafolyamataiba és alkalmazásaiba egy intuitív CLI, SDK és REST API segítségével.


![arch](../../../../../translated_images/hu/foundry-local-arch.8823e321dd8258d7.webp)

### Miért válassza a Foundry Local-t?

A Foundry Local előnyeinek megértése segít megalapozott döntéseket hozni AI bevezetési stratégiája kapcsán:

- **Eszközön futó következtetés:** Modellek helyi futtatása saját hardverén, csökkentve a költségeket, miközben az összes adat az eszközön marad.

- **Modell testreszabás:** Kiválaszthatja az előre beállított modelleket vagy használhatja sajátját specifikus követelmények és használati esetek kielégítésére.

- **Költséghatékonyság:** Megszünteti a felhőszolgáltatások ismétlődő költségeit meglévő hardvere használatával, így az AI hozzáférhetőbbé válik.

- **Zökkenőmentes integráció:** SDK, API végpontok vagy CLI segítségével kapcsolódhat alkalmazásaihoz, egyszerű skálázással a Microsoft Foundry-ra növekedés esetén.

> **Kezdő lépések megjegyzés:** Ez az útmutató a Foundry Local CLI és SDK interfészein keresztüli használatra fókuszál. Mindkét megközelítést megismerheti, hogy kiválaszthassa az Ön esete számára legjobb módszert.

## 1. rész: Foundry Local CLI beállítása

### 1. lépés: Telepítés

A Foundry Local CLI kapuja a helyi AI modellek kezeléséhez és futtatásához. Kezdjük azzal, hogy telepítjük a rendszerére.

**Támogatott platformok:** Windows és macOS

A részletes telepítési utasításokért kérjük, tekintse meg a [hivatalos Foundry Local dokumentációt](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### 2. lépés: Elérhető modellek felfedezése

Miután telepítette a Foundry Local CLI-t, felfedezheti, hogy milyen modellek állnak rendelkezésre az Ön használati esetéhez. Ez a parancs megmutatja az összes támogatott modellt:


```bash
foundry model list
```

### 3. lépés: Phi-család modellek megértése

A Phi-család különböző modelleket kínál eltérő használati esetekre és hardverkonfigurációkra optimalizálva. Íme a Foundry Local-ban elérhető Phi modellek:

**Elérhető Phi modellek:** 

- **phi-3.5-mini** - Kompakt modell alapvető feladatokhoz
- **phi-3-mini-128k** - Kiterjesztett kontextusú változat hosszabb beszélgetésekhez
- **phi-3-mini-4k** - Standard kontextusú modell általános használathoz
- **phi-4** - Fejlett modell továbbfejlesztett képességekkel
- **phi-4-mini** - A Phi-4 könnyűsúlyú változata
- **phi-4-mini-reasoning** - Speciális összetett érvelési feladatokhoz

> **Hardver kompatibilitás:** Minden modellt különböző hardver gyorsítási beállításokra (CPU, GPU) konfigurálhat a rendszer képességei szerint.

### 4. lépés: Első Phi modell futtatása

Kezdjük egy gyakorlati példával. A `phi-4-mini-reasoning` modellt fogjuk futtatni, amely kiválóan alkalmas komplex problémák lépésenkénti megoldására.


**Parancs a modell futtatásához:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Első futtatás beállítása:** Amikor először futtat egy modellt, a Foundry Local automatikusan letölti azt az eszközére. A letöltési idő hálózati sebességétől függ, kérjük, legyen türelemmel az első beállítás során.

### 5. lépés: A modell tesztelése valós problémával

Most teszteljük a modellünket egy klasszikus logikai problémával, hogy lássuk, hogyan működik a lépésenkénti érvelés:

**Példa probléma:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Várt viselkedés:** A modell bontsa le a problémát logikai lépésekre, használva azt a tényt, hogy a fácánnak 2 lába, a nyúlnak pedig 4 lába van a egyenletrendszer megoldásához.

## 2. rész: Alkalmazások építése a Foundry Local SDK-val

### Miért használja az SDK-t?

Míg a CLI tökéletes teszteléshez és gyors interakciókhoz, az SDK lehetővé teszi a Foundry Local programozott integrációját alkalmazásaiba. Ez megnyitja a lehetőségeket:

- Egyedi, AI-alapú alkalmazások létrehozására
- Automatizált munkafolyamatok kialakítására
- AI képességek beépítésére meglévő rendszerekbe
- Chatbotok és interaktív eszközök fejlesztésére

### Támogatott programozási nyelvek

A Foundry Local több programozási nyelvhez biztosít SDK támogatást, hogy megfeleljen fejlesztési preferenciáinak:

**📦 Elérhető SDK-k:**

- **C# (.NET):** [SDK dokumentáció és példák](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK dokumentáció és példák](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK dokumentáció és példák](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK dokumentáció és példák](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### Következő lépések

1. **Válassza ki az Ön fejlesztési környezetéhez legmegfelelőbb SDK-t**
2. **Kövesse az SDK-specifikus dokumentációt** a részletes megvalósítási útmutatókért
3. **Kezdje egyszerű példákkal** mielőtt összetett alkalmazásokat építene
4. **Fedezze fel a mintakódokat**, amelyeket az egyes SDK tárolókban talál
