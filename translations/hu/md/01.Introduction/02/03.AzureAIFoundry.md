## Phi család a Microsoft Foundry-ban

A [Microsoft Foundry](https://ai.azure.com) egy megbízható platform, amely lehetővé teszi a fejlesztők számára, hogy biztonságos, megbízható és felelősségteljes módon hajtsanak végre innovációkat és formálják a jövőt az MI segítségével.

A [Microsoft Foundry](https://ai.azure.com) fejlesztők számára készült, hogy:

- Generatív MI alkalmazásokat építsenek vállalati szintű platformon.
- Felfedezzenek, építsenek, teszteljenek és telepítsenek a legkorszerűbb MI eszközöket és gépi tanulás modelljeit használva, amelyek a felelős MI gyakorlatokon alapulnak.
- Egy csapattal együttműködjenek az alkalmazásfejlesztés teljes életciklusa alatt.

A Microsoft Foundry-val számos modellt, szolgáltatást és képességet fedezhet fel, és elkezdheti építeni azokat az MI alkalmazásokat, amelyek legjobban szolgálják céljait. A Microsoft Foundry platform megkönnyíti a bizonyítékokon alapuló koncepciók zökkenőmentes átültetését teljes körű éles alkalmazásokba. A folyamatos monitorozás és finomhangolás támogatja a hosszú távú sikert.

### **Python kód futtatása a Microsoft Foundry Phi-4 hívásához**

```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Tartalmazza a beszéd eredményét, ha a beszéd engedélyezve van
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
