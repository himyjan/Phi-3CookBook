# **استخدام Microsoft Foundry للتقييم**

![aistudo](../../../../../translated_images/ar/AIFoundry.9e0b513e999a1c5a.webp)

كيفية تقييم تطبيق الذكاء الاصطناعي التوليدي الخاص بك باستخدام [Microsoft Foundry](https://ai.azure.com?WT.mc_id=aiml-138114-kinfeylo). سواء كنت تقوم بتقييم محادثات ذات دورة واحدة أو متعددة الدورات، يوفر Microsoft Foundry أدوات لتقييم أداء النموذج والسلامة.

![aistudo](../../../../../translated_images/ar/AIPortfolio.69da59a8e1eaa70f.webp)

## كيفية تقييم تطبيقات الذكاء الاصطناعي التوليدي باستخدام Microsoft Foundry
لمزيد من التعليمات التفصيلية راجع [توثيق Microsoft Foundry](https://learn.microsoft.com/azure/ai-studio/how-to/evaluate-generative-ai-app?WT.mc_id=aiml-138114-kinfeylo)

فيما يلي خطوات البدء:

## تقييم نماذج الذكاء الاصطناعي التوليدي في Microsoft Foundry

**المتطلبات المسبقة**

- مجموعة بيانات اختبار إما بصيغة CSV أو JSON.
- نموذج ذكاء اصطناعي توليدي منشور (مثل Phi-3، GPT 3.5، GPT 4، أو نماذج Davinci).
- بيئة تشغيل مع جهاز حساب لتشغيل التقييم.

## مقاييس التقييم المدمجة

يتيح لك Microsoft Foundry تقييم كل من المحادثات ذات الدورة الواحدة والمعقدة متعددة الدورات.  
في سيناريوهات التوليد المعزز بالاستخراج (RAG)، حيث النموذج مستند إلى بيانات محددة، يمكنك تقييم الأداء باستخدام مقاييس التقييم المدمجة.  
بالإضافة إلى ذلك، يمكنك تقييم سيناريوهات الإجابة على أسئلة ذات دورة واحدة عامة (غير RAG).

## إنشاء جلسة تقييم

من واجهة Microsoft Foundry، انتقل إلى صفحة التقييم أو صفحة تدفق المطالبات.  
اتبع معالج إنشاء التقييم لإعداد جلسة تقييم. قدم اسمًا اختياريًا لتقييمك.  
اختر السيناريو الذي يتماشى مع أهداف تطبيقك.  
حدد مقياسًا واحدًا أو أكثر لتقييم نواتج النموذج.

## تدفق تقييم مخصص (اختياري)

لمزيد من المرونة، يمكنك إنشاء تدفق تقييم مخصص. خصص عملية التقييم استنادًا إلى متطلباتك الخاصة.

## عرض النتائج

بعد تشغيل التقييم، سجّل النتائج واطلع عليها وحلل مقاييس التقييم التفصيلية في Microsoft Foundry. احصل على رؤى حول قدرات وحدود تطبيقك.

**ملاحظة** Microsoft Foundry في الوقت الحالي في المعاينة العامة، لذا استخدمه لأغراض التجربة والتطوير. بالنسبة لأعباء العمل الإنتاجية، فكر في خيارات أخرى. استكشف [توثيق AI Foundry الرسمي](https://learn.microsoft.com/azure/ai-studio/?WT.mc_id=aiml-138114-kinfeylo) لمزيد من التفاصيل والتعليمات خطوة بخطوة.
