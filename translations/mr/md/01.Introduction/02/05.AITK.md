# AITK मधील Phi कुटुंब

[VS कोडसाठी AI टूलकिट](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) जनरेटिव AI अॅप विकास सुलभ करते जे Microsoft Foundry Catalog आणि Hugging Face सारख्या इतर कॅटलॉगमधील अत्याधुनिक AI विकास साधने आणि मॉडेल एकत्र आणते. तुम्ही GitHub मॉडेल्स आणि Microsoft Foundry मोडेल कॅटलॉगद्वारे चालवलेले AI मॉडेल कॅटलॉग ब्राउझ करू शकता, स्थानिक किंवा रिमोट डाउनलोड करू शकता, फाईन-ट्यून करू शकता, चाचणी घेऊ शकता आणि तुमच्या अॅप्लिकेशनमध्ये वापरू शकता.

AI टूलकिट पूर्वावलोकन स्थानिकरित्या चालेल. स्थानिक इन्फरन्स किंवा फाईन-ट्यूनिंग, निवडलेल्या मॉडेलवर अवलंबून आहे, तुम्हाला NVIDIA CUDA GPU सारखा GPU आवश्यक असू शकतो. तुम्ही सीधे GitHub मॉडेल्स AITK बरोबरही चालवू शकता.

### AI टूलकिट इन्स्टॉल करा

AI टूलकिट [व्हिज्युअल स्टुडिओ कोड एक्सटेंशन](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions) म्हणून वितरित केला जातो, म्हणून तुम्हाला आधी [VS कोड](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo) इन्स्टॉल करावे लागेल, आणि नंतर [VS मार्केटप्लेस](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) वरून AI टूलकिट डाउनलोड करावे लागेल.
[AI टूलकिट व्हिज्युअल स्टुडिओ मार्केटप्लेस मध्ये उपलब्ध आहे](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) आणि इतर कोणत्याही VS कोड एक्सटेंशन्स प्रमाणे इन्स्टॉल केले जाऊ शकते.

जर तुम्हाला VS कोड एक्सटेंशन्स इन्स्टॉल करण्याचा अनुभव नसेल तर, खालील टप्पे फॉलो करा:

### साइन इन करा

1. VS कोडमधील Activity Bar मध्ये **Extensions** निवडा
1. Extensions सर्च बार मध्ये "AI Toolkit" टाइप करा
1. "AI Toolkit for Visual Studio code" निवडा
1. **इन्स्टॉल** निवडा

आता, तुम्ही एक्सटेंशन वापरण्यास तयार आहात!

तुम्हाला GitHub मध्ये साइन इन करण्यासाठी विनंती केली जाईल, कृपया "Allow" क्लिक करा. तुम्हाला GitHub साइनिंग पृष्ठावर पुनर्निर्देशित केले जाईल.

कृपया साइन इन करा आणि प्रक्रिया पूर्ण करा. यशस्वी झाल्यानंतर तुम्हाला VS कोडकडे पुनर्निर्देशित केले जाईल.

एकदा एक्सटेंशन इन्स्टॉल झाल्यावर, तुमच्या Activity Bar मध्ये AI Toolkit चिन्ह दिसेल.

चला, उपलब्ध क्रिया पाहू या!

### उपलब्ध क्रिया

AI Toolkit चे प्राथमिक साइडबार खालीलप्रमाणे संघटित आहे

- **Models**
- **Resources**
- **Playground**  
- **Fine-tuning**
- **Evaluation**

Resources विभागात उपलब्ध आहेत. सुरुवात करण्यासाठी **Model Catalog** निवडा.

### कॅटलॉगमधून मॉडेल डाउनलोड करा

VS कोड साइडबारमधून AI Toolkit सुरू करतांना, तुम्ही खालील पर्याय निवडू शकता:

![AI toolkit model catalog](../../../../../translated_images/mr/AItoolkitmodel_catalog.7a7be6a7d8468d31.webp)

- **Model Catalog** मधून समर्थित मॉडेल शोधा आणि स्थानिक डाउनलोड करा
- **Model Playground** मध्ये मॉडेल इन्फरन्स चाचणी करा
- **Model Fine-tuning** मध्ये स्थानिक किंवा रिमोट फाईन-ट्यूनिंग करा
- AI Toolkit साठी कमांड पॅलेट वापरून क्लाउडवर फाईन-ट्यून केलेले मॉडेल्स डिप्लॉय करा
- मॉडेल्सचे मूल्यांकन करा

> [!NOTE]
>
> **GPU वि CPU**
>
> तुम्हाला लक्षात येईल की मॉडेल कार्ड्समध्ये मॉडेलचा आकार, प्लॅटफॉर्म आणि अॅक्सेलेरेटर प्रकार (CPU, GPU) दाखवलेले असतात. **विंडोज डिव्हाइसेस ज्यांच्याकडे किमान एक GPU आहे** अशा उपकरणांवर अधिक उत्तम कार्यक्षमतेसाठी, फक्त Windows लक्ष ठेवणारे मॉडेल आवृत्त्या निवडा.
>
> यामुळे तुम्हाला DirectML अॅक्सेलेरेटरसाठी ऑप्टिमाइझ केलेले मॉडेल मिळते.
>
> मॉडेल नावे खालील स्वरूपात असतात
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
>तुमच्या विंडोज डिव्हाइसवर GPU आहे की नाही हे तपासण्यासाठी, **Task Manager** उघडा आणि नंतर **Performance** टॅब निवडा. जर GPU(s) असतील तर ते "GPU 0" किंवा "GPU 1" सारख्या नावाखाली सूचीबद्ध असतील.

### तुमच्या अॅप्लिकेशनमध्ये REST API वापरा

AI टूलकिट स्थानिक REST API वेब सर्वरसह येतो **पोर्ट 5272** वर जो [OpenAI च्या चॅट पूर्णत्व स्वरूपाचा](https://platform.openai.com/docs/api-reference/chat/create) वापर करतो.

हे तुम्हाला क्लाउड AI मॉडेल सेवांवर अवलंबून न राहता स्थानिकरित्या अॅप्लिकेशन तपासण्याची परवानगी देते. उदाहरणार्थ, खालील JSON फाईल विनंतीचे बॉडी कसे कॉन्फिगर करायचे दाखवते:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```

तुम्ही REST API चाचणीसाठी (उदा.) [Postman](https://www.postman.com/) किंवा CURL (Client URL) उपयुक्तता वापरू शकता:

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```


### Python साठी OpenAI क्लायंट लायब्ररी वापरणे

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # API साठी आवश्यक आहे परंतु वापरलेले नाही
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```


### .NET साठी Azure OpenAI क्लायंट लायब्ररी वापरणे

NuGet वापरून प्रोजेक्टमध्ये [Azure OpenAI क्लायंट लायब्ररी](https://www.nuget.org/packages/Azure.AI.OpenAI/) जोडा:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```

**OverridePolicy.cs** नावाची C# फाईल तुमच्या प्रोजेक्टमध्ये जोडा आणि खालील कोड पेस्ट करा:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```

नंतर, तुमच्या **Program.cs** फाईलमध्ये खालील कोड पेस्ट करा:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
