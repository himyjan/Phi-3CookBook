## मायक्रोसॉफ्ट फाउंड्रीमधील फाई फॅमिली

[मायक्रोसॉफ्ट फाउंड्री](https://ai.azure.com) हे एक विश्वासार्ह प्लॅटफॉर्म आहे जे विकासकांना सुरक्षित, संरक्षित आणि जबाबदार पद्धतीने AI सोबत नवकल्पना घडविण्यास आणि भविष्याची रचना करण्यास सक्षम करते.

[मायक्रोसॉफ्ट फाउंड्री](https://ai.azure.com) विकासकांसाठी तयार केले गेले आहे जेणेकरून ते:

- एंटरप्राइज-ग्रेड प्लॅटफॉर्मवर जनरेटिव्ह AI अनुप्रयोग तयार करू शकतील.
- जबाबदार AI पद्धतींवर आधारित अत्याधुनिक AI साधने आणि ML मॉडेल्स वापरून शोध, तयार करणे, चाचणी करणे आणि तैनात करणे.
- अनुप्रयोग विकासाच्या पूर्ण जीवनचक्रासाठी संघासोबत सहकार्य करणे.

मायक्रोसॉफ्ट फाउंड्रीसह, तुम्ही मॉडेल्स, सेवा आणि क्षमता यांची विस्तृत विविधता शोधू शकता आणि तुमच्या उद्दिष्टांचे सर्वोत्तम सेवा करणारे AI अनुप्रयोग तयार करू शकता. मायक्रोसॉफ्ट फाउंड्री प्लॅटफॉर्म प्रूफ ऑफ कॉन्सेप्ट्सना पूर्ण-प्रकारच्या उत्पादन अनुप्रयोगांमध्ये रूपांतरित करण्यासाठी स्केलेबिलिटी सुलभ करतो. सतत निरीक्षण आणि सुधारणा दीर्घकालीन यशाला समर्थन देतात.

![portal](../../../../../translated_images/mr/AIFoundryPorral.6b1094b101dd499e.webp)

मायक्रोसॉफ्ट फाउंड्रीमध्ये Azure AOAI सेवा वापरण्याव्यतिरिक्त, तुम्ही मायक्रोसॉफ्ट फाउंड्री मॉडेल कॅटलॉगमध्ये तृतीय-पक्ष मॉडेल्स देखील वापरू शकता. जर तुम्हाला मायक्रोसॉफ्ट फाउंड्री तुमच्या AI सोल्यूशन प्लॅटफॉर्म म्हणून वापरायचे असेल तर हा एक चांगला पर्याय आहे.

आपण मायक्रोसॉफ्ट फाउंड्रीमध्ये मॉडेल कॅटलॉगद्वारे फाई फॅमिली मॉडेल्स त्वरीत तैनात करू शकतो.

### **मायक्रोसॉफ्ट फाउंड्री फाई-4 कॉल करण्यासाठी पाइथन कोड चालवताना**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# भाषण सक्षम असल्यास भाषण निकाल समाविष्ट करा
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
