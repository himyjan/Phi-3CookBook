## Foundry Local मध्ये Phi-Family मॉडेलसह प्रारंभ करणे

### Foundry Local परिचय

Foundry Local हा एक शक्तिशाली ऑन-डिव्हाइस AI इनफरन्स सोल्यूशन आहे जो तुमच्या स्थानिक हार्डवेअरवर थेट एंटरप्राइज-ग्रेड AI क्षमता आणतो. हा ट्यूटोरियल तुम्हाला Foundry Local सह Phi-Family मॉडेल्स स्थापित करण्याची आणि वापरण्याची प्रक्रिया मार्गदर्शन करेल, ज्यामुळे तुम्हाला तुमच्या AI कामांच्या संपूर्ण नियंत्रणाबरोबरच गोपनीयता राखण्यास आणि खर्च कमी करण्यास मदत होईल.

Foundry Local तुमच्या उपकरणावर स्थानिकपणे AI मॉडेल्स चालवून कामगिरी, गोपनीयता, सानुकूलन आणि खर्चाचे फायदे प्रदान करतो. तो एक सुलभ CLI, SDK आणि REST API द्वारे तुमच्या विद्यमान कार्यप्रवाह आणि अनुप्रयोगांमध्ये सुलभपणे इंटीग्रेट होतो.


![arch](../../../../../translated_images/mr/foundry-local-arch.8823e321dd8258d7.webp)

### Foundry Local का निवडावे?

Foundry Local चे फायदे समजून घेणे तुम्हाला तुमच्या AI तैनाती धोरणाबद्दल सुज्ञ निर्णय घेण्यास मदत करेल:

- **ऑन-डिव्हाइस इनफरन्स:** तुमच्या स्वतःच्या हार्डवेअरवर मॉडेल्स स्थानिकपणे चालवा, ज्यामुळे तुमचा खर्च कमी होईल आणि सर्व डेटा तुमच्या डिव्हाइसवर राहील.

- **मॉडेल सानुकूलन:** आधीच्या सेट केलेल्या मॉडेल्सपैकी निवडा किंवा तुमच्या विशिष्ट गरजांसाठी तुमचे स्वतःचे मॉडेल वापरा.

- **खर्च कार्यक्षमते:** तुमच्या विद्यमान हार्डवेअरचा वापर करून पुनरावृत्ती होणाऱ्या क्लाउड सेवा खर्चांना टाळा, ज्यामुळे AI अधिक प्रवेशयोग्य होतो.

- **सुलभ इंटीग्रेशन:** SDK, API एंडपॉइंट्स किंवा CLI वापरून तुमच्या अनुप्रयोगांशी कनेक्ट करा, आणि तुमच्या गरजा वाढल्यावर Microsoft Foundry कडे सहज स्केल करा.

> **प्रारंभ करण्याची नोंद:** हा ट्यूटोरियल CLI आणि SDK इंटरफेस वापरण्यावर लक्ष केंद्रित करतो. तुम्हाला दोन्ही पद्धती शिकवली जातील जेणेकरून तुम्हाला तुमच्या वापरासाठी सर्वोत्तम पद्धत निवडता येईल.

## भाग 1: Foundry Local CLI सेटअप करणे

### पायर 1: स्थापना

Foundry Local CLI हा तुमचा स्थानिकपणे AI मॉडेल्स व्यवस्थापित करण्याचा आणि चालवण्याचा प्रवेशद्वार आहे. चला तुमच्या सिस्टमवर ते स्थापित करण्यापासून सुरू करूया.

**समर्थित प्लॅटफॉर्म:** Windows आणि macOS

सविस्तर स्थापना सूचना साठी, कृपया [अधिकृत Foundry Local दस्तऐवज](https://github.com/microsoft/Foundry-Local/blob/main/README.md) पहा.

### पायर 2: उपलब्ध मॉडेल्सची तपासणी

Foundry Local CLI स्थापित केल्यानंतर, तुम्ही तुमच्या वापरासाठी कोणती मॉडेल्स उपलब्ध आहेत ते शोधू शकता. हा आदेश तुम्हाला सर्व समर्थित मॉडेल्स दाखवेल:


```bash
foundry model list
```

### पायर 3: Phi Family मॉडेल्स समजून घेणे

Phi Family वेगवेगळ्या वापर प्रकरणांसाठी आणि हार्डवेअर संरचनेसाठी अनुकूलित विविध मॉडेल्सची श्रेणी ऑफर करते. येथे Foundry Local मध्ये उपलब्ध Phi मॉडेल्स आहेत:

**उपलब्ध Phi मॉडेल्स:** 

- **phi-3.5-mini** - मूलभूत कार्यांसाठी कॉम्पॅक्ट मॉडेल
- **phi-3-mini-128k** - लांब संभाषणांसाठी विस्तारित संदर्भ आवृत्ती
- **phi-3-mini-4k** - सामान्य वापरासाठी मानक संदर्भ मॉडेल
- **phi-4** - सुधारित क्षमता असलेले प्रगत मॉडेल
- **phi-4-mini** - Phi-4 चे हलके आवृत्ती
- **phi-4-mini-reasoning** - जटिल कारणे सोडविण्यासाठी खास

> **हार्डवेअर सुसंगतता:** प्रत्येक मॉडेल वेगवेगळ्या हार्डवेअर अ‍ॅक्सेलरेशनसाठी (CPU, GPU) कॉन्फिगर केले जाऊ शकते तुमच्या सिस्टमच्या क्षमतेनुसार.

### पायर 4: तुमचा पहिला Phi मॉडेल चालवणे

चला एका व्यावहारिक उदाहरणाने सुरुवात करूया. आपण `phi-4-mini-reasoning` मॉडेल चालवणार आहोत, जे जटिल समस्या टप्प्याटप्प्याने सोडवायला उत्कृष्ट आहे.


**मॉडेल चालवण्याचा आदेश:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **पहिल्यांदा सेटअप:** मॉडेल प्रथम वेळ चालवताना, Foundry Local ते तुमच्या स्थानिक उपकरणावर स्वयंचलित डाउनलोड करेल. डाउनलोड वेळ तुमच्या नेटवर्क स्पीडनुसार वेगळा असू शकतो, त्यामुळे कृपया प्रारंभी संयम ठेवा.

### पायर 5: वास्तविक समस्येवर मॉडेलची चाचणी करणे

आता चला आपल्या मॉडेलची चाचणी एका पारंपरिक लॉजिक समस्येवर करूया आणि त्याची टप्प्याटप्प्याने कारणे कशी सोडवते ते पाहूया:

**उदाहरण समस्या:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**अपेक्षित वर्तन:** मॉडेलने ही समस्या लॉजिकच्या टप्प्यांमध्ये विभागली पाहिजे, ज्यात pheasants चे 2 पाय आणि rabbits चे 4 पाय यांचा वापर करून समीकरणांचे समाधान केले जाईल.

## भाग 2: Foundry Local SDK सह अनुप्रयोग बनवणे

### SDK का वापरावे?

CLI चाचणी आणि जलद संवादांसाठी योग्य असले तरी SDK Foundry Local चे प्रोग्रामॅटिक इंटीग्रेशन अनुप्रयोगांमध्ये करण्यास सक्षम करते. यामुळे शक्यतांचा विस्तार होतो:

- सानुकूल AI-समर्थित अनुप्रयोग तयार करणे
- स्वयंचलित कार्यप्रवाह तयार करणे
- विद्यमान प्रणालींमध्ये AI क्षमता एकत्रित करणे
- चॅटबॉट्स आणि संवादात्मक साधने विकसित करणे

### समर्थित प्रोग्रामिंग भाषा

Foundry Local विविध प्रोग्रामिंग भाषांसाठी SDK समर्थन प्रदान करतो जेणेकरून तुमच्या विकास प्राधान्यांशी जुळू शकेल:

**📦 उपलब्ध SDKs:**

- **C# (.NET):** [SDK दस्तऐवज आणि उदाहरणे](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK दस्तऐवज आणि उदाहरणे](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK दस्तऐवज आणि उदाहरणे](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK दस्तऐवज आणि उदाहरणे](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### पुढील पावले

1. तुमच्या विकास वातावरणानुसार तुमचा पसंतीचा SDK निवडा
2. सविस्तर अंमलबजावणी मार्गदर्शकासाठी SDK-विशिष्ट दस्तऐवज अनुसरा
3. गुंतागुंतीचे अनुप्रयोग तयार करण्यापूर्वी सोप्या उदाहरणांपासून सुरुवात करा
4. प्रत्येक SDK रिपॉझिटरीमध्ये दिलेला नमुना कोड एक्सप्लोर करा

## निष्कर्ष

तुम्ही आता कसे करायचे शिकले:

- ✅ Foundry Local CLI स्थापित आणि सेटअप करणे
- ✅ Phi Family मॉडेल्स शोधणे आणि चालवणे
- ✅ वास्तविक समस्यांसह मॉडेल्सची चाचणी करणे
- ✅ अनुप्रयोग विकासासाठी SDK पर्याय समजून घेणे

Foundry Local तुम्हाला AI क्षमता थेट तुमच्या स्थानिक वातावरणात आणण्यासाठी एक शक्तिशाली आधार प्रदान करतो, ज्यामुळे तुम्हाला कामगिरी, गोपनीयता आणि खर्चांवर नियंत्रण ठेवण्याचा अधिकार मिळतो, तसेच गरजेनुसार क्लाउड सोल्यूशन्सकडे स्केल करण्याची लवचीकता टिकवून ठेवतो.
