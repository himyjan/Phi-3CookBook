# Phi హార్డ్‌వేర్ మద్దతు

Microsoft Phi ONNX Runtime కొరకు ఆప్టిమైజ్ చేయబడింది మరియు Windows DirectML ను మద్దతు ఇస్తుంది. ఇది వివిధ హార్డ్‌వేర్ రకాలపై బాగా పనిచేస్తుంది, GPUలు, CPUలు మరియు మోబైల్ పరికరాలను కూడా కలుపుకొని.

## పరికరం హార్డ్‌వేర్  
స్పష్టంగా, మద్దతు ఇచ్చే హార్డ్‌వేర్ లో ఇవి ఉన్నాయి:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPUs, 128 GiB మెమరీ)

## మోబైల్ SKU

- ఆండ్రాయిడ్ - సామ్‌సంగ్ గెలాక్సీ S21
- ఆపిల్ ఐఫోన్ 14 లేదా పైగా A16/A17 ప్రాసెసర్

## Phi హార్డ్‌వేర్ స్పెసిఫికేషన్

- కనీస కాంఫిగరేషన్ అవసరం.
- విండోస్: DirectX 12-సమర్థ GPU మరియు కనీసం 4GB కలిగిన కలిపిన RAM

CUDA: Compute Capability >= 7.02 ఉన్న NVIDIA GPU

![HardwareSupport](../../../../../translated_images/te/01.phihardware.5d51b2377cba18af.webp)

## బహుళ GPUలపై onnxruntime ని నడపడం

ప్రస్తుతం అందుబాటులో ఉన్న Phi ONNX నమూనాలు కేవలం 1 GPU కోసం మాత్రమే ఉన్నాయి. Phi మోడల్ కోసం బహుళ GPU మద్దతు ఇవ్వడం సాధ్యం, కానీ 2 GPUలతో ORT 2 ORT ఇన్స్టాన్స్లకు పైగా throughput ఇస్తుందనే హామీ ఇవ్వదు. తాజా నవీకరణలకు దయచేసి [ONNX Runtime](https://onnxruntime.ai/) ను చూడండి.

[Build 2024 the GenAI ONNX Team](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) వారు Phi మోడల్స్ కోసం బహుళ GPU కాకుండా బహుళ ఇన్స్టాన్స్లను ప్రారంభించడాన్ని ప్రకటించారు.

ప్రస్తుతం ఇది ఈ విధంగా CUDA_VISIBLE_DEVICES ఎన్‌విరాన్‌మెంట్ వేరియబుల్ తో ఒక onnnxruntime లేదా onnxruntime-genai ఇన్స్టాన్స్ నడపడానికి అనుమతిస్తుంది.

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

ఆసక్తి ఉన్నవారు [Microsoft Foundry](https://ai.azure.com) లో Phi ని మరింత అవలోకనం చేసుకోవచ్చు.
