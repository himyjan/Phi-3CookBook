## Microsoft Foundryలో Phi కుటుంబం

[Microsoft Foundry](https://ai.azure.com) అనేది అభివృద్ధికారులకు AIతో భద్రంగా, సురక్షితంగా, బాధ్యతగల మార్గంలో ఆవిష్కరణను నడిపించడానికి మరియు భవిష్యత్తును ఆకృతీకరించడానికి శక్తివంతమైన ప్లాట్‌ఫారమ్.

[Microsoft Foundry](https://ai.azure.com) అభివృద్ధికారుల కోసం రూపొందించబడింది:

- ఎంటర్‌ప్రైజ్-గ్రేడ్ ప్లాట్‌ఫారమ్‌లో జనరేటివ్ AI అప్లికేషన్లు నిర్మించండి.
- బాధ్యతగల AI ఆచరణలపై ఆధారపడి ఆధునిక AI ఉపకరణాలు మరియు ML మోడళ్లను అన్వేషించండి, నిర్మించండి, పరీక్షించండి మరియు నియమించండి.
- అప్లికేషన్ అభివృద్ది యొక్క పూర్తి అభ్యాసం కోసం ఒక బృందంతో సహకరించండి.

Microsoft Foundryతో, మీరు వివిధ రకాల మోడళ్లు, సేవలు మరియు సామర్థ్యాలను అన్వేషించవచ్చు, మరియు మీ లక్ష్యాలకు ఉత్తమంగా సేవలందించే AI అప్లికేషన్లు తయారు చేయవచ్చు. Microsoft Foundry ప్లాట్‌ఫారమ్ ప్రూఫ్ ఆఫ్ కాన్సెప్ట్‌లను పూర్తి స్థాయి ఉత్పత్తి అప్లికేషన్లుగా మార్చేందుకు స్కేలబిలిటీని సులభతరం చేస్తుంది. నిరంతర మానిటరింగ్ మరియు మెరుగుదల దీర్ఘకాల విజయాన్ని తోత్సందిస్తాయి.

![portal](../../../../../translated_images/te/AIFoundryPorral.6b1094b101dd499e.webp)

Microsoft Foundryలో Azure AOAI సర్వీస్ ఉపయోగించడం తో పాటు, మీరు Microsoft Foundry మోడల్ క్యాటలాగ్‌లో మూడవ పార్టీ మోడళ్లను కూడా ఉపయోగించవచ్చు. మీరు Microsoft Foundryని మీ AI పరిష్కార ప్లాట్‌ఫారంగా ఉపయోగించాలనుకుంటే ఇది మంచి ఎంపిక.

### **Microsoft Foundry Phi-4ను పిలవడానికి Python కోడ్ నడిపించడం**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# మాట Enabled అయితే మాట ఫలితాన్ని చేర్చండి
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
