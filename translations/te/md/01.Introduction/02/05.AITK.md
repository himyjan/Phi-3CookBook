# AITKలో ఫై ఫ్యామిలీ

[AI Toolkit for VS Code](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) మైక్రోసాఫ్ట్ ఫౌండ్రీ క్యాటలాగ్ మరియు Hugging Face వంటి ఇతర క్యాటలాగ్‌ల నుండి ఆధునిక AI అభివృద్ధి సాధనాలు మరియు నమూనాలను ఒకసారిగా తీసుకువచ్చి జనరేటివ్ AI యాప్ అభివృద్ధినుంది సులభతరం చేస్తుంది. మీరు GitHub Models మరియు Microsoft Foundry Model Catalogs శక్తివంతమైన AI మోడల్స్ క్యాటలాగ్‌ను బ్రౌజ్ చేయగలరు, వాటిని లోకల్ లేదా రిమోట్‌గా డౌన్లోడ్ చేసుకుని, ఫైన్-ట్యూన్ చేసి, పరీక్షించి, మీ అప్లికేషన్‌లో ఉపయోగించవచ్చు.

AI Toolkit ప్రివ్యూ లోకలీలో నడుస్తుంది. లోకల్ ఇన్ఫరెన్సు లేదా ఫైన్-ట్యూన్, మీరు ఎంచుకున్న మోడల్‌పై ఆధారపడి ఉంటుంది, మీరు NVIDIA CUDA GPU వంటి GPU అవసరం ఉండొచ్చు. మీరు GitHub Models ను నేరుగా AITK తో కూడా నడపవచ్చు.

### AI Toolkit ఇన్స్టాల్ చేయండి

AI Toolkit [Visual Studio Code విస్తరణగా](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions) పంపిణీ చేయబడుతుంది, కాబట్టి ముందుగా [VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo) ని ఇన్స్టాల్ చేసుకుని, [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) నుండి AI Toolkit ను డౌన్లోడ్ చేసుకోండి.  
[AI Toolkit Visual Studio Marketplace లో అందుబాటులో ఉంది](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) మరియు అది ఇతర VS Code విస్తరణలపோలా ఇన్స్టాల్ చేయవచ్చు.

మీకు VS Code విస్తరణలు ఎలా ఇన్స్టాల్ చేయాలో తెలీదు అయితే ఈ దశలను అనుసరించండి:

### సైన్ ఇన్

1. VS Code లో Activity Bar లో **Extensions** ఎంచుకోండి  
1. Extensions Search బార్‌లో "AI Toolkit" టైప్ చేయండి  
1. "AI Toolkit for Visual Studio code" ఎంచుకోండి  
1. **Install** ఎంచుకోండి  

ఇప్పుడు మీరు విస్తరణ ఉపయోగించడానికి సిద్ధంగా ఉన్నారు!

GitHubలో సైన్ ఇన్ చేయమని మీరు ప్రాంప్ట్ తీస్తారు, కాబట్టి "Allow" క్లిక్ చేసి కొనసాగండి. మీరు GitHub సైన్ ఇన్ పేజీకి రీడైరెక్ట్ అవుతారు.

దయచేసి సైన్ ఇన్ చేసి ప్రక్రియ దశలను అనుసరించండి. విజయవంతంగా పూర్తిచేశాక, మీరు VS Codeకి తిరిగి పంపబడతారు.

విస్తరణ ఇన్స్టాల్ అయిన తర్వాత మీరు Activity Barలో AI Toolkit ఐకాన్‌ను చూడగలుగుతారు.

ఇప్పుడు అందుబాటులో ఉన్న చర్యలను చూద్దాం!

### అందుబాటులో ఉన్న చర్యలు

AI Toolkit యొక్క ప్రాథమిక సైడ్‌बार క్రింద ఇవ్వబడ్డ విధంగా ఉండింది  

- **Models**  
- **Resources**  
- **Playground**  
- **Fine-tuning**  
- **Evaluation**

Resources భాగంలో అందుబాటులో ఉన్నాయి. ప్రారంభమవ్వడంకోసం **Model Catalog** ఎంచుకోండి.

> [!NOTE]
> 
> **GPU Vs CPU**  
> 
> మోడల్ కార్డులు మోడల్ పరిమాణం, ప్లాట్ఫాం మరియు యాక్సలరేటర్ రకం (CPU, GPU) చూపిస్తాయని మీరు గమనిస్తారు. కనీసం ఒక GPU ఉన్న **Windows పరికరాలలో** గరిష్ట ప్రదర్శన కోసం, కేవలం Windows లక్ష్యం ఉండే మోడల్ వేరియంట్‌లను ఎంచుకోండి.  
> 
> ఇది DirectML యాక్సలరేటర్‌కు ఆప్టిమైజడ్ మోడల్ కలిగి ఉంటుందని నిర్ధారిస్తుంది.  
> 
> మోడల్ పేర్లు ఇలా ఉంటాయి  
> 
> - `{model_name}-{accelerator}-{quantization}-{format}`.  
> 
> మీ Windows పరికరంలో GPU ఉందా లేదా తెలుసుకోవడానికి, **Task Manager** తేరుస్తూ **Performance** టాబ్ ఎంచుకోండి. GPU ఉంటే, అవి "GPU 0" లేదా "GPU 1" అనే పేర్ల క్రింద చూపబడతాయి.

### మీ అప్లికేషన్‌లో REST API ఉపయోగించండి

AI Toolkit లోకల్ REST API వెబ్ సర్వర్ **పోర్ట్ 5272** మీద ఉంటుంది, ఇది [OpenAI చాట్ కంప్లిషన్స్ ఫార్మాట్](https://platform.openai.com/docs/api-reference/chat/create) ఉపయోగిస్తుంది.

ఇది మీ అప్లికేషన్‌ను లోకల్‌గా పరీక్షించడానికి క్లౌడుపై ఆధారపడకుండా అవకాశం కల్పిస్తుంది. ఉదాహరణకు, ఈ JSON ఫైల్ రిక్వెస్ట్ బాడీ ఎలా కాన్ఫిగర్ చేయాలో చూపిస్తుంది:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```
  
[Postman](https://www.postman.com/) లేదా CURL (Client URL) యుటిలిటీ ఉపయోగించి REST API పరీక్షించవచ్చు:

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```
  
### Python కోసం OpenAI క్లయింట్ లైబ్రరీ ఉపయోగించడం

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # APIకి అవసరమయినది కానీ ఉపయోగించలేదు
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```
  
### .NET కోసం Azure OpenAI క్లయింట్ లైబ్రరీ ఉపయోగించడం

మీ ప్రాజెక్టుకు [Azure OpenAI క్లయింట్ లైబ్రరీ .NET కోసం](https://www.nuget.org/packages/Azure.AI.OpenAI/) NuGet ఉపయోగించి జోడించండి:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```
  
మీ ప్రాజెక్టులో **OverridePolicy.cs** అనే C# ఫైల్ జోడించి క్రింది కోడ్ పేస్ట్ చేయండి:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```
  
తర్వాత, మీ **Program.cs** ఫైల్‌లో ఈ క్రింది కోడ్ పేస్ట్ చేయండి:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
