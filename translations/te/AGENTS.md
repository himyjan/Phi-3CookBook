# AGENTS.md

## ప్రాజెక్ట్ అవలోకనం

PhiCookBook అనేది Microsoft యొక్క Phi స్మాల్ లాంగ్వేజ్ మోడల్స్ (SLMs) కుటుంబంతో పనిచేసే ప్రాక్టికల్ ఉదాహరణలు, శిక్షణా సలహాలు మరియు డాక్యుమెంటేషన్ ఉన్న సమగ్ర కుక్‌బుక్ రిపాజిటరీ. ఈ రిపాజిటరీ వివిధ వాడుకలు చూపిస్తుంది, అందులో ఇన్ఫరెన్స్, ఫైన్-ట్యూనింగ్, క్వాంటైజేషన్, RAG అమలులు, మరియు విభిన్న ప్లాట్‌ఫామ్‌లు మరియు ఫ్రేమ్‌వర్క్‌లలో మల్టీమోడల్ అప్లికేషన్లు ఉన్నాయి.

**ప్రధాన సాంకేతికతలు:**
- **భాషలు:** Python, C#/.NET, JavaScript/Node.js
- **ఫ్రేమ్‌వర్క్‌లు:** ONNX Runtime, PyTorch, Transformers, MLX, OpenVINO, Semantic Kernel
- **ప్లాట్‌ఫామ్‌లు:** Microsoft Foundry, GitHub Models, Hugging Face, Ollama
- **మోడల్ రకాలు:** Phi-3, Phi-3.5, Phi-4 (టెక్స్ట్, విజన్, మల్టీమోడల్, రీజనింగ్ వేరియంట్లు)

**రిపాజిటరీ రూపకల్పన:**
- `/code/` - పనిచేసే కోడ్ ఉదాహరణలు మరియు సాంపిల్ అమలులు
- `/md/` - సమగ్రమైన డాక్యుమెంటేషన్, శిక్షణలు మరియు ఎలా చేయాలో గైడ్లు  
- `/translations/` - బహుభాషా అనువాదాలు (50+ భాషలు ఆటోమేటిక్ వర్క్‌ఫ్లో ద్వారా)
- `/.devcontainer/` - డెవ్ కంటైనర్ కాన్ఫిగరేషన్ (Python 3.12 తో Ollama)

### లోకల్ అమరిక

**అవసరాలు:**
- Python 3.12 లేదా ఆపై
- .NET 8.0 SDK (C# ఉదాహరణల కోసం)
- Node.js 18+ మరియు npm (JavaScript ఉదాహరణల కోసం)
- కనీసం 16GB RAM సిఫార్సు చేయబడింది

**ఇన్‌స్టాలేషన్:**
```bash
git clone https://github.com/microsoft/PhiCookBook.git
cd PhiCookBook
```

**Python ఉదాహరణల కోసం:**
నిర్దిష్ట ఉదాహరణ డైరెక్టరీలకు వెళ్లి ఆధారాలను ఇన్‌స్టాల్ చేయండి:
```bash
cd code/<example-directory>
pip install -r requirements.txt  # requirements.txt ఉన్నట్లయితే
```

**.NET ఉదాహరణల కోసం:**
```bash
cd md/04.HOL/dotnet/src
dotnet restore LabsPhi.sln
dotnet build LabsPhi.sln
```

**JavaScript/Web ఉదాహరణల కోసం:**
```bash
cd code/08.RAG/rag_webgpu_chat
npm install
npm run dev  # అభివృద్ధి సర్వర్ ప్రారంభించండి
npm run build  # ఉత్పత్తి కోసం బిల్డ్ చేయండి
```

## పరీక్షలు

ఈ రిపాజిటరీలో సాధారణ సాఫ్ట్‌వేర్ ప్రాజెక్ట్‌లవంటిది యూనిట్ టెస్టులు కాకుండా ఉదాహరణ కోడ్ మరియు శిక్షణలు ఉన్నాయి. ధృవీకరణ సాధారణంగా ఇలా జరుగుతుంది:

1. **ఉదాహరణలు నడపడం** - ప్రతి ఉదాహరణ తప్పులేని రీతిలో అమలవ్వాలి
2. **ఫలితాలను తనిఖీ చేయడం** - మోడల్ స్పందనలు సరైనవి కాబోతున్నాయో చూడండి
3. **శిక్షణలను అనుసరించడం** - రాత గైడ్లు ప్రకారం డ్రెడ్‌లు పనిచేయాలి

**సాధారణ ధృవీకరణ విధానం:**
- లక్ష్య వాతావరణంలో ఉదాహరణలను పరీక్షించడం
- డిపెండెన్సీలు సరిగా ఇన్‌స్టాల్ అవుతున్నాయో చూడటం
- మోడల్ డౌన్లోడ్లు / లోడ్ చేయడం విజయవంతమైందో ధృవీకరించడం
- అంచనా వేయబడిన ప్రవర్తన డాక్యుమెంటేషన్‌తో సరిపోలుతుందో తనిఖీ
