# การสนับสนุนฮาร์ดแวร์ Phi

Microsoft Phi ได้รับการปรับให้เหมาะสมสำหรับ ONNX Runtime และรองรับ Windows DirectML ทำงานได้ดีบนฮาร์ดแวร์หลายประเภท รวมถึง GPU, CPU และแม้แต่บนอุปกรณ์เคลื่อนที่

## ฮาร์ดแวร์ของอุปกรณ์  
โดยเฉพาะฮาร์ดแวร์ที่รองรับได้แก่:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPU, หน่วยความจำ 128 GiB)

## Mobile SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 หรือสูงกว่า พร้อมโปรเซสเซอร์ A16/A17

## สเปคฮาร์ดแวร์ Phi

- การกำหนดค่าขั้นต่ำที่ต้องการ
- Windows: GPU ที่รองรับ DirectX 12 และ RAM รวมอย่างน้อย 4GB

CUDA: NVIDIA GPU ที่มี Compute Capability >= 7.02

![HardwareSupport](../../../../../translated_images/th/01.phihardware.5d51b2377cba18af.webp)

## การรัน onnxruntime บนหลาย GPU

โมเดล Phi ONNX ที่มีในปัจจุบันรองรับเพียง 1 GPU เท่านั้น เป็นไปได้ที่จะรองรับ multi-gpu สำหรับโมเดล Phi แต่ ORT กับ 2 GPU ไม่รับประกันว่าจะให้ประสิทธิภาพสูงขึ้นเมื่อเทียบกับ 2 อินสแตนซ์ของ ORT โปรดดูที่ [ONNX Runtime](https://onnxruntime.ai/) สำหรับข้อมูลอัปเดตล่าสุด

ในงาน [Build 2024 the GenAI ONNX Team](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) ได้ประกาศว่าพวกเขาได้เปิดใช้งาน multi-instance แทน multi-gpu สำหรับโมเดล Phi

ในปัจจุบันนี้ช่วยให้คุณสามารถรันอินสแตนซ์ onnnxruntime หรือ onnxruntime-genai ตัวใดตัวหนึ่งโดยใช้ตัวแปรสภาพแวดล้อม CUDA_VISIBLE_DEVICES ดังนี้

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

รู้สึกอิสระที่จะสำรวจ Phi เพิ่มเติมได้ที่ [Microsoft Foundry](https://ai.azure.com)
