## Phi Family ใน Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) เป็นแพลตฟอร์มที่เชื่อถือได้ซึ่งช่วยให้นักพัฒนาสามารถขับเคลื่อนนวัตกรรมและสร้างอนาคตด้วย AI อย่างปลอดภัย มั่นคง และมีความรับผิดชอบ



[Microsoft Foundry](https://ai.azure.com) ถูกออกแบบมาเพื่อนักพัฒนาเพื่อ:

- สร้างแอปพลิเคชัน AI เชิงสร้างสรรค์บนแพลตฟอร์มระดับองค์กร
- สำรวจ สร้าง ทดสอบ และปรับใช้โดยใช้เครื่องมือ AI และโมเดล ML ที่ล้ำสมัย โดยยึดหลักปฏิบัติ AI อย่างรับผิดชอบ
- ทำงานร่วมกับทีมตลอดวงจรชีวิตการพัฒนาแอปพลิเคชัน

ด้วย Microsoft Foundry คุณสามารถสำรวจโมเดล บริการ และความสามารถหลากหลาย และเริ่มสร้างแอปพลิเคชัน AI ที่เหมาะสมกับเป้าหมายของคุณ แพลตฟอร์ม Microsoft Foundry ช่วยให้ขยายขนาดได้เพื่อเปลี่ยนแนวคิดสู่แอปพลิเคชันการผลิตที่สมบูรณ์ได้อย่างง่ายดาย การติดตามและปรับปรุงอย่างต่อเนื่องช่วยสนับสนุนความสำเร็จในระยะยาว

![portal](../../../../../translated_images/th/AIFoundryPorral.6b1094b101dd499e.webp)

### **เรียกใช้โค้ด Python เพื่อเรียก Microsoft Foundry Phi-4**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# รวมผลลัพธ์เสียงพูดหากเปิดใช้งานเสียงพูด
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
