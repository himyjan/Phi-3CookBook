# AITK இல் Phi குடும்பம்

[VS கோடுக்கான AI கருவிப்பெட்டி](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) Microsoft Foundry கருவிப்பெட்டி மற்றும் Hugging Face போன்ற மற்ற கருவிப்பெட்டிகள் மூலம் சிறந்த AI மேம்பாட்டு கருவிகள் மற்றும் மாதிரிகளை ஒன்றாக கொண்டு வந்து உருவாக்கும் AI செயலிகளை எளிதாக்குகிறது. GitHub மாதிரிகள் மற்றும் Microsoft Foundry மாதிரி கருவிப்பெட்டிகள் மூலம் இயக்கப்படும் AI மாதிரிகள் கருவிப்பெட்டியை உலாவி, அவற்றை உள்ளூர் அல்லது தொலைபேசி பதிவிறக்கம் செய்து, நுணுக்கம் செய்தல், சோதனை செய்தல் மற்றும் உங்கள் செயலியில் பயன்படுத்தலாம்.

AI கருவிப்பெட்டி முன்னோட்டம் உள்ளூரே இயங்கும். உள்ளூர் கருதுப்பு அல்லது நுணுக்கம் செய்வது நீங்கள் தேர்ந்தெடுத்த மாதிரியின் பொருள்படி Nvidia CUDA GPU போன்ற GPU தேவைப்படலாம். GitHub மாதிரிகளை நேராக AITK உடனும் இயக்கலாம்.

### AI கருவிப்பெட்டியை நிறுவுதல்

AI கருவிப்பெட்டி [Visual Studio Code விரிவாக்கமாக](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions) வழங்கப்படுகிறது, ஆகையால் முதலில் [VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo) ஐ நிறுவி, அதன் பிறகு [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) இலிருந்து AI கருவிப்பெட்டியை பதிவிறக்கம் செய்ய வேண்டும்.
[AI கருவிப்பெட்டி Visual Studio Marketplace இல் கிடைக்கிறது](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) மற்றும் மற்ற VS Code விரிவாக்கங்களோடு அதே விதமாக நிறுவலாம்.

VS Code விரிவாக்கங்களை நிறுவுவதில் நீங்கள் பரிச்சயமாக இல்லாவிட்டால், கீழ்க்கண்ட படிகளை பின்பற்றவும்:

இப்போது, நீங்கள் விரிவாக்கத்தை பயன்படுத்த தயார்!

உள்நுழைந்த பிறகு செயல்முறை படிகளை பின்பற்றவும். வெற்றிகரமாக முடிந்த பின், நீங்கள் VS Code க்கு திருப்பி அனுப்பப்படும்.

விரிவாக்கம் நிறுவப்பட்ட பின் AI கருவிப்பெட்டி ஐகான் உங்கள் செயல்பாட்டு பட்டியில் தோன்றும்.

### கிடைக்கும் செயல்பாடுகள்

AI கருவிப்பெட்டியின் முதன்மை பக்கப்பக்க பட்டி கீழ்க்காணும் வகையில் அமைக்கப்பட்டுள்ளது:

- **Models**
- **Resources**
- **Playground**  
- **Fine-tuning**
- **Evaluation**

Resources பகுதியில்தான் கிடைக்கின்றன. துவக்கமாக **Model Catalog** ஐ தேர்வு செய்யவும்.

### கருவிப்பெட்டியில் இருந்து ஒரு மாதிரியை பதிவிறக்கம் செய்துகொள்ளுதல்

VS Code பக்கப்பக்க பட்டியிலிருந்து AI கருவிப்பெட்டியை தொடங்கி, கீழ்க்கண்ட விருப்பங்களில் ஒன்றை தேர்ந்தெடுக்கலாம்:

![AI toolkit model catalog](../../../../../translated_images/ta/AItoolkitmodel_catalog.7a7be6a7d8468d31.webp)

- **Model Catalog** இல் இருந்து ஆதரவு கிடைக்கும் மாதிரியை கண்டுபிடித்து உள்ளூர் பதிவிறக்கம் செய்யவும்
- **Model Playground** இல் மாதிரி கருதுப்பதைச் சோதனை செய்யவும்
- **Model Fine-tuning** இல் உள்ளூரோ அல்லது தொலைதூர நுணுக்கம் செய்யவும்
- நூல் மலர்களை (fine-tuned models) AI கருவிப்பெட்டி கட்டளை பட்டியலால் மேகத்தில் பிரசுரிக்கவும்
- மாதிரிகளை மதிப்பீடு செய்யவும்

> [!NOTE]
>
> **GPU Vs CPU**
>
> மாதிரிக் கார்டுகள் மாதிரி அளவு, தளம் மற்றும் வேகம் அதிகரிப்பான் வகை (CPU, GPU) ஆகியவற்றை காட்டும். **ஒரு GPU கொண்ட விண்டோஸ் சாதனங்களில் சிறந்த செயல்திறனைப் பெற,** Windows ஐ குறிவைக்கும் மாதிரி பதிப்புகளைத் தேர்வு செய்யவும்.
>
> இது DirectML வேகம் அதிகரிப்பானுக்கு மேம்படுத்தப்பட்ட மாதிரியை உறுதி செய்யும்.
>
> மாதிரி பெயர்கள் வடிவில் இருக்கும்
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
> உங்கள் விண்டோஸ் சாதனத்தில் GPU உள்ளதா എന്ന് பார்வையிட, **Task Manager** திறந்து **Performance** தாவலை தேர்வு செய்யவும். GPUகள் "GPU 0" அல்லது "GPU 1" போன்ற பெயர்களில் பட்டியலிடப்படும்.

### உங்கள் செயலியில் REST API ஐ பயன்படுத்துதல்

AI கருவிப்பெட்டி உள்ளூர் REST API வலைசேவையகம் **port 5272** இல் வருகிறது, இது [OpenAI chat completions format](https://platform.openai.com/docs/api-reference/chat/create) ஐ பயன்படுத்துகிறது.

இது உங்கள் செயலியை உள்ளூர் சோதிக்க மேக AI மாதிரி சேவையைப் பகிர வேண்டியதில்லை. உதாரணமாக, கீழ்காணும் JSON கோப்பு கோரிக்கை உட்பொருளை கையாளும் விதத்தை காட்டுகிறது:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```

[Postman](https://www.postman.com/) அல்லது CURL (Client URL) உபகரணத்தைப் பயன்படுத்தி REST API ஐ சோதிக்கலாம்:

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```

### Python க்கு OpenAI கிளையண்ட் நூலகத்தைப் பயன்படுத்துதல்

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # API க்காக தேவையானது ஆனால் பயன்படவில்லை
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```

### .NET க்கு Azure OpenAI கிளையண்ட் நூலகத்தைப் பயன்படுத்துதல்

NuGet வழியாக உங்கள் திட்டத்தில் [Azure OpenAI கிளையண்ட் நூலகத்தை](https://www.nuget.org/packages/Azure.AI.OpenAI/) சேர்க்கவும்:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```

**OverridePolicy.cs** என்ற C# கோப்பை உங்கள் திட்டத்தில் சேர்த்து பின்வரும் குறியீட்டை ஒட்டவும்:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```

பின்னர், உங்கள் **Program.cs** கோப்பில் பின்வரும் குறியீட்டை ஒட்டவும்:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
