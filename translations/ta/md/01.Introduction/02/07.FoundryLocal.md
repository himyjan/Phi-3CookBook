## Foundry Local இல் Phi-Family மாதிரிகள் பயன்படுத்துதல் ஆரம்பிப்பது

### Foundry Local அறிமுகம்

Foundry Local என்பது அதிகாரப்பூர்வ-தர AI திறன்களை நேரடியாக உங்கள் உள்ளூர் ஹார்டゑர் மீது கொண்டுவரும் சக்திவாய்ந்த சாதி மேல்-AI தீர்வு ஆகும். இந்த கையேடு, Phi-Family மாதிரிகளை Foundry Local உடன் அமைத்து பயன்படுத்துவதற்கு வழிகாட்டும், உங்களின் AI பணிகளை முழுமையாக கட்டுப்படுத்தி, தனியுரிமையை பாதுகாத்து, செலவுகளை குறைக்க உதவும்.

Foundry Local உங்கள் கனசாட்சிகாரத்தில் உள்ள AI மாதிரிகளை இயக்கி செயல்திறன், தனியுரிமை, மாற்றமாற்றவை மற்றும் செலவு தகுதிகளை வழங்குகிறது. இது உங்கள் தற்போது உள்ள பணிமுறைகளில் மற்றும் பயன்பாடுகளில் CLI, SDK மற்றும் REST API மூலம் சீராக இணைகிறது.


![arch](../../../../../translated_images/ta/foundry-local-arch.8823e321dd8258d7.webp)

### Foundry Local ஐ ஏன் தேர்ந்தெடுக்க வேண்டும்?

Foundry Local இன் நன்மைகளை புரிந்துகொள்வது உங்கள் AI பராமரிப்பு திட்டங்களை தகவல் அடிப்படையில் மேற்கொள்ள உதவியாக இருக்கும்:

- **சாதியில் இயங்கும் கண்டறிதல்:** உங்கள் சொந்த ஹார்டゑர் மீது மாதிரிகளை இயக்கி, செலவுகளை குறைத்து, அனைத்து தரவையும் உங்களது சாதியிலேயே வைக்கலாம்.

- **மாதிரி தனிப்பயன்மைமை:** முன்பிரதிபட்ட மாதிரிகளிலிருந்து தேர்ந்தெடுக்கவும் அல்லது உங்கள் சொந்த மாதிரியை பயன்படுத்தி, குறிப்பிட்ட தேவைகள் மற்றும் பயன்பாடுகளை பூர்த்திசெய்யவும்.

- **செலவு திறன்:** தொடர்ச்சியான மேக சேவைகள் செலவுகளை நீக்கி, உங்கள் தற்போதைய சாதியைக் கொண்டு AI ஐ மேலும் அணுகக்கூடியதாக மாற்றுதல்.

- **சீரான ஒருங்கிணைப்பு:** SDK, API எண்ட்பாயின்ட்ஸ் அல்லது CLI மூலமாக பயன்பாடுகளுடன் இணைத்து, தேவைகள் அதிகரிக்கும் போது Microsoft Foundryக்கு சுலபமாக விரிவாக்கம் செய்யலாம்.

> **தொடக்க குறிப்பு:** இந்த கையேடு Foundry Local ஐ CLI மற்றும் SDK இடைமுகங்கள் மூலம் பயன்படுத்துவதை மையமாகக் கொண்டது. உங்கள் பயன்பாட்டிற்கு சிறந்த முறையை தேர்வுசெய்ய இரண்டு வழிகளையும் கற்றுக்கொள்வீர்கள்.

## பகுதி 1: Foundry Local CLI அமைத்து இயக்குதல்

### படி 1: நிறுவல்

Foundry Local CLI உங்கள் உள்ளூர் AI மாதிரிகளை நிர்வகித்து இயக்குகை செய்யும் வாயிலாகும். உங்கள் கணினியில் அதை நிறுவுவோம்.

**ஆதரவு கிடைக்கும் தளம்:** Windows மற்றும் macOS

நிறுவல் வழிமுறைகளுக்காக, [அதிகாரப்பூர்வ Foundry Local ஆவணத்தினை](https://github.com/microsoft/Foundry-Local/blob/main/README.md) பார்க்கவும்.

### படி 2: கிடைக்கும் மாதிரிகளைக் காண்பதும்

Foundry Local CLI நிறுவப்பட்டதும், உங்களது பயன்பாட்டிற்கான கிடைக்கும் மாதிரிகளை கண்டுபிடிக்க முடியும். இந்த கட்டளையால் அனைத்து ஆதரவு பெற்ற மாதிரிகளும் காண்பிக்கப்படும்:


```bash
foundry model list
```

### படி 3: Phi குடும்ப மாதிரிகளை புரிந்துகொள்ளுதல்

Phi குடும்பம் பல்வேறு பயன்பாடுகள் மற்றும் ஹார்டゑர் அமைப்புகளுக்கான சிறந்தசெய்யப்பட்ட மாதிரிகளைக் கொண்டுள்ளது. Foundry Local இல் கிடைக்கும் Phi மாதிரிகள் இவை:

**கிடைக்கும் Phi மாதிரிகள்:**

- **phi-3.5-mini** - அடிப்படை பணிகளுக்கான சுருக்கமான மாதிரி
- **phi-3-mini-128k** - நீண்ட உரையாடல்களுக்கான விரிவடைந்த சூழல் பதிப்பு
- **phi-3-mini-4k** - பொதுவான பயன்பாட்டுக்கு நிலையான சூழல் மாதிரி
- **phi-4** - மேம்பட்ட திறன்களுடன் முன்னேற்றிய மாதிரி
- **phi-4-mini** - Phi-4 இன் எளிதான பதிப்பு
- **phi-4-mini-reasoning** - சிக்கலான கருதுபாவ பணிகளுக்கு சிறப்பு

> **ஹார்டゑர் பொருந்துதல்:** உங்கள் கணினி திறன்படைக்கு ஏற்ப ஒவ்வொரு மாதிரியும் CPU, GPU போன்ற வன்பொருள் விரைவுப் பூஜ்ஜியங்களுக்காக அமைக்கப்படலாம்.

### படி 4: உங்கள் முதலாவது Phi மாதிரியை ஓட்டுதல்

வழக்கமாக ஆரம்பிப்போம். `phi-4-mini-reasoning` மாதிரியை ஓட்டி பார்ப்போம், இது படி படியாக கடுமையான பிரச்சனைகளை தீர்க்க சிறந்தது.


**மாதிரியை இயக்க கட்டளை:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **முதல் முறையாக அமைப்பு:** ஒரு மாதிரியை முதற்கட்டமாக இயக்கும்போது, Foundry Local அதை தானாக உங்கள் உள்ளூர் சாதியிலே பதிவிறக்கம் செய்யும். உங்கள் நெட்வொர்க் வேகத்தைப் பொறுத்து பதிவிறக்கம் நேரம் மாறும், ஆகவே ஆரம்ப அமைப்பின் போது பொறுமையாக இருங்கள்.

### படி 5: மாதிரியை உண்மையான பிரச்சனையுடன் சோதனை செய்தல்

இப்போது, படி படியாக கருதுபாவம் செய்தல் எப்படி நடைபெறும் என்பதை பார்க்கும் வகையில், ஒரு அறிவியல் விதி பிரச்சனையை எடுத்து சோதிப்போம்:

**உதாரண பிரச்சனை:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**எதிர்பார்க்கப்படும் நடத்தை:** இந்த பிரச்சனையை, pheasants க்கு 2 கால் மற்றும் rabbits க்கு 4 கால் உள்ளதை பயன்படுத்தி, லாஜிக்கல் படிவங்களில் உடைத்து, சமன்பாட்டு அமைப்பை தீர்க்கும்.

## பகுதி 2: Foundry Local SDK பயன்படுத்தி பயன்பாடுகள் உருவாக்குதல்

### SDKஐ ஏன் பயன்படுத்த வேண்டும்?

CLI சோதனை மற்றும் விரைவு தொடர்புகளுக்கு சிறந்தது என்றாலும், SDK Foundry Local ஐ உங்கள் பயன்பாடுகளில் நிரல்பூர்வமாக ஒருங்கிணைக்க உதவுகிறது. இதனால்:

- தனிப்பயன் AI சக்தியுள்ள பயன்பாடுகளை உருவாக்கலாம்
- தானியங்கி பணிமுறைகளை உருவாக்கலாம்
- ஏற்கனவே உள்ள அமைப்புகளில் AI திறன்களை இணைக்கலாம்
- உரையாடல் பாட்டுகள் மற்றும் தொடர்புடைய கருவிகளை உருவாக்கலாம்

### ஆதரவு பெறும் நிரல்பாம்கள்

Foundry Local பல நிரல்பாம்களுக்கு SDK ஆதரவு வழங்குகிறது, உங்கள் வளர்ச்சி விருப்பங்களுக்கு பொருந்துமாறு:

**📦 கிடைக்கும் SDKகள்:**

- **C# (.NET):** [SDK ஆவணம் மற்றும் எடுத்துக்காட்டுகள்](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK ஆவணம் மற்றும் எடுத்துக்காட்டுகள்](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK ஆவணம் மற்றும் எடுத்துக்காட்டுகள்](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK ஆவணம் மற்றும் எடுத்துக்காட்டுகள்](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### அடுத்த படிகள்

1. உங்கள் வளர்ச்சி சூழலுக்கு ஏற்ப விருப்பமான SDKஐ தேர்வு செய்யவும்
2. விரிவான நடைமுறை வழிகாட்டல்களுக்காக SDK-அனுபவ ஆவணங்களை பின்பற்றவும்
3. சிக்கலான பயன்பாடுகளை உருவாக்குவதற்கு முன் எளிய எடுத்துக்காட்டுகளுடன் தொடங்கவும்
4. ஒவ்வொரு SDK சேமிப்பிடத்திலும் கொடுக்கப்பட்ட மாதிரி குறியீட்டைக் கற்றறிந்து காணவும்

- ✅ பயன்பாட்டை உருவாக்க SDK விருப்பங்களை புரிந்து கொள்வது
