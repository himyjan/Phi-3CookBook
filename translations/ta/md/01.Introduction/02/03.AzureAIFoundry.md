## மைக்ரோசாஃப்ட் ஃபவுண்ட்ரியில் பை குடும்பம்

[Microsoft Foundry](https://ai.azure.com) என்பது பாதுகாப்பான, நம்பகமான மற்றும் பொறுப்புமிக்க முறையில் AI மூலம் புதுமையை ஊக்குவித்து எதிர்காலத்தை வடிவமைக்க டெவலப்பர்களுக்கு அதிகாரம் வழங்கும் ஒரு நம்பகமான தளமாகும்.



[Microsoft Foundry](https://ai.azure.com) டெவலப்பர்களுக்காக வடிவமைக்கப்பட்டுள்ளது:

- எண்டர்பிரைஸ் தரத்தின் மீது புதுமை AI செயலிகளை கட்டமைக்க.
- நவீன AI கருவிகள் மற்றும் இயந்திர கற்று மாதிரிகள் மூலம் ஆராய்ந்து, கட்டமைத்து, சோதித்து, மற்றும் பிரசுரிக்க, பொறுப்பான AI நடைமுறைகளை அடிப்படையாகக் கொண்டு.
- செயலி மேம்பாட்டின் முழு வாழ்நாளுக்கான குழுவுடன் ஒத்துழைக்க.

Microsoft Foundry மூலம், நீங்கள் பலவகை மாதிரிகள், சேவைகள் மற்றும் திறன் வாய்ந்தவற்றைக் கண்டறிந்து, உங்கள் நோக்கங்களுக்கு மிகவும் பொருந்தும் AI செயலிகளை கட்டமைக்கலாம். மைக்ரோசாஃப்ட் ஃபவுண்ட்ரி தளம், சான்றிதழ் கொள்கைகளை முழுமையான உற்பத்தி செயலிகளாக மாற்ற ஸ்கெயிலபிலிட்டியை வசதியாக்குகிறது. தொடர்ச்சியான கண்காணிப்பு மற்றும் மேம்படுத்தல் நீண்டகால வெற்றிக்கு ஆதரவு அளிக்கிறது.

Microsoft Foundry இல் Azure AOAI சேவையை பயன்படுத்துவதுடன், நீங்கள் Microsoft Foundry மாதிரி பதிவியிலுள்ள மூன்றாம் தரப்புப் மாதிரிகளையும் பயன்படுத்தலாம். உங்கள் AI தீர்வு தளமாக Microsoft Foundry-ஐ பயன்படுத்த விரும்பினால் இது நல்ல தேர்வாகும்.

Microsoft Foundry உள்ள மாதிரி பதிவியிலிருந்து பை குடும்ப மாதிரிகளை விரைவில் பிரசுரிக்கலாம்

### **Microsoft Foundry Phi-4 ஐ அழைக்க Python குறியீட்டைக் இயக்குதல்**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# பேச்சு இயக்கப்பட்டிருந்தால் பேச்சு முடிவு சேர்க்கவும்
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
