# பை ஹார்ட்வேர் ஆதரவு

Microsoft Phi ONNX Runtimeக்கு உகந்ததாக உருவாக்கப்பட்டுள்ளது மற்றும் Windows DirectMLஐ ஆதரிக்கிறது. இது GPUகள், CPUகள் மற்றும் மொபைல் சாதனங்களையும் உள்ளடக்கிய பல்வேறு ஹார்ட்வேர் வகைகளில் சிறப்பாக வேலை செய்கிறது.

## சாதனம் ஹார்ட்வேர்
குறிப்பாக, ஆதரவான ஹார்ட்வேர் கீழ்காணும் வகையில் உள்ளது:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPUs, 128 GiB நினைவகம்)

## மொபைல் SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 அல்லது அதற்கு மேல் A16/A17 செயலி

## பை ஹார்ட்வேர் விவரங்கள்

- குறைந்தபட்ச கட்டமைப்பு தேவை.
- Windows: DirectX 12-க்கு உகந்த GPU மற்றும் குறைந்தபட்சம் 4GB இணைந்த RAM

CUDA: NVIDIA GPU கணினித் திறன் >= 7.02

![HardwareSupport](../../../../../translated_images/ta/01.phihardware.5d51b2377cba18af.webp)

## பல GPUகளில் onnxruntime ஓட்டுதல்

தற்போது கிடைக்கும் Phi ONNX மாதிரிகள் 1 GPUக்கே ஆகும். Phi மாதிரிக்கு பல GPU ஆதரவு வழங்கிச் செய்யக்கூடும், ஆனாலும் 2 GPU கொண்ட ORT அதே அளவு throughputஐ 2 ORT instanceக்குச் சமமா தருமென்று உறுதி இல்லை. புதிய தகவல்களுக்கு [ONNX Runtime](https://onnxruntime.ai/) பாருங்கள்.

[Build 2024 the GenAI ONNX Team](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) அதில் Phi மாதிரிகளுக்கான multi-gpuவுக்கு பதிலாக multi-instanceஐ இயலுமைப்படுத்தியதாக அறிவித்தனர்.

தற்போது இது CUDA_VISIBLE_DEVICES மாறிலியுடன் ஒரு onnxruntime அல்லது onnxruntime-genai instanceஐ இயக்க அனுமதிக்கிறது.

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

[Microsoft Foundry](https://ai.azure.com) இல் Phiவை மேலும் ஆராய்ந்து பார்க்கவும்.
