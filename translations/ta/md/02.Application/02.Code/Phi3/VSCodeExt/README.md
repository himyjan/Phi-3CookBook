# **மைக்ரோசாஃப்ட் பை-3 குடும்பத்துடன் உங்கள் சொந்த Visual Studio Code GitHub Copilot Chat ஐ உருவாக்கவும்**

GitHub Copilot Chat இல் வேலைப்பயிற்சி முகவரியை நீங்கள் பயன்படுத்தியுள்ளீர்களா? உங்கள் குழுவில் உங்கள் சொந்தக் கோடு முகவரியை உருவாக்க விரும்புகிறீர்களா? இந்த கைகள்-ஆன் ஆய்வகம் திறந்த மூல மாதிரியை இணைத்து ஒரு நிறுவனத் தரம் கொண்ட கோடு வணிக முகவரியை உருவாக்கும் நோக்கில் உள்ளது.

## **அடித்தளம்**

### **மைக்ரோசாஃப்ட் பை-3ஐ ஏன் தேர்ந்தெடுக்க வேண்டும்**

பை-3 என்பது குடும்பத் தொடர், இதில் phi-3-mini, phi-3-small, மற்றும் phi-3-medium ஆகியவை வெவ்வேறு பயிற்சி அளவுருக்களுக்கு அடிப்படையாக கொண்டவை, உரை உருவாக்கம், உரையாடல் நிறைவு மற்றும் கோடு உருவாக்கத்துக்காக. அதேபோல் Vision அடிப்படையிலான phi-3-vision கூட உள்ளது. இது நிறுவனங்கள் அல்லது வெவ்வேறு குழுக்களுக்கு ஆஃப்லைன் உருவாக்கும் AI தீர்வுகளை உருவாக்க ஏற்றது.

இந்த இணைப்பை படிக்க பரிந்துரைக்கப்படுகிறது [https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/01/01.PhiFamily.md](https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/01/01.PhiFamily.md)

### **மைக்ரோசாஃப்ட் GitHub Copilot Chat**

GitHub Copilot Chat நீங்களும் GitHub Copilot உடன் தொடர்பு கொண்டு குறியீடு தொடர்பான கேள்விகளுக்கு பதிலளிக்கும் ஒரு உரையாடல் இடைமுகத்தை VS Code இல் நேரடியாக வழங்கும் விருத்தி ஆகும், இது ஆவணங்களை தேடவோ ஆன்லைன் கருத்தரங்குகளை ஆராயவோ தேவையில்லாமல் உள்ளது.

Copilot Chat உருவாக்கப்பட்ட பதிலுக்கு தெளிவைச் சேர்க்கவும், உரையியல் வெளிச்சம், இடைநீக்கம் மற்றும் பிற வடிவமைப்பு அம்சங்களை பயன்படுத்தக்கூடும். பயனர் கேள்வியின் வகையின் அடிப்படையில், பதிலில் Copilot பயன்படுத்திய உரைகோப்பு கோப்புகள் அல்லது ஆவணங்கள் போன்ற தொடர்புகள் அல்லது VS Code செயல்பாட்டிற்கான பொத்தான்கள் இடப்பட்டிருப்பதல்லவோ இருக்கலாம்.

- Copilot Chat உங்கள் மென்பொருள் வளர்ச்சித் திட்டத்தில் ஒருங்கிணைக்கபட்டு தேவையான உதவியைக் கொடுக்கும்:

- குறியீடு எழுதும் போது உதவிக்காக பதிப்பி அல்லது டெர்மினலில் நேரடியாக ஒரு வகையில் உரையாடலைத் தொடங்கவும்

- உட்பக்க AI உதவியாளரை எந்த நேரத்திலும் உதவிக்கு பயன்படுத்த உரையாடல் காட்சி

- விரைந்த கேள்வி கேட்டு மீண்டும் உங்கள் பணியில் ஈடுபட விரைந்து உரையாடலைத் தொடங்கு

GitHub Copilot Chat ஐ பல்வேறு சூழல்களில் பயன்படுத்தலாம், உதாரணமாக:

- பிரச்சனையை சிறந்த முறையில் எவ்வாறு தீர்க்க வேண்டும் என்று குறியீடு கேள்விகளுக்கு பதிலளிக்கும்

- மற்றவரின் குறியீட்டை விளக்கும் மற்றும் மேம்படுத்துகைகளைக் குறிப்பது

- குறியீடு திருத்தங்களை முன்மொழிவது

- அலகு சோதனை வழக்குகளை உருவாக்கவும்

- குறியீடு ஆவணங்களை உருவாக்கவும்

இந்த இணைப்பை படிக்க பரிந்துரைக்கப்படுகிறது [https://code.visualstudio.com/docs/copilot/copilot-chat](https://code.visualstudio.com/docs/copilot/copilot-chat?WT.mc_id=aiml-137032-kinfeylo)


###  **Microsoft GitHub Copilot Chat @workspace**

Copilot Chat இல் **@workspace** குறிப்பு உங்கள் முழுக் குறியீடு அடிமளத்தைப் பற்றி கேள்வி கேட்க அனுமதிக்கிறது. கேள்வியின் அடிப்படையில், Copilot பொதுவான கோப்புகள் மற்றும் குறியீடு சின்னங்களை திறம்பட கண்டுபிடித்து, பின்னர் பதில்லில் அவற்றை தொடர்புகள் மற்றும் குறியீடு எடுத்துக்காட்டுகளாக குறிப்பிடுகிறது.

உங்கள் கேள்விக்கு பதில் அளிக்க, **@workspace** VS Code இல் ஒரு டெவலப்பர் குறியீடு அடிமளை வழிசெய்ய பயன்படுத்தும் அதே மூலங்களைத் தேடும்:

- .gitignore கோப்பால் புறக்கணிக்கப்பட்ட கோப்புகளை தவிர வேலைப்பளத்தில் உள்ள அனைத்து கோப்புகளும்

- அடைவகமைப்பு உள்ள கோப்புகள் மற்றும் அடைவுகளின் பெயர்கள்

- வேலைப்பளம் GitHub रिपாசிடரி ஆகவும், குறியீடு தேடல் மூலம் குறியிடப்பட்ட இருந்தால் GitHub-ன் குறியீடு தேடல் குறியீடு

- வேலைப்பளத்திலுள்ள குறியீடு சின்னங்கள் மற்றும் வரையறைகள்

- தற்போது தேர்ந்தெடுக்கப்பட்ட உரை அல்லது செயலில் உள்ள தொகுப்பாளியில் காணப்படும் உரை

குறிப்பு: .gitignore புறக்கணிக்கப்பட்ட கோப்புக்குள் கோப்பை திறந்தால் அல்லது உரையை தேர்ந்தெடுத்தால் அது புறக்கணிக்கப்படவில்லை.

இந்த இணைப்பை படிக்க பரிந்துரைக்கப்படுகிறது [[https://code.visualstudio.com/docs/copilot/copilot-chat](https://code.visualstudio.com/docs/copilot/workspace-context?WT.mc_id=aiml-137032-kinfeylo)]


## **இந்த ஆய்வகத்தைப் பற்றி மேலும் அறியவும்**

GitHub Copilot நிறுவனங்களின் நிரலாக்க திறனை பெரிதும் மேம்படுத்தியுள்ளது, மேலும் எவ்வொரு நிறுவனமும் GitHub Copilot இன் தொடர்புடைய அம்சங்களை தனிப்பயனாக்க விரும்புகிறது. பல நிறுவனங்கள் தங்கள் வணிக சூழல்களுக்கு மற்றும் திறந்த மூல மாதிரிகளுக்கு ஏற்ப GitHub Copilot போன்ற விருத்திகள் தனிப்பயனாக்கியுள்ளன. நிறுவனங்களுக்கு தனிப்பயனாக்கப்பட்ட விருத்திகள் கட்டுப்படுத்த எளிதாக இருக்கின்றன, ஆனால் இது பயனர் அனுபவத்தையும் பாதிக்கிறது. GitHub Copilot பொதுவான சூழல்களை மற்றும் திறமைகளை குறியீட்டு நேர்மை மற்றும் தொழில்முறை ரீதியில் சிறப்பாக கையாள்கிறது. அனுபவம் இடையில்லாமல் இருந்தால், அதன் நிறுவனம் தனிப்பயனாக்கிய விருத்தி இன்னும் சிறந்ததாக இருக்கும். GitHub Copilot Chat நிறுவங்களுக்கு உரையாடலின் அனுபவங்களை விரிவுபடுத்த தொடர்புடைய APIs வழங்குகிறது. ஒரு நிலையான அனுபவத்தை உருவாக்கி தனிப்பயனாக்கப்பட்ட அம்சங்களின் மூலம் சிறந்த பயனர் அனுபவம் வழங்கப்படுகிறது.

இந்த ஆய்வகம் முக்கியமாக Phi-3 மாதிரியை உள்ளூர் NPU மற்றும் Azure கலவையை இணைத்து GitHub Copilot Chat இல் தனிப்பயன் முகவர் ***@PHI3*** உருவாக்கி நிறுவன டெவலப்பர்களுக்கு குறியீடு உருவாக்கத்திலும் ***(@PHI3 /gen)*** படங்களின் அடிப்படையில் குறியீடு உருவாக்கத்திலும் ***(@PHI3 /img)*** உதவுகிறது.
