# Microsoft Foundry இல் Prompt flow உடன் தனிப்பயன் Phi-3 மாதிரிகளை நுட்பமாகச் சரிசெய்து ஒருங்கிணைக்கவும்

Microsoft Tech Community இல் இருந்து "[Microsoft Foundry இல் Prompt Flow உடன் தனிப்பயன் Phi-3 மாதிரிகளை நுட்பமாகச் சரிசெய்து ஒருங்கிணைக்கும் முறை](https://techcommunity.microsoft.com/t5/educator-developer-blog/fine-tune-and-integrate-custom-phi-3-models-with-prompt-flow-in/ba-p/4191726?WT.mc_id=aiml-137032-kinfeylo)" என்ற வழிகாட்டலை அடிப்படையாக கொண்ட இந்த முற்றிலும் முடிந்த (E2E) மாதிரிப்பாகும். இது Microsoft Foundry இல் Prompt flow உடன் தனிப்பயன் Phi-3 மாதிரிகளை நுட்பமாகச் சரிசெய்தல், இடைநீக்கம் செய்தல் மற்றும் ஒருங்கிணைக்கும் செயல்முறைகளை அறிமுகப்படுத்துகிறது. E2E மாதிரிப்புடன் வேறுபடுமெனில், "[Prompt Flow உடன் தனிப்பயன் Phi-3 மாதிரிகளை நுட்பமாகச் சரிசெய்தலும் ஒருங்கிணைப்பும்](./E2E_Phi-3-FineTuning_PromptFlow_Integration.md)" என்னும் மாதிரி உள்ளூர் இடத்தில் குறியீட்டை இயக்கியது, இந்தப் பயிற்சிக்கட்டுரையில் Azure AI / ML ஸ்டுடியோவில் உங்கள் மாதிரியை நுட்பமாகச் சரிசெய்தலும் ஒருங்கிணைப்பும் மட்டுமே உள்ளது.

    - பயன்படுத்த வேண்டிய **கண்டெயினர் பதிவு மையத்தை** தேர்ந்தெடுக்கவும் (தேவைப்பட்டால் புதிதாய் உருவாக்கவும்).

### பங்கு ஒதுக்கீட்டைச் சேர்க்கவும்

    ```python
        """
        # கோப்புறை இல்லையெனில் உருவாக்கவும்
        os.makedirs(os.path.dirname(filepath), exist_ok=True)
        
        # கோப்பை எழுதும் முறையில் திறக்கவும்
        with open(filepath, 'w', encoding='utf-8') as f:
            # தரவுத்தொகுப்பில் உள்ள ஒவ்வொரு பதிவிலும் மடக்கவும்
            for record in dataset:
                # பதிவை JSON பொருளாக மாற்றி அதைக் கோப்பில் எழுதவும்
                json.dump(record, f)
                # பதிவுகளை பிரிக்க புதிய வரிசை எழுத்தை எழுதவும்
                f.write('\n')
        
        print(f"Dataset saved to {filepath}")

    def main():
        """
        Main function to load, split, and save the dataset.
        """
        # குறிப்பிட்ட கட்டமைப்பு மற்றும் பிரிப்புக் கீழ் ULTRACHAT_200k தரவுத்தொகுப்பை ஏற்றவும் மற்றும் பிரிக்கவும்
        dataset = load_and_split_dataset("HuggingFaceH4/ultrachat_200k", 'default', 'train_sft[:1%]')
        
        # பிரிப்பில் இருந்து பயிற்சி மற்றும் சோதனை தரவுத்தொகுப்புகளை பிரித்தெடுக்கவும்
        train_dataset = dataset['train']
        test_dataset = dataset['test']

        # பயிற்சி தரவுத்தொகுப்பை JSONL கோப்பாகச் சேமிக்கவும்
        save_dataset_to_jsonl(train_dataset, "data/train_data.jsonl")
        
        # சோதனை தரவுத்தொகுப்பை தனியான JSONL கோப்பாகச் சேமிக்கவும்
        save_dataset_to_jsonl(test_dataset, "data/test_data.jsonl")

    if __name__ == "__main__":
        main()

    ```

1. Prompt flow இல், நிறுவிய உள்ள ஓட்டத்தை பின்வரும் வேலைகளுடன் மீண்டும் கட்டமைக்கவும்:


    - **சேமிக்கவும்** ஐ தேர்வு செய்யவும்.

    ![Select raw file mode.](../../../../../../translated_images/ta/08-15-select-raw-file-mode.61d988b41df28985.webp)

1. Prompt flow இல் தனிப்பயன் Phi-3 மாடலை பயன்படுத்த *integrate_with_promptflow.py* கோப்புக்குள் பின்வரும் கோடுகளைச் சேர்க்கவும்.

    ```python
    import logging
    import requests
    from promptflow import tool
    from promptflow.connections import CustomConnection

    # பதிவு அமைப்பு
    logging.basicConfig(
        format="%(asctime)s - %(levelname)s - %(name)s - %(message)s",
        datefmt="%Y-%m-%d %H:%M:%S",
        level=logging.DEBUG
    )
    logger = logging.getLogger(__name__)

    def query_phi3_model(input_data: str, connection: CustomConnection) -> str:
        """
        Send a request to the Phi-3 model endpoint with the given input data using Custom Connection.
        """

        # "connection" என்பது தனிப்பயன் இணைப்பின் பெயர், "endpoint", "key" ஆகியவை தனிப்பயன் இணைப்பில் உள்ள விசைகளாகும்
        endpoint_url = connection.endpoint
        api_key = connection.key

        headers = {
            "Content-Type": "application/json",
            "Authorization": f"Bearer {api_key}"
        }
        data = {
            "input_data": {
                "input_string": [
                    {"role": "user", "content": input_data}
                ],
                "parameters": {
                    "temperature": 0.7,
                    "max_new_tokens": 128
                }
            }
        }
        try:
            response = requests.post(endpoint_url, json=data, headers=headers)
            response.raise_for_status()
            
            # முழு JSON பதிலை பதிவு செய்
            logger.debug(f"Full JSON response: {response.json()}")

            result = response.json()["output"]
            logger.info("Successfully received response from Azure ML Endpoint.")
            return result
        except requests.exceptions.RequestException as e:
            logger.error(f"Error querying Azure ML Endpoint: {e}")
            raise

    def my_python_tool(input_data: str, connection: CustomConnection) -> str:
        """
        Tool function to process input data and query the Phi-3 model.
        """
        return query_phi3_model(input_data, connection)
    ```
