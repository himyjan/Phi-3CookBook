### பாதுகாப்பு மதிப்பாய்வு அறிமுகம்

Microsoft Foundry இல் பாதுகாப்பு மதிப்பாய்வுகள், உங்கள் மாதிரி தலைமையிலான அமைப்பு பிணைப்பு தாக்குதல்களுக்கு எதிர்வினையுடையதா, மற்றும் அது பாதிப்பூட்டும் உள்ளடக்கம் உருவாக்கும் வாய்ப்புக்களை மதிப்பாய்வு செய்ய உதவுகிறது. இது நேரடியாக இ 원칙ங்களுடன் ஒத்துப்போகின்றன.

#### Microsoft–இன் பொறுப்பான செயற்கை நுண்ணறிவு 원칙ங்கள்

திட்டமிடும் தொழில்நுட்ப படிகள் துவங்குவதற்கு முன்பு, Microsoft–இன் பொறுப்பான செயற்கை நுண்ணறிவு 원칙ங்களை புரிந்து கொள்வது அவசியம். இவை செயற்கை நுண்ணறிவு அமைப்புக்களின் பொறுப்பான உருவாக்கம், பராமரிப்பு மற்றும் செயல்பாட்டை வழிநடத்தும் ஒரு நெறிமுறை கட்டமைப்பாகும். இ 원칙ங்கள் எவ்வாறு செயற்கை நுண்ணறிவு அமைப்புகள் நியாயமான, வெளிப்படையான மற்றும் ஒற்றுமையான முறையில் கட்டமைக்கப்பட வேண்டும் என்பதனை வழிகாட்டுகின்றன. இ 원칙ங்கள் செயற்கை நுண்ணறிவு மாதிரிகளின் பாதுகாப்பு மதிப்பாய்விற்கு அடித்தளம்.

Microsoft-இன் பொறுப்பான செயற்கை நுண்ணறிவு 원칙ங்கள்:

- **நியாயமும் ஒருங்கிணைப்பும்**: செயற்கை நுண்ணறிவு அமைப்புகள் அனைவருக்கும் நியாயமாக நடத்த வேண்டும் மற்றும் ஒரே நிலைமையிலுள்ள மக்கள் குழுக்களை வேறுபடுத்தி பாதிக்கக் கூடாது. உதாரணமாக, மருத்துவ சிகிச்சை, கடன் விண்ணப்பம் அல்லது வேலை வாய்ப்புகளிற்கு வழிகாட்டும் போது, ஒரே அறிகுறிகள், பொருளாதார நிலைகள், அல்லது தொழில் தகுதிகள் உள்ள அனைவருக்கும் ஒரே பரிந்துரைகளை வழங்க வேண்டும்.

- **நம்பகத்தன்மை மற்றும் பாதுகாப்பு**: நம்பிக்கை உருவாக்க, செயற்கை நுண்ணறிவு அமைப்புகள் நம்பகமாக, பாதுகாப்பாக மற்றும் நிலையாக செயல்பட வேண்டும். இவை திட்டமிடப்பட்டபடி செயல்படும், எதிர்பாராத சூழ்நிலைகளில் பாதுகாப்பாக பதிலளிக்கும், மற்றும் தீய விளைவுகளைத் தடுக்கும். இவை வடிவமைப்பு மற்றும் சோதனைக்குப் பின் எதிர்பார்த்த சூழ்நிலைகள் மற்றும் நிலைமைகளின் பரப்பு-பரப்பை பிரதிபலிக்கும்.

- **வெளிப்படை தன்மை**: மனிதர்களின் வாழ்க்கையில் பெரிய தாக்கம் உண்டாக்கும் முடிவுகளை தகவலோடு வழங்கும் போது, அந்த முடிவுகள் எவ்வாறு எடுக்கப்பட்டன என்பதை மக்கள் புரிந்துகொள்ள வேண்டும். உதாரணமாக, ஒரு வங்கி கடன் வழங்க AI அமைப்பைப் பயன்படுத்தினால், அது எப்படி கடனதாரணம் கண்டறிந்து தேர்வு செய்தது என்பதை மக்கள் புரிந்துகொள்ள வேண்டும்.

- **தனியுரிமை மற்றும் பாதுகாப்பு**: செயற்கை நுண்ணறிவு பரவலாகியதுடன், தனியுரிமை பாதுகாப்பதும் தனிப்பட்ட மற்றும் வணிகத் தகவல்களை பாதுகாத்தலும் முக்கியமாகவும் சிக்கலானதாகவும் மாறிவிட்டன. AI உடன், தனியுரிமை மற்றும் தரவு பாதுகாப்பு கவனமாக பராமரிக்கப்பட வேண்டும் ஏனெனில் AI அமைப்புகள் மனிதர்களைப் பற்றி சரியான மற்றும் அறிவார்ந்த கணிப்புகள் மற்றும் முடிவுகளை எடுக்க தரவுக்கு அணுகல் அவசியம்.

- **பொறுப்புத் திருப்பு**: AI அமைப்புகளை வடிவமைத்து மொத்தமாக செயல்படுத்தும் மக்கள் அதன் செயல்பாடுகளுக்குப் பொறுப்பாக இருக்க வேண்டும். நிறுவனங்கள் தொழிற்துறை தரக்குறிப்புகளைப் பயன்படுத்தி பொறுப்புத் திருப்பு நடைமுறைகளை உருவாக்க வேண்டும். இவைகள் மனித வாழ்வின் மீது தாக்கம் ஏற்படும் எந்த முடிவிலும் AI அமைப்புகள் இறுதி அதிகாரம் ஆகாமிருக்க உதவும். மேலும், முற்றிலும் சுயமரியாதை கொண்ட AI அமைப்புகளுக்கும் மனிதர்கள் பொருத்தமான கட்டுப்பாட்டை வைத்திருக்க உறுதி செய்யும்.

- **வெறுக்கும் மற்றும் அநியாயமான உள்ளடக்கம்**: மாதிரி வெறுக்கும் அல்லது அநியாயமான உள்ளடக்கம் உருவாக்கும் சாத்தியத்தை மதிப்பாய்வு செய்கிறது.
- **கடுமையான உள்ளடக்கம்**: மாதிரி கடுமையான உள்ளடக்கம் உருவாக்கும் சாத்தியத்தை மதிப்பாய்வு செய்கிறது.
- **பாலியல் உள்ளடக்கம்**: மாதிரி அசட்டையான பாலியல் உள்ளடக்கம் உருவாக்குவதை மதிப்பாய்வு செய்கிறது.

இந்த அம்சங்களை மதிப்பாய்வது AI மாதிரி பாதிப்பூட்டும் அல்லது அவமானகர உள்ளடக்கம் உருவாக்காமல் சமூக மதிப்பீடுகள் மற்றும் கட்டுப்பாட்டு ஒழுங்குகளுடன் இணைந்திருப்பதை உறுதி செய்கிறது.

#### செயல்திறன் அளவுகோல்கள்

இந்த பாடத்தில், Microsoft Foundry–இன் செயல்திறன் அளவுகோல்களைப் பயன்படுத்தி நுட்பமாக கட்டமைக்கப்பட்ட Phi-3 / Phi-3.5 மாதிரியின் செயல்திறனை மதிப்பாய்வு செய்வீர்கள். இந்த அளவுகோல்கள் மாதிரி சரியான, பொருத்தமான மற்றும் இணக்கமான பதில்களை உருவாக்குவதில் எவ்வளவு திறமையானது என்பதை மதிப்பிட உதவுகின்றன. செயல்திறன் அளவுகோல்கள்:

- **அடிப்படைத்தன்மை (Groundedness)**: உருவாக்கப்பட்ட பதில்கள் உள்ளீடு மூலத் தகவலுடன் எவ்வளவு பொருந்துகிறது என்பதை மதிப்பாய்வு செய்கிறது.
- **பொருத்தம் (Relevance)**: கொடுக்கப்பட்ட கேள்விகளுக்கு உருவாக்கப்பட்ட பதில்களின் பொருத்தத்தை மதிப்பாய்வு செய்கிறது.
- **இணக்கம் (Coherence)**: உருவாக்கப்பட்ட உரையின் ஓடல் நுட்பம், இயற்கையாக வாசிக்கும் விதம் மற்றும் மனித மொழி போன்ற தன்மையை மதிப்பாய்வு செய்கிறது.
- **சுவைநிலை (Fluency)**: உருவாக்கப்பட்ட உரையின் மொழி திறமையை மதிப்பாய்வு செய்கிறது.
- **GPT ஒப்பீடு (GPT Similarity)**: உருவாக்கப்பட்ட பதிலை உண்மை தரவுடன் ஒப்பிடுகிறது.
- **F1 மதிப்பெண்**: உருவாக்கப்பட்ட பதிலும் மூல தரவுமுள்ள பகிரப்பட்ட சொற்களின் விகிதத்தை கணக்கிடுகிறது.

இந்த அளவுகோல்கள் மாதிரி சரியான, பொருத்தமான மற்றும் இணக்கமான பதில்களை உருவாக்குவதில் எவ்வளவு திறமையானது என்பதை மதிப்பிட உதவுகின்றன.

1. Prompt flow இல் தனிப்பயன் Phi-3 / Phi-3.5 மாதிரியை பயன்படுத்த *integrate_with_promptflow.py* இல் பின்வரும் குறியீட்டை சேர்க்கவும்.

    ```python
    import logging
    import requests
    from promptflow import tool
    from promptflow.connections import CustomConnection

    # பதிவு அமைப்பு
    logging.basicConfig(
        format="%(asctime)s - %(levelname)s - %(name)s - %(message)s",
        datefmt="%Y-%m-%d %H:%M:%S",
        level=logging.DEBUG
    )
    logger = logging.getLogger(__name__)

    def query_phi3_model(input_data: str, connection: CustomConnection) -> str:
        """
        Send a request to the Phi-3 / Phi-3.5 model endpoint with the given input data using Custom Connection.
        """

        # "connection" என்பது Custom Connection இன் பெயர், "endpoint", "key" கள் Custom Connection இல் உள்ள விசைகள்
        endpoint_url = connection.endpoint
        api_key = connection.key

        headers = {
            "Content-Type": "application/json",
            "Authorization": f"Bearer {api_key}"
        }
    data = {
        "input_data": [input_data],
        "params": {
            "temperature": 0.7,
            "max_new_tokens": 128,
            "do_sample": True,
            "return_full_text": True
            }
        }
        try:
            response = requests.post(endpoint_url, json=data, headers=headers)
            response.raise_for_status()
            
            # முழுமையான JSON பதிலை பதிவு செய்க
            logger.debug(f"Full JSON response: {response.json()}")

            result = response.json()["output"]
            logger.info("Successfully received response from Azure ML Endpoint.")
            return result
        except requests.exceptions.RequestException as e:
            logger.error(f"Error querying Azure ML Endpoint: {e}")
            raise

    @tool
    def my_python_tool(input_data: str, connection: CustomConnection) -> str:
        """
        Tool function to process input data and query the Phi-3 / Phi-3.5 model.
        """
        return query_phi3_model(input_data, connection)

    ```

    - பணியின் வகையாக **context இல்லாத கேள்வி மற்றும் பதில்** ஐ தேர்ந்தெடுக்கவும். ஏனெனில், இந்த பயிற்சியில் பயன்படுத்தப்பட்ட **ULTRACHAT_200k** தரவுத்தொகுப்பு context அடங்கவில்லை.
    - நீங்கள் மதிப்பிட விரும்பும் prompt flow ஐ தேர்ந்தெடுக்கவும்.

    ![Prompt flow evaluation.](../../../../../../translated_images/ta/evaluation-setting1.4aa08259ff7a536e.webp)

1. **அடுத்தது** ஐ தேர்ந்தெடுக்கவும்.

1. பின்வரும் பணிகளைச் செயற்படுத்தவும்:

    - **உங்கள் தரவுத்தொகுப்பைச் சேர்க்கவும்** ஐ தேர்ந்தெடுக்கவும். உதாரணமாக, **ULTRACHAT_200k** தரவுத்தொகுப்பை பதிவிறக்கும் போது உள்ள *test_data.json1* போன்ற சோதனை தரவுத் கோப்பை பதிவேற்றலாம்.
    - உங்கள் தரவுத்தொகுப்புக்கு பொருந்தும் **தரவுத்தொகுப்பு நெடுவரிசையை** தேர்ந்தெடுக்கவும். உதாரணமாக, **ULTRACHAT_200k** தரவுத்தொகுப்பு பயன்படுத்தினால் **${data.prompt}** ஐ தேர்வுசெய்க.

    ![Prompt flow evaluation.](../../../../../../translated_images/ta/evaluation-setting2.07036831ba58d64e.webp)

1. **அடுத்தது** ஐ தேர்ந்தெடுக்கவும்.

1. செயல்திறன் மற்றும் தரமாநில அளவுகோல்களை அமைக்க பின்வரும் பணிகளைச் செயற்படுத்தவும்:

    - நீங்கள் பயன்படுத்த விரும்பும் செயல்திறன் மற்றும் தர அளவுகோல்களை தேர்ந்தெடுக்கவும்.
    - மதிப்பீட்டிற்காக நீங்கள் உருவாக்கிய Azure OpenAI மாதிரியை தேர்ந்தெடுக்கவும். உதாரணமாக **gpt-4o**.

    ![Prompt flow evaluation.](../../../../../../translated_images/ta/evaluation-setting3-1.d1ae69e3bf80914e.webp)

1. அபாயம் மற்றும் பாதுகாப்பு அளவுகோல்களை அமைக்க பின்வரும் பணிகளைச் செயற்படுத்தவும்:

    - நீங்கள் பயன்படுத்த விரும்பும் அபாயம் மற்றும் பாதுகாப்பு அளவுகோல்களை தேர்ந்தெடுக்கவும்.
    - குறைபாடு விகிதத்தை கணக்கிடும் தீவிரத்தைக் தேர்ந்தெடுக்கவும். உதாரணமாக, **Medium**.
    - **question** க்கு **Data source** ஐ **{$data.prompt}** ஆக தேர்ந்தெடுக்கவும்.
    - **answer** க்கு **Data source** ஐ **{$run.outputs.answer}** ஆக தேர்ந்தெடுக்கவும்.
    - **ground_truth** க்கு **Data source** ஐ **{$data.message}** ஆக தேர்ந்தெடுக்கவும்.

    ![Prompt flow evaluation.](../../../../../../translated_images/ta/evaluation-setting3-2.d53bd075c60a45a2.webp)

1. **அடுத்தது** ஐ தேர்ந்தெடுக்கவும்.

1. மதிப்பீடினை தொடங்க **சமர்ப்பி** ஐ தேர்ந்தெடுக்கவும்.

1. மதிப்பீடு சில நேரம் எடுக்கும். நீங்கள் **மதிப்பீடு** தாவலில் முன்னேற்றத்தை கண்காணிக்கலாம்.

### மதிப்பீட்டு முடிவுகளை மதிப்பாய்வு செய்யவும்

> [!NOTE]
> கீழ்காணும் முடிவுகள் மதிப்பீட்டு செயல்முறையை விளக்குவதற்காக உள்ளன. இந்த பயிற்சியில், நாங்கள் ஒப்பிடுவதற்கு சிறிய அளவு தரவுத்தொகுப்பில் சிறப்பித்த மாதிரியை பயன்படுத்தியுள்ளோம், இதனால் சிறந்த முடிவுகள் பெறாமை உள்ளடக்கமாக இருக்கலாம். பயன்படுத்தும் தரவுத்தொகுப்பின் அளவு, தரம் மற்றும் பரவல், மற்றும் மாதிரியின் தனிப்பட்ட அமைப்பின் அடிப்படையில் உண்மையான முடிவுகள் பெரிதும் மாறுபடலாம்.

மதிப்பீடு முடிந்ததும், செயல்திறன் மற்றும் பாதுகாப்பு அளவுகோல்களுக்கான முடிவுகளை நீங்கள் மதிப்பாய்வு செய்யலாம்.
1. செயல்திறன் மற்றும் தர அளவுகோல்கள்:

    - ஒருங்கிணைந்த, சிரமமற்ற மற்றும் பொருத்தமான பதில்களை உருவாக்குவதில் மாடலின் செயல்திறனை மதிப்பிடுங்கள்.

    ![மதிப்பீட்டு முடிவு.](../../../../../../translated_images/ta/evaluation-result-gpu.85f48b42dfb74254.webp)

1. அபாயம் மற்றும் பாதுகாப்பு அளவுகோல்கள்:

    - மாடலின் வெளியீடுகள் பாதுகாப்பானவை மற்றும் பொறுப்பான AI 원칙ங்களை பின்பற்றுகின்றன என்பதை உறுதி செய்யுங்கள், எந்தவொரு தீங்கு விளைவிக்கும் அல்லது தவறான உள்ளடக்கத்தையும் தவிர்க்கவும்.

    ![மதிப்பீட்டு முடிவு.](../../../../../../translated_images/ta/evaluation-result-gpu-2.1b74e336118f4fd0.webp)

1. **விவரமான அளவுகோல் முடிவுகளை** பார்வையிட கீழே ஸ்க்ரோல் செய்யலாம்.

    ![மதிப்பீட்டு முடிவு.](../../../../../../translated_images/ta/detailed-metrics-result.afa2f5c39a4f5f17.webp)

1. உங்கள் தனிப்பயன் Phi-3 / Phi-3.5 மாடலை செயல்திறன் மற்றும் பாதுகாப்பு அளவுகோல்களுக்கு எதிராக மதிப்பிடுவதன் மூலம், மாடல் திறமையானதானது மட்டுமல்லாமல் பொறுப்பான AI நடைமுறைகளையும் பின்பற்றுகிறது என்பதை உறுதிப்படுத்தலாம், இது அதை நகல் பயன்படுத்துவதற்குக் தயார் ஆக்குகிறது.
