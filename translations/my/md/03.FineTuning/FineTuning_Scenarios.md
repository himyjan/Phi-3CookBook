## အတိကျညှိနှိုင်းခြင်း ကိစ္စများ

![FineTuning with MS Services](../../../../translated_images/my/FinetuningwithMS.3d0cec8ae693e094.webp)

ဤအပိုင်းတွင် Microsoft Foundry နှင့် Azure ပတ်ဝန်းကျင်များရှိ အတိကျညှိနှိုင်းခြင်းကိစ္စများ၊ ထားရှိမှု ပုံစံများ၊ အခြေခံအဆောက်အအုံအလွှာများနှင့် ပုံမှန်အသုံးပြုသော ပေါင်းစည်းတိုးတက်မှုနည်းပညာများကို အကျဉ်းချုပ် ပြသထားပါသည်။

**ပလက်ဖောင်း**  
Microsoft Foundry (ယခင် Microsoft Foundry) နှင့် Azure Machine Learning ကဲ့သို့ စီမံခန့်ခွဲမှုဝန်ဆောင်မှုများ ဖြစ်ပြီး၊ မော်ဒယ်စီမံခန့်ခွဲမှု၊ မတုံးစီမံခန့်ခွဲမှု၊ စမ်းသပ်မှုအခြေအနေ စစ်ဆေးခြင်း၊ ပြန်လည်ဖြန့်ချိမှု လုပ်ငန်းစဉ်များကို ပံ့ပိုးပေးပါသည်။

**အဆောက်အအုံ**  
အတိကျညှိနှိုင်းခြင်းအတွက် ကျယ်ပြန့်သည့် ကွန်ပြူတာအရင်းအမြစ်များ လိုအပ်ပါသည်။ Azure ပတ်ဝန်းကျင်များတွင် အများအားဖြင့် GPU အခြေပြု အရာ့ဂျင် ဗားချွင်းများနှင့် CPU အရင်းအမြစ်များကို ပေါင်းစည်းထားသည်၊ အလေးချိန်သက်သာသောအလုပ်များအတွက်ရှိပြီး၊ ဒေတာစုစည်းမှုများနှင့် checkpoint များအတွက် ကျယ်ပြန့်စွာသိုလှောင်နိုင်သော အရင်းအမြစ်များပါဝင်သည်။

**ကိရိယာများနှင့် ဖရိမ်းဝတ်**  
အတိကျညှိနှိုင်းခြင်း လုပ်ငန်းစဉ်များတွင် ပုံမှန်အားဖြင့် Hugging Face Transformers၊ DeepSpeed နှင့် PEFT (Parameter-Efficient Fine-Tuning) ကဲ့သို့ ဖရိမ်းဝတ်များနှင့် ပေါင်းစည်းတိုးတက်မှု စာကြည့်တိုက်များကို အသုံးပြုပါသည်။

Microsoft နည်းပညာများဖြင့် အတိကျညှိနှိုင်းခြင်း လုပ်ငန်းစဉ်သည် ပလက်ဖောင်း ဝန်ဆောင်မှုများ၊ ကွန်ပြူတာ အဆောက်အအုံနှင့် လေ့ကျင့်ရေး ဖရိမ်းဝတ်များ အကွာအဝေး ပြောင်းလဲမှု ဖြစ်သည်။ ဤအစိတ်အပိုင်းများဟာ ဘယ်လို ပူးပေါင်းဆောင်ရွက်ကြောင်း နားလည်ခြင်းဖြင့် ဖန်တီးသူများသည် အခြေခံမော်ဒယ်များကို အထူး အလုပ်များနှင့် ထုတ်လုပ်မှု ကိစ္စများအတွက် ထိရောက်စွာ ချိန်ညှိနိုင်မည် ဖြစ်သည်။

## မော်ဒယ်အား ဝန်ဆောင်မှုအဖြစ်

မော်ဒယ်ကို ကွန်ပြူတာ စီမံခန့်ခွဲရန် မလိုအပ်ဘဲ မှတ်ပုံတင်ထားသော အတိကျညှိနှိုင်းခြင်းဖြင့် အတိကျညှိနှိုင်းနိုင်သည်။

![MaaS Fine Tuning](../../../../translated_images/my/MaaSfinetune.3eee4630607aff0d.webp)

Phi-3, Phi-3.5, နှင့် Phi-4 မော်ဒယ် များအတွက် serverless အတိကျညှိနှိုင်းခြင်း ယခုရရှိနိုင်ပြီး၊ ပလက်ဖောင်းနှင့် အနားအရပ် စာရင်းများအတွက် အလွယ်တကူ မော်ဒယ်များကို ဆက်ပြင်နိုင်ရန် ဖန်တီးသူများအား အချိန်တိုအတွင်း ဖန်တီးနိုင်ပြီး ကွန်ပြူတာစီစဉ်ရန် လိုအပ်မှုမရှိပါ။

## မော်ဒယ်အား ပလက်ဖောင်းအဖြစ်

အသုံးပြုသူများသည် မိမိတို့၏ ကွန်ပြူတာကို စီမံခန့်ခွဲကာ မော်ဒယ်များကို အတိကျညှိနှိုင်းကြသည်။
