# **Microsoft Foundry ကို သုံးပြီး သုံးသပ်ခြင်း**

![aistudo](../../../../../translated_images/my/AIFoundry.9e0b513e999a1c5a.webp)

သင့် generative AI အက်ပလီကေးရှင်းကို [Microsoft Foundry](https://ai.azure.com?WT.mc_id=aiml-138114-kinfeylo) ဖြင့် ဘယ်လိုသုံးသပ်နိုင်သည်ကို ပြသသည်။ တစ်ဖောင်းပြောဆိုမှု(single-turn) ဖြစ်စေ၊ မျိုးစုံပြောဆိုမှု(multi-turn) ဖြစ်စေ Microsoft Foundry သည် မော်ဒယ်လုပ်ဆောင်ချက်နှင့် လုံခြုံရေးကို သုံးသပ်ရန် ကိရိယာများပေးဆောင်သည်။

![aistudo](../../../../../translated_images/my/AIPortfolio.69da59a8e1eaa70f.webp)

## Microsoft Foundry ဖြင့် generative AI အက်ပလီကေးရှင်းများကို ဘယ်လိုသုံးသပ်မလဲ
အသေးစိတ်ညွှန်ကြားချက်များအတွက် [Microsoft Foundry အထောက်အပံ့စာရွက်စာတမ်း](https://learn.microsoft.com/azure/ai-studio/how-to/evaluate-generative-ai-app?WT.mc_id=aiml-138114-kinfeylo) ကို ကြည့်ပါ။

စတင်ရန် လုပ်ဆောင်ရမည့် အဆင့်များမှာ -

## Microsoft Foundry တွင် Generative AI မော်ဒယ်များကို သုံးသပ်ခြင်း

**လိုအပ်ချက်များ**

- CSV သို့မဟုတ် JSON ပုံစံဖြင့် စမ်းသပ်ဒေတာစုစည်းမှု။
- ထုတ်ပြန်ထားသော generative AI မော်ဒယ်တစ်ခု (ဥပမာ Phi-3, GPT 3.5, GPT 4, သို့မဟုတ် Davinci မော်ဒယ်များ)။
- သုံးသပ်မှုကို လုပ်ဆောင်ရန် ရှိသည့် နောက်ခံ ကွန်ပျူတာ ပတ်၀န်းကျင်။

## အတွင်းတွင် ပါသော သုံးသပ်မှု မီထရစ်များ

Microsoft Foundry သည် တစ်ဖောင်းပြောဆိုမှုနှင့် ရှုပ်ထွေးသော မျိုးစုံပြောဆိုမှုများကို သုံးသပ်နိုင်သည်။
Retrieval Augmented Generation (RAG) ဇာတ်ကြောင်းအတွက် မော်ဒယ်သည် အချက်အလက်တိ တစ်သတ်မှတ်ထားသော အခြေခံဖြစ်သော့ယ်တွင်၊ မွမ်းမံထားသည့် သုံးသပ်မှု မီထရစ်များကို အသုံးပြု၍ လုပ်ဆောင်ချက်ကို သုံးသပ်နိုင်သည်။
ထို့အပြင် အထွေထွေ တစ်ဖောင်းမေးခွန်းဖြေခြင်း (non-RAG) ကိုလည်း သုံးသပ်နိုင်သည်။

## သုံးသပ်မှု မျက်နှာပြင် တည်ဆောက်ခြင်း

Microsoft Foundry UI မှာ Evaluate စာမျက်နှာ သို့မဟုတ် Prompt Flow စာမျက်နှာကို ဝင်ကြည့်ပါ။
သုံးသပ်မှုရေးသားမှု ဝိဇ္ဇာကို လိုက်နာ၍ သုံးသပ်မှု လည်ပတ်မှု တည်ဆောက်ပါ။ သင့်သုံးသပ်မှုအတွက် အမည်တစ်ခု ထည့်သွင်းနိုင်သည်။
သင့်အက်ပလီကေးရှင်း ရည်ရွယ်ချက်နှင့် ကိုက်ညီသော ဇာတ်ကြောင်းကို ရွေးချယ်ပါ။
မော်ဒယ်ရလဒ်ကို သုံးသပ်ရန် မီထရစ်တစ်ခု သို့မဟုတ် မီထရစ်များစွာ ရွေးချယ်ပါ။

## အထူးပြုသုံးသပ်မှု လည်ပတ်မှု (optional)

ပိုမို ကွဲပြားသော လုပ်ငန်းစဉ်လိုအပ်ချက်များအတွက် အထူးပြုသုံးသပ်မှု လည်ပတ်မှုတစ်ခု တည်ဆောက်နိုင်သည်။ သင့်လိုအပ်ချက်များအရ သုံးသပ်မှု လုပ်ငန်းစဉ်ကို ပြင်ဆင်နိုင်သည်။

## ရလဒ် ကြည့်ရှုခြင်း

သုံးသပ်မှု လည်ပတ်ပြီးပါက Microsoft Foundry တွင် အချက်အလက်မှတ်တမ်း၊ ကြည့်ရှုခြင်းနှင့် အသေးစိတ် သုံးသပ်မှု မီထရစ်များကို စူးစမ်းနိုင်သည်။ သင့်အက်ပလီကေးရှင်း၏ တိုးတက်မှုနှင့် ကန့်သတ်ချက်များကို သိရှိရမည်။

**Note** Microsoft Foundry သည် လောလောဆယ် ပြည်သူ့ကြိုတင်သုံးသပ်မှု အဆင့်တွင် ရှိနေပြီး စမ်းသပ်မှုနှင့် ဖွံ့ဖြိုးရေးအတွက်သာ အသုံးပြုသင့်ပါသည်။ ထုတ်လုပ်မှု အလုပ်ပေါ်များအတွက် ခြားနားသော ရွေးချယ်စရာများကို စဉ်းစားပါ။ အသေးစိတ်နှင့် လုပ်ဆောင်ချက်အစီအစဉ်များအတွက် တရားဝင် [AI Foundry စာရွက်စာတမ်း](https://learn.microsoft.com/azure/ai-studio/?WT.mc_id=aiml-138114-kinfeylo) ကို စူးစမ်းကြည့်ပါ။
