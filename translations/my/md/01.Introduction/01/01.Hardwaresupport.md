# Phi Hardware Support

Microsoft Phi သည် ONNX Runtime အတွက်အကောင်းဆုံး စနစ်ချထားပြီး Windows DirectML ကို ထောက်ပံ့ပေးသည်။ ယင်းသည် GPU, CPU နှင့် ဖုန်းစက်များအပါအဝင် အမျိုးမျိုးသော hardware များပေါ်တွင် ကောင်းစွာ အလုပ်လုပ်သည်။

## Device Hardware  
အထူးသဖြင့် ထောက်ပံ့ထားသော hardware များမှာ -

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPUs, 128 GiB memory)

## Mobile SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 သို့မဟုတ် ဧရာ A16/A17 Processor ပါသော ဌာန

## Phi Hardware Specification

- အနည်းဆုံး လိုအပ်ချက်များ။
- Windows: DirectX 12 ကိုထောက်ပံ့သော GPU နှင့် အနည်းဆုံး 4GB RAM ပေါင်းစပ်မှုရှိရမည်။

CUDA: Compute Capability >= 7.02 ရှိသော NVIDIA GPU

## Running onnxruntime on multiple GPUs

လက်ရှိ Phi ONNX မော်ဒယ်များမှာ GPU တစ်ခုအတွက်သာ အသုံးပြုနိုင်သည်။ Phi မော်ဒယ်များအတွက် multi-GPU ထောက်ပံ့မှု ရနိုင်သည်၊ သို့သော် ORT ၂ခု GPU ဖြင့် အသုံးပြုလျှင် ORT ၂ခု instance နှင့် နှိုင်းပါက throughput ပိုမိုများမဖြစ်မည်ကို မအာမခံနိုင်ပါ။ နောက်ဆုံး အပ်ဒိတ်များအတွက် [ONNX Runtime](https://onnxruntime.ai/) ကို ကြည့်ပါ။

[Build 2024 the GenAI ONNX Team](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) တွင် Phi မော်ဒယ်များအတွက် multi-GPU မဟုတ် multi-instance ကို ဖွင့်ပေးခဲ့ကြောင်း ကြေညာခဲ့သည်။

လောလောဆယ်တွင် CUDA_VISIBLE_DEVICES environment variable ကို အသုံးပြု၍ onnxruntime သို့မဟုတ် onnxruntime-genai instance တစ်ခုကို ဒီအတိုင်း လည်ပတ်နိုင်သည်။

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

Phi ကို ပိုမိုလေ့လာရန် [Microsoft Foundry](https://ai.azure.com) တွင် လမ်းညွှန်တိုက်ရိုက် လေ့လာနိုင်ပါသည်။
