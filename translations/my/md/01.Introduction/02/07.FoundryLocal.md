## Foundry Local တွင် Phi-Family မော်ဒယ်များဖြင့် စတင်အသုံးပြုခြင်း

### Foundry Local အကြောင်း အနှစ်ချုပ်

Foundry Local သည် သင့်ဒေသတွင်း ဟားဒ်ဝဲပေါ်တွင် အနက်စုံ AI စွမ်းရည်များကို တိုက်ရိုက်ပေးဆောင်သည့် အင်ဖားရန့်စ်ဖြေရှင်းချက်ဖြစ်သည်။ ဤလေ့ကျင့်ခန်းတစ်ခုသည် Foundry Local နှင့် Phi-Family မော်ဒယ်များကို စတင်တပ်ဆင်အသုံးပြုနည်းကို လမ်းပြပေးမှာဖြစ်ပြီး၊ သင့် AI လုပ်ငန်းများကို လုံခြုံမှုအရ ပြဿနာကင်းစွာ ထိန်းချုပ်နိုင်ရန်နှင့် ကုန်ကျစရိတ် လျော့ချနိုင်ရန် အကောင်းဆုံးနည်းလမ်းများကို အကြံပြုပါသည်။

Foundry Local သည် သင့်ပစ္စည်းပေါ်တွင် AI မော်ဒယ်များကို တိုက်ရိုက်စစ်ဆေးအသုံးပြု၍ တိုးတက်စွာ လုပ်ဆောင်မှု၊ လုံခြုံမှု၊ စိတ်ကြိုက် ပြင်ဆင်နိုင်မှု နှင့် ကုန်ကျစရိတ် သက်သာမှုများကို ပေးစွမ်းသည်။ CLI, SDK နှင့် REST API များမှတဆင့် သင့် ရှိပြီးသား လူမှုစီမံချက်များနှင့် အပလီကေးရှင်းများတွင် ညှိနှိုင်းအသုံးချနိုင်သည်။


![arch](../../../../../translated_images/my/foundry-local-arch.8823e321dd8258d7.webp)

### Foundry Local ကို ရွေးချယ်သင့်ကြောင်း

Foundry Local ၏ အကျိုးကျေးဇူးများကို နားလည်ခြင်းဖြင့် သင့် AI တပ်ဆင်မှုအတွက် မှန်ကန်သော ဆုံးဖြတ်ချက်ကို ချနိုင်မည်ဖြစ်သည်-

- **ပစ္စည်းတွင် အင်ဖားရန့်စ်လုပ်ခြင်း** - မော်ဒယ်များကို သင့်ပစ္စည်းပေါ်တွင် တိုက်ရိုက် စမ်းသပ်မှုကို ကျင့်သုံးပြီး ကုန်ကျစရိတ်လျော့ပါးစေသည်။

- **မော်ဒယ် စိတ်ကြိုက် အဆင်ပြေမှု** - အလိုအလျောက် ရွေးချယ်နိုင်သော မော်ဒယ်များ သို့မဟုတ် သင့်လိုအပ်ချက်နှင့် ကိုက်ညီသော မော်ဒယ်ကို အသုံးပြုနိုင်သည်။

- **ကုန်ကျစရိတ် ထိရောက်မှု** - ရှိပြီးသား ဟားဒ်ဝဲကို အသုံးပြုပြီး ပျော်မွေ့စွာ တင်သွင်းကုန်ကျစရိတ်ကို ဖယ်ရှားပေးသည်။

- **ညှိနှိုင်းပေါင်းစည်းမှု အဆင်ပြေမှု** - SDK, API နှင့် CLI များနှင့် ချိတ်ဆက်၍ Microsoft Foundry အသုံးပြုမှု တိုးတက်လာရာ အရွယ်အစား ပြောင်းလဲနိုင်သည်။

> **စတင်အသုံးပြုခြင်း သတိပေးချက်:** ဤ လမ်းညွှန်မှုအပိုင်းတွင် CLI နှင့် SDK ကို အသုံးပြုနည်းကို အဓိကထား ဖော်ပြထားပြီး သင့် အသုံးပြုပုံအတွက် အကောင်းဆုံး နည်းလမ်းကို ရွေးချယ်နိုင်ပါသည်။

## အပိုင်း ၁: Foundry Local CLI ကို တပ်ဆင်ခြင်း

### ခြေလှမ်း ၁: တပ်ဆင်ခြင်း

Foundry Local CLI သည် သင့်ဒေသတွင်း AI မော်ဒယ်များကို စီမံခန့်ခွဲရန် နှင့် ဖျော်ဖြေရန်သော ဝင်ခန်းငယ်ဖြစ်သည်။ စက်မှာ တပ်ဆင်ခြင်း အစမှ စလိုက်ပါ။

**ထောက်ခံပလက်ဖောင်းများ:** Windows နှင့် macOS

အသေးစိတ် တပ်ဆင်ရန် လမ်းညွှန်ချက်များအတွက် [Foundry Local တရားဝင်စာတမ်း](https://github.com/microsoft/Foundry-Local/blob/main/README.md) ကို ကြည့်ပါ။

### ခြေလှမ်း ၂: ရရှိနိုင်သော မော်ဒယ်များ ကို စူးစမ်းရန်

Foundry Local CLI ကို တပ်ဆင်ပြီးနောက် သင့်လိုအပ်ချက်များအတွက် ရရှိနိုင်သော မော်ဒယ်များအား အောက်ပါ အမိန့်ဖြင့် ကြည့်ရှုနိုင်သည်-


```bash
foundry model list
```

### ခြေလှမ်း ၃: Phi Family မော်ဒယ်များကို နားလည်ခြင်း

Phi Family သည် အသုံးပြုမှုနှင့် ဟားဒ်ဝဲ ဖွဲ့စည်းမှုများအတွက် ပြင်ဆင်ထားသော မော်ဒယ်စုံမျိုးစုံကို ပေးစွမ်းသည်။ Foundry Local တွင် ရနိုင်သော Phi မော်ဒယ်များမှာ-

**ရရှိနိုင်သော Phi မော်ဒယ်များ:** 

- **phi-3.5-mini** - အခြေခံ အလုပ်များအတွက် စွမ်းအင်သေးသွားသော မော်ဒယ်
- **phi-3-mini-128k** - ထိပ်တန်း အပြောအဆိုများအတွက် ကြီးမားသော စကားဝိုင်း
- **phi-3-mini-4k** - ယေဘုယျ အသုံးပြုမှုအတွက် စံမော်ဒယ်
- **phi-4** - အခြားကျွမ်းကျင်မှုများ အထူးပြု မော်ဒယ်
- **phi-4-mini** - Phi-4 ၏ အလင်းလေး ဗားရှင်း
- **phi-4-mini-reasoning** - ခက်ခဲသော စဉ်းစားမှုများအတွက် အထူးပြု

> **ဟားဒ်ဝဲ ကိုက်ညီမှု:** မော်ဒယ်တိုင်းကို သင့်စနစ်အရ CPU, GPU ချိန်ညှိ၍ အသုံးပြုနိုင်သည်။

### ခြေလှမ်း ၄: ပထမဆုံး Phi မော်ဒယ်ကို ပြေးဆွဲခြင်း

သုံးသပ်ကြည့်ရအောင်။ `phi-4-mini-reasoning` မော်ဒယ်ကို အသုံးပြုပြီး ခက်ခဲသောပြဿနာများကို အဆင့်ဆင့် ဖြေရှင်းပုံကို တင်ပြပါမည်။


**မော်ဒယ်ပြေးဆွဲရန် အမိန့်:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **ပထမဆုံးတပ်ဆင်ခြင်း:** မော်ဒယ်ကို ပထမဆုံးအသုံးပြုသောအခါ Foundry Local သည် မော်ဒယ်ကို သင့်ဒေသတွင်းစက်သို့ တောင်းယူဒေါင်းလုပ်လုပ်ပါမည်။ ဒေါင်းလုပ် အချိန်သည် သင့်ကွန်ရက်အရှိန်ပမာဏပေါ် မူတည်သည်၊ သို့ကြောင့် စောင့်ဆိုင်းရန်လိုအပ်ပါသည်။

### ခြေလှမ်း ၅: မော်ဒယ်အား လက်တွေ့ ပြဿနာဖြေရှင်းမှုဖြင့် စမ်းသပ်ခြင်း

ယခု မော်ဒယ်အား အလွန်အမင်း ရိုးရာ မြင်သာသော မှန်ကန်ချက်ဖြင့် စမ်းသပ်ကြည့်မည်-

**ဥပမာပြဿနာ:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**မျှော်လင့်ထားသော ကျင့်ကြံမှု:** pheasant များတွင် ခြေတစ်စုံ ၂ ခြေလှမ်း ရှိပြီး နွားမြင်းများတွင် ၄ ခြေလှမ်းရှိကြောင်းကို အသုံးပြု၍ မျဉ်းဆွဲဆိုင်ရာ ပြဿနာကို တွဲဖက် ဖြေကြဉ်းရန် ဖြစ်သည်။

## အပိုင်း ၂: Foundry Local SDK ဖြင့် အပလီကေးရှင်းများ တည်ဆောက်ခြင်း

### SDK ကို အသုံးပြုရခြင်း အကြောင်း

CLI သည် စမ်းသပ်မှုများနှင့် အလျင်အမြန် အပြန်အလှန် အတွက် ထူးခြားသော်လည်း SDK သည် Foundry Local ကို သင့်အပလီကေးရှင်းများထဲတွင် ပရိုဂရမ်းမှုနည်းလမ်းဖြင့် ပေါင်းစည်းရန် အခွင့်အလမ်းများပေးသည်။ ဤသည်သည် -

- စိတ်ကြိုက် AI အပလီကေးရှင်းများ တည်ဆောက်ခြင်း
- အလိုအလျောက် လုပ်ငန်းစဉ်များ ဖန်တီးခြင်း
- ရှိပြီးသား စနစ်များဖြင့် AI ဆက်စပ်ထားခြင်း
- စကားပြောမြှုပ်နှံစက်နှင့် အပြန်အလှန်ကိရိယာများ ဖန်တီးခြင်း

### ထောက်ခံထားသော ပရိုဂရမ်မင်ဘာသာစကားများ

Foundry Local သည် သင့် ဖွံ့ဖြိုးရေး စိတ်ကြိုက်များနှင့် ကိုက်ညီစေရန် SDK အများကြီး ပံ့ပိုးပေးသည်-

**📦 ရရှိနိုင်သော SDK များ:**

- **C# (.NET):** [SDK စာတမ်းနှင့် နမူနာများ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK စာတမ်းနှင့် နမူနာများ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK စာတမ်းနှင့် နမူနာများ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK စာတမ်းနှင့် နမူနာများ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### နောက် တစ်ဆင့်များ

1. သင့်ဖွံ့ဖြိုးရေး ပတ်ဝန်းကျင်အရ စိတ်ကြိုက် SDK ကို ရွေးချယ်ပါ
2. SDK အထူးစာရွက်များနှင့် လမ်းညွှန်ချက်များကို လိုက်နာပါ
3. ရိုးရှင်းသော နမူနာများမှ စတင်ပြီး ပြန်လည်တည်ဆောက်ပါ
4. SDK ကိုဒ်များ repo တစ်ခုချင်းစီတွင် ပံ့ပိုးထားသော နမူနာကို စူးစမ်းပါ
