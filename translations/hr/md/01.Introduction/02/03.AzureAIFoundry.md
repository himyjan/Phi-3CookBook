## Phi obitelj u Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) je pouzdana platforma koja omogućava programerima da potiču inovacije i oblikuju budućnost s AI na siguran, zaštićen i odgovoran način.

[Microsoft Foundry](https://ai.azure.com) je dizajniran za programere da:

- Izgrade generativne AI aplikacije na platformi razine poduzeća.
- Istražuju, razvijaju, testiraju i raspoređuju koristeći najmodernije AI alate i ML modele, utemeljene na praksama odgovornog AI-ja.
- Suradjuju s timom kroz cijeli životni ciklus razvoja aplikacije.

Uz Microsoft Foundry, možete istražiti širok spektar modela, usluga i mogućnosti te početi graditi AI aplikacije koje najbolje služe vašim ciljevima. Microsoft Foundry platforma olakšava skalabilnost za transformaciju prototipova u potpuno funkcionalne produkcijske aplikacije. Kontinuirano praćenje i poboljšavanje podupiru dugoročni uspjeh.

### **Pokretanje Python koda za pozivanje Microsoft Foundry Phi-4**

```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Uključi rezultat govora ako je govor omogućen
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
