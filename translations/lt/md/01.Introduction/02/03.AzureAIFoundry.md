## Phi šeima Microsoft Foundry platformoje

[Microsoft Foundry](https://ai.azure.com) yra patikima platforma, kuri leidžia kūrėjams skatinti inovacijas ir formuoti ateitį su DI saugiu, patikimu ir atsakingu būdu.



[Microsoft Foundry](https://ai.azure.com) skirta kūrėjams, kad jie galėtų:

- Kurti generatyvinius DI sprendimus įmonių lygio platformoje.
- Tyrinėti, kurti, testuoti ir diegti naudojant pažangius DI įrankius ir ML modelius, pagrįstus atsakingos DI praktika.
- Bendradarbiauti komandoje viso programos kūrimo ciklo metu.

Naudodami Microsoft Foundry galite tyrinėti daugybę modelių, paslaugų ir galimybių bei kurti DI programas, geriausiai atitinkančias jūsų tikslus. Microsoft Foundry platforma palengvina mastelio didinimą, leidžiant lengvai paversti koncepcijas pilna apimtimi veikiančiomis gamybinėmis programomis. Nuolatinė priežiūra ir tobulinimas padeda užtikrinti ilgalaikę sėkmę.

### **Paleisti Python kodą, kviečiantį Microsoft Foundry Phi-4**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Įtraukti kalbos rezultatą, jei kalba įjungta
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
