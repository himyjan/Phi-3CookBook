# Phi šeima AITK

[AI Toolkit for VS Code](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) supaprastina generatyvinių dirbtinio intelekto programų kūrimą, sujungdamas pažangiausius DI kūrimo įrankius ir modelius iš Microsoft Foundry katalogo bei kitų katalogų, tokių kaip Hugging Face. Galėsite naršyti DI modelių katalogą, kurį palaiko GitHub modeliai ir Microsoft Foundry modelių katalogai, juos atsisiųsti vietoje arba nuotoliniu būdu, atlikti tikslinimą, testuoti ir naudoti savo programoje.

AI Toolkit peržiūra veiks vietoje. Vietinė inferencija arba tikslinimas priklauso nuo pasirinkto modelio, gali prireikti turėti GPU, tokį kaip NVIDIA CUDA GPU. Taip pat galite tiesiogiai paleisti GitHub modelius naudodami AITK.

### Įdiekite AI Toolkit

AI Toolkit pateikiamas kaip [Visual Studio Code plėtinys](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions), todėl pirmiausia turite įdiegti [VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo) ir atsisiųsti AI Toolkit iš [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio).

[AI Toolkit yra prieinamas Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) ir jį galima įdiegti kaip bet kurią kitą VS Code plėtinį.

Jei nesate susipažinę su VS Code plėtinių diegimu, sekite šiuos veiksmus:

### Prisijungimas

1. VS Code veiklų juostoje pasirinkite **Extensions**
1. Plėtinių paieškos juostoje įrašykite "AI Toolkit"
1. Pasirinkite "AI Toolkit for Visual Studio code"
1. Paspauskite **Install**

Dabar esate pasiruošę naudoti plėtinį!

Jums bus pasiūlyta prisijungti prie GitHub, todėl paspauskite "Allow" tęsti. Būsite nukreipti į GitHub prisijungimo puslapį.

Prašome prisijungti ir vykdyti procesą. Sėkmingai užbaigus, būsite nukreipti atgal į VS Code.

Įdiegus plėtinį, veiklų juostoje pasirodys AI Toolkit piktograma.

Tyrinėkime prieinamas funkcijas!

### Prieinamos funkcijos

AI Toolkit pagrindinis šoninis meniu suskirstytas į  

- **Models**
- **Resources**
- **Playground**  
- **Fine-tuning**
- **Evaluation**

Jos prieinamos skiltyje Resources. Norėdami pradėti, pasirinkite **Model Catalog**.

### Atsisiųskite modelį iš katalogo

Paleidus AI Toolkit iš VS Code šoninės juostos, galite pasirinkti iš šių variantų:

![AI toolkit model catalog](../../../../../translated_images/lt/AItoolkitmodel_catalog.7a7be6a7d8468d31.webp)

- Surasti palaikomą modelį iš **Model Catalog** ir atsisiųsti vietoje
- Išbandyti modelio inferenciją **Model Playground**
- Tikslinti modelį vietoje arba nuotoliniu būdu **Model Fine-tuning**
- Diegti tikslintus modelius debesyje per AI Toolkit komandų paletę
- Vertinti modelius

> [!NOTE]
>
> **GPU vs CPU**
>
> Pastebėsite, kad modelių kortelėse rodomas modelio dydis, platforma ir akceleratoriaus tipas (CPU, GPU). Optimizuotam veikimui **Windows įrenginiuose, turinčiuose bent vieną GPU**, pasirinkite tik Windows skirtas modelio versijas.
>
> Tai užtikrina modelio optimizavimą DirectML akceleratoriui.
>
> Modelių pavadinimai yra formatu
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
> Norėdami sužinoti, ar jūsų Windows įrenginyje yra GPU, atidarykite **Task Manager** ir pasirinkite **Performance** skirtuką. Jei turite GPU(-ių), jie bus išvardyti kaip "GPU 0" arba "GPU 1".

### Naudokite REST API savo programoje

AI Toolkit turi vietinį REST API tinklo serverį **prievade 5272**, kuris naudoja [OpenAI pokalbių užbaigimo formatą](https://platform.openai.com/docs/api-reference/chat/create).

Tai leidžia patikrinti programą vietoje, nereikalaujant prisijungimo prie debesų DI modelių paslaugų. Pavyzdžiui, šis JSON failas rodo, kaip sukonfigūruoti užklausos turinį:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```
  
REST API galite išbandyti naudodami, pavyzdžiui, [Postman](https://www.postman.com/) arba CURL (Client URL) įrankį:

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```
  
### Naudojant OpenAI klientų biblioteką Python kalba

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # reikalinga API, bet nenaudojama
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```
  
### Naudojant Azure OpenAI klientų biblioteką .NET kalbai

Pridėkite [Azure OpenAI klientų biblioteką .NET](https://www.nuget.org/packages/Azure.AI.OpenAI/) prie savo projekto naudodami NuGet:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```
  
Pridėkite C# failą pavadinimu **OverridePolicy.cs** savo projekte ir įklijuokite šį kodą:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```
  
Tada į savo **Program.cs** failą įklijuokite šį kodą:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
