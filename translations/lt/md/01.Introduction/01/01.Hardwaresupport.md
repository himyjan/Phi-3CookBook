# Phi aparatinė įranga

Microsoft Phi yra optimizuotas ONNX Runtime ir palaiko Windows DirectML. Jis gerai veikia įvairių tipų aparatinėje įrangoje, įskaitant GPU, CPU ir net mobiliuosius įrenginius.

## Įrenginio aparatinė įranga  
Specifiškai, palaikoma aparatinė įranga apima:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPU, 128 GiB atmintis)

## Mobilioji SKU

- Android – Samsung Galaxy S21
- Apple iPhone 14 arba naujesnis A16/A17 procesorius

## Phi aparatinės įrangos specifikacija

- Reikalinga minimali konfigūracija.
- Windows: DirectX 12 palaikantis GPU ir bent 4GB bendros RAM

CUDA: NVIDIA GPU su skaičiavimo galimybe >= 7.02

## onnxruntime paleidimas keliose GPU

Šiuo metu prieinami Phi ONNX modeliai yra tik 1 GPU naudojimui. Yra galimybė palaikyti daugiagrafikį Phi modeliui, tačiau ORT su 2 GPU negarantuoja didesnio pralaidumo lyginant su 2 atskiromis ort egzempliorių instancijomis. Naujausius atnaujinimus rasite [ONNX Runtime](https://onnxruntime.ai/).

Konferencijoje [Build 2024 the GenAI ONNX Team](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) paskelbė, kad Phi modeliams jie įgalino daugiainstanciją vietoje daugiagrafikiškumo.

Šiuo metu tai leidžia paleisti vieną onnxruntime arba onnxruntime-genai egzempliorių su CUDA_VISIBLE_DEVICES aplinkos kintamuoju tokiu būdu.

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

Džiaugiamės, jei norite plačiau pažinti Phi [Microsoft Foundry](https://ai.azure.com) svetainėje.
