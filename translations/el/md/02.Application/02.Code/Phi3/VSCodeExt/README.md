# **Δημιουργήστε το δικό σας Visual Studio Code GitHub Copilot Chat με την οικογένεια Microsoft Phi-3**

Έχετε χρησιμοποιήσει τον πράκτορα εργασιακού χώρου στο GitHub Copilot Chat; Θέλετε να δημιουργήσετε τον δικό σας πράκτορα κώδικα της ομάδας σας; Αυτό το εργαστήριο έχει ως στόχο να συνδυάσει το ανοιχτού κώδικα μοντέλο για να δημιουργήσει έναν επιχειρησιακό πράκτορα κώδικα επιπέδου επιχείρησης.

## **Βασικά**

### **Γιατί να επιλέξετε το Microsoft Phi-3**

Το Phi-3 είναι μια σειρά οικογένειας, που περιλαμβάνει τα phi-3-mini, phi-3-small και phi-3-medium βασισμένα σε διαφορετικές παραμέτρους εκπαίδευσης για την παραγωγή κειμένου, την ολοκλήρωση διαλόγου και την παραγωγή κώδικα. Υπάρχει επίσης το phi-3-vision βασισμένο στην όραση (Vision). Είναι κατάλληλο για επιχειρήσεις ή διαφορετικές ομάδες για τη δημιουργία εκτός σύνδεσης λύσεων γεννητικής AI.

Συνιστάται να διαβάσετε αυτόν τον σύνδεσμο [https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/01/01.PhiFamily.md](https://github.com/microsoft/PhiCookBook/blob/main/md/01.Introduction/01/01.PhiFamily.md)

### **Microsoft GitHub Copilot Chat**

Η επέκταση GitHub Copilot Chat σας παρέχει ένα περιβάλλον συνομιλίας που σας επιτρέπει να αλληλεπιδράτε με τον GitHub Copilot και να λαμβάνετε απαντήσεις σε ερωτήσεις σχετικά με τον κώδικα απευθείας μέσα στο VS Code, χωρίς να χρειάζεται να πλοηγηθείτε σε τεκμηρίωση ή να αναζητήσετε σε διαδικτυακά φόρουμ.

Το Copilot Chat μπορεί να χρησιμοποιεί επισημάνσεις σύνταξης, εσοχές και άλλες λειτουργίες μορφοποίησης για να προσθέσει σαφήνεια στις παραγόμενες απαντήσεις. Ανάλογα με τον τύπο της ερώτησης από τον χρήστη, το αποτέλεσμα μπορεί να περιλαμβάνει συνδέσμους προς το πλαίσιο που χρησιμοποιήθηκε για τη δημιουργία της απάντησης, όπως αρχεία πηγαίου κώδικα ή τεκμηρίωση, ή κουμπιά για πρόσβαση στη λειτουργικότητα του VS Code.

- Το Copilot Chat ενσωματώνεται στη ροή εργασίας σας ως προγραμματιστής και σας παρέχει βοήθεια όπου τη χρειάζεστε:

- Ξεκινήστε μια συνομιλία inline απευθείας από τον επεξεργαστή ή το τερματικό για βοήθεια ενώ γράφετε κώδικα

- Χρησιμοποιήστε την προβολή Chat για να έχετε έναν βοηθό AI στο πλάι για βοήθεια ανά πάσα στιγμή

- Εκκινήστε το Quick Chat για να κάνετε μια σύντομη ερώτηση και να επιστρέψετε σε αυτό που κάνετε

Μπορείτε να χρησιμοποιήσετε το GitHub Copilot Chat σε διάφορα σενάρια, όπως:

- Απάντηση σε ερωτήσεις προγραμματισμού σχετικά με τον καλύτερο τρόπο επίλυσης ενός προβλήματος

- Εξήγηση του κώδικα κάποιου άλλου και πρόταση βελτιώσεων

- Πρόταση διορθώσεων κώδικα

- Δημιουργία περιπτώσεων μονάδας ελέγχου (unit test cases)

- Δημιουργία τεκμηρίωσης κώδικα

Συνιστάται να διαβάσετε αυτόν τον σύνδεσμο [https://code.visualstudio.com/docs/copilot/copilot-chat](https://code.visualstudio.com/docs/copilot/copilot-chat?WT.mc_id=aiml-137032-kinfeylo)


###  **Microsoft GitHub Copilot Chat @workspace**

Η αναφορά σε **@workspace** στο Copilot Chat σας επιτρέπει να κάνετε ερωτήσεις για ολόκληρη τη βάση κώδικα σας. Βασιζόμενο στην ερώτηση, το Copilot ανακτά έξυπνα σχετικά αρχεία και σύμβολα, τα οποία στη συνέχεια αναφέρει στην απάντηση ως συνδέσμους και παραδείγματα κώδικα.

Για να απαντήσει στην ερώτησή σας, το **@workspace** αναζητά μέσα στις ίδιες πηγές που θα χρησιμοποιούσε ένας προγραμματιστής όταν πλοηγείται σε μια βάση κώδικα στο VS Code:

- Όλα τα αρχεία στον χώρο εργασίας, εκτός από αρχεία που αγνοούνται από ένα αρχείο .gitignore

- Τη δομή καταλόγου με τις φάκελους και τα αρχεία, ακόμη και μέσα σε υποφακέλους

- Την ευρετηρίαση αναζήτησης κώδικα του GitHub, αν ο χώρος εργασίας είναι αποθετήριο GitHub και είναι ευρετηριασμένο μέσω αναζήτησης κώδικα

- Σύμβολα και ορισμούς στον χώρο εργασίας

- Το τρέχον επιλεγμένο κείμενο ή ορατό κείμενο στον ενεργό επεξεργαστή

Σημείωση: Το .gitignore παρακάμπτεται αν έχετε ανοιχτό αρχείο ή επιλεγμένο κείμενο σε αρχείο που κανονικά αγνοείται.

Συνιστάται να διαβάσετε αυτόν τον σύνδεσμο [[https://code.visualstudio.com/docs/copilot/copilot-chat](https://code.visualstudio.com/docs/copilot/workspace-context?WT.mc_id=aiml-137032-kinfeylo)]


## **Μάθετε περισσότερα για αυτό το Εργαστήριο**

Το GitHub Copilot έχει βελτιώσει σημαντικά την αποδοτικότητα προγραμματισμού των επιχειρήσεων, και κάθε επιχείρηση ελπίζει να προσαρμόσει τις σχετικές λειτουργίες του GitHub Copilot. Πολλές επιχειρήσεις έχουν προσαρμόσει επεκτάσεις παρόμοιες με το GitHub Copilot βάσει των δικών τους επιχειρησιακών σεναρίων και ανοιχτών μοντέλων. Για τις επιχειρήσεις, οι προσαρμοσμένες επεκτάσεις είναι ευκολότερες στη διαχείριση, αλλά αυτό επηρεάζει και την εμπειρία των χρηστών. Μετά από όλα, το GitHub Copilot διαθέτει πιο ισχυρές λειτουργίες στη διαχείριση γενικών σεναρίων και επαγγελματισμού. Αν η εμπειρία μπορεί να διατηρηθεί συνεπής, θα είναι καλύτερο να προσαρμόσετε τη δική σας επέκταση επιχείρησης. Το GitHub Copilot Chat παρέχει σχετικές APIs για τις επιχειρήσεις να επεκταθούν στην εμπειρία συνομιλίας. Η διατήρηση μιας συνεπούς εμπειρίας και η ύπαρξη προσαρμοσμένων λειτουργιών αποτελεί καλύτερη εμπειρία χρήστη.

Αυτό το εργαστήριο χρησιμοποιεί κυρίως το μοντέλο Phi-3 σε συνδυασμό με το τοπικό NPU και τη συνδυασμένη χρήση Azure για να δημιουργήσει έναν προσαρμοσμένο Πράκτορα στο GitHub Copilot Chat ***@PHI3*** για να βοηθήσει τους προγραμματιστές επιχειρήσεων στην ολοκλήρωση παραγωγής κώδικα***(@PHI3 /gen)*** και στη δημιουργία κώδικα βασισμένου σε εικόνες ***(@PHI3 /img)***.
