# **Χρήση του Microsoft Foundry για αξιολόγηση**

![aistudo](../../../../../translated_images/el/AIFoundry.9e0b513e999a1c5a.webp)

Πώς να αξιολογήσετε την εφαρμογή γεννητικής AI σας χρησιμοποιώντας το [Microsoft Foundry](https://ai.azure.com?WT.mc_id=aiml-138114-kinfeylo). Είτε αξιολογείτε συνομιλίες με έναν γύρο είτε με πολλαπλούς γύρους, το Microsoft Foundry παρέχει εργαλεία για την αξιολόγηση της απόδοσης και της ασφάλειας του μοντέλου.

![aistudo](../../../../../translated_images/el/AIPortfolio.69da59a8e1eaa70f.webp)

## Πώς να αξιολογήσετε εφαρμογές γεννητικής AI με το Microsoft Foundry
Για περισσότερες αναλυτικές οδηγίες δείτε την [Τεκμηρίωση του Microsoft Foundry](https://learn.microsoft.com/azure/ai-studio/how-to/evaluate-generative-ai-app?WT.mc_id=aiml-138114-kinfeylo)

Ακολουθούν τα βήματα για να ξεκινήσετε:

## Αξιολόγηση Μοντέλων Γεννητικής AI στο Microsoft Foundry

**Προαπαιτούμενα**

- Ένα σύνολο δεδομένων δοκιμής σε μορφή CSV ή JSON.
- Ένα αναπτυγμένο μοντέλο γεννητικής AI (όπως τα μοντέλα Phi-3, GPT 3.5, GPT 4 ή Davinci).
- Ένα runtime με μια υπολογιστική μονάδα για να εκτελέσει την αξιολόγηση.

## Ενσωματωμένοι Δείκτες Αξιολόγησης

Το Microsoft Foundry σας επιτρέπει να αξιολογήσετε τόσο συνομιλίες ενός γύρου όσο και σύνθετες, πολλαπλών γύρων.
Για σενάρια Retrieval Augmented Generation (RAG), όπου το μοντέλο βασίζεται σε συγκεκριμένα δεδομένα, μπορείτε να αξιολογήσετε την απόδοση χρησιμοποιώντας ενσωματωμένους δείκτες αξιολόγησης.
Επιπλέον, μπορείτε να αξιολογήσετε γενικές ερωτήσεις απάντησης ενός γύρου (μη RAG).

## Δημιουργία Μιας Εκτέλεσης Αξιολόγησης

Από το περιβάλλον χρήστη του Microsoft Foundry, μεταβείτε είτε στη σελίδα Evaluate είτε στη σελίδα Prompt Flow.
Ακολουθήστε τον οδηγό δημιουργίας αξιολόγησης για να ρυθμίσετε μια εκτέλεση αξιολόγησης. Παρέχετε προαιρετικά ένα όνομα για την αξιολόγησή σας.
Επιλέξτε το σενάριο που ευθυγραμμίζεται με τους στόχους της εφαρμογής σας.
Επιλέξτε έναν ή περισσότερους δείκτες αξιολόγησης για να εκτιμήσετε τα αποτελέσματα του μοντέλου.

## Προσαρμοσμένη Ροή Αξιολόγησης (Προαιρετικό)

Για μεγαλύτερη ευελιξία, μπορείτε να δημιουργήσετε μια προσαρμοσμένη ροή αξιολόγησης. Προσαρμόστε τη διαδικασία αξιολόγησης βάσει των συγκεκριμένων απαιτήσεών σας.

## Προβολή Αποτελεσμάτων

Μετά την εκτέλεση της αξιολόγησης, καταγράψτε, προβάλετε και αναλύστε λεπτομερείς δείκτες αξιολόγησης στο Microsoft Foundry. Αποκτήστε πληροφορίες σχετικά με τις δυνατότητες και τους περιορισμούς της εφαρμογής σας.

**Σημείωση** Το Microsoft Foundry βρίσκεται αυτή τη στιγμή σε δημόσια προεπισκόπηση, οπότε χρησιμοποιήστε το για πειραματισμό και ανάπτυξη. Για παραγωγικές εργασίες, εξετάστε άλλες επιλογές. Εξερευνήστε την επίσημη [τεκμηρίωση του AI Foundry](https://learn.microsoft.com/azure/ai-studio/?WT.mc_id=aiml-138114-kinfeylo) για περισσότερες λεπτομέρειες και βήματα.
