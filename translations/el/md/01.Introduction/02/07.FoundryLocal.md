## Ξεκινώντας με τα Μοντέλα Phi-Family στο Foundry Local

### Εισαγωγή στο Foundry Local

Το Foundry Local είναι μια ισχυρή λύση AI inference στη συσκευή που φέρνει εταιρικής κλάσης δυνατότητες AI απευθείας στο τοπικό σας υλικό. Αυτό το σεμινάριο θα σας καθοδηγήσει στη ρύθμιση και χρήση των μοντέλων Phi-Family με το Foundry Local, προσφέροντάς σας πλήρη έλεγχο των εργασιών AI σας, διατηρώντας παράλληλα την ιδιωτικότητα και μειώνοντας το κόστος.

Το Foundry Local παρέχει πλεονεκτήματα σε απόδοση, ιδιωτικότητα, προσαρμογή και κόστος εκτελώντας τα μοντέλα AI τοπικά στη συσκευή σας. Ενσωματώνεται ομαλά στις υπάρχουσες ροές εργασίας και εφαρμογές σας μέσω ενός διαισθητικού CLI, SDK και REST API.


![arch](../../../../../translated_images/el/foundry-local-arch.8823e321dd8258d7.webp)

### Γιατί να Επιλέξετε το Foundry Local;

Η κατανόηση των πλεονεκτημάτων του Foundry Local θα σας βοηθήσει να πάρετε ενημερωμένες αποφάσεις για την στρατηγική ανάπτυξης AI:

- **Inference στη Συσκευή:** Εκτελέστε μοντέλα τοπικά στο δικό σας υλικό, μειώνοντας το κόστος σας ενώ διατηρείτε όλα τα δεδομένα σας στη συσκευή.

- **Προσαρμογή Μοντέλων:** Επιλέξτε από προκαθορισμένα μοντέλα ή χρησιμοποιήστε τα δικά σας για να καλύψετε συγκεκριμένες απαιτήσεις και περιπτώσεις χρήσης.

- **Οικονομία Κόστους:** Εξαλείψτε τα επαναλαμβανόμενα κόστη υπηρεσιών cloud χρησιμοποιώντας το υπάρχον υλικό σας, καθιστώντας το AI πιο προσιτό.

- **Απρόσκοπτη Ενσωμάτωση:** Συνδεθείτε με τις εφαρμογές σας μέσω SDK, API endpoints ή CLI, με εύκολη κλιμάκωση στο Microsoft Foundry όσο μεγαλώνουν οι ανάγκες σας.

> **Σημείωση Έναρξης:** Αυτό το σεμινάριο επικεντρώνεται στη χρήση του Foundry Local μέσω διεπαφών CLI και SDK. Θα μάθετε και τις δύο προσεγγίσεις για να σας βοηθήσουμε να επιλέξετε τον καλύτερο τρόπο για την περίπτωσή σας.

## Μέρος 1: Ρύθμιση Foundry Local CLI

### Βήμα 1: Εγκατάσταση

Το Foundry Local CLI είναι η πύλη σας για τη διαχείριση και εκτέλεση μοντέλων AI τοπικά. Ας ξεκινήσουμε με την εγκατάστασή του στο σύστημά σας.

**Υποστηριζόμενες Πλατφόρμες:** Windows και macOS

Για λεπτομερείς οδηγίες εγκατάστασης, παρακαλώ ανατρέξτε στην [επίσημη τεκμηρίωση Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Βήμα 2: Εξερεύνηση Διαθέσιμων Μοντέλων

Μόλις εγκαταστήσετε το Foundry Local CLI, μπορείτε να δείτε ποια μοντέλα είναι διαθέσιμα για τη χρήση σας. Η εντολή αυτή θα σας δείξει όλα τα υποστηριζόμενα μοντέλα:


```bash
foundry model list
```

### Βήμα 3: Κατανόηση των Μοντέλων Phi Family

Η οικογένεια Phi προσφέρει μια σειρά μοντέλων βελτιστοποιημένων για διαφορετικές περιπτώσεις χρήσης και διαμορφώσεις υλικού. Εδώ είναι τα μοντέλα Phi που είναι διαθέσιμα στο Foundry Local:

**Διαθέσιμα Μοντέλα Phi:** 

- **phi-3.5-mini** - Συμπαγές μοντέλο για βασικές εργασίες
- **phi-3-mini-128k** - Εκτεταμένη έκδοση συμφραζομένων για πιο μακροχρόνιες συνομιλίες
- **phi-3-mini-4k** - Πρότυπο μοντέλο συμφραζομένων για γενική χρήση
- **phi-4** - Προχωρημένο μοντέλο με βελτιωμένες δυνατότητες
- **phi-4-mini** - Ελαφριά έκδοση του Phi-4
- **phi-4-mini-reasoning** - Εξειδικευμένο για σύνθετες εργασίες λογικής

> **Συμβατότητα Υλικού:** Κάθε μοντέλο μπορεί να ρυθμιστεί για διαφορετική επιτάχυνση υλικού (CPU, GPU) ανάλογα με τις δυνατότητες του συστήματός σας.

### Βήμα 4: Εκτέλεση του Πρώτου σας Μοντέλου Phi

Ας ξεκινήσουμε με ένα πρακτικό παράδειγμα. Θα εκτελέσουμε το μοντέλο `phi-4-mini-reasoning`, που διακρίνεται στην επίλυση σύνθετων προβλημάτων βήμα προς βήμα.


**Εντολή εκτέλεσης μοντέλου:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Πρώτη Ρύθμιση:** Κατά την πρώτη εκτέλεση ενός μοντέλου, το Foundry Local θα το κατεβάσει αυτόματα στη συσκευή σας. Ο χρόνος λήψης εξαρτάται από την ταχύτητα του δικτύου σας, οπότε παρακαλώ να είστε υπομονετικοί κατά την αρχική ρύθμιση.

### Βήμα 5: Δοκιμή του Μοντέλου με Ένα Πραγματικό Πρόβλημα

Τώρα ας δοκιμάσουμε το μοντέλο μας με ένα κλασικό πρόβλημα λογικής για να δούμε πώς εκτελεί βήμα προς βήμα συλλογισμό:

**Παράδειγμα Προβλήματος:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Αναμενόμενη Συμπεριφορά:** Το μοντέλο θα πρέπει να διασπάσει αυτό το πρόβλημα σε λογικά βήματα, χρησιμοποιώντας το γεγονός ότι οι φασιανοί έχουν 2 πόδια και τα κουνέλια 4 πόδια για να λύσει το σύστημα των εξισώσεων.

## Μέρος 2: Δημιουργία Εφαρμογών με το Foundry Local SDK

### Γιατί να Χρησιμοποιήσετε το SDK;

Ενώ το CLI είναι ιδανικό για δοκιμές και γρήγορες αλληλεπιδράσεις, το SDK σας επιτρέπει να ενσωματώσετε το Foundry Local στις εφαρμογές σας προγραμματιστικά. Αυτό ανοίγει δυνατότητες για:

- Δημιουργία προσαρμοσμένων εφαρμογών με δυνατότητα AI
- Δημιουργία αυτοματοποιημένων ροών εργασίας
- Ενσωμάτωση δυνατοτήτων AI σε υπάρχοντα συστήματα
- Ανάπτυξη chatbots και διαδραστικών εργαλείων

### Υποστηριζόμενες Γλώσσες Προγραμματισμού

Το Foundry Local παρέχει υποστήριξη SDK για πολλές γλώσσες προγραμματισμού ώστε να ταιριάζει στις προτιμήσεις ανάπτυξης σας:

**📦 Διαθέσιμα SDKs:**

- **C# (.NET):** [Τεκμηρίωση & Παραδείγματα SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [Τεκμηρίωση & Παραδείγματα SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [Τεκμηρίωση & Παραδείγματα SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [Τεκμηρίωση & Παραδείγματα SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### Επόμενα Βήματα

1. **Επιλέξτε το προτιμώμενο SDK** με βάση το περιβάλλον ανάπτυξής σας
2. **Ακολουθήστε την τεκμηρίωση κάθε SDK** για λεπτομερείς οδηγίες υλοποίησης
3. **Ξεκινήστε με απλά παραδείγματα** πριν δημιουργήσετε σύνθετες εφαρμογές
4. **Εξερευνήστε τον δείγμα κώδικα** που παρέχεται σε κάθε αποθετήριο SDK
