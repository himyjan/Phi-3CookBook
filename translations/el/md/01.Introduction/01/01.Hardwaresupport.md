# Υποστήριξη Υλικού Phi

Το Microsoft Phi έχει βελτιστοποιηθεί για το ONNX Runtime και υποστηρίζει το Windows DirectML. Λειτουργεί καλά σε διάφορους τύπους υλικού, συμπεριλαμβανομένων των GPU, CPU, και ακόμη και κινητών συσκευών.

## Υλικό Συσκευής
Συγκεκριμένα, το υποστηριζόμενο υλικό περιλαμβάνει:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPUs, 128 GiB μνήμη)

## Mobile SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 ή ανώτερο με επεξεργαστή A16/A17

## Προδιαγραφές Υλικού Phi

- Ελάχιστη Απαραίτητη Διαμόρφωση.
- Windows: GPU συμβατή με DirectX 12 και τουλάχιστον 4GB συνδυασμένης μνήμης RAM

CUDA: NVIDIA GPU με Compute Capability >= 7.02

## Εκτέλεση onnxruntime σε πολλαπλές GPUs

Τα τρέχοντα διαθέσιμα μοντέλα Phi ONNX είναι μόνο για 1 GPU. Είναι δυνατό να υποστηριχθεί multi-gpu για το μοντέλο Phi, αλλά το ORT με 2 GPU δεν εγγυάται ότι θα παρέχει μεγαλύτερη απόδοση σε σύγκριση με 2 περιπτώσεις του ort. Παρακαλώ δείτε το [ONNX Runtime](https://onnxruntime.ai/) για τις τελευταίες ενημερώσεις.

Στο [Build 2024 η ομάδα GenAI ONNX](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) ανακοίνωσε ότι ενεργοποίησαν την πολυ-περίπτωση αντί για multi-gpu για τα μοντέλα Phi.

Προς το παρόν, αυτό επιτρέπει να εκτελέσετε μία περίπτωση onnnxruntime ή onnxruntime-genai με τη μεταβλητή περιβάλλοντος CUDA_VISIBLE_DEVICES όπως αυτή.

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

Νιώστε ελεύθεροι να εξερευνήσετε περαιτέρω το Phi στο [Microsoft Foundry](https://ai.azure.com)
