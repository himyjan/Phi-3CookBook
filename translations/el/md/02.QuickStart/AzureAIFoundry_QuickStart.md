# **Χρήση του Phi-3 στο Microsoft Foundry**

Με την ανάπτυξη της Γενετικής Τεχνητής Νοημοσύνης, ελπίζουμε να χρησιμοποιήσουμε μια ενοποιημένη πλατφόρμα για τη διαχείριση διαφορετικών LLM και SLM, την ολοκλήρωση δεδομένων επιχειρήσεων, τις εργασίες fine-tuning/RAG, και την αξιολόγηση διαφορετικών επιχειρήσεων μετά την ενσωμάτωση LLM και SLM, κ.λπ., ώστε να μπορεί η γενετική ΤΝ να υλοποιηθεί καλύτερα σε έξυπνες εφαρμογές. Το [Microsoft Foundry](https://ai.azure.com) είναι μια πλατφόρμα γενετικών εφαρμογών τεχνητής νοημοσύνης σε επίπεδο επιχείρησης.

![aistudo](../../../../translated_images/el/aifoundry_home.f28a8127c96c7d93.webp)

Με το Microsoft Foundry, μπορείτε να αξιολογήσετε τις απαντήσεις μεγάλων γλωσσικών μοντέλων (LLM) και να οργανώσετε συστατικά εφαρμογών prompt με prompt flow για καλύτερη απόδοση. Η πλατφόρμα διευκολύνει την κλιμάκωση για τη μετατροπή αποδείξεων έννοιας σε πλήρη παραγωγή με ευκολία. Η συνεχής παρακολούθηση και βελτίωση υποστηρίζουν τη μακροπρόθεσμη επιτυχία.

Μπορούμε να αναπτύξουμε γρήγορα το μοντέλο Phi-3 στο Microsoft Foundry μέσω απλών βημάτων και στη συνέχεια να χρησιμοποιήσουμε το Microsoft Foundry για να ολοκληρώσουμε εργασίες όπως Playground/Chat, Fine-tuning, αξιολόγηση και άλλα σχετικά με το Phi-3.

## **1. Προετοιμασία**

Εάν έχετε ήδη εγκαταστήσει το [Azure Developer CLI](https://learn.microsoft.com/azure/developer/azure-developer-cli/overview?WT.mc_id=aiml-138114-kinfeylo) στον υπολογιστή σας, η χρήση αυτού του προτύπου είναι απλή όπως το να εκτελέσετε την εντολή σε έναν νέο κατάλογο.

## Χειροκίνητη Δημιουργία

Η δημιουργία ενός έργου και hub στο Microsoft Foundry είναι ένας εξαιρετικός τρόπος για να οργανώσετε και να διαχειριστείτε το έργο τεχνητής νοημοσύνης σας. Ακολουθεί ένας οδηγός βήμα-βήμα για να ξεκινήσετε:

### Δημιουργία έργου στο Microsoft Foundry

1. **Μεταβείτε στο Microsoft Foundry**: Συνδεθείτε στην πύλη Microsoft Foundry.
2. **Δημιουργήστε ένα έργο**:
   - Εάν βρίσκεστε ήδη σε ένα έργο, επιλέξτε "Microsoft Foundry" πάνω αριστερά στη σελίδα για να μεταβείτε στην Αρχική σελίδα.
   - Επιλέξτε "+ Δημιουργία έργου".
   - Εισάγετε ένα όνομα για το έργο.
   - Εάν έχετε ήδη ένα hub, θα επιλεγεί αυτόματα. Εάν έχετε πρόσβαση σε περισσότερα από ένα hubs, μπορείτε να επιλέξετε κάποιο διαφορετικό από το αναπτυσσόμενο μενού. Εάν θέλετε να δημιουργήσετε νέο hub, επιλέξτε "Δημιουργία νέου hub" και δώστε ένα όνομα.
   - Επιλέξτε "Δημιουργία".

### Δημιουργία hub στο Microsoft Foundry

1. **Μεταβείτε στο Microsoft Foundry**: Συνδεθείτε με το λογαριασμό Azure σας.
2. **Δημιουργία hub**:
   - Επιλέξτε το Κέντρο Διαχείρισης από το αριστερό μενού.
   - Επιλέξτε "Όλοι οι πόροι", στη συνέχεια το βέλος κάτω από το "+ Νέο έργο" και επιλέξτε "+ Νέο hub".
   - Στο διάλογο "Δημιουργία νέου hub", εισάγετε ένα όνομα για το hub σας (π.χ., contoso-hub) και τροποποιήστε τα υπόλοιπα πεδία όπως επιθυμείτε.
   - Επιλέξτε "Επόμενο", ελέγξτε τις πληροφορίες και στη συνέχεια επιλέξτε "Δημιουργία".

Για πιο λεπτομερείς οδηγίες, μπορείτε να ανατρέξετε στην επίσημη [τεκμηρίωση Microsoft](https://learn.microsoft.com/azure/ai-studio/how-to/create-projects).

Μετά την επιτυχή δημιουργία, μπορείτε να έχετε πρόσβαση στο studio που δημιουργήσατε μέσω του [ai.azure.com](https://ai.azure.com/)

Μπορεί να υπάρχουν πολλαπλά έργα σε ένα AI Foundry. Δημιουργήστε ένα έργο στο AI Foundry για να προετοιμαστείτε.

Δημιουργία Microsoft Foundry [QuickStarts](https://learn.microsoft.com/azure/ai-studio/quickstarts/get-started-code)


## **2. Ανάπτυξη μοντέλου Phi στο Microsoft Foundry**

Κάντε κλικ στην επιλογή Explore του έργου για να εισέλθετε στον Κατάλογο Μοντέλων και επιλέξτε Phi-3.

Επιλέξτε Phi-3-mini-4k-instruct.

Πατήστε 'Deploy' για να αναπτύξετε το μοντέλο Phi-3-mini-4k-instruct.

> [!NOTE]
>
> Μπορείτε να επιλέξετε υπολογιστική ισχύ κατά την ανάπτυξη.

## **3. Παιδική Χαρά Chat Phi στο Microsoft Foundry**

Μεταβείτε στη σελίδα ανάπτυξης, επιλέξτε Playground και συνομιλήστε με το Phi-3 του Microsoft Foundry.

## **4. Ανάπτυξη Μοντέλου από το Microsoft Foundry**

Για να αναπτύξετε ένα μοντέλο από τον Κατάλογο Μοντέλων Azure, μπορείτε να ακολουθήσετε τα παρακάτω βήματα:

- Συνδεθείτε στο Microsoft Foundry.
- Επιλέξτε το μοντέλο που θέλετε να αναπτύξετε από τον κατάλογο μοντέλων του Microsoft Foundry.
- Στη σελίδα Λεπτομερειών του μοντέλου, επιλέξτε Ανάπτυξη και στη συνέχεια επιλέξτε Serverless API με Azure AI Content Safety.
- Επιλέξτε το έργο στο οποίο θέλετε να αναπτύξετε τα μοντέλα σας. Για να χρησιμοποιήσετε την προσφορά Serverless API, ο χώρος εργασίας σας πρέπει να ανήκει στην περιοχή East US 2 ή Sweden Central. Μπορείτε να προσαρμόσετε το όνομα Ανάπτυξης.
- Στον οδηγό ανάπτυξης, επιλέξτε Τιμολόγηση και Όρους για να ενημερωθείτε σχετικά με την τιμολόγηση και τους όρους χρήσης.
- Επιλέξτε Ανάπτυξη. Περιμένετε μέχρι να ετοιμαστεί η ανάπτυξη και να ανακατευθυνθείτε στη σελίδα Αναπτύξεων.
- Επιλέξτε Άνοιγμα στο playground για να αρχίσετε να αλληλεπιδράτε με το μοντέλο.
- Μπορείτε να επιστρέψετε στη σελίδα Αναπτύξεων, να επιλέξετε την ανάπτυξη και να σημειώσετε το URL-στόχο του endpoint και το Μυστικό Κλειδί, τα οποία μπορείτε να χρησιμοποιήσετε για να καλέσετε την ανάπτυξη και να δημιουργήσετε απαντήσεις.
- Μπορείτε πάντα να βρείτε τις λεπτομέρειες του endpoint, το URL και τα κλειδιά πρόσβασης μεταβαίνοντας στην καρτέλα Build και επιλέγοντας Αναπτύξεις από το τμήμα Components.

> [!NOTE]
> Σημειώστε ότι ο λογαριασμός σας πρέπει να διαθέτει τα δικαιώματα ρόλου Azure AI Developer στο Resource Group για να εκτελέσετε αυτά τα βήματα.

## **5. Χρήση του Phi API στο Microsoft Foundry**

Μπορείτε να έχετε πρόσβαση στο https://{Το όνομα του έργου σας}.region.inference.ml.azure.com/swagger.json μέσω Postman με μέθοδο GET και να το συνδυάσετε με το Κλειδί για να μάθετε σχετικά με τα παρεχόμενα interfaces.

Μπορείτε πολύ εύκολα να λάβετε τις παραμέτρους αιτήματος καθώς και τις παραμέτρους απάντησης.
