## Σενάρια Βελτιστοποίησης

![FineTuning with MS Services](../../../../translated_images/el/FinetuningwithMS.3d0cec8ae693e094.webp)

Αυτή η ενότητα παρέχει μια επισκόπηση των σεναρίων βελτιστοποίησης στο Microsoft Foundry και περιβάλλοντα Azure, συμπεριλαμβανομένων των μοντέλων ανάπτυξης, επιπέδων υποδομής και κοινώς χρησιμοποιούμενων τεχνικών βελτιστοποίησης.

**Πλατφόρμα**  
Αυτή περιλαμβάνει διαχειριζόμενες υπηρεσίες όπως το Microsoft Foundry (παλαιότερα Microsoft Foundry) και το Azure Machine Learning, που προσφέρουν διαχείριση μοντέλων, ορχήστρωση, παρακολούθηση πειραμάτων και ροές εργασίας ανάπτυξης.

**Υποδομή**  
Η βελτιστοποίηση απαιτεί κλιμακούμενους υπολογιστικούς πόρους. Στα περιβάλλοντα Azure, αυτό συνήθως περιλαμβάνει εικονικές μηχανές βασισμένες σε GPU και πόρους CPU για ελαφριά φορτία εργασίας, μαζί με κλιμακούμενη αποθήκευση για σύνολα δεδομένων και σημεία ελέγχου.

**Εργαλεία & Πλαίσια**  
Οι ροές εργασίας βελτιστοποίησης βασίζονται συνήθως σε πλαίσια και βιβλιοθήκες βελτιστοποίησης όπως τα Hugging Face Transformers, DeepSpeed και PEFT (Parameter-Efficient Fine-Tuning).

Η διαδικασία βελτιστοποίησης με τις τεχνολογίες της Microsoft εκτείνεται σε υπηρεσίες πλατφόρμας, υποδομή υπολογιστών και πλαίσια εκπαίδευσης. Κατανοώντας πώς αυτά τα στοιχεία συνεργάζονται, οι προγραμματιστές μπορούν να προσαρμόσουν αποτελεσματικά τα βασικά μοντέλα σε συγκεκριμένα καθήκοντα και σενάρια παραγωγής.

## Μοντέλο ως Υπηρεσία

Κάντε λεπτομερή ρύθμιση του μοντέλου χρησιμοποιώντας φιλοξενούμενη βελτιστοποίηση, χωρίς την ανάγκη δημιουργίας και διαχείρισης υπολογιστικών πόρων.

![MaaS Fine Tuning](../../../../translated_images/el/MaaSfinetune.3eee4630607aff0d.webp)

Η serverless βελτιστοποίηση είναι πλέον διαθέσιμη για τις οικογένειες μοντέλων Phi-3, Phi-3.5 και Phi-4, δίνοντας τη δυνατότητα στους προγραμματιστές να προσαρμόζουν γρήγορα και εύκολα τα μοντέλα για σενάρια cloud και edge χωρίς να χρειάζεται να οργανώσουν υπολογιστικούς πόρους.

## Μοντέλο ως Πλατφόρμα

Οι χρήστες διαχειρίζονται τους δικούς τους υπολογιστικούς πόρους για να κάνουν βελτιστοποίηση των μοντέλων τους.
