## Memulai dengan Model Keluarga Phi di Foundry Local

### Pengenalan Foundry Local

Foundry Local adalah solusi inferensi AI yang kuat langsung di perangkat yang menghadirkan kemampuan AI tingkat perusahaan langsung ke perangkat keras lokal Anda. Tutorial ini akan memandu Anda melalui pengaturan dan penggunaan model Keluarga Phi dengan Foundry Local, menawarkan kontrol penuh atas beban kerja AI Anda sambil menjaga privasi dan mengurangi biaya.

Foundry Local menyediakan keunggulan performa, privasi, kustomisasi, dan biaya dengan menjalankan model AI secara lokal di perangkat Anda. Ia terintegrasi dengan mulus ke dalam alur kerja dan aplikasi Anda yang sudah ada melalui CLI, SDK, dan REST API yang intuitif.


![arch](../../../../../translated_images/id/foundry-local-arch.8823e321dd8258d7.webp)

### Mengapa Memilih Foundry Local?

Memahami manfaat Foundry Local akan membantu Anda membuat keputusan yang tepat tentang strategi penerapan AI Anda:

- **Inferensi di Perangkat:** Jalankan model secara lokal di perangkat keras Anda sendiri, mengurangi biaya sambil menjaga semua data tetap di perangkat Anda.

- **Kustomisasi Model:** Pilih dari model yang sudah ada atau gunakan model Anda sendiri untuk memenuhi kebutuhan dan kasus penggunaan spesifik.

- **Efisiensi Biaya:** Hilangkan biaya layanan cloud berulang dengan menggunakan perangkat keras yang sudah Anda miliki, membuat AI lebih terjangkau.

- **Integrasi Mulus:** Hubungkan dengan aplikasi Anda melalui SDK, endpoint API, atau CLI, dengan skala mudah menuju Microsoft Foundry sesuai kebutuhan Anda.

> **Catatan Memulai:** Tutorial ini fokus pada penggunaan Foundry Local melalui antarmuka CLI dan SDK. Anda akan mempelajari kedua pendekatan untuk membantu memilih metode terbaik sesuai kebutuhan Anda.

## Bagian 1: Menyiapkan Foundry Local CLI

### Langkah 1: Instalasi

Foundry Local CLI adalah pintu gerbang Anda untuk mengelola dan menjalankan model AI secara lokal. Mari mulai dengan memasangnya di sistem Anda.

**Platform yang Didukung:** Windows dan macOS

Untuk instruksi instalasi yang lebih rinci, silakan lihat [dokumentasi resmi Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Langkah 2: Menjelajahi Model yang Tersedia

Setelah Foundry Local CLI terpasang, Anda dapat menemukan model apa saja yang tersedia untuk kasus penggunaan Anda. Perintah ini akan menampilkan semua model yang didukung:


```bash
foundry model list
```

### Langkah 3: Memahami Model Keluarga Phi

Keluarga Phi menawarkan berbagai model yang dioptimalkan untuk berbagai kasus penggunaan dan konfigurasi perangkat keras. Berikut adalah model Phi yang tersedia di Foundry Local:

**Model Phi yang Tersedia:**

- **phi-3.5-mini** - Model kompak untuk tugas dasar
- **phi-3-mini-128k** - Versi konteks diperluas untuk percakapan lebih panjang
- **phi-3-mini-4k** - Model konteks standar untuk penggunaan umum
- **phi-4** - Model lanjutan dengan kemampuan yang ditingkatkan
- **phi-4-mini** - Versi ringan dari Phi-4
- **phi-4-mini-reasoning** - Khusus untuk tugas penalaran kompleks

> **Kompatibilitas Perangkat Keras:** Setiap model dapat dikonfigurasi untuk akselerasi perangkat keras berbeda (CPU, GPU) tergantung kemampuan sistem Anda.

### Langkah 4: Menjalankan Model Phi Pertama Anda

Mari mulai dengan contoh praktis. Kita akan menjalankan model `phi-4-mini-reasoning`, yang unggul dalam menyelesaikan masalah kompleks langkah demi langkah.


**Perintah untuk menjalankan model:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Pengaturan Pertama Kali:** Saat menjalankan model untuk pertama kalinya, Foundry Local akan mengunduh model secara otomatis ke perangkat lokal Anda. Waktu unduhan bervariasi sesuai kecepatan jaringan Anda, jadi harap bersabar selama pengaturan awal.

### Langkah 5: Menguji Model dengan Masalah Nyata

Sekarang mari uji model kami dengan masalah logika klasik untuk melihat bagaimana model melakukan penalaran langkah-demi-langkah:

**Contoh Masalah:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Perilaku yang Diharapkan:** Model seharusnya memecah masalah ini menjadi langkah-langkah logis, menggunakan fakta bahwa burung pegar mempunyai 2 kaki dan kelinci 4 kaki untuk menyelesaikan sistem persamaan.

## Bagian 2: Membangun Aplikasi dengan Foundry Local SDK

### Mengapa Menggunakan SDK?

Walaupun CLI sempurna untuk pengujian dan interaksi cepat, SDK memungkinkan Anda mengintegrasikan Foundry Local ke dalam aplikasi Anda secara pemrograman. Ini membuka kemungkinan untuk:

- Membangun aplikasi dengan kecerdasan buatan kustom
- Membuat alur kerja otomatis
- Mengintegrasikan kemampuan AI ke dalam sistem yang sudah ada
- Mengembangkan chatbot dan alat interaktif

### Bahasa Pemrograman yang Didukung

Foundry Local menyediakan dukungan SDK untuk berbagai bahasa pemrograman agar sesuai dengan preferensi pengembangan Anda:

**📦 SDK yang Tersedia:**

- **C# (.NET):** [Dokumentasi & Contoh SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [Dokumentasi & Contoh SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [Dokumentasi & Contoh SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [Dokumentasi & Contoh SDK](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### Langkah Selanjutnya

1. **Pilih SDK sesuai preferensi pengembangan Anda**
2. **Ikuti dokumentasi spesifik SDK** untuk panduan implementasi terperinci
3. **Mulai dengan contoh sederhana** sebelum membuat aplikasi kompleks
4. **Jelajahi kode contoh** yang disediakan di setiap repositori SDK
