## Початок роботи з моделями Phi-Family у Foundry Local

### Вступ до Foundry Local

Foundry Local — це потужне рішення для AI-інференсу на пристрої, яке надає можливості штучного інтелекту корпоративного рівня безпосередньо на вашому локальному обладнанні. Цей посібник допоможе вам налаштувати та використовувати моделі Phi-Family з Foundry Local, забезпечуючи повний контроль над AI-навантаженням, зберігаючи конфіденційність і знижуючи витрати.

Foundry Local забезпечує переваги в продуктивності, конфіденційності, налаштуванні та вартості, запускаючи AI-моделі локально на вашому пристрої. Він бездоганно інтегрується у ваші існуючі робочі процеси та застосунки через інтуїтивно зрозумілий CLI, SDK та REST API.

### Чому варто обрати Foundry Local?

Розуміння переваг Foundry Local допоможе вам приймати обґрунтовані рішення щодо стратегії розгортання AI:

- **Інференс на пристрої:** Запускайте моделі локально на власному обладнанні, знижуючи витрати та зберігаючи всі дані на своєму пристрої.

- **Налаштування моделей:** Вибирайте серед попередньо встановлених моделей або використовуйте власні для задоволення конкретних вимог та випадків застосування.

- **Економічність:** Позбавтеся від повторюваних витрат на хмарні сервіси, використовуючи наявне обладнання, роблячи AI доступнішим.

- **Безшовна інтеграція:** Підключайтеся до своїх застосунків через SDK, API або CLI із легким масштабуванням до Microsoft Foundry, коли ваші потреби зростають.

> **Примітка для початку роботи:** Цей посібник зосереджений на використанні Foundry Local через інтерфейси CLI та SDK. Ви дізнаєтесь обидва підходи, щоб допомогти обрати найкращий метод для вашого випадку.

## Частина 1: Налаштування Foundry Local CLI

### Крок 1: Встановлення

Foundry Local CLI — це ваш доступ до керування та запуску AI-моделей локально. Почнемо з встановлення на вашу систему.

**Підтримувані платформи:** Windows та macOS

Для детальних інструкцій із встановлення зверніться до [офіційної документації Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Крок 2: Огляд доступних моделей

Після встановлення Foundry Local CLI ви можете дізнатися, які моделі доступні для вашого випадку використання. Ця команда покаже всі підтримувані моделі:


```bash
foundry model list
```

### Крок 3: Розуміння моделей Phi Family

Phi Family пропонує широкий вибір моделей, оптимізованих для різних випадків використання та конфігурацій обладнання. Ось моделі Phi, доступні у Foundry Local:

**Доступні моделі Phi:** 

- **phi-3.5-mini** - Компактна модель для базових завдань
- **phi-3-mini-128k** - Версія з розширеним контекстом для довших розмов
- **phi-3-mini-4k** - Стандартна контекстна модель для загального використання
- **phi-4** - Розвинена модель з покращеними можливостями
- **phi-4-mini** - Полегшена версія Phi-4
- **phi-4-mini-reasoning** - Спеціалізована для складних завдань логічного мислення

> **Сумісність з обладнанням:** Кожну модель можна налаштовувати для різного апаратного прискорення (CPU, GPU) залежно від можливостей вашої системи.

### Крок 4: Запуск вашої першої моделі Phi

Почнемо з практичного прикладу. Запустимо модель `phi-4-mini-reasoning`, яка добре справляється з поетапним розв'язанням складних задач.


**Команда запуску моделі:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Налаштування при першому запуску:** Під час першого запуску модель автоматично завантажиться на ваш пристрій. Час завантаження залежить від швидкості вашого інтернет-з'єднання, тому під час початкового налаштування потрібно бути терплячим.

### Крок 5: Перевірка моделі на реальній задачі

Тепер перевіримо нашу модель на класичній логічній задачі, щоб побачити, як вона виконує поетапне мислення:

**Приклад задачі:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Очікувана поведінка:** Модель має розбити цю задачу на логічні кроки, використовуючи факт, що у фазанів 2 ноги, а у кроликів 4 ноги, щоб розв’язати систему рівнянь.

- Автоматизації робочих процесів
- Інтеграції AI у існуючі системи
- Розробки чат-ботів та інтерактивних інструментів
