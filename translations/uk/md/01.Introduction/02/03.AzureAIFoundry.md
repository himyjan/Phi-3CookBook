## Родина Phi у Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) — це надійна платформа, яка дає змогу розробникам впроваджувати інновації та формувати майбутнє з AI у безпечний, захищений та відповідальний спосіб.

[Microsoft Foundry](https://ai.azure.com) створена для розробників, щоб:

- Створювати генеративні AI-додатки на корпоративній платформі.
- Досліджувати, будувати, тестувати та розгортати за допомогою найсучасніших інструментів AI та моделей машинного навчання, базуючись на відповідальних практиках AI.
- Співпрацювати з командою на всьому життєвому циклі розробки додатків.

З Microsoft Foundry ви можете досліджувати широкий спектр моделей, сервісів і можливостей, а також розробляти AI-додатки, які найкраще відповідають вашим цілям. Платформа Microsoft Foundry сприяє масштабуванню, полегшуючи трансформацію концепцій у повноцінні виробничі додатки. Постійне моніторинг і вдосконалення підтримують довгостроковий успіх.

### **Запуск коду на Python для виклику Microsoft Foundry Phi-4**

```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Включити результат розпізнавання мови, якщо мова увімкнена
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
