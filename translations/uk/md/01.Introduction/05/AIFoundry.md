# **Використання Microsoft Foundry для оцінювання**

![aistudo](../../../../../translated_images/uk/AIFoundry.9e0b513e999a1c5a.webp)

Як оцінити ваш генеративний AI-додаток за допомогою [Microsoft Foundry](https://ai.azure.com?WT.mc_id=aiml-138114-kinfeylo). Незалежно від того, чи ви оцінюєте одноголосні чи багатоголосні розмови, Microsoft Foundry надає інструменти для оцінки продуктивності моделі та безпеки.

![aistudo](../../../../../translated_images/uk/AIPortfolio.69da59a8e1eaa70f.webp)

## Як оцінити генеративні AI-додатки за допомогою Microsoft Foundry
Для детальної інструкції дивіться [документацію Microsoft Foundry](https://learn.microsoft.com/azure/ai-studio/how-to/evaluate-generative-ai-app?WT.mc_id=aiml-138114-kinfeylo)

Ось кроки для початку:

## Оцінка генеративних AI-моделей у Microsoft Foundry

**Передумови**

- Тестовий набір даних у форматі CSV або JSON.
- Розгорнута генеративна AI-модель (наприклад, Phi-3, GPT 3.5, GPT 4 або моделі Davinci).
- Виконавче середовище з обчислювальним інстансом для запуску оцінки.

## Вбудовані метрики оцінювання

Microsoft Foundry дозволяє оцінювати як одноголосні, так і складні багатоголосні розмови.
Для сценаріїв Retrieval Augmented Generation (RAG), де модель базується на конкретних даних, ви можете оцінити продуктивність за допомогою вбудованих метрик.
Крім того, можна оцінити загальні сценарії відповіді на одноголосні запитання (не RAG).

## Створення запуску оцінювання

У інтерфейсі Microsoft Foundry перейдіть на сторінку Evaluate або Prompt Flow.
Слідуйте майстру створення оцінювання, щоб налаштувати запуск оцінювання. Вкажіть необов’язкову назву для вашого оцінювання.
Виберіть сценарій, який відповідає цілям вашого додатку.
Виберіть одну або декілька метрик оцінювання для аналізу результатів моделі.

## Користувацький потік оцінювання (необов’язково)

Для більшої гнучкості ви можете створити користувацький потік оцінювання. Налаштуйте процес оцінки відповідно до ваших конкретних вимог.

## Перегляд результатів

Після запуску оцінювання реєструйте, переглядайте та аналізуйте детальні метрики оцінювання у Microsoft Foundry. Отримуйте уявлення про можливості та обмеження вашого додатка.

**Note** Microsoft Foundry наразі знаходиться у публічному прев’ю, тому використовуйте його для експериментів та розробки. Для продуктивних навантажень розгляньте інші опції. Ознайомтеся з офіційною [документацією AI Foundry](https://learn.microsoft.com/azure/ai-studio/?WT.mc_id=aiml-138114-kinfeylo) для додаткових деталей та покрокових інструкцій.
