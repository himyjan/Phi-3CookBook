# Родина Phi від Microsoft

Моделі Phi є найпотужнішими та економічно ефективними Малими Мовними Моделями (SLM), які перевершують моделі того ж розміру та наступного розміру за різними показниками для мови, логіки, кодування, аудіо, зору та математики. Це випуск розширює вибір високоякісних моделей для клієнтів, пропонуючи більш практичні варіанти для створення та побудови генеративних AI-додатків.

Родина Phi почалася з Phi-1 для генерації коду на Python, продовжилася моделями Phi-1.5/2 на основі тексту та чат-комплітації, Phi-3-mini/small/medium-instruct та Phi-3.5/4-mini-instruct, розвинулася до Phi-3/3.5-vision для зору, Phi-4 на основі складного логічного мислення, Phi-3.5-MoE для MoE, і тепер повноформатна мультимодальна модель Phi-4-multimodal. Завдяки високоякісним наборам даних, ці моделі можуть бути навчені до рівня моделей з більшим числом параметрів.
