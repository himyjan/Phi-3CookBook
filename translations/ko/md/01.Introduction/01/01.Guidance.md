### Guidance-AI 및 Phi 모델 서비스(MaaS)
우리는 [Guidance](https://github.com/guidance-ai/guidance)를 Microsoft Foundry의 Phi-3.5-mini 서버리스 엔드포인트 제공에 도입하여 애플리케이션에 맞게 구조를 정의함으로써 출력의 예측 가능성을 높였습니다. Guidance를 사용하면 비용이 많이 드는 재시도를 없앨 수 있으며, 예를 들어 사전 정의된 목록(예: 의료 코드)에서 선택하도록 모델을 제한하거나, 제공된 컨텍스트에서 직접 인용한 출력만 허용하거나, 특정 정규식에 따라 출력하도록 제약할 수 있습니다. Guidance는 추론 스택에서 토큰 단위로 모델을 조종하여 비용과 대기 시간을 30-50% 절감하며, 이는 [Phi-3-mini 서버리스 엔드포인트](https://aka.ms/try-phi3.5mini)에 매우 독특하고 유용한 추가 기능이 됩니다.

## [**Guidance-AI**](https://github.com/guidance-ai/guidance)는 개발자가 AI 모델을 효율적으로 생성하고 배포할 수 있도록 도와주는 프레임워크입니다. 견고한 AI 애플리케이션을 구축하기 위한 도구와 모범 사례를 제공하는 데 중점을 둡니다.

<strong>Phi Models as a Service (MaaS)</strong>와 결합하면, 비용 효율적이고 고성능인 소형 언어 모델(SLM)을 배포하는 강력한 솔루션을 제공합니다.

<strong>Guidance-AI</strong>는 개발자가 대형 언어 모델(LLM)을 보다 효과적으로 제어하고 조종할 수 있도록 설계된 프로그래밍 프레임워크입니다. 출력 구성을 정밀하게 제어하여 기존의 프롬프트 기반 방법이나 미세 조정 방식에 비해 대기 시간과 비용을 줄입니다.

### Guidance-AI 주요 기능:
- **효율적인 제어**: 개발자가 언어 모델의 텍스트 생성 방식을 제어하여 고품질의 관련성 높은 출력을 보장합니다.
- **비용 및 대기 시간 절감**: 생성 프로세스를 최적화하여 더 비용 효율적이고 빠르게 처리합니다.
- **유연한 통합**: Transformers, llama.cpp, AzureAI, VertexAI, OpenAI 등 다양한 백엔드와 연동 가능합니다.
- **풍부한 출력 구조**: 조건문, 반복문, 도구 사용 등 복잡한 출력 구조를 지원하여 명확하고 파싱 가능한 결과 생성이 용이합니다.
- <strong>호환성</strong>: 하나의 Guidance 프로그램을 여러 백엔드에서 실행할 수 있어 유연성과 사용 편의성을 높입니다.

### 예제 활용 사례:
- **제약 있는 생성**: 정규식 및 문맥 자유 문법을 활용하여 모델 출력 제어.
- **도구 통합**: 예를 들어, 텍스트 생성 작업 중 계산기를 자동으로 교차 사용하여 제어와 생성을 병행.

자세한 정보와 예제는 [Guidance-AI GitHub 저장소](https://github.com/guidance-ai/guidance)를 참고하세요.

### Phi 모델 주요 기능:
1. **비용 효율성**: 높은 성능을 유지하면서도 합리적인 비용으로 설계됨.
2. <strong>저지연</strong>: 빠른 응답이 필요한 실시간 애플리케이션에 적합.
3. <strong>유연성</strong>: 클라우드, 에지, 오프라인 환경 등 다양한 환경에 배포 가능.
4. <strong>맞춤화</strong>: 도메인별 데이터로 미세 조정하여 성능 향상 가능.
5. **보안 및 규정 준수**: Microsoft의 AI 원칙에 따라 책임성, 투명성, 공정성, 신뢰성, 안전성, 개인정보 보호 및 포용성을 보장.

### Phi 모델 서비스(MaaS):
Phi 모델은 추론 API를 통한 종량제 과금 시스템으로 제공되어 초기 비용 부담 없이 애플리케이션에 쉽게 통합할 수 있습니다.

### Phi-3 시작하기:
Phi 모델 사용을 시작하려면 [Azure AI 모델 카탈로그](https://ai.azure.com/explore/models) 또는 GitHub Marketplace Models](https://github.com/marketplace/models)에서 미리 구축되었거나 맞춤형 모델을 탐색해 보세요. 또한 [Microsoft Foundry](https://ai.azure.com)와 같은 도구를 사용해 AI 애플리케이션을 개발하고 배포할 수 있습니다.
