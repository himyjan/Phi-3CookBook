# AITK의 Phi 패밀리

[AI Toolkit for VS Code](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio)는 Microsoft Foundry 카탈로그와 Hugging Face와 같은 다른 카탈로그의 최첨단 AI 개발 도구 및 모델을 통합하여 생성 AI 앱 개발을 간소화합니다. GitHub 모델과 Microsoft Foundry 모델 카탈로그를 기반으로 한 AI 모델 카탈로그를 탐색하고, 로컬 또는 원격으로 다운로드하며, 미세 조정, 테스트 및 애플리케이션에서 사용할 수 있습니다.

AI Toolkit 미리보기는 로컬에서 실행됩니다. 로컬 추론 또는 미세 조정은 선택한 모델에 따라 NVIDIA CUDA GPU와 같은 GPU가 필요할 수 있습니다. 또한 GitHub 모델도 AITK로 직접 실행할 수 있습니다.

### AI Toolkit 설치

AI Toolkit은 [Visual Studio Code 확장](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions)으로 제공되므로, 먼저 [VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo)를 설치하고 [VS 마켓플레이스](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio)에서 AI Toolkit을 다운로드해야 합니다.
[AI Toolkit은 Visual Studio Marketplace에서 사용할 수 있으며](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) 다른 VS Code 확장과 동일한 방식으로 설치할 수 있습니다.

VS Code 확장 설치에 익숙하지 않은 경우, 다음 절차를 따르세요:

### 로그인

1. VS Code 활동 표시줄에서 **확장 기능** 선택
2. 확장 검색창에 "AI Toolkit" 입력
3. "AI Toolkit for Visual Studio code" 선택
4. <strong>설치</strong> 선택

이제 확장을 사용할 준비가 되었습니다!

GitHub 로그인이 필요하다는 메시지가 표시되면 계속하려면 "허용"을 클릭하세요. GitHub 로그인 페이지로 리디렉션됩니다.

로그인하고 절차를 따라 진행하세요. 완료되면 VS Code로 다시 리디렉션됩니다.

확장이 설치되면 활동 표시줄에 AI Toolkit 아이콘이 나타납니다.

사용 가능한 작업들을 살펴봅시다!

- <strong>리소스</strong>
- <strong>플레이그라운드</strong>

### 카탈로그에서 모델 다운로드

VS Code 사이드바에서 AI Toolkit을 실행하면 다음 옵션을 선택할 수 있습니다:

![AI toolkit model catalog](../../../../../translated_images/ko/AItoolkitmodel_catalog.7a7be6a7d8468d31.webp)

- <strong>모델 카탈로그</strong>에서 지원되는 모델 찾고 로컬에 다운로드
- <strong>모델 플레이그라운드</strong>에서 모델 추론 테스트
- <strong>모델 미세 조정</strong>에서 로컬 또는 원격으로 미세 조정 실행
- AI Toolkit 명령 팔레트를 통해 클라우드에 미세 조정 모델 배포
- 모델 평가

> [!NOTE]
>
> **GPU vs CPU**
>
> 모델 카드에는 모델 크기, 플랫폼 및 가속기 유형(CPU, GPU)이 표시됩니다. <strong>GPU가 하나 이상 있는 Windows 장치</strong>에서 최적의 성능을 위해서는 Windows만 지원하는 모델 버전을 선택하세요.
>
> 이렇게 하면 DirectML 가속기에 최적화된 모델을 사용할 수 있습니다.
>
> 모델 이름 형식은
>
> - `{model_name}-{accelerator}-{quantization}-{format}` 입니다.
>
> Windows 장치에 GPU가 있는지 확인하려면, <strong>작업 관리자</strong>를 열고 <strong>성능</strong> 탭을 선택하세요. GPU가 있다면 "GPU 0" 또는 "GPU 1" 같은 이름으로 표시됩니다.

### 플레이그라운드에서 모델 실행

모든 매개변수를 설정한 후 <strong>프로젝트 생성</strong>을 클릭하세요.

- 모델 다운로드 시작
- 모든 전제 조건 및 종속성 설치
- VS Code 작업 공간 생성

### 애플리케이션에서 REST API 사용

AI Toolkit은 포트 5272에서 로컬 REST API 웹 서버를 제공하며, [OpenAI 채팅 완성 형식](https://platform.openai.com/docs/api-reference/chat/create)을 사용합니다.

이를 통해 클라우드 AI 모델 서비스에 의존하지 않고도 로컬에서 애플리케이션을 테스트할 수 있습니다. 예를 들어 다음 JSON 파일은 요청 본문 구성 방법을 보여줍니다:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```

Postman(https://www.postman.com/)이나 CURL(클라이언트 URL) 도구를 사용하여 REST API를 테스트할 수 있습니다:

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```

### Python용 OpenAI 클라이언트 라이브러리 사용하기

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # API에 필요하지만 사용되지 않음
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```

### .NET용 Azure OpenAI 클라이언트 라이브러리 사용하기

NuGet을 사용하여 [.NET용 Azure OpenAI 클라이언트 라이브러리](https://www.nuget.org/packages/Azure.AI.OpenAI/)를 프로젝트에 추가하세요:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```

<strong>OverridePolicy.cs</strong>라는 C# 파일을 프로젝트에 추가하고 다음 코드를 붙여넣으세요:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```

다음으로, **Program.cs** 파일에 다음 코드를 붙여넣으세요:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
