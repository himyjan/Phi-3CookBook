## Getting Started with Phi-Family Models in Foundry Local

### Introduction to Foundry Local

Ang Foundry Local ay isang makapangyarihang solusyon para sa on-device AI inference na nagdadala ng enterprise-grade AI capabilities direkta sa iyong lokal na hardware. Gabayan ka ng tutorial na ito sa pag-setup at paggamit ng Phi-Family models gamit ang Foundry Local, na nag-aalok sa iyo ng kumpletong kontrol sa iyong AI workloads habang pinananatili ang privacy at binabawasan ang mga gastos.

Nagbibigay ang Foundry Local ng performance, privacy, customization, at cost advantages sa pamamagitan ng pagpapatakbo ng AI models nang lokal sa iyong device. Ito ay seamlessly na nag-iintegrate sa iyong umiiral na workflows at applications sa pamamagitan ng isang intuitive na CLI, SDK, at REST API.

### Why Choose Foundry Local?

Ang pag-unawa sa mga benepisyo ng Foundry Local ay tutulong sa iyo na gumawa ng may kaalamang desisyon tungkol sa iyong AI deployment strategy:

- **On-Device Inference:** Patakbuhin ang mga modelo nang lokal sa sarili mong hardware, binabawasan ang iyong mga gastos habang pinananatili ang lahat ng iyong data sa iyong device.

- **Model Customization:** Pumili mula sa mga preset na modelo o gamitin ang sarili mong mga modelo upang matugunan ang mga partikular na pangangailangan at use cases.

- **Cost Efficiency:** Alisin ang mga paulit-ulit na gastos sa cloud service sa pamamagitan ng paggamit ng iyong umiiral na hardware, na ginagawang mas accessible ang AI.

- **Seamless Integration:** Kumonekta sa iyong mga application sa pamamagitan ng SDK, API endpoints, o CLI, na may madaling scaling patungo sa Microsoft Foundry habang lumalaki ang iyong mga pangangailangan.

> **Getting Started Note:** Nakatuon ang tutorial na ito sa paggamit ng Foundry Local sa pamamagitan ng CLI at SDK interfaces. Matututuhan mo ang parehong paraan upang makatulong sa pagpili ng pinakamahusay na pamamaraan para sa iyong use case.

## Part 1: Setting Up Foundry Local CLI

### Step 1: Installation

Ang Foundry Local CLI ang iyong gateway sa pamamahala at pagpapatakbo ng AI models nang lokal. Magsimula tayo sa pag-install nito sa iyong system.

**Supported Platforms:** Windows at macOS

Para sa detalyadong mga tagubilin sa pag-install, mangyaring tingnan ang [opisyal na dokumentasyon ng Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Step 2: Exploring Available Models

Kapag na-install mo na ang Foundry Local CLI, maaari mong tuklasin kung anong mga modelo ang available para sa iyong use case. Ipapakita ng command na ito ang lahat ng suportadong mga modelo:


```bash
foundry model list
```

### Step 3: Understanding Phi Family Models

Nag-aalok ang Phi Family ng iba't ibang mga modelo na na-optimize para sa iba't ibang use cases at hardware configurations. Narito ang mga Phi models na available sa Foundry Local:

**Available Phi Models:** 

- **phi-3.5-mini** - Compact na modelo para sa mga pangunahing gawain
- **phi-3-mini-128k** - Extended context na bersyon para sa mas mahabang usapan
- **phi-3-mini-4k** - Standard context model para sa pangkalahatang paggamit
- **phi-4** - Advanced na modelo na may pinahusay na kakayahan
- **phi-4-mini** - Magaan na bersyon ng Phi-4
- **phi-4-mini-reasoning** - Espesyal na para sa mga kumplikadong reasoning tasks

> **Hardware Compatibility:** Ang bawat modelo ay maaaring i-configure para sa iba't ibang hardware acceleration (CPU, GPU) depende sa kakayahan ng iyong sistema.

### Step 4: Running Your First Phi Model

Magsimula tayo sa isang praktikal na halimbawa. Patatakbuhin natin ang `phi-4-mini-reasoning` model, na mahusay sa paglutas ng mga komplikadong problema nang sunud-sunod.


**Command to run the model:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **First-Time Setup:** Kapag nagpapatakbo ka ng modelo sa unang pagkakataon, awtomatikong ida-download ito ng Foundry Local sa iyong lokal na device. Nag-iiba ang oras ng pag-download depende sa bilis ng iyong network, kaya mangyaring maghintay nang matiwasay sa unang setup.

### Step 5: Testing the Model with a Real Problem

Ngayon subukan natin ang ating modelo gamit ang isang klasikong problemang lohikal upang makita kung paano nito ginagawa ang step-by-step reasoning:

**Example Problem:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Expected Behavior:** Ang modelo ay dapat hatiin ang problemang ito sa mga lohikal na hakbang, gamit ang katotohanan na ang mga pheasant ay may 2 paa at ang mga kuneho ay may 4 na paa upang malutas ang sistema ng mga ekwasyon.
