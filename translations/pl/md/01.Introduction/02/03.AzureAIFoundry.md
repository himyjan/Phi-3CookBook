## Rodzina Phi w Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) to zaufana platforma, która umożliwia programistom napędzanie innowacji i kształtowanie przyszłości z AI w bezpieczny, zabezpieczony i odpowiedzialny sposób.



[Microsoft Foundry](https://ai.azure.com) jest zaprojektowana dla programistów, aby:

- Budować aplikacje generatywne AI na platformie klasy korporacyjnej.
- Eksplorować, budować, testować i wdrażać, korzystając z najnowocześniejszych narzędzi AI i modeli ML, opartych na praktykach odpowiedzialnej AI.
- Współpracować z zespołem przez cały cykl życia tworzenia aplikacji.

Dzięki Microsoft Foundry możesz eksplorować szeroką gamę modeli, usług i możliwości oraz tworzyć aplikacje AI, które najlepiej służą Twoim celom. Platforma Microsoft Foundry ułatwia skalowanie, umożliwiając łatwe przekształcanie proof of concept w w pełni funkcjonalne aplikacje produkcyjne. Ciągły monitoring i udoskonalanie wspierają długoterminowy sukces.

### **Uruchamianie kodu Pythona wywołującego Microsoft Foundry Phi-4**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Uwzględnij wynik mowy, jeśli mowa jest włączona
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
