## Microsoft FoundryのPhiファミリー

[Microsoft Foundry](https://ai.azure.com)は、安全で安心かつ責任ある方法でAIを用いてイノベーションを推進し、未来を形作るための開発者向け信頼プラットフォームです。



[Microsoft Foundry](https://ai.azure.com)は開発者が次のことを行えるよう設計されています：

- エンタープライズグレードのプラットフォーム上で生成AIアプリケーションを構築する。
- 最先端のAIツールと機械学習モデルを使用して探索、構築、テスト、展開を行い、責任あるAIの実践に基づく。
- アプリケーション開発の全ライフサイクルを通じてチームで協力する。

Microsoft Foundryを利用することで、さまざまなモデル、サービス、機能を探索でき、目的に最適なAIアプリケーション構築を開始できます。Microsoft Foundryプラットフォームは、概念実証から本格的なプロダクションアプリケーションへのスケールアップを容易にします。継続的な監視と改良により長期的な成功を支援します。

![portal](../../../../../translated_images/ja/AIFoundryPorral.6b1094b101dd499e.webp)

Microsoft FoundryでAzure AOAIサービスを利用するだけでなく、Microsoft Foundryモデルカタログ内のサードパーティモデルも利用できます。これは、Microsoft FoundryをAIソリューションプラットフォームとして使用したい場合に適した選択肢です。

### **Microsoft Foundry Phi-4を呼び出すPythonコードの実行**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# 音声が有効な場合、音声結果を含める
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
