## Foundry LocalでのPhiファミリーモデルの始め方

### Foundry Localの紹介

Foundry Localは、エンタープライズグレードのAI機能を直接ローカルハードウェアに提供する強力なオンデバイスAI推論ソリューションです。このチュートリアルでは、Foundry LocalでPhiファミリーモデルをセットアップして使用する方法をご案内し、プライバシーを守りつつコストを削減し、AIワークロードの完全な制御を可能にします。

Foundry Localは、AIモデルをデバイス上でローカルに実行することでパフォーマンス、プライバシー、カスタマイズ性、およびコストの利点を提供します。直感的なCLI、SDK、およびREST APIを通じて既存のワークフローやアプリケーションにシームレスに統合します。


![arch](../../../../../translated_images/ja/foundry-local-arch.8823e321dd8258d7.webp)

### なぜFoundry Localを選ぶのか？

Foundry Localの利点を理解することで、AI展開戦略についてより良い判断が可能になります：

- **オンデバイス推論:** 自分のハードウェア上でモデルをローカルに実行し、すべてのデータをデバイス内に保持しながらコストを削減します。

- **モデルのカスタマイズ:** 既成モデルから選択するか、自分のモデルを使って特定の要件やユースケースに対応可能です。

- **コスト効率:** 既存のハードウェアを使用することでクラウドサービスの継続的な費用を排除し、AIをより手軽に利用できます。

- **シームレスな統合:** SDK、APIエンドポイント、またはCLIを通じてアプリケーションに接続し、必要に応じてMicrosoft Foundryへのスケールアップも容易です。

> **始めるにあたっての注意:** このチュートリアルではCLIとSDK双方の使用方法に焦点を当てています。どちらのアプローチも学び、ご自身のユースケースに最適な方法を選択してください。

## パート1: Foundry Local CLIのセットアップ

### ステップ1: インストール

Foundry Local CLIは、AIモデルをローカルで管理・実行するための入り口です。まずはシステムにインストールしましょう。

**対応プラットフォーム:** WindowsとmacOS

詳しいインストール手順は[公式Foundry Localドキュメント](https://github.com/microsoft/Foundry-Local/blob/main/README.md)を参照してください。

### ステップ2: 利用可能なモデルの確認

Foundry Local CLIをインストールしたら、ご自身のユースケースに利用可能なモデルを調べることができます。以下のコマンドで対応モデルをすべて表示します：


```bash
foundry model list
```

### ステップ3: Phiファミリーモデルの理解

Phiファミリーは、さまざまなユースケースやハードウェア構成に最適化されたモデル群を提供しています。Foundry Localで利用可能なPhiモデルは以下の通りです：

**利用可能なPhiモデル:** 

- **phi-3.5-mini** - 基本的なタスク用のコンパクトモデル
- **phi-3-mini-128k** - 長めの会話用の拡張コンテキスト版
- **phi-3-mini-4k** - 一般的な用途の標準コンテキストモデル
- **phi-4** - 改良された能力を持つ高度なモデル
- **phi-4-mini** - Phi-4の軽量版
- **phi-4-mini-reasoning** - 複雑な推論タスクに特化

> **ハードウェアの互換性:** 各モデルはご使用のシステムの能力に応じて異なるハードウェアアクセラレーション（CPU、GPU）向けに設定できます。

### ステップ4: 最初のPhiモデルの実行

実践的な例として、複雑な問題をステップごとに解くのに優れた`phi-4-mini-reasoning`モデルを実行してみましょう。


**モデル実行コマンド：**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **初回セットアップ:** モデルを初めて実行すると、Foundry Localが自動的にローカルデバイスにダウンロードします。ネットワーク速度によりダウンロード時間は異なりますので、初回はしばらくお待ちください。

### ステップ5: 実際の問題でのモデルテスト

続いて、クラシックな論理問題を使ってモデルがステップごとの推論をどのように行うかテストします。

**例題：**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**期待される動作:** この問題を論理的なステップに分解し、キジが2本足、ウサギが4本足であるという事実を使って連立方程式を解くことが求められます。
