# Famiglia Phi in AITK

[AI Toolkit per VS Code](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) semplifica lo sviluppo di app AI generativa riunendo strumenti e modelli AI all'avanguardia da Microsoft Foundry Catalog e altri cataloghi come Hugging Face. Potrai sfogliare il catalogo di modelli AI alimentato da GitHub Models e Microsoft Foundry Model Catalogs, scaricarli localmente o da remoto, fare il fine-tuning, testarli e usarli nella tua applicazione.

AI Toolkit Preview funziona in locale. Inferenza o fine-tuning locale, a seconda del modello selezionato, potresti aver bisogno di una GPU come NVIDIA CUDA GPU. Puoi eseguire direttamente i modelli GitHub con AITK anche.

### Installare AI Toolkit

AI Toolkit è distribuito come [Estensione di Visual Studio Code](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions), quindi devi prima installare [VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo) e scaricare AI Toolkit dal [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio).
[AI Toolkit è disponibile nel Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) e può essere installato come qualsiasi altra estensione di VS Code.

Se non hai familiarità con l'installazione di estensioni VS Code, segui questi passaggi:

### Accedi

1. Nella barra delle attività di VS Code seleziona **Estensioni**
1. Nella barra di ricerca delle estensioni digita "AI Toolkit"
1. Seleziona "AI Toolkit for Visual Studio code"
1. Seleziona **Installa**

Ora sei pronto per usare l'estensione!

Ti verrà chiesto di accedere a GitHub, quindi fai clic su "Consenti" per continuare. Sarai reindirizzato alla pagina di accesso di GitHub.

Accedi e segui i passaggi del processo. Dopo il completamento con successo, sarai reindirizzato su VS Code.

Una volta installata l'estensione vedrai l'icona di AI Toolkit apparire nella barra delle attività.

Esploriamo le azioni disponibili!

### Azioni disponibili

La barra laterale principale di AI Toolkit è organizzata in  

- **Modelli**
- **Risorse**
- **Playground**  
- **Fine-tuning**
- **Valutazione**

Sono disponibili nella sezione Risorse. Per iniziare seleziona **Catalogo Modelli**.

### Scaricare un modello dal catalogo

Avviando AI Toolkit dalla barra laterale di VS Code, puoi selezionare tra le seguenti opzioni:

![Catalogo modelli AI toolkit](../../../../../translated_images/it/AItoolkitmodel_catalog.7a7be6a7d8468d31.webp)

- Trova un modello supportato dal **Catalogo Modelli** e scaricalo localmente
- Testa l'inferenza del modello nel **Playground Modello**
- Esegui il fine-tuning locale o remoto nel **Fine-tuning Modello**
- Distribuisci modelli fine-tuning nel cloud tramite la palette dei comandi di AI Toolkit
- Valutazione modelli

> [!NOTE]
>
> **GPU vs CPU**
>
> Noterai che le schede modello mostrano la dimensione del modello, la piattaforma e il tipo di acceleratore (CPU, GPU). Per prestazioni ottimizzate su **dispositivi Windows che hanno almeno una GPU**, seleziona versioni del modello che mirano solo a Windows.
>
> Ciò garantisce un modello ottimizzato per l'acceleratore DirectML.
>
> I nomi dei modelli sono nel formato
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
>Per verificare se hai una GPU sul dispositivo Windows, apri **Task Manager** e poi seleziona la scheda **Prestazioni**. Se hai GPU, saranno elencate con nomi come "GPU 0" o "GPU 1".

### Usa l'API REST nella tua applicazione

AI Toolkit viene fornito con un server web REST API locale **sulla porta 5272** che usa il [formato OpenAI chat completions](https://platform.openai.com/docs/api-reference/chat/create).

Questo ti permette di testare la tua applicazione localmente senza dover dipendere da un servizio AI cloud. Per esempio, il seguente file JSON mostra come configurare il corpo della richiesta:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```

Puoi testare la REST API usando (per esempio) [Postman](https://www.postman.com/) o l'utilità CURL (Client URL):

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```

### Usare la libreria client OpenAI per Python

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # richiesto per l'API ma non utilizzato
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```

### Usare la libreria client Azure OpenAI per .NET

Aggiungi la [libreria client Azure OpenAI per .NET](https://www.nuget.org/packages/Azure.AI.OpenAI/) al tuo progetto usando NuGet:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```

Aggiungi un file C# chiamato **OverridePolicy.cs** al tuo progetto e incolla il seguente codice:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```

Quindi incolla il seguente codice nel tuo file **Program.cs**:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
