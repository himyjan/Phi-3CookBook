## Gia đình Phi trong Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) là một nền tảng tin cậy giúp các nhà phát triển thúc đẩy đổi mới và định hình tương lai với AI một cách an toàn, bảo mật và có trách nhiệm.

[Microsoft Foundry](https://ai.azure.com) được thiết kế cho các nhà phát triển để:

- Xây dựng các ứng dụng AI tạo sinh trên nền tảng cấp doanh nghiệp.
- Khám phá, xây dựng, thử nghiệm và triển khai bằng cách sử dụng các công cụ AI tiên tiến và mô hình ML, dựa trên các thực hành AI có trách nhiệm.
- Hợp tác với đội ngũ trong suốt vòng đời phát triển ứng dụng.

Với Microsoft Foundry, bạn có thể khám phá nhiều loại mô hình, dịch vụ và khả năng, và bắt đầu xây dựng các ứng dụng AI phù hợp nhất với mục tiêu của bạn. Nền tảng Microsoft Foundry hỗ trợ khả năng mở rộng để chuyển đổi các nguyên mẫu thành các ứng dụng sản xuất hoàn chỉnh một cách dễ dàng. Việc giám sát liên tục và tinh chỉnh hỗ trợ thành công lâu dài.

### **Chạy mã Python để gọi Microsoft Foundry Phi-4**

```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Bao gồm kết quả giọng nói nếu giọng nói được bật
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
