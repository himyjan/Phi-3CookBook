# Família Phi da Microsoft

Os modelos Phi são os Modelos de Linguagem Pequena (SLMs) mais capazes e custo-efetivos disponíveis, superando modelos do mesmo tamanho e do próximo tamanho em uma variedade de benchmarks de linguagem, raciocínio, codificação, áudio, visão e matemática. Esta versão expande a seleção de modelos de alta qualidade para os clientes, oferecendo escolhas mais práticas para compor e construir aplicações de IA generativa.

A Família Phi começou com o Phi-1 para geração de código Python, continuou com Phi-1.5 /2 baseado em conclusão de texto e chat, Phi-3-mini/pequeno/médio-instruct e Phi-3.5/4-mini-instruct, e se desenvolveu para Phi-3/3.5-vision para visão, Phi-4 baseado em forte raciocínio, e Phi-3.5-MoE para MoE, e agora o modelo multimodal completo Phi-4-multimodal. Por meio de conjuntos de dados de alta qualidade, o benchmark pode ser treinado para ser comparável a modelos com maiores parâmetros de treinamento.

## Exemplo de Seleção de Modelo

| | | | |
|-|-|-|-|
|Necessidade do Cliente|Tarefa|Começar com|Mais Detalhes|
|Necessita de um modelo que simplesmente resuma um fio de mensagens|Resumo de Conversa|Modelo de texto Phi-3 / 3.5|O fator decisivo aqui é que o cliente tem uma tarefa de linguagem bem definida e direta|
|Um aplicativo gratuito de tutor de matemática para crianças|Matemática e Raciocínio|Modelos de texto Phi-3 / 3.5 / 4|Por ser um aplicativo gratuito, os clientes querem uma solução que não lhes custe de forma recorrente|
|Câmera Autônoma de Patrulha de Carro|Análise de Visão|Phi-3 /3.5 -Visão ou Phi-4-multimodal|Necessita de uma solução que possa funcionar na borda sem internet|
|Quer construir um agente de reservas de viagem baseado em IA|Necessita planejamento complexo, chamada de função e orquestração|Modelos GPT|Necessita capacidade de planejar, chamar APIs para coletar informações e executar|
|Quer construir um copiloto para seus funcionários|RAG, múltiplos domínios, complexo e aberto|Modelos GPT + Família Phi |Cenário aberto, necessita de conhecimento mais amplo do mundo, por isso um modelo maior é mais adequado. Você precisa fracionar o conteúdo do conhecimento, talvez o SLM seja ideal para você |
