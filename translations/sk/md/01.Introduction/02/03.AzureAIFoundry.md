## Phi rodina v Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) je dôveryhodná platforma, ktorá umožňuje vývojárom poháňať inovácie a formovať budúcnosť pomocou AI bezpečným, zabezpečeným a zodpovedným spôsobom.

[Microsoft Foundry](https://ai.azure.com) je navrhnutý pre vývojárov, ktorí chcú:

- Vytvárať generatívne AI aplikácie na podnikovej platforme.
- Preskúmavať, stavať, testovať a nasadzovať pomocou najmodernejších AI nástrojov a ML modelov, zakotvených v zodpovedných praktikách AI.
- Spolupracovať s tímom počas celého životného cyklu vývoja aplikácie.

S Microsoft Foundry môžete preskúmať širokú škálu modelov, služieb a kapacít a začať budovať AI aplikácie, ktoré najlepšie slúžia vašim cieľom. Platforma Microsoft Foundry uľahčuje škálovanie pri transformácii dôkazov koncepcií na plnohodnotné produkčné aplikácie. Neustále monitorovanie a zdokonaľovanie podporuje dlhodobý úspech.

### **Spustenie Python kódu na volanie Microsoft Foundry Phi-4**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Zahrnúť výsledok reči, ak je reč povolená
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
