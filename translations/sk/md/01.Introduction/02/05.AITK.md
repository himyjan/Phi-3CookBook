# Rodina Phi v AITK

[AI Toolkit pre VS Code](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) zjednodušuje vývoj generatívnych AI aplikácií tým, že spája špičkové nástroje a modely pre vývoj AI od Microsoft Foundry Catalog a ďalších katalógov ako Hugging Face. Budete môcť prezerať katalóg AI modelov poháňaný GitHub Models a Microsoft Foundry Model Catalogs, sťahovať ich lokálne alebo vzdialene, doladiť, otestovať a používať vo svojej aplikácii.

AI Toolkit Preview bude bežať lokálne. Lokálny inference alebo doladenie závisí od vybraného modelu, môžete potrebovať GPU ako NVIDIA CUDA GPU. Modely GitHub môžete tiež spúšťať priamo pomocou AITK.

### Inštalácia AI Toolkitu

AI Toolkit je dodávaný ako [rozšírenie Visual Studio Code](https://code.visualstudio.com/docs/setup/additional-components#_vs-code-extensions), takže najskôr potrebujete nainštalovať [VS Code](https://code.visualstudio.com/docs/setup/windows?WT.mc_id=aiml-137032-kinfeylo), a stiahnuť AI Toolkit z [VS Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio).
[AI Toolkit je dostupný v Visual Studio Marketplace](https://marketplace.visualstudio.com/items?itemName=ms-windows-ai-studio.windows-ai-studio) a môžete ho nainštalovať ako ktorékoľvek iné rozšírenie VS Code.

Ak nemáte skúsenosti s inštaláciou rozšírení pre VS Code, postupujte podľa týchto krokov:

### Prihlásenie

1. V paneli Aktivít vo VS Code vyberte **Rozšírenia**
1. Do vyhľadávacieho poľa rozšírení napíšte "AI Toolkit"
1. Vyberte "AI Toolkit for Visual Studio code"
1. Kliknite na **Inštalovať**

Teraz ste pripravení používať rozšírenie!

Budete vyzvaný na prihlásenie do GitHubu, kliknite na "Povoliť" pre pokračovanie. Budete presmerovaný na prihlasovaciu stránku GitHub.

Prihláste sa a postupujte podľa krokov procesu. Po úspešnom dokončení budete presmerovaný späť do VS Code.

Po inštalácii rozšírenia uvidíte ikonu AI Toolkit v paneli Aktivít.

Poďme preskúmať dostupné akcie!

### Dostupné akcie

Hlavný bočný panel AI Toolkitu je rozdelený na

- **Modely**
- **Zdroje**
- **Ihrisko**  
- **Doladenie**
- **Vyhodnotenie**

Dostupné v sekcii Zdroje. Pre začiatok vyberte **Katalóg modelov**.

> [!NOTE]
>
> **GPU vs CPU**
>
> Všimnete si, že kartičky modelov ukazujú veľkosť modelu, platformu a typ akcelerátora (CPU, GPU). Pre optimalizovaný výkon na **Windows zariadeniach, ktoré majú aspoň jednu GPU**, vyberte modely určené len pre Windows.
>
> To zabezpečí model optimalizovaný pre akcelerátor DirectML.
>
> Názvy modelov majú formát
>
> - `{model_name}-{accelerator}-{quantization}-{format}`.
>
> Pre kontrolu, či máte GPU na vašom zariadení s Windows, otvorte **Správcu úloh** a vyberte kartu **Výkon**. Ak máte GPU, budú zobrazené pod názvami ako "GPU 0" alebo "GPU 1".

### Používanie REST API vo vašej aplikácii

AI Toolkit prichádza s lokálnym REST API web serverom **na porte 5272**, ktorý používa [formát OpenAI chat completions](https://platform.openai.com/docs/api-reference/chat/create).

To vám umožňuje testovať vašu aplikáciu lokálne bez potreby spoliehať sa na cloudovú AI službu. Napríklad nasledujúci JSON súbor ukazuje, ako nakonfigurovať telo požiadavky:

```json
{
    "model": "Phi-4",
    "messages": [
        {
            "role": "user",
            "content": "what is the golden ratio?"
        }
    ],
    "temperature": 0.7,
    "top_p": 1,
    "top_k": 10,
    "max_tokens": 100,
    "stream": true
}
```

REST API môžete testovať napríklad pomocou [Postman](https://www.postman.com/) alebo CURL (Client URL) nástroja:

```bash
curl -vX POST http://127.0.0.1:5272/v1/chat/completions -H 'Content-Type: application/json' -d @body.json
```

### Použitie OpenAI knižnice pre Python

```python
from openai import OpenAI

client = OpenAI(
    base_url="http://127.0.0.1:5272/v1/", 
    api_key="x" # vyžaduje sa pre API, ale nepoužíva sa
)

chat_completion = client.chat.completions.create(
    messages=[
        {
            "role": "user",
            "content": "what is the golden ratio?",
        }
    ],
    model="Phi-4",
)

print(chat_completion.choices[0].message.content)
```

### Použitie Azure OpenAI knižnice pre .NET

Pridajte [Azure OpenAI knižnicu pre .NET](https://www.nuget.org/packages/Azure.AI.OpenAI/) do projektu pomocou NuGet:

```bash
dotnet add {project_name} package Azure.AI.OpenAI --version 1.0.0-beta.17
```

Pridajte do projektu súbor C# s názvom **OverridePolicy.cs** a vložte nasledujúci kód:

```csharp
// OverridePolicy.cs
using Azure.Core.Pipeline;
using Azure.Core;

internal partial class OverrideRequestUriPolicy(Uri overrideUri)
    : HttpPipelineSynchronousPolicy
{
    private readonly Uri _overrideUri = overrideUri;

    public override void OnSendingRequest(HttpMessage message)
    {
        message.Request.Uri.Reset(_overrideUri);
    }
}
```

Ďalej skopírujte nasledujúci kód do vášho súboru **Program.cs**:

```csharp
// Program.cs
using Azure.AI.OpenAI;

Uri localhostUri = new("http://localhost:5272/v1/chat/completions");

OpenAIClientOptions clientOptions = new();
clientOptions.AddPolicy(
    new OverrideRequestUriPolicy(localhostUri),
    Azure.Core.HttpPipelinePosition.BeforeTransport);
OpenAIClient client = new(openAIApiKey: "unused", clientOptions);

ChatCompletionsOptions options = new()
{
    DeploymentName = "Phi-4",
    Messages =
    {
        new ChatRequestSystemMessage("You are a helpful assistant. Be brief and succinct."),
        new ChatRequestUserMessage("What is the golden ratio?"),
    }
};

StreamingResponse<StreamingChatCompletionsUpdate> streamingChatResponse
    = await client.GetChatCompletionsStreamingAsync(options);

await foreach (StreamingChatCompletionsUpdate chatChunk in streamingChatResponse)
{
    Console.Write(chatChunk.ContentUpdate);
}
```
