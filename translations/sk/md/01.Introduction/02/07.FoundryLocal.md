## Začíname s modelmi Phi-Family vo Foundry Local

### Úvod do Foundry Local

Foundry Local je výkonné riešenie pre AI inferenciu priamo na zariadení, ktoré prináša AI schopnosti úrovne podnikov priamo na váš lokálny hardvér. Tento návod vás prevedie nastavením a používaním modelov Phi-Family vo Foundry Local, pričom vám ponúkne úplnú kontrolu nad vašimi AI pracovnými záťažami, zároveň zachováva súkromie a znižuje náklady.

Foundry Local poskytuje výhody výkonu, súkromia, prispôsobenia a nákladovej efektívnosti tým, že spúšťa AI modely lokálne na vašom zariadení. Integruje sa hladko do vašich existujúcich pracovných tokov a aplikácií prostredníctvom intuitívneho CLI, SDK a REST API.


![arch](../../../../../translated_images/sk/foundry-local-arch.8823e321dd8258d7.webp)

### Prečo zvoliť Foundry Local?

Pochopenie výhod Foundry Local vám pomôže urobiť informované rozhodnutia o vašej stratégii nasadenia AI:

- **Inferencia na zariadení:** Spúšťajte modely lokálne na vlastnom hardvéri, znižujte náklady a zároveň udržujte všetky dáta na vašom zariadení.

- **Prispôsobenie modelu:** Vyberte si z prednastavených modelov alebo použite vlastné, ktoré spĺňajú špecifické požiadavky a prípady použitia.

- **Nákladová efektívnosť:** Eliminujte opakujúce sa náklady na cloudové služby použitím existujúceho hardvéru, čím sprístupníte AI širšiemu okruhu používateľov.

- **Bezproblémová integrácia:** Spojte sa s vašimi aplikáciami prostredníctvom SDK, API endpointov alebo CLI, s ľahkým škálovaním na Microsoft Foundry podľa vašich potrieb.

> **Poznámka pre začiatočníkov:** Tento tutoriál sa zameriava na používanie Foundry Local cez rozhrania CLI a SDK. Naučíte sa oba prístupy, aby ste si mohli vybrať najlepší spôsob pre váš prípad použitia.

## Časť 1: Nastavenie Foundry Local CLI

### Krok 1: Inštalácia

Foundry Local CLI je vašou bránou na správu a spúšťanie AI modelov lokálne. Začneme jeho inštaláciou na vašom systéme.

**Podporované platformy:** Windows a macOS

Pre podrobné inštrukcie inštalácie si prosím prečítajte [oficiálnu dokumentáciu Foundry Local](https://github.com/microsoft/Foundry-Local/blob/main/README.md).

### Krok 2: Preskúmanie dostupných modelov

Akonáhle máte Foundry Local CLI nainštalované, môžete zistiť, ktoré modely sú k dispozícii pre váš prípad použitia. Tento príkaz vám zobrazí všetky podporované modely:


```bash
foundry model list
```

### Krok 3: Pochopenie modelov Phi Family

Phi Family ponúka rad modelov optimalizovaných pre rôzne prípady použitia a hardvérové konfigurácie. Tu sú modely Phi dostupné vo Foundry Local:

**Dostupné Phi modely:** 

- **phi-3.5-mini** - Kompaktný model pre základné úlohy
- **phi-3-mini-128k** - Rozšírená verzia s kontextom pre dlhšie rozhovory
- **phi-3-mini-4k** - Štandardný model s kontextom pre všeobecné použitie
- **phi-4** - Pokročilý model so zlepšenými schopnosťami
- **phi-4-mini** - Odľahčená verzia Phi-4
- **phi-4-mini-reasoning** - Špecializovaný na komplexné úlohy uvažovania

> **Kompatibilita hardvéru:** Každý model môže byť konfigurovaný pre rôzne hardvérové akcelerácie (CPU, GPU) podľa možností vášho systému.

### Krok 4: Spustenie prvého modelu Phi

Začnime praktickým príkladom. Spustíme model `phi-4-mini-reasoning`, ktorý vyniká v riešení komplexných problémov krok za krokom.


**Príkaz na spustenie modelu:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```

> **Nastavenie pri prvom použití:** Pri prvom spustení modelu si ho Foundry Local automaticky stiahne do vášho zariadenia. Čas sťahovania závisí od rýchlosti vašej siete, preto prosím buďte trpezliví počas počiatočného nastavenia.

### Krok 5: Testovanie modelu na reálnom probléme

Teraz otestujeme náš model na klasickom logickom probléme, aby sme videli, ako postupuje krok za krokom pri uvažovaní:

**Príklad problému:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```

**Očakávané správanie:** Model by mal tento problém rozložiť na logické kroky, využívajúc skutočnosť, že bažanty majú 2 nohy a králiky 4 nohy, aby vyriešil sústavu rovníc.
