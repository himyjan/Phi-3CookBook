# Podpora hardvéru Phi

Microsoft Phi bol optimalizovaný pre ONNX Runtime a podporuje Windows DirectML. Funguje dobre naprieč rôznymi typmi hardvéru, vrátane GPU, CPU a dokonca aj mobilných zariadení.

## Hardvér zariadenia
Konkrétne podporovaný hardvér zahŕňa:

- GPU SKU: RTX 4090 (DirectML)
- GPU SKU: 1 A100 80GB (CUDA)
- CPU SKU: Standard F64s v2 (64 vCPU, 128 GiB pamäte)

## Mobilný SKU

- Android - Samsung Galaxy S21
- Apple iPhone 14 alebo vyšší procesor A16/A17

## Špecifikácia hardvéru Phi

- Minimálna požadovaná konfigurácia.
- Windows: GPU schopné DirectX 12 a minimálne 4GB kombinovanej RAM

CUDA: NVIDIA GPU s výpočtovou schopnosťou >= 7.02

## Spúšťanie onnxruntime na viacerých GPU

Momentálne dostupné Phi ONNX modely sú len pre 1 GPU. Je možné podporovať multi-gpu pre Phi model, ale ORT s 2 GPU nezaručuje, že poskytne väčší priepust ako 2 inštancie ORT. Pozrite si prosím [ONNX Runtime](https://onnxruntime.ai/) pre najnovšie aktualizácie.

Na [Build 2024 tím GenAI ONNX](https://youtu.be/WLW4SE8M9i8?si=EtG04UwDvcjunyfC) oznámil, že povolili multi-instance namiesto multi-gpu pre Phi modely.

V súčasnosti to umožňuje spustiť jednu inštanciu onnxruntime alebo onnxruntime-genai s premennou prostredia CUDA_VISIBLE_DEVICES takto.

```Python
CUDA_VISIBLE_DEVICES=0 python infer.py
CUDA_VISIBLE_DEVICES=1 python infer.py
```

Neváhajte ďalej skúmať Phi na [Microsoft Foundry](https://ai.azure.com)
