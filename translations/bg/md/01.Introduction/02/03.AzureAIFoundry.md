## Phi Family в Microsoft Foundry

[Microsoft Foundry](https://ai.azure.com) е доверена платформа, която дава възможност на разработчиците да стимулират иновациите и да оформят бъдещето с ИИ по безопасен, сигурен и отговорен начин.



[Microsoft Foundry](https://ai.azure.com) е проектирана за разработчици, за да:

- Създават генеративни AI приложения на корпоративно ниво.
- Изследват, изграждат, тестват и внедряват, използвайки най-съвременни AI инструменти и ML модели, основани на отговорни AI практики.
- Сътрудничат в екип през целия жизнен цикъл на разработка на приложения.

С Microsoft Foundry можете да разглеждате разнообразие от модели, услуги и възможности и да започнете да изграждате AI приложения, които най-добре обслужват вашите цели. Платформата Microsoft Foundry улеснява мащабирането за превръщане на доказателства за концепции в пълноценни производствени приложения с лекота. Непрекъснатият мониторинг и усъвършенстване подкрепят дългосрочния успех.

### **Изпълнение на Python код за извикване на Microsoft Foundry Phi-4**


```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# Включете резултат от реч, ако речта е активирана
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
