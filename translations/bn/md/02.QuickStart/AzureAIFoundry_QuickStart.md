# **Microsoft Foundry-এ Phi-3 ব্যবহার করা**

Generative AI-এর উন্নয়নের সাথে, আমরা আশা করি একটি একক প্ল্যাটফর্ম ব্যবহার করে বিভিন্ন LLM এবং SLM, এন্টারপ্রাইজ ডেটা ইন্টিগ্রেশন, ফাইন-টিউনিং/RAG অপারেশন এবং LLM ও SLM একত্রিত করার পরে বিভিন্ন এন্টারপ্রাইজ ব্যবসার মূল্যায়ন ইত্যাদি পরিচালনা করতে পারি, যাতে জেনেরেটিভ AI স্মার্ট অ্যাপ্লিকেশনগুলো আরও ভালভাবে বাস্তবায়িত হতে পারে। [Microsoft Foundry](https://ai.azure.com) একটি এন্টারপ্রাইজ-লেভেলের জেনেরেটিভ AI অ্যাপ্লিকেশন প্ল্যাটফর্ম।

![aistudo](../../../../translated_images/bn/aifoundry_home.f28a8127c96c7d93.webp)

Microsoft Foundry-এর মাধ্যমে, আপনি বড় ভাষার মডেল (LLM) প্রতিক্রিয়াগুলো মূল্যায়ন করতে এবং প্রম্পট ফ্লো সহ প্রম্পট অ্যাপ্লিকেশন উপাদানগুলোকে সমন্বয় করতে পারেন আরও ভালো পারফরম্যান্সের জন্য। এই প্ল্যাটফর্মটি প্রুফ অফ কনসেপ্ট থেকে সম্পূর্ণ উৎপাদনে রূপান্তরের জন্য স্কেলেবিলিটি সহজ করে। ধারাবাহিক মনিটরিং এবং সূক্ষ্মকরণের মাধ্যমে দীর্ঘমেয়াদী সফলতা নিশ্চিত করে।

আমরা সহজ ধাপে Microsoft Foundry-এ দ্রুত Phi-3 মডেল মোতায়েন করতে পারি, এবং তারপর Microsoft Foundry ব্যবহার করে Phi-3 সংক্রান্ত Playground/Chat, ফাইন-টিউনিং, মূল্যায়ন ও অন্যান্য সম্পর্কিত কাজ সম্পন্ন করতে পারি।

## **1. প্রস্তুতি**

যদি আপনার মেশিনে ইতিমধ্যেই [Azure Developer CLI](https://learn.microsoft.com/azure/developer/azure-developer-cli/overview?WT.mc_id=aiml-138114-kinfeylo) ইনস্টল করা থাকে, তবে এই টেমপ্লেট ব্যবহার করা খুব সহজ, একটি নতুন ডিরেক্টরিতে নিচের কমান্ডটি চালান।

## ম্যানুয়াল তৈরি

Microsoft Foundry প্রকল্প এবং হাব তৈরি একটি দুর্দান্ত উপায় আপনার AI কাজ সংগঠিত এবং পরিচালনা করার জন্য। শুরু করার জন্য ধাপে ধাপে গাইড এখানে:

### Microsoft Foundry-এ একটি প্রকল্প তৈরি করা

1. **Microsoft Foundry-এ যান**: Microsoft Foundry পোর্টালে সাইন ইন করুন।
2. **একটি প্রকল্প তৈরি করুন**:
   - যদি আপনি একটি প্রকল্পের মধ্যে থাকেন, পৃষ্ঠার উপরে বাম দিকে "Microsoft Foundry" নির্বাচন করে হোম পেজে যান।
   - "+ Create project" নির্বাচন করুন।
   - প্রকল্পের জন্য একটি নাম দিন।
   - যদি আপনার একটি হাব থাকে, এটি ডিফল্টরূপে নির্বাচিত থাকবে। যদি আপনার একাধিক হাবে অ্যাক্সেস থাকে, তাহলে ড্রপডাউনে থেকে অন্য একটি হাব নির্বাচন করতে পারেন। নতুন একটি হাব তৈরি করতে চাইলে "Create new hub" নির্বাচন করে একটি নাম দিন।
   - "Create" নির্বাচন করুন।

### Microsoft Foundry-এ একটি হাব তৈরি করা

1. **Microsoft Foundry-এ যান**: আপনার Azure অ্যাকাউন্ট দিয়ে সাইন ইন করুন।
2. **একটি হাব তৈরি করুন**:
   - বাম মেনু থেকে Management center নির্বাচন করুন।
   - "All resources" নির্বাচন করুন, তারপর "+ New project" এর পাশের ডাউন অ্যারোতে ক্লিক করে "+ New hub" নির্বাচন করুন।
   - "Create a new hub" ডায়ালগে, আপনার হাবের জন্য একটি নাম দিন (যেমন contoso-hub) এবং অন্যান্য ফিল্ড প্রয়োজন অনুযায়ী পরিবর্তন করুন।
   - "Next" নির্বাচন করুন, তথ্যগুলো রিভিউ করুন, তারপর "Create" নির্বাচন করুন।

বিস্তারিত নির্দেশনার জন্য, আপনি অফিসিয়াল [Microsoft ডকুমেন্টেশন](https://learn.microsoft.com/azure/ai-studio/how-to/create-projects) দেখতে পারেন।

সফলভাবে তৈরি হওয়ার পরে, আপনি [ai.azure.com](https://ai.azure.com/) থেকে তৈরি করা স্টুডিও অ্যাক্সেস করতে পারেন।

একটি AI Foundry-তে একাধিক প্রকল্প থাকতে পারে। AI Foundry-তে একটি প্রকল্প তৈরি করে প্রস্তুতি নিন।

Microsoft Foundry [QuickStarts](https://learn.microsoft.com/azure/ai-studio/quickstarts/get-started-code) তৈরি করুন।

## **2. Microsoft Foundry-এ একটি Phi মডেল মোতায়েন করা**

প্রকল্পের Explore অপশন ক্লিক করে Model Catalog-তে প্রবেশ করুন এবং Phi-3 নির্বাচন করুন।

Phi-3-mini-4k-instruct নির্বাচন করুন।

Phi-3-mini-4k-instruct মডেল মোতায়েন করার জন্য 'Deploy' ক্লিক করুন।

> [!NOTE]
>
> মোতায়েনের সময় আপনি কম্পিউটিং পাওয়ার নির্বাচন করতে পারেন।

## **3. Microsoft Foundry-এ Playground Chat Phi**

মোতায়েন পৃষ্ঠায় যান, Playground নির্বাচন করুন এবং Microsoft Foundry এর Phi-3-এর সাথে চ্যাট করুন।

## **4. Microsoft Foundry থেকে মডেল মোতায়েন করা**

Azure Model Catalog থেকে একটি মডেল মোতায়েন করতে, আপনি নিম্নলিখিত ধাপগুলি অনুসরণ করতে পারেন:

- Microsoft Foundry-তে সাইন ইন করুন।
- Microsoft Foundry মডেল ক্যাটালগ থেকে মোতায়েন করতে ইচ্ছুক মডেল নির্বাচন করুন।
- মডেলের Details পৃষ্ঠায়, Deploy নির্বাচন করুন এবং তারপর Serverless API with Azure AI Content Safety নির্বাচন করুন।
- আপনার মডেলগুলো মোতায়েন করার জন্য প্রকল্প নির্বাচন করুন। Serverless API অফার ব্যবহারের জন্য, আপনার ওয়ার্কস্পেসটি East US 2 অথবা Sweden Central অঞ্চলের হতে হবে। আপনি Deployment এর নাম কাস্টমাইজ করতে পারবেন।
- Deployment wizard-এ, Pricing and terms নির্বাচন করে মূল্য এবং ব্যবহারের শর্তাবলী সম্পর্কে জানুন।
- Deploy নির্বাচন করুন। মোতায়েন প্রস্তুত হওয়া পর্যন্ত অপেক্ষা করুন এবং আপনাকে Deployments পৃষ্ঠায় পুনঃনির্দেশিত করা হবে।
- মডেলের সাথে ইন্টারঅ্যাক্ট শুরু করতে Open in playground নির্বাচন করুন।
- Deployments পৃষ্ঠায় ফিরে যান, মোতায়েন নির্বাচন করুন, এবং endpoint এর Target URL এবং Secret Key নোট করুন, যা আপনি মোতায়েন কল করতে এবং সম্পূর্ণতা তৈরি করতে ব্যবহার করতে পারবেন।
- আপনি সবসময় Build ট্যাবে গিয়ে Components সেকশনের Deployments নির্বাচন করে endpoint এর বিবরণ, URL এবং access key পেতে পারেন।

> [!NOTE]
> এই ধাপগুলি সম্পাদনের জন্য আপনার অ্যাকাউন্টে Resource Group এ Azure AI Developer রোল পারমিশন থাকতে হবে।

## **5. Microsoft Foundry-এ Phi API ব্যবহার করা**

আপনি https://{Your project name}.region.inference.ml.azure.com/swagger.json এই URL-এ Postman GET ব্যবহার করে Key-এর সঙ্গে একত্রিত করে প্রদত্ত ইন্টারফেসগুলো সম্পর্কে জানতে পারেন।

আপনি খুব সহজেই রিকোয়েস্ট প্যারামিটার এবং রেসপন্স প্যারামিটার পেতে পারবেন।
