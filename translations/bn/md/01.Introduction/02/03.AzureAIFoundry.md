## মাইক্রোসফ্ট ফাউন্ড্রিতে ফাই পরিবার

[Microsoft Foundry](https://ai.azure.com) একটি বিশ্বাসযোগ্য প্ল্যাটফর্ম যা ডেভেলপারদের নিরাপদ, সুরক্ষিত এবং দায়বদ্ধ উপায়ে এআই দিয়ে উদ্ভাবন চালানোর এবং ভবিষ্যত গড়ার ক্ষমতা দেয়।

[Microsoft Foundry](https://ai.azure.com) ডেভেলপারদের জন্য ডিজাইন করা হয়েছে:

- একটি এন্টারপ্রাইজ-গ্রেড প্ল্যাটফর্মে জেনারেটিভ এআই অ্যাপ্লিকেশন তৈরি করতে।
- দায়বদ্ধ এআই অনুশীলনের ভিত্তিতে আধুনিক এআই সরঞ্জাম এবং এমএল মডেল ব্যবহার করে অন্বেষণ, গঠন, পরীক্ষা এবং মোতায়েন করতে।
- অ্যাপ্লিকেশন উন্নয়নের পূর্ণ জীবনচক্রের জন্য একটি দলের সাথে সহযোগিতা করতে।

মাইক্রোসফ্ট ফাউন্ড্রির মাধ্যমে, আপনি বিস্তৃত ধরণের মডেল, পরিষেবা এবং সক্ষমতা অন্বেষণ করতে পারবেন এবং এমন এআই অ্যাপ্লিকেশন তৈরি করতে পারবেন যা আপনার লক্ষ্যগুলি সর্বোত্তম সেবা দেয়। মাইক্রোসফ্ট ফাউন্ড্রি প্ল্যাটফর্ম প্রুফ অফ কনসেপ্টকে সহজেই পূর্ণাঙ্গ উৎপাদন অ্যাপ্লিকেশনে রূপান্তরের জন্য স্কেলেবিলিটি সুবিধা প্রদান করে। দীর্ঘমেয়াদী সফলতার জন্য অবিচ্ছিন্ন নজরদারি এবং পরিমার্জন সমর্থিত।

![portal](../../../../../translated_images/bn/AIFoundryPorral.6b1094b101dd499e.webp)

মাইক্রোসফ্ট ফাউন্ড্রিতে Azure AOAI সার্ভিস ব্যবহার করার পাশাপাশি, আপনি মাইক্রোসফ্ট ফাউন্ড্রি মডেল ক্যাটালগে তৃতীয় পক্ষের মডেলগুলোও ব্যবহার করতে পারেন। যদি আপনি মাইক্রোসফ্ট ফাউন্ড্রিকে আপনার এআই সমাধান প্ল্যাটফর্ম হিসেবে ব্যবহার করতে চান, তবে এটি একটি ভালো বিকল্প।

আমরা মাইক্রোসফ্ট ফাউন্ড্রিতে মডেল ক্যাটালগের মাধ্যমে দ্রুত ফাই পরিবার মডেলগুলোর ডিপ্লয়মেন্ট করতে পারি

### **মাইক্রোসফ্ট ফাউন্ড্রি Phi-4 কল করতে পাইথন কোড চালানো**

```python

import os  
import base64
from openai import AzureOpenAI  
from azure.identity import DefaultAzureCredential, get_bearer_token_provider  
        
endpoint = os.getenv("ENDPOINT_URL", "Your Azure AOAI Service Endpoint")  
deployment = os.getenv("DEPLOYMENT_NAME", "Phi-4")  
      
token_provider = get_bearer_token_provider(  
    DefaultAzureCredential(),  
    "https://cognitiveservices.azure.com/.default"  
)  
  
client = AzureOpenAI(  
    azure_endpoint=endpoint,  
    azure_ad_token_provider=token_provider,  
    api_version="2024-05-01-preview",  
)  
  

chat_prompt = [
    {
        "role": "system",
        "content": "You are an AI assistant that helps people find information."
    },
    {
        "role": "user",
        "content": "can you introduce yourself"
    }
] 
    
# বক্তৃতা সক্ষম থাকলে বক্তৃতার ফলাফল অন্তর্ভুক্ত করুন
messages = chat_prompt 

completion = client.chat.completions.create(  
    model=deployment,  
    messages=messages,
    max_tokens=800,  
    temperature=0.7,  
    top_p=0.95,  
    frequency_penalty=0,  
    presence_penalty=0,
    stop=None,  
    stream=False  
)  
  
print(completion.to_json())  

```
