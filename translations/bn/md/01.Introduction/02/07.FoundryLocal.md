## Foundry Local-এ Phi-Family মডেল দিয়ে শুরু করা

### Foundry Local পরিচিতি

Foundry Local একটি শক্তিশালী অন-ডিভাইস AI ইনফারেন্স সমাধান যা এন্টারপ্রাইজ-গ্রেড AI ক্ষমতা সরাসরি আপনার স্থানীয় হার্ডওয়্যারে নিয়ে আসে। এই টিউটোরিয়ালে আপনাকে Foundry Local এর সাথে Phi-Family মডেল সেটআপ এবং ব্যবহারের মাধ্যমে গাইড করা হবে, যা আপনাকে আপনার AI কাজের উপর সম্পূর্ণ নিয়ন্ত্রণ প্রদান করে, গোপনীয়তা রক্ষা করে এবং খরচ কমায়।

Foundry Local আপনার ডিভাইসে স্থানীয়ভাবে AI মডেল চালানোর মাধ্যমে কর্মক্ষমতা, গোপনীয়তা, কাস্টমাইজেশন, এবং খরচের সুবিধা প্রদান করে। এটি একটি সহজবোধ্য CLI, SDK, এবং REST API এর মাধ্যমে আপনার বিদ্যমান ওয়ার্কফ্লো এবং অ্যাপ্লিকেশনগুলোর সাথে নির্বিঘ্নে সংযোগ স্থাপন করে।

![arch](../../../../../translated_images/bn/foundry-local-arch.8823e321dd8258d7.webp)

### কেন Foundry Local বেছে নেবেন?

Foundry Local এর সুবিধাগুলো বুঝে নেওয়া আপনাকে আপনার AI ডিপ্লয়মেন্ট কৌশল সম্পর্কে সচেতন সিদ্ধান্ত নিতে সাহায্য করবে:

- **অন-ডিভাইস ইনফারেন্স:** আপনার নিজস্ব হার্ডওয়্যারে মডেলগুলি স্থানীয়ভাবে চালান, খরচ কমান এবং সমস্ত ডেটা আপনার ডিভাইসে রাখুন।

- **মডেল কাস্টমাইজেশন:** নির্দিষ্ট প্রয়োজন এবং ব্যবহারের জন্য প্রিসেট মডেল থেকে বাছাই করুন অথবা আপনার নিজস্ব মডেল ব্যবহার করুন।

- **খরচ সাশ্রয়:** আপনার বিদ্যমান হার্ডওয়্যার ব্যবহার করে পুনরাবৃত্তিমান ক্লাউড সার্ভিস খরচ বাতিল করুন, যাতে AI আরও প্রবেশযোগ্য হয়।

- **নিরবিচ্ছিন্ন একীভূতকরণ:** SDK, API এন্ডপয়েন্ট, বা CLI এর মাধ্যমে আপনার অ্যাপ্লিকেশনের সাথে সংযোগ করুন, এবং প্রয়োজনে Microsoft Foundry তে সহজেই স্কেল আপ করুন।

> **শুরু করার নোট:** এই টিউটোরিয়ালটি Foundry Local CLI এবং SDK ইন্টারফেস ব্যবহার করার ওপর কেন্দ্রিক। আপনি উভয় উপায় সম্পর্কে জানবেন, যাতে আপনার ব্যবহার পরিস্থিতির জন্য সেরা পদ্ধতি বেছে নিতে পারেন।

## অংশ ১: Foundry Local CLI সেটআপ করা

### পদক্ষেপ ১: ইনস্টলেশন

Foundry Local CLI হল স্থানীয়ভাবে AI মডেল পরিচালনা এবং চালনার আপনার প্রবেশপথ। চলুন এটি আপনার সিস্টেমে ইনস্টল করি।

**সমর্থিত প্ল্যাটফর্ম:** Windows এবং macOS

বিস্তারিত ইনস্টলেশন নির্দেশিকার জন্য, দয়া করে [আধিকারিক Foundry Local ডকুমেন্টেশন](https://github.com/microsoft/Foundry-Local/blob/main/README.md) দেখুন।

### পদক্ষেপ ২: উপলব্ধ মডেলগুলো অন্বেষণ

Foundry Local CLI ইনস্টল করার পরে, আপনি জানতে পারবেন আপনার ব্যবহারের জন্য কোন মডেলগুলো উপলব্ধ। এই কমান্ডটি আপনাকে সমস্ত সমর্থিত মডেল দেখাবে:

```bash
foundry model list
```


### পদক্ষেপ ৩: Phi Family মডেলগুলো বোঝা

Phi Family বিভিন্ন ব্যবহারের ক্ষেত্র এবং হার্ডওয়্যার কনফিগারেশনের জন্য অপ্টিমাইজড মডেল সরবরাহ করে। Foundry Local-এ উপলব্ধ Phi মডেলগুলো হল:

**উপলব্ধ Phi মডেলগুলো:** 

- **phi-3.5-mini** - মৌলিক কাজের জন্য কম্প্যাক্ট মডেল
- **phi-3-mini-128k** - দীর্ঘ আলাপের জন্য বিস্তৃত প্রসঙ্গ সংস্করণ
- **phi-3-mini-4k** - সাধারণ ব্যবহারের জন্য স্ট্যান্ডার্ড প্রসঙ্গ মডেল
- **phi-4** - উন্নত ক্ষমতাসম্পন্ন মডেল
- **phi-4-mini** - Phi-4 এর হালকা সংস্করণ
- **phi-4-mini-reasoning** - জটিল যুক্তির কাজের জন্য বিশেষায়িত

> **হার্ডওয়্যার কম্প্যাটিবিলিটি:** প্রতিটি মডেল আপনার সিস্টেম শক্তির ওপর নির্ভর করে বিভিন্ন হার্ডওয়্যার অ্যাকসিলারেশন (CPU, GPU) জন্য কনফিগার করা যেতে পারে।

### পদক্ষেপ ৪: আপনার প্রথম Phi মডেল চালানো

একটি ব্যবহারিক উদাহরণ দিয়ে শুরু করি। আমরা চালাবো `phi-4-mini-reasoning` মডেল, যা পর্যায়ক্রমে জটিল সমস্যাগুলো সমাধানে দক্ষ।

**মডেল চালানোর কমান্ড:**

```bash
foundry model run Phi-4-mini-reasoning-generic-cpu
```


> **প্রথমবারের সেটআপ:** প্রথমবার কোনও মডেল চালানো হলে, Foundry Local সেটি স্বয়ংক্রিয়ভাবে আপনার স্থানীয় ডিভাইসে ডাউনলোড করবে। ডাউনলোড সময় আপনার নেটওয়ার্ক স্পিডের ওপর নির্ভর করে পরিবর্তিত হতে পারে, অতএব প্রথম সেটআপের সময় ধৈর্য ধরুন।

### পদক্ষেপ ৫: বাস্তব সমস্যার মাধ্যমে মডেল পরীক্ষা

এখন আসুন একটি ক্লাসিক লজিক সমস্যার মাধ্যমে আমাদের মডেলটিকে পরীক্ষা করি এবং দেখাই এটি কীভাবে পর্যায়ক্রমে যুক্তি প্রদর্শন করে:

**উদাহরণ সমস্যা:**

```txt
Please calculate the following step by step: Now there are pheasants and rabbits in the same cage, there are thirty-five heads on top and ninety-four legs on the bottom, how many pheasants and rabbits are there?
```


**প্রত্যাশিত আচরণ:** মডেলটি এই সমস্যাকে যুক্তিযুক্ত ধাপে ভাগ করবে, pheasants এর ২ পা এবং rabbits এর ৪ পা থাকার তথ্য ব্যবহার করে সমীকরণ সমাধান করবে।

## অংশ ২: Foundry Local SDK দিয়ে অ্যাপ্লিকেশন তৈরি করা

### কেন SDK ব্যবহার করবেন?

যখন CLI দ্রুত পরীক্ষার এবং তাড়াতাড়ি মিথস্ক্রিয়ার জন্য উপযুক্ত, SDK Foundry Local কে প্রোগ্রাম্যাটিকভাবে আপনার অ্যাপ্লিকেশনে একত্রিত করার সুযোগ দেয়। এর ফলে আপনি পারেন:

- কাস্টম AI চালিত অ্যাপ্লিকেশন তৈরি করা
- স্বয়ংক্রিয় ওয়ার্কফ্লো তৈরি করা
- বিদ্যমান সিস্টেমে AI ক্ষমতা সংযুক্ত করা
- চ্যাটবট এবং ইন্টারেক্টিভ টুলস তৈরি করা

### সমর্থিত প্রোগ্রামিং ভাষাসমূহ

Foundry Local আপনার ডেভেলপমেন্ট পছন্দের জন্য একাধিক প্রোগ্রামিং ভাষার SDK সাপোর্ট প্রদান করে:

**📦 উপলব্ধ SDK:**

- **C# (.NET):** [SDK ডকুমেন্টেশন এবং উদাহরণ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/cs)
- **Python:** [SDK ডকুমেন্টেশন এবং উদাহরণ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/python)
- **JavaScript:** [SDK ডকুমেন্টেশন এবং উদাহরণ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/js)
- **Rust:** [SDK ডকুমেন্টেশন এবং উদাহরণ](https://github.com/microsoft/Foundry-Local/tree/main/sdk/rust)

### পরবর্তী পদক্ষেপ

1. আপনার ডেভেলপমেন্ট পরিবেশের ভিত্তিতে প্রিয় SDK নির্বাচন করুন  
2. বিস্তারিত বাস্তবায়ন গাইডের জন্য SDK-নির্দিষ্ট ডকুমেন্টেশন অনুসরণ করুন  
3. জটিল অ্যাপ্লিকেশন তৈরি করার আগে সহজ উদাহরণ দিয়ে শুরু করুন  
4. প্রতিটি SDK রিপোজিটরিতে প্রদত্ত নমুনা কোড অন্বেষণ করুন  

## উপসংহার

এখন আপনি শিখেছেন কীভাবে:  
- ✅ Foundry Local CLI ইনস্টল এবং সেটআপ করবেন  
- ✅ Phi Family মডেল আবিষ্কার এবং চালাবেন  
- ✅ বাস্তব জীবনের সমস্যার মাধ্যমে মডেল পরীক্ষা করবেন  
- ✅ অ্যাপ্লিকেশন ডেভেলপমেন্টের জন্য SDK বিকল্পগুলো বুঝবেন  

Foundry Local একটি শক্তিশালী ভিত্তি সরবরাহ করে যা সরাসরি আপনার স্থানীয় পরিবেশে AI ক্ষমতা নিয়ে আসে, কর্মক্ষমতা, গোপনীয়তা, এবং খরচের উপর নিয়ন্ত্রণ দেয় এবং প্রয়োজনে ক্লাউড সমাধানে স্কেল করার নমনীয়তাও বজায় রাখে।
