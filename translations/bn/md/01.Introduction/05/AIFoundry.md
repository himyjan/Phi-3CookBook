# **মাইক্রোসফট ফাউন্ড্রি ব্যবহার করে মূল্যায়ন**

![aistudo](../../../../../translated_images/bn/AIFoundry.9e0b513e999a1c5a.webp)

কিভাবে আপনার জেনারেটিভ AI অ্যাপ্লিকেশনকে [Microsoft Foundry](https://ai.azure.com?WT.mc_id=aiml-138114-kinfeylo) ব্যবহার করে মূল্যায়ন করবেন। আপনি যদি সিঙ্গেল-টার্ন অথবা মাল্টি-টার্ন কথোপকথন মূল্যায়ন করেন, Microsoft Foundry মডেল পারফরম্যান্স এবং সেফটির জন্য মূল্যায়ন করার টুল সরবরাহ করে।

![aistudo](../../../../../translated_images/bn/AIPortfolio.69da59a8e1eaa70f.webp)

## Microsoft Foundry এর মাধ্যমে জেনারেটিভ AI অ্যাপ্লিকেশন কিভাবে মূল্যায়ন করবেন  
বিস্তারিত নির্দেশনার জন্য দেখুন [Microsoft Foundry ডকুমেন্টেশন](https://learn.microsoft.com/azure/ai-studio/how-to/evaluate-generative-ai-app?WT.mc_id=aiml-138114-kinfeylo)

শুরু করার জন্য ধাপগুলো হলো:

## Microsoft Foundry-এ জেনারেটিভ AI মডেল মূল্যায়ন

**প্রয়োজনীয় শর্তসমূহ**

- CSV অথবা JSON ফরম্যাটে একটি টেস্ট ডেটাসেট।
- একটি ডিপ্লয় করা জেনারেটিভ AI মডেল (যেমন Phi-3, GPT 3.5, GPT 4, অথবা Davinci মডেল)।
- একটি রানটাইম যার মাধ্যমে মূল্যায়ন চালানো হবে।

## বিল্ট-ইন মূল্যায়ন মেট্রিক্স

Microsoft Foundry আপনাকে সিঙ্গেল-টার্ন এবং জটিল, মাল্টি-টার্ন কথোপকথন উভয়ই মূল্যায়ন করার সুযোগ দেয়।  
রিট্রিভাল অগমেন্টেড জেনারেশন (RAG) সিনারিওর ক্ষেত্রে, যেখানে মডেল নির্দিষ্ট ডেটার উপর ভিত্তি করে, আপনি বিল্ট-ইন মূল্যায়ন মেট্রিক্স ব্যবহার করে পারফরম্যান্স মূল্যায়ন করতে পারবেন।  
অতিরিক্তভাবে, আপনি সাধারণ সিঙ্গেল-টার্ন প্রশ্নোত্তর (নন-RAG) সিনারিওও মূল্যায়ন করতে পারেন।

## একটি মূল্যায়ন রান তৈরি করা

Microsoft Foundry UI থেকে, Evaluate পৃষ্ঠা অথবা Prompt Flow পৃষ্ঠা থেকে শুরু করুন।  
মূল্যায়ন তৈরি উইজার্ড অনুসরণ করে একটি মূল্যায়ন রান সেট আপ করুন। আপনার মূল্যায়নের জন্য ঐচ্ছিক একটি নাম দিন।  
আপনার অ্যাপ্লিকেশনের উদ্দেশ্য অনুযায়ী সঠিক সিনারিও নির্বাচন করুন।  
মডেলের আউটপুট মূল্যায়নের জন্য এক বা একাধিক মূল্যায়ন মেট্রিক্স বেছে নিন।

## কাস্টম মূল্যায়ন ফ্লো (ঐচ্ছিক)

অধিক নমনীয়তার জন্য, আপনি একটি কাস্টম মূল্যায়ন ফ্লো প্রতিষ্ঠা করতে পারেন। আপনার নির্দিষ্ট প্রয়োজন অনুযায়ী মূল্যায়ন প্রক্রিয়াটি কাস্টমাইজ করুন।

## ফলাফল দেখা

মূল্যায়ন চালানোর পর Microsoft Foundry তে বিস্তারিত মূল্যায়ন মেট্রিক্স লগ, দেখুন এবং বিশ্লেষণ করুন। আপনার অ্যাপ্লিকেশনের সক্ষমতা এবং সীমাবদ্ধতাগুলো সম্পর্কে অন্তর্দৃষ্টি পান।

**Note** Microsoft Foundry বর্তমানে পাবলিক প্রিভিউ অবস্থায় আছে, তাই এটি পরীক্ষামূলক এবং উন্নয়ন উদ্দেশ্যে ব্যবহার করুন। প্রোডাকশন ওয়ার্কলোডের জন্য অন্যান্য বিকল্পগুলি বিবেচনা করুন। আরও বিস্তারিত এবং ধাপে ধাপে নির্দেশনার জন্য অফিসিয়াল [AI Foundry ডকুমেন্টেশন](https://learn.microsoft.com/azure/ai-studio/?WT.mc_id=aiml-138114-kinfeylo) দেখুন।
