# AGENTS.md

## প্রকল্প পরিচিতি

PhiCookBook হলো একটি সমগ্র কুকবুক রেপোজিটরি যা Microsoft এর Phi পরিবারের ছোট ভাষা মডেল (SLM) এর সাথে কাজ করার জন্য হাতে-কলমে উদাহরণ, টিউটোরিয়াল এবং ডকুমেন্টেশন ধারণ করে। রেপোজিটরিটি বিভিন্ন ব্যবহার ক্ষেত্র প্রদর্শন করে যেমন ইনফারেন্স, ফাইন-টিউনিং, কোয়ান্টাইজেশন, RAG ইমপ্লিমেন্টেশন, এবং বিভিন্ন প্ল্যাটফর্ম ও ফ্রেমওয়ার্ক জুড়ে মাল্টিমোডাল অ্যাপ্লিকেশন।

**মূল প্রযুক্তি:**
- **ভাষা:** Python, C#/.NET, JavaScript/Node.js
- **ফ্রেমওয়ার্ক:** ONNX Runtime, PyTorch, Transformers, MLX, OpenVINO, Semantic Kernel
- **প্ল্যাটফর্ম:** Microsoft Foundry, GitHub Models, Hugging Face, Ollama
- **মডেল টাইপ:** Phi-3, Phi-3.5, Phi-4 (টেক্সট, ভিশন, মাল্টিমোডাল, রিজনিং ভ্যারিয়েন্ট)

**রেপোজিটরি স্ট্রাকচার:**
- `/code/` - কাজের কোড উদাহরণ ও স্যাম্পল ইমপ্লিমেন্টেশন
- `/md/` - বিস্তারিত ডকুমেন্টেশন, টিউটোরিয়াল, এবং হাউ-টু গাইড  
- `/translations/` - বহু ভাষার অনুবাদ (স্বয়ংক্রিয় ওয়ার্কফ্লো মাধ্যমে ৫০+ ভাষা)
- `/.devcontainer/` - ডেভ কন্টেইনার কনফিগারেশন (Python 3.12 সহ Ollama)

### লোকাল সেটআপ

**পূর্বশর্ত:**
- Python 3.12 বা এর উপর
- .NET 8.0 SDK (C# উদাহরণের জন্য)
- Node.js 18+ এবং npm (JavaScript উদাহরণের জন্য)
- ন্যূনতম ১৬GB RAM সুপারিশকৃত

**ইনস্টলেশন:**
```bash
git clone https://github.com/microsoft/PhiCookBook.git
cd PhiCookBook
```

**Python উদাহরণের জন্য:**
নির্দিষ্ট উদাহরণ ডিরেক্টরিতে যান এবং ডিপেনডেন্সি ইনস্টল করুন:
```bash
cd code/<example-directory>
pip install -r requirements.txt  # যদি requirements.txt বিদ্যমান থাকে
```

**.NET উদাহরণের জন্য:**
```bash
cd md/04.HOL/dotnet/src
dotnet restore LabsPhi.sln
dotnet build LabsPhi.sln
```

**JavaScript/Web উদাহরণের জন্য:**
```bash
cd code/08.RAG/rag_webgpu_chat
npm install
npm run dev  # ডেভেলপমেন্ট সার্ভার শুরু করুন
npm run build  # প্রোডাকশনের জন্য বিল্ড করুন
```

## উদাহরণের সাথে কাজ করা


```bash
cd code/<example-directory>
pip install -r requirements.txt
python <script-name>.py
```

### .NET উদাহরণ চালানো

```bash
cd md/04.HOL/dotnet/src/<project-name>
```
